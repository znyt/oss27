　　第298章
<br/>　　程大雷一时无语，看阴重慷慨激昂的模样，是不是对公道二字有什么误解。
<br/>　　樊梨花此刻忍不住了，开口怒斥道：“阴重，家父生前当你是兄弟，你却勾结外人，屠戮山寨弟兄，亏你还口口声声，谈什么公道！”
<br/>　　“我是为了那般，无非是为了让山寨弟兄有个好日子过。那些死去的人，都是为山寨的未来而死，我知道我愧对他们，但牺牲不是没有意义。只有你，光顾着自己，不肯牺牲，真正害他们死的是你啊。”
<br/>　　“你……”樊梨花竟被气得哑口无言。
<br/>　　“好啦，好啦。”程大雷挥挥手：“押下去杀了便是，和他废话做什么。”
<br/>　　“哈哈哈。”阴重仰天大笑：“你们虽能杀我，永远不可磨灭我心中的正义。老寨主，我来啦，你若在天有灵，就睁开眼睛看看，老夫心有余而力不足，无颜去见你啊。”
<br/>　　“……”
<br/>　　屋子内鸦雀无声，似乎什么古怪的气氛在蔓延。
<br/>　　“好啦，好啦。”程大雷挥挥手：“咱们继续聊正事。”
<br/>　　所谓正事就是如何处理山寨中的叛贼。罚不可不重，也不可没有，太重容易激起众怒，太轻容易失去威信。各人的惩罚视情况而定，和阴重一起策划叛变的，直接处死，绝不留情。而那些被煽动，蛊惑的人，则是被赶出山寨，自谋生路。
<br/>　　而对于那些墙头草，也不能置之不理，关键是如何进行精神文明建设。
<br/>　　“这些人该死，为什么该死，因为他们背叛了山寨，背叛了弟兄，背叛了道义。”
<br/>　　在众人中间，徐神机慷慨激昂，吐沫星子飞溅。
<br/>　　“人不知义，与畜生何异，以德报恶，何以报德，所以这些人不得不杀，不能不杀！”
<br/>　　随着徐神机手重重挥下，几十颗脑袋落地，恐惧在人群中弥漫。
<br/>　　“当然，我们既然来了，就是给大家带来好日子的。日后还有人敢生二心，这些人就是下场，不过，只要大家忠心耿耿，程寨主也绝不会亏待诸位兄弟。”
<br/>　　血淋淋的人头，让人不得不惧，徐神机打一棒子给个甜枣，令所有人都忐忑难安。
<br/>　　程大雷开始处理善后工作，赵子龙、高飞豹、张肥、关鱼四队人马先回去，毕竟，蛤蟆寨还空着呢，若是有人打过来怎么办。而秦蛮的蛮字队却留下来了，协助樊梨花一起处理后续事宜。
<br/>　　当然，徐神机也留下了，将他放在大本营，程大雷无论如何也放心不下。
<br/>　　刀就是法，法就是刀，程大雷今日在落玉寨露了露肌肉，想要确立威信还是很容易的。
<br/>　　现在大家都知道，落玉寨明面上的寨主是樊梨花，可真正的主人却是程大雷。
<br/>　　程大雷留在落玉寨，也不是无事可做，他对于山寨的建设，还是可以提出些指导意见的。
<br/>　　山寨地形如何利用得上，焚毁的聚义厅要重新建起来，还有其他一些建筑布局。当然，山寨建设要抓，精神建设也要抓，像这次叛变的事情再也不要发生了。
<br/>　　秦蛮率领的蛮字队，充当了教官的工作。他们把曾经在程大雷手中遭受的折磨，统统报复到这些人身上，手段残忍，有过之而无不及。
<br/>　　“你们也配称为山贼么，孬种，懦夫，娘们都不如。”
<br/>　　肉体折磨和精神折磨双管齐下，撑得过去的才是好汉。
<br/>　　而樊梨花一直看着这一幕发生，见本来山寨的弟兄被凝聚在一起，开始露出一支队伍的精气神。而这些，则是落在程大雷手中，三五天内发生的。
<br/>　　樊梨花的功夫是不弱的，不然也没可能击败秦蛮、高飞豹、张肥。然而，当发生叛变后，老实说，她有些手足无措，而只有几十个弟兄支持自己，更是令她觉得自己这个寨主当得有些不够格。
<br/>　　而这个山寨落到程大雷手中，没几天就换了一副模样，杀伐之果断，手段之干净，令樊梨花不得不对程大雷刮目相看。
<br/>　　程大雷正在清点落玉寨的财物，日后，程大雷打算将这里发展成自己的马场，专门爆骑兵。这几天，徐灵儿也被调了过来，她有养马的技能，此刻正好发挥作用。而经过这些日子的历练，徐灵儿也升级了，从普通晋级到优秀，一名优秀的马夫。
<br/>　　这一日夜里，程大雷洗过脚后准备上床睡觉，他已经决定要回蛤蟆寨了。毕竟，那里才是大本营，主要升级精力，要放在蛤蟆寨。
<br/>　　咚！
<br/>　　房门被一脚踹开，程大雷吓了一跳，看到樊梨花拎着一柄钢刀闯进屋，钢刀对准程大雷，刀尖泛着寒光。
<br/>　　“樊家姑娘……”程大雷打了个哆嗦：“你是要杀我，还是要睡我？”
<br/>　　“姓程的，你别废话！”樊梨花冷喝道：“今日我就是来找你算账的？”
<br/>　　“算什么帐？”程大雷此刻双脚还浸在洗脚盆中，他一边回答，一边看着周围，看有什么可以当武器反击。
<br/>　　“落玉寨是家父辛苦经营的，并不是你想占就能占的，你不是想要我身子么，只有你能胜过我。”樊梨花脸冷得像霜。
<br/>　　“哪里胜过你？”
<br/>　　“要么比刀法，要么比喝酒。”
<br/>　　程大雷此刻才注意到，樊梨花手中还拎着一坛酒，她砰地摆在桌子上。
<br/>　　“只要你有一样胜过我，落玉寨，包括我这个人就都是你的！”樊梨花道：“你要胜不了我，就乖乖滚蛋！”
<br/>　　“呃……”
<br/>　　程大雷久久无语，盯着樊梨花冷若寒霜的一张脸，突然抿起嘴角笑了笑。草原上的姑娘呐，的确敢爱敢恨，洒脱得紧。
<br/>　　樊梨花被程大雷盯的有些不好意思，强忍住慌乱的眼神，道：“你笑什么，喝酒还是比刀，你选一个？”
<br/>　　“喝酒，我喝酒。”程大雷忙举手投降。
<br/>　　樊梨花这几日，在落玉寨完全成了摆设，程大雷俨然成了山寨主人。这程大雷如此可恶会不会占了自己山寨，继续图谋自己这个人……以他的可恶，不是没可能啊。
<br/>　　草原儿女么，自然没有那么多九曲十八弯的心思，不就是嫁人么，嫁便嫁，又有何所谓。
<br/>　　于是，不知道她是想通了还是没想明白，二杆子精神大爆发，来找程大雷拼酒来了。
<br/>(本章完)
