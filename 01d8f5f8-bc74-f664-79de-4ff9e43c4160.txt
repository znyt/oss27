　　踏入这一方宫殿，面前便是天旋地转，化为了一片群山。
<br/>　　楚风眠一下子，就感觉到了一股惊人的灵气，在这群山之中的灵气，比起剑道门如今一方小世界之中，还要浓郁的多。
<br/>　　楚风眠一眼望去，便是见到了，在这山脉之下，一座巨大无比的灵泉。
<br/>　　这灵泉，比起剑道门之下的灵泉，要足足大上十倍，远远不断的灵气，从这灵泉之中凝聚出来。
<br/>　　以至于才有如此惊人的灵气。
<br/>　　这一方群山，比起圣龙秘境的圣地，都要好得多。
<br/>　　大量的灵药，生长在其中。
<br/>　　也许是因为灵气的关系，这一片群山之中，树木皆高大无比，足以千丈之高。
<br/>　　只是似乎是除了草木，却是没有一丝丝其他的生机。
<br/>　　没有妖兽，没有人踪。
<br/>　　楚风眠现在正身处于一处山谷之中，在这山谷之内，有着一座山洞。
<br/>　　“这里应该就是狮霸武祖的真正洞府所在。”
<br/>　　楚风眠看了这山洞，眼神之中露出了几分笑意。
<br/>　　这狮霸武祖，纵然是成为了一代武祖，但是他的骨子里，依然只是一头狮子。
<br/>　　纵然是外面林立着无数的宫殿，但是他的真正洞府所在，就是这样一座山洞。
<br/>　　楚风眠看了一眼，便是一步踏入了其中，这山洞从外看起，漆黑无比，但是随着楚风眠一步踏入其中。
<br/>　　山洞之中的禁制阵法，突然间的都被激活了一般，点亮了整座山洞。
<br/>　　在这山洞之内，只有简简单单的一座石床，在这石床之上，曾经应该还有着一张兽皮。
<br/>　　只是不知道是不是因为岁月流逝的关系，上面的兽皮已经是腐朽消散，只有一点点的痕迹留下。
<br/>　　除此之外，在这山洞之内，几乎是看不见任何不凡的东西，简简单单的一座石洞。
<br/>　　简单到了一种不可思议的地步，让人都无法相信，这里曾经是一位武祖的洞府。
<br/>　　不过……
<br/>　　楚风眠目光扫视了一眼这山洞。
<br/>　　他可不相信，楚风眠通过了这狮霸武祖独桥之上的考核，会一无所获。
<br/>　　一旦是踏入独桥两边的光幕之中，都足以是得到大量的宝藏好处，在这洞府内，不可能一无所获。
<br/>　　楚风眠的目光，扫视了一圈，落入到了这石床之下，突然之间手心之中凝聚灵力，一掌轰击在这了石床上。
<br/>　　这一张石床，本就是以寻常的青石打造而成的，楚风眠的一掌轰击过去，这青石却是纹丝不动。
<br/>　　要知道以现在楚风眠的实力，哪怕是随意一掌，拍碎一方山脉都是轻轻松松。
<br/>　　这样的青石，就算是千万块加在一起，楚风眠出手轻易一掌都可以拍的粉碎。
<br/>　　“果然有些古怪。”
<br/>　　楚风眠嘴角露出了几分笑容，他刚刚那一掌轰击在这了青石之上的时候，力量却是被突然泄去，没有打在这青石之上。
<br/>　　这青石，的确是寻常随处可见的青石，但是这青石的背后却另有乾坤。
<br/>　　“剑出！”
<br/>　　楚风眠冷喝一声，祖龙至尊剑出现在楚风眠的手心之中，刹那之间楚风眠手持灵剑，一剑向着那青石斩下。
<br/>　　手持灵剑，这一剑的威力，比起刚刚楚风眠的随手一击恐怖无数倍。
<br/>　　咔嚓！
<br/>　　剑锋闪过的瞬间，在这青石之上，一层禁制，缓缓的浮现了出来。
<br/>　　在楚风眠的剑下，这青石之上隐藏的禁制，也被楚风眠一剑斩了出来。
<br/>　　“禁制，腐朽了无数岁月的禁制，还想要抵挡住我的剑？”
<br/>　　楚风眠脸色冰冷无比，看到这禁制的浮现，脸色依然冷漠，只是在楚风眠的手心之上的剑锋，上面的锋芒变的更加的锋利起来。
<br/>　　“破开！”
<br/>　　嘶拉！
<br/>　　剑锋闪过，蕴含极致锋芒，斩在了这禁制之上。
<br/>　　一剑之下，这本来就腐朽不堪的禁制，突然崩灭开来。
<br/>　　轰！
<br/>　　这被禁制所守护的青石，随着禁制的破碎，也随即断裂开来，在楚风眠的剑光之下，被当即震成了一团粉末。
<br/>　　楚风眠的目光看了过去，在这青石之下，却是隐藏着一个密室。
<br/>　　在这密室之中，有着一口棺材。
<br/>　　这棺材，通体白玉所做。
<br/>　　楚风眠看了一眼这棺材，手心一点，轰隆之间将这棺材打开。
<br/>　　旋即楚风眠的目光看了过去。
<br/>　　这棺材之中，并没有骸骨。
<br/>　　却是摆放着一个玉瓶。
<br/>　　这玉瓶之中盛放的，却是一瓶精血。
<br/>　　这狮霸武祖的真正宝藏，就是这样一瓶。
<br/>　　楚风眠看了一眼这一瓶精血，手心一动，抓了过去，这一瓶精血便是没有任何反抗能力的，落入到了楚风眠的手心之中。
<br/>　　楚风眠手心抓着这一瓶精血，隔着玉瓶，楚风眠都仿佛可以感觉到这精血之中蕴含的恐怖力量。
<br/>　　极致的武道，无敌的力量，蕴含在这精血之中。
<br/>　　“狮霸武祖的精血，这狮霸武祖，居然是留下了这样一瓶精血？”
<br/>　　楚风眠看了一眼，也就明白了过来，这一瓶精血的来历，狮霸武祖的精血。
<br/>　　一尊武祖，曾经踏入天人境级别强者的一瓶精血。
<br/>　　在楚风眠拿起这一瓶精血的一刹那，在楚风眠的脑海之中，还浮现出了一段文字。
<br/>　　“我以寿元将尽，故留下这洞府，还有这精血，如我族后裔所得到，望你以此振兴我族。”
<br/>　　“若外族得到，望你以后遇到我族，可以照顾一二，这一瓶精血，随你使用。”
<br/>　　狮霸武祖，也逃不出寿元的灭亡。
<br/>　　楚风眠听到这一段话，也是感叹不已，狮霸武祖已经是九域历史之中绝对无敌的霸主了。
<br/>　　能够被称为武祖，他的实力不容置疑，甚至是这狮霸武祖，纵然是在天人境之中，都是一位强者。
<br/>　　可依然是逃不过一个寿尽而亡的下场。
<br/>　　这洞府，是狮霸武祖故意留下的，想要被他的后人所得到。
<br/>　　可后来九域的狮族之中，却没有出现什么强者，以至于这洞府经历数十万年的岁月，再度开启，被人瓜分。
<br/>　　“这精血我要了，日后遇到狮族碰到危机，我会出手帮他们一次。”
<br/>　　楚风眠轻声说道。
<br/>　　楚风眠得到了这洞府之中的宝藏，也算是得到了狮霸武祖的半个传承，得到了好处，这恩情日后楚风眠自然会报。
