
<br/>　　<br/>　　    “我要是知道，有什么是你能图的话，我还真就能防着点了，有个人说过一句话，我觉得特别在理，那就是尽量离那些蓄意接近你的女人远一点，特别还是个漂亮的女人……”<br/>　　<br/>　　    李红拂笑呵呵地问道：“哪个女人，说的这么有理啊？”<br/>　　<br/>　　    “我姐说的，因为她就挺漂亮的，但凡是接近他的男人，最后都被忽悠瘸了”王惊蛰一本正经地说道。<br/>　　<br/>　　    “啪”李红拂推了下王惊蛰的胳膊，转身就拉开车门坐进了斯马特里，然后抬头说道：“王惊蛰，不可否认的是，你这样的人被别人接触了确实该有点提防的心思，其实我还真有点别的原因”<br/>　　<br/>　　    “啥啊？”王惊蛰挠了挠鼻子问道。<br/>　　<br/>　　    “以后告诉你，只希望我找你的时候，你别拒绝就行了”李红拂发动车子，升起车窗扔下一句话后就走了。<br/>　　<br/>　　    林汶骐走过来，看着远去的斯马特，皱眉问道：“你俩扯到哪一步了？我刚才看见，你们都有肢体接触了呢？怎么的，刑讯逼供了啊？”<br/>　　<br/>　　    “我和她止步于此，还没等我探出她的深浅呢，她就打了个马虎眼走了”<br/>　　<br/>　　    “哎，哥们在这点上也没帮不了你什么，不过我送你句话希望你能听进去”<br/>　　<br/>　　    “啥啊？”<br/>　　<br/>　　    “离那些别有用心的女人远一点，特别是这还是个漂亮的女人”<br/>　　<br/>　　    王惊蛰瞬间懵逼，眨着眼睛问道：“哪，哪个女人说的这么有道理啊？”<br/>　　<br/>　　    “那是我曾经的故事，后来却变成了事故”林汶骐眼神深邃地说道。<br/>　　<br/>　　    两天之后，沪海虹桥机场。<br/>　　<br/>　　    林汶骐和陈臣返回了川中，临走之时陈臣的状态倒是没有默默无语两眼泪的，但是其中的神色却让王惊蛰看着挺不好意思的，这个世上有两件事其实是最难拒绝的，一个是贫穷，另一个是有人对你的深情。<br/>　　<br/>　　    陈臣鼓着俏脸两边的腮帮子，说道：“你就没有什么想说的了？”<br/>　　<br/>　　    王惊蛰憋了半天，硬着头皮说道：“一路走好”<br/>　　<br/>　　    “唰”陈臣忽然伸出两根手指，就拧在了他的胳膊上：“个瓜娃子的，我讨厌你！”<br/>　　<br/>　　    林汶骐拍了拍他的肩膀，说道：“我们走了，希望你能一直安好，回见吧哥们……”<br/>　　<br/>　　    王惊蛰离开机场，是乘坐地铁走的，沪海的地铁跟京城一样，得算是国内最拥挤的两个城市了，他从机场回来的这个点还不算是早晚高峰，但上了地铁的时候还挺挤的。<br/>　　<br/>　　    地铁晃晃悠悠地开着，王惊蛰穿着一身大褂手抓着栏杆，然后跟他挨着的是个大概五十岁左右的老头，穿着挺破旧的，一身的粗布衣裳和布鞋，脚下放着个蛇皮的袋子，眼角还有眼屎没有扣干净，一脸的老褶子，一看这就是个饱经沧桑的贫苦大众。<br/>　　<br/>　　    开始的时候，王惊蛰也没有留意，地铁后来开了几站这老头接了个电话，王惊蛰才留神到对方，因为他说的话恰好就是朝歌那边的方言，偶然间在异地能碰到个老乡，王惊蛰也觉得挺有缘的，特亲切。<br/>　　<br/>　　    “我还有四五站就要到地方了，人民广场下车嘛？对，对，我看到了，看到了”<br/>　　<br/>　　    “不用你接，我自己过去就可以了，找得到的，放心……”老人似乎要去什么地方，在电话里一直聊着这事，等他挂了电话，过了四站地后，地铁就到了人民广场这一战。<br/>　　<br/>　　    “叮咚，叮咚”地铁门开，老人弯腰拎起地上的蛇皮袋子就扛在了肩上，然后笑着操着普通话，点头跟王惊蛰说道：“借过，麻烦借过一下，谢谢”<br/>　　<br/>　　    王惊蛰往旁边挪了下跟老人错过身子，他看着对方往出走向屏障门，略微皱了下眉头，等老人出去以后他想了想，也迈步跟了出去。<br/>　　<br/>　　    “唉，大爷”从地铁里出来后，他就叫了对方一声，说的也是朝歌话。<br/>　　<br/>　　    老人顿时停住脚，好奇地转过了头，他也没想到会碰见个家乡的人，就好奇地问道：“小哥，你叫我？”<br/>　　<br/>　　    王惊蛰点了点头，说道：“大爷，从朝歌哪过来的啊？”<br/>　　<br/>　　    “我是曹王庄的人，你……”<br/>　　<br/>　　    王惊蛰哦了一声，说道：“我刚才听见你说的朝歌话，就想着是老乡了，我住王村那边的，还别说咱俩离的真挺近的，坐车也就半个小时吧？”<br/>　　<br/>　　    “是了，是了，蛮近的呢……”老人见王惊蛰主动搭话，也来了兴趣，就边走跟王惊蛰边聊了一会，两人从地铁站出来，就到了人民广场。<br/>　　<br/>　　    “小哥你这是要去哪里啊？”<br/>　　<br/>　　    王惊蛰说了个地方，就是电话里那老人说的地址，对方觉得还很巧的，就挺高兴的跟他做伴往那边走了过去。<br/>　　<br/>　　    路程大概有二十分钟左右，一路上王惊蛰有一搭没一搭的跟老头随意的聊着，期间他一直都在留意周围的人。<br/>　　<br/>　　    刚刚，这老头打完电话要下地铁的时候，王惊蛰就发觉这老人今天有破财的相，而且还是破大财，他就从地铁上跟了下来，出于老乡见老乡的热乎劲，他真不忍心看着对方损失这一把，因为他明显瞧出来了，这老人挣点钱不容易，可能身上带着的是他说不上几年才攒下的积蓄，要是一下子全都没了的话，这恐怕都能要他的老命了。<br/>　　<br/>　　    人啊，能不作恶千万别作恶，但要是能做点善事的话，也千万别犹豫。<br/>　　<br/>　　    上一次在骊山秦村，向缺就和他聊过这个，说是人在做天在看，你做啥事老天爷都能看得见，做点好事给自己积累点阴德，也算是功德无量了，可能对于王惊蛰面对天谴会有点帮助，毕竟老天爷不眼瞎啊。<br/>　　<br/>　　    赊刀一脉和古井观出世的哲学还是有很多不同的，所以向缺跟王家父子一沟通之后，他俩也觉得挺有道理的。<br/>　　<br/>　　    王惊蛰一见这老头要破财，就起了恻隐之心，能帮对方度过这个坎，对他来讲不过就是举手之劳罢了。<br/>　　<br/>　　    两人闲聊着走了大概能有七八分钟之后，从他们后面就呼啦一下走过来四个人，这几个人都是分开的，好像彼此间都不认识，有男也有女，他们步伐挺快的从两人后面经过，其中一个大概三十岁左右的女子突然跌跌撞撞地就撞向了那个老人，老头脚步一个踉跄往旁边躲了一下，随即另外一人快速的搀扶了他一下。<br/>　　<br/>　　    “哎呦，大爷您看着点，小心啊”<br/>　　<br/>　　    老人连忙道谢，点头哈腰的说着对不起，王惊蛰则是眯了下眼睛，看着摆手说无所谓的这人，心里想着这他么的还是团伙作案呢啊？<br/>　　<br/>　　    对方下手的速度，快准狠，那人搀扶上老头的时候，手就伸到了他的口袋里，然后迅速掏出一叠钱，就放在了自己的身上。
