　　第160章
<br/>　　而可能就连献计的程大雷都没想到，歪打正着，杨龙停当年的确因打仗时受了些伤，一直没有子嗣，不然他也不会如此重视罗铁。
<br/>　　所以，每当杨龙停走出帐外，看到兵丁窃窃私语时，心情是如何也就可想而知了。就算他们不是在议论自己，杨龙停也总觉的对方看自己的眼神有些异样。
<br/>　　“攻城！攻城！攻城！”杨龙停此刻的杀心已经难以遏制。
<br/>　　在这一日黎明，杨龙停开始攻城。
<br/>　　大军向前推进，四路兵同时发动攻击。投石机如春笋拔地而起，飞石如蝗，城墙上落石如雨，士兵靠着竹梯冲上去，又变成尸体落下来。
<br/>　　一切的一切，都是为了杀死更多的同类。
<br/>　　而这，也仅仅是佯攻。
<br/>　　杨龙停从未幻想过，能靠一场战斗打下黑石城。杨龙停发动今天的战斗，主要目的是判断薛半川对四面城墙兵力的调度。
<br/>　　四面城墙，防守总是有强有弱，判断清楚对方的虚实，才好对症下药。
<br/>　　结果，不太理想。
<br/>　　黑石城薛半川能调动的兵力不过五千，可这是在平时，若是逼到绝境，比如现在，那么城中的十万平民，都可以算作兵员。
<br/>　　而根据今天都战斗判断，虽然兵力己方占优，但在士兵的悍勇程度上，幽州兵却不如守军。
<br/>　　毕竟，黑石城紧挨着戎族，时不时就要打一场，虽然输多胜少吧，但他们对打仗、死人是不陌生对。可幽州这边，不知多少年没发生过大规模战争了。
<br/>　　黎明攻城，黄昏收兵，城内紧张的忙碌着，医治伤员，埋葬死尸，修补损坏的城墙。
<br/>　　夜来，幽州兵的营地内，也是哀鸿一片，受伤的士兵窝在被褥上，他们有的丢了眼睛，有的丢了胳膊……时不时发出阵阵哀嚎。
<br/>　　战争，从来不是一件浪漫的事情。万军丛中取上将首级的确威风，可是一将功成后几万人几万人的死亡，落在史书上不过几个数字而已。
<br/>　　杨龙停的大帐内，几员大将正讨论着今天的战斗，并对之后该如何破城发表看法。
<br/>　　杨龙停坐在帅案后，一直没有作声。
<br/>　　幽州已经有三十年没有打过大仗，自己的手下还能不能打仗，那些是纸上谈兵，那些平日不吭声却是关键时候的猛人，都能从一场战斗中看出来。
<br/>　　没有什么比真正的战争更能锻炼人才。
<br/>　　“主公，我刚得到一条消息，是从青牛山传来的？”李善言起身递过来一份信笺。
<br/>　　杨龙停打开看了看，看罢之后，本来稳坐军中帐的他瞬间三尸神暴跳，五灵豪气腾空。
<br/>　　手掌重重拍在帅案上，冲手下大将道：“谁与我点五千兵，提程大雷的脑袋见我！”
<br/>　　……
<br/>　　程大雷一直待在蛤蟆寨中，耳朵却是听着四面八方的动静，为保万无一失，他派人将苏樱和小蝶从落叶城接了回来，湖底捞的弟兄也撤了。
<br/>　　至于苏家的产业，湖底捞的生意……兵荒马乱的年景，活着比什么都重要，难道还想做点生意，安稳做个富家翁，想什么呢。
<br/>　　等这段风波过去了，自己再重新把生意做起来，也没什么困难，自己和方伯山关系处得还不错。
<br/>　　况且，程大雷现在有日进斗金的生意，要说发财还有什么比发战争财来得更快。
<br/>　　把苏樱接过来后，程大雷彻底放开了手脚。
<br/>　　这一日，一支运粮的队伍来到青牛山，他们的头目名叫王孙鱼。
<br/>　　刚进入青牛山，只听一阵铜锣声响，从山林中杀出一拨人来。
<br/>　　“此山是我开
<br/>　　此树是我载
<br/>　　要想从此过
<br/>　　留下买路财
<br/>　　牙嘣半个不
<br/>　　管杀不管埋”
<br/>　　为首之人大喊。
<br/>　　王孙鱼愣了愣，忽然笑道：“程将军，您开什么玩笑，喔，我明白了，您肯定是好久不干这一行了，想过过瘾，哈哈，还是真幽默。”
<br/>　　王孙鱼不是第一次从青牛山经过，自然也不是第一次看到程大雷。往日程大雷都很客气，当然经过罗铁的事情后，王孙鱼更加客气。
<br/>　　“谁和你开玩笑了，我在打劫不懂么！”程大雷肩上扛着斧头。
<br/>　　“可您不是已经归顺幽州王了么？”
<br/>　　“不干啦！”程大雷大手一挥。
<br/>　　“说不干……就不干啦。”王孙鱼有些无语。
<br/>　　“王兄弟，我也不难为你，粮食我要两成，这是蛤蟆寨从来的规矩。”
<br/>　　“规矩……还从来的？”
<br/>　　“对呀！”徐神机拎着几个人把一块木牌扎在路边：“看到没有，这规矩不是第一天立下的，现在你们想从此地过，咱们还是按照规矩来。”
<br/>　　王孙鱼看木牌上的字迹，的的确确，存在已经很久了。他想不通的是，程大雷竟敢把这规矩用在幽州王身上。用那句话说：汝不畏死乎！
<br/>　　程大雷态度还算不错：“王老弟，我实话给你说，我要抢了你，也就是抢了你的，你留下两成粮食，咱们当什么事情都没发生过。你可要知道，上次没把粮食运到的人，可是被幽州王斩了。”
<br/>　　王孙鱼一激灵，就觉得脑后一寒，他可不想用自己的脑袋去撞幽州王的刀。
<br/>　　最后，王孙鱼守了程大雷的规矩，并带走了程大雷给幽州王的一封信。
<br/>　　这封信又从李善言手中，传到杨龙停手中，随着杨龙停的重击，飘飘落在地上，大帐内所有人都看清了上面都内容。
<br/>　　信上只有四个字：我不干了！
<br/>　　听明白李善言的转述后，大帐内的众人面面相觑。程大雷一直都是很懂事的，可他不懂事的时候，就显得这么疯狂。疯狂到要向幽州王收过路费。
<br/>　　杨龙停叹口气，恍然意识到，自己是被程大雷糊弄了。
<br/>　　他装疯卖傻，胡言乱语……虽然各种方式有些可笑，但变相让大家轻视了程大雷。让大家觉得程大雷可笑而不是可怕。
<br/>　　如果不是轻视此人，早该攻下蛤蟆寨，扫平后患，也不会落到如今上不去下不来的尴尬境地。
<br/>　　如今，程大雷把着自己的粮道，此人非除不可。
<br/>　　幽州王传下帅令，由齐德强领兵一万，必要在三天之内，打下蛤蟆寨，把程大雷那颗讨厌的脑袋提过来见自己。
<br/>　　
<br/>(本章完)
