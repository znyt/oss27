陆寒霆来到了隔壁的诊疗室里，史密斯教授温和的笑道，“陆总，关于你的情况我大致了解了，现在请你躺在上面，我先给你做一下身体检查。”
<br/>
<br/>        陆寒霆来到了单床上，然后躺了下来。
<br/>
<br/>        史密斯教授走过来，很是友善的征求意见，“陆总，是我帮你脱裤子，还是你自己脱裤子？”
<br/>
<br/>        “…”
<br/>
<br/>        陆寒霆英俊的眉心里像凝结了一层寒霜，刚才小女佣要脱他的裤子，现在又被一个男教授给围观，他觉得糟糕透了。
<br/>
<br/>        “陆总，还是我帮你吧。”这时史密斯教授伸手去解他的皮带。
<br/>
<br/>        下一秒陆寒霆就坐起了身，将薄唇抿成一道森冷泛白的弧线，他快速的下床了，直接离开。
<br/>
<br/>        史密斯教授愣了一下，“陆总，你去哪里，我们还没有开始治疗。”
<br/>
<br/>        陆寒霆那高大英挺的身影已经消失在了视线里。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆拎着车钥匙上了劳斯莱斯幻影豪车，一路将豪车开到了醉玉欢公寓楼下，他知道夏夕绾最近都住在叶翎这里。
<br/>
<br/>        副驾驶车窗缓缓滑了下来，他抬眸向上看去，窗户那里拉上了窗幔，不过里面晕黄的灯光透了出来，一片温暖。
<br/>
<br/>        以前在幽兰苑，哪怕她先睡着了，她都会给他留一盏灯的。
<br/>
<br/>        现在她走了，再也没有人给他留灯了。
<br/>
<br/>        他的卧室里，总裁办公室的休息室里，都没有了她纤柔的身影。
<br/>
<br/>        陆寒霆将挺括的后背慵懒的倚靠进了椅座里，就这样静静的守在她的楼下，他心里的阴郁暴躁稍稍缓解一些。
<br/>
<br/>        这时一串悠扬的手机铃声响起了，是秘书严毅打来的电话。
<br/>
<br/>        陆寒霆按键接通，严毅在那端汇报道，“总裁，刚才得到消息，太太已经在整理东西了，她向枢密研究院递出了辞呈，订好了飞往帝都城的机票，太太两天后就会离开海城飞往帝都城了。”
<br/>
<br/>        陆寒霆拽着手机，满耳都是两天后她就飞走了！
<br/>
<br/>        他知道她会走的，但是没有想到这么快！
<br/>
<br/>        陆寒霆将手机丢在了副驾驶座里，精健的胸膛开始上下起伏，多日未眠让他狭长的眼梢里布满了可怕的红血丝，还有一层戾气。
<br/>
<br/>        他迅速拿出了安眠药，倒出三四粒放在嘴巴里咽下去，这段时间他找到治疗失眠障碍和心理疾病的捷径了，那就是服用安眠药。
<br/>
<br/>        只要闭上眼睛沉睡，他就能克制自己不去打扰她，离她远远的。
<br/>
<br/>        ……
<br/>
<br/>        楼上公寓里，夏夕绾已经洗了澡，她坐在台灯前翻开了妈咪留下来的医典，开始研究妈咪的针法。
<br/>
<br/>        房间里静谧无声，夏夕绾越往下翻越觉得心惊，妈咪的针法精湛玄妙，里面蕴蓄着无比强大的力量，让她震撼。
<br/>
<br/>        她觉得自己以前就是一个井底之蛙，这本医典好像打开了她的新世界。
<br/>
<br/>        李文清院长说的对，现在的医学已经无法满足夏夕绾的学习了，她是真正的天之骄女，需要更玄妙深奥的东西来引领她突破自己，进入更高的境界。
