        网上最近也没有关于洛羽的消息，而他的微博上也未更新最近状态。
<br/>
<br/>        连星儿看完微博后，收起了手机，返回教室，但她坐在座位上时，指腹却情不自禁地摩挲着手腕上这串洛羽送给她的粉晶手链，还一边看着一边发起了呆。
<br/>
<br/>        她对洛羽真的只是粉丝对偶像的那种喜欢，为什么洛羽却突然间向她表白？
<br/>
<br/>        是她的什么行为让他产生了误解吗？
<br/>
<br/>        而就在这个时候，明司寒走到了她的教室门外，就站在教室门口，她的一举一动，尽收眼底。
<br/>
<br/>        “明学长，是来找咱家星儿的吧？我帮你喊她哈！”顾千凝上了一趟厕所回来后，看到明司寒站在教室前门口，主动打了声招呼。
<br/>
<br/>        明司寒看都未看顾千凝一眼，二话不说便转身离去。
<br/>
<br/>        顾千凝知道明司寒这臭脾气，除了连星儿，他好像真的不把任何女生放眼里，真的是一点礼貌也没有。
<br/>
<br/>        哼！
<br/>
<br/>        顾千凝闷哼了声，回到座位上后，跟坐在一旁的连星儿抱怨了起来：“刚刚你家明司寒就站在咱们教室门口，我跟他打招呼，他还不理我了！真的是拽上天了噢！”
<br/>
<br/>        “明司寒？！”连星儿回过神来，本能地抬起头来，伸直了脖子向斜前方的教室前门看去，“他人呢？”
<br/>
<br/>        “走了啊！不知道他来是干嘛的！我说要帮他喊你，他也没吭声，就那么掉头走了。太没绅士风度了！”顾千凝继续埋怨。
<br/>
<br/>        连星儿微笑着安慰道：“你别跟他计较哈！他就那样，好像谁都欠了他钱似的！”
<br/>
<br/>        “我听学姐们说，明司寒一直对女生就不大友善。他大一刚入学那年，学校里好多长得漂亮的女生都追过他，同年级的也好，比他大的学姐也好，都被他毫不留情地拒绝了。所以，后来追他的女生就少了，现在几乎为零。”顾千凝一边说着，一边又啧了啧舌，“啧啧啧，算了，我不跟他计较了。”
<br/>
<br/>        “原来如此啊！”连星儿忽然间感慨起来。
<br/>
<br/>        怪不得她一入学，他就觉得她很碍眼来着。原来不是因为他针对她，而是因为她主动去搭讪了他，让他觉得厌烦。
<br/>
<br/>        所以说，明司寒是不喜欢主动的女生？！
<br/>
<br/>        连星儿找到原因所在后，暗自决定，以后在明司寒的面前，一定要被动。
<br/>
<br/>        不主动给他打电话，不主动给他发短信，也不主动跟他打招呼便是。
<br/>
<br/>        就这样连着三天过去了，男生宿舍里，明司寒有了想摔手机的冲动。
<br/>
<br/>        因为他发现，如果他不主动给连星儿那丫头发短信或者是打电话，连星儿那丫头就当他不存在一样。
<br/>
<br/>        这样也就算了，那天，他和她在同一栋教学楼里上课，课间休息的时候，他还特意跑楼下去看她。
<br/>
<br/>        哪知道，她竟然看着洛羽送给她的那条粉晶手链发呆！
<br/>
<br/>        想想这事到现在，明司寒心里便极其不爽，不爽也就算了，而且还很难受。
<br/>
<br/>        明司寒躺在床上心烦气躁地翻来覆去。
<br/>
<br/>        坐在床铺下的书桌前看书的邢乐游忍不住念叨了一句：“阿寒，你身上是长虱子了吗？只听到你弄得床铺‘咚咚’响。”
<br/>
<br/>        “问你个事。”明司寒索性坐起身来，俯视那边正在翻看计算机编程语言的邢乐游。
<br/>
<br/>        “你说。”邢乐游应道。
<br/>
<br/>        明司寒忽然一本正经起来：“怎样让一个心里已经有喜欢的人的女生喜欢上自己？”
<br/>
<br/>        “你这个问题很复杂。”邢乐游也一本正经起来。
<br/>
<br/>        “怎么个复杂法？”
<br/>
<br/>        “首先，你自身要比那个女生喜欢的人要优秀；第二，你自身要比那个女生喜欢的人对那个女生还要好；第三，要让那个女生发现她自己喜欢的人是个渣男。同时满足这三点，那女生也不一定能喜欢上你。”邢乐游条条是道地说。
<br/>
<br/>        明司寒嘴角抽了抽。
<br/>
<br/>        “连星儿她不喜欢你啊？”邢乐游接着狐疑地问道。
<br/>
<br/>        明司寒白了邢乐游一眼，心虚地反驳：“我替我哥问的。”
<br/>
<br/>        “你二哥不是大明星吗？应该不存在这种问题。”邢乐游笑道。
<br/>
<br/>        明司寒微眯起黑眸：“是我大哥。”
<br/>
<br/>        “好吧！随你说你哪个哥，反正你哥哥多。”邢乐游咧了咧嘴。
<br/>
<br/>        明司寒却一脸很是苦恼的样子。
<br/>
<br/>        邢乐游心知肚明，微笑道：“我觉得吧！那个连星儿应该是对你有意思。你是不知道那天咱们打篮球时，她那双眼睛，可是直勾勾地盯在你身上。普通的看球赛的女孩子，一边眼睛都是跟着球走。如果是特意关注球场上的某个人，那双眼睛就会全程盯着某个人。”
