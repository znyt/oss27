<br/>        至于加入战场，其他的宗门压根没想过，甚至于古仙门没有一点让燕山直接加入战场的意思。

<br/>        一个以炼丹为主业的宗门，发起战争的力量多半来源于人脉，就如同陈明在九幽寒泉之中炼制阴阳和合丹，对于那些受惠的诸王，可谓是一呼百应！

<br/>        炼丹师就好好炼丹，没事儿来凑什么热闹。

<br/>        而就在这个时候，古仙门已经通过各处的斥候知道了燕山出兵的消息，众多古仙门的诸王也是疑惑不解，不是，你们这群炼丹的，能不能老实一点，好好在家里炼丹啊，没事儿跑到战场来凑什么热闹？

<br/>        等到燕山来到进攻这些鬼仙诸王的营地，和燕山商量一番，让他们打道回府算了。

<br/>        苏青阳在进攻古仙门腹地诸王的营地之中等待了良久，还是不见卓青瑶来到营地之中会盟，根据消息，卓青瑶应该在一天前就到了啊，怎么还没到？

<br/>        就在这时，他忽然听到外面有人禀报道:“报，在南方一百里处发现剧烈的战斗，应该是燕山入场了！”

<br/>        苏青阳露出一抹焦急之色，“怎么直接就上去开战了？快快点齐兵马，随我出去救援！”

<br/>        要是其他古仙门直接参战了，苏青阳才懒得管，古仙门之间是竞争关系，让他们去打呗，燕山就不一样了，这种专职炼丹的宗门，属于附属宗门，这种宗门，是不能得罪的，不然，可没有丹药供应。

<br/>        苏青阳愣了片刻，这到底是什么鬼，几个月前，燕山可是只有十万大军左右啊，怎么现在直接就五六十万了？
