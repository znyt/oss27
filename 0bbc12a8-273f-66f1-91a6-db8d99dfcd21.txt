　　刚才，在听到来者与清微掌门有关时，紫萱心间猛的波动了一下，但听到不是徐长卿后，眼底满是失望之色。
<br/>　　“圣姑，我的事情，自己有分寸。”
<br/>　　“有分寸就好，他似乎没有离开的意思，看来是有目的性的，你在这里待着，我出去看看。”
<br/>　　瞥了眼心不在焉的紫萱，圣姑无奈的摇了摇头，站起身来缓步朝竹楼外走去。
<br/>　　此时，站在灵溪谷上空，慕清霄望着脚下的结界，陷入沉思，是直接破开，或者是开启空间门穿过去呢？
<br/>　　正当他沉思之际，一道身影却从结界内部走了出来。
<br/>　　抬起头来，慕清霄目光一眯。
<br/>　　来者是一名女子，容貌极美，拥有一头雪白的发丝，更让他在意的并非后者容貌，而是她那深不可测的气息。
<br/>　　此女子不具备地仙强者那种绝对的压迫感，看来是渡劫期修士。
<br/>　　后者给他的感觉不一般，绝不是紫萱，灵溪谷中就只有女娲后裔与保护她的圣姑，那么此人的身份不言明了。
<br/>　　慕清霄打量圣姑的时候，圣姑也在打量着他，两人站在空中，相对而立，谁也没有率先开口说话。
<br/>　　“此处是有主之地，阁下不打一声招呼就来此打扰，恐怕有失礼数吧？”
<br/>　　对持许久，圣姑终于有些不耐烦，徐徐开口说道。
<br/>　　慕清霄瞥了她一眼，面色如常，道：“我要找的人不是你，可否将路让开？”
<br/>　　“哼，区区大乘修士，口气倒是不小！”
<br/>　　圣姑脸色变得有些愤怒，显然，她已经猜出，慕清霄就是冲着紫萱来的。
<br/>　　而她的职责是负责保护紫萱，没有允许自然不可能让他进灵溪谷，更何况慕清霄如此猖狂，更不可能让他进去。
<br/>　　慕清霄可不是一个拥有耐心的人，道：“看来，你是不打算让开了？”
<br/>　　“本圣姑很想知道，区区一个大乘修士，到底是谁给你的勇气在我的面前如此狂妄，难道是蜀山？”
<br/>　　慕清霄微微一愣，后者不笨，居然知道抬出蜀山来压他。
<br/>　　“我确实算是蜀山中人，但你拿清微老头子来压我，可就真的失策了，到底是谁给我的勇气，你马上就会知道了。”
<br/>　　话语刚落，慕清霄体内灵气翻滚，恐怖的威压以他为中心，向四周席卷而出。
<br/>　　感受着慕清霄身上席卷而出的恐怖威压，圣姑眼中闪过一抹诧异，这股威压已经不亚于普通渡劫期修士，此子有些本事。
<br/>　　立于上空，慕清霄吐出一口气浪，道：“给你一次机会，将路让开。”
<br/>　　“想要进灵溪谷，就看你有没有本事了，本圣姑还没有沦落到给区区大乘修士低头的地步，要战便战吧。”
<br/>　　话语刚落，慕清霄右手一握，周围灵气就汇聚而来，最终凝聚出一柄三尺的虚幻长剑。
<br/>　　慕清霄抬起头来，看向对面的女子，既然多说无用，那就只能动手了，长剑一挥，剑鸣声响彻长空，剑刃所过，空间皆是扭曲了一下。
<br/>　　圣姑虽面目平静的望着慕清霄的身影，但心中已经谨慎起来。
<br/>　　她境界虽高，但在战斗上却不占优势，她更善于无声无息的杀人。
<br/>　　恐怖的剑意袭来，圣姑终于有所动作，长袖一挥，一股清风自袖中席卷而出，旋即携着浑厚的掌力向慕清霄印去。
<br/>　　霎时间，风云色变……
<br/>　　慕清霄对自然界的任何东西都非常敏感，刚才圣姑挥动长袖舞动的清风，其中却参杂着一些别的东西。
<br/>　　虚化长剑与圣姑掌力相撞的同时，他也调动灵气一直防备着。
<br/>　　“铿铿……”
<br/>　　剑尖与掌心相撞，慕清霄瞬间感觉到，其上有一股恐怖的力量席卷而来，旋即，恐怖的劲气风暴以两人为中心向四周席卷而出。
<br/>　　爆退几步，慕清霄低头看了一眼手中的长剑，灵气凝聚而成的长剑已经崩碎成几截，渡劫不愧是渡劫。
<br/>　　刚才的一击，让慕清霄的心一下子沉重起来，后者可能是渡劫六重以上的强者，对付起来比较有难度，最重要的是……
<br/>　　“你是毒修？”
<br/>　　闻言，圣姑眼中满是惊讶，道：“小子，毒修如何？你可知道，修炼邪派无上法，妖蛊巫毒可回春，念在你是蜀山弟子的份上，离开这里吧？”
<br/>　　“并非偏见，只是有些意外，三千大道，毒乃其一，我的妻子也是一名毒修。”
<br/>　　说罢，魂剑已经悬浮在了身前，慕清霄道：“况且，我可并非蜀山弟子，念在你心地不坏的份上，能接下这招，便饶你一命。”
<br/>　　“绕我一命？”
<br/>　　圣姑真的是被气笑了，眼前的青年简直狂妄的不知天高地厚啊，刚才只是试探，难道他真以为，自己杀不了他？
<br/>　　不再理会后者，慕清霄双手结出剑印，魂剑轻鸣，一股无比恐怖的剑意威压乍然而现。
<br/>　　圣姑瞳孔一缩，惊骇失声，道：“仙法！”
<br/>　　唯有超越了最顶尖修炼法门，才有资格称之为仙法，而面前青年施展出的招式，威压令她这个渡劫九重的修士都感到余悸。
<br/>　　惊骇的同时，圣姑心中满是不敢置信，区区一个大乘九重巅峰的修士居然能施展出仙法，让她有种恍如梦境的感觉。
<br/>　　“你这个疯子，住手，这样下去你也会因灵气枯竭而死，难道你想与我同归于尽吗？”
<br/>　　圣姑咬牙切齿的嘶吼道，难以想象她先前平静的模样，虽说是同归于尽，但死的可不止是他们两人，灵溪谷中还有一名紫萱。
<br/>　　这一招未出，恐怖威压已经降临，她是渡劫九重巅峰的修士，非常清楚，结界根本不可能挡住这一击。
<br/>　　因为，就连她本人都已经被剑意锁定，这种感觉，简直就像是对上了地仙境强者，根本无处可躲。
<br/>　　望着她焦急的模样，慕清霄面色如常，体内阴阳玉下，阴阳灵气却急速翻滚起来。
<br/>　　魂剑轻鸣，剑尖指向圣姑，其上携着慕清霄最强的一击。
<br/>　　阴阳湖中的阴阳灵气几乎被抽取五分之一，灵气翻滚不休，无上剑意贯入魂剑当中。
<br/>　　“我说过了，你能接下这招，就饶你一命，此剑诀名《御剑伏魔》！”
