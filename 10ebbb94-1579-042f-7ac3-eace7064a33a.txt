<br/>				    各处正往这地方而来的强者们感觉到整个岛心都在震动着，不由一惊，纷纷惊问着怎么回事？然而，他们离岛心处却还极远，就算有心想知道，也无可奈何。<br/>　　<br/>　　    他们心头震惊，觉得应该是湖心那里出了什么事，可那地方还能出什么事？莫不是登仙梯提前出现？<br/>　　<br/>　　    想到这一点，他们加快了速度往那地方而去，然而，在赶路的过程时，却发现原本震动的岛心不再震动了，仿佛平静下来一般恢复如初。<br/>　　<br/>　　    虽是如此，但他们只是怔了片刻，便往那地方赶去。<br/>　　<br/>　　    而在那岛心处，原本在凤九面前的湖泊只剩下一个深深的大窟窿，如同干枯了一般半点水也没有，就连原本站在一旁的凤九也消失不见了。<br/>　　<br/>　　    此时的凤九其实是被卷进了空间的湖泊之中，当她将水引进空间里，在最后里面竟出现了一条巨大的白蟒，那条白蟒伴随着水源被引进空间时尾巴一卷将她也给卷进进来。<br/>　　<br/>　　    在空间里面，一身湿渌渌的她从湖泊中爬了起来，在她的手中，仍握着一把匕首，匕首上的血已经被湖水洗干净，光洁如初。<br/>　　<br/>　　    几头契约兽围站在湖边，见她出来便连忙拉了她一把，纷纷询问着：“主人，你怎样？有没受伤？”<br/>　　<br/>　　    “啊！主人，你快看，那条白蟒的血不见了。”老白惊呼一声，指着那湖中渐渐消失的血迹。<br/>　　<br/>　　    凤九坐在边上回头看了一眼，见那条被她所杀的白蟒在水中荡开的鲜血一点点的消失，湖水碧绿如初，若不是那飘浮在一旁的白蟒尸体，还真看不出刚才沾染过鲜血。<br/>　　<br/>　　    “去将那尸体捞出来，别污了这水。”凤九示意着，收起匕首后，取出一枚丹药服下。<br/>　　<br/>　　    倒没想到这里面会有这么一条白蟒，当时那条白蟒将她卷进来后，强行将她按在水中，张开大嘴就想吞了她，于是，她想也不想的便将它给杀了。<br/>　　<br/>　　    灵力消耗太大，再加上在水底的一战，让她整个人显得有些疲惫，于是换下一身湿衣后，她对几头契约**待着：“不准动金莲。”便沉沉的睡了过去。<br/>　　<br/>　　    她这一睡，吓得几头契约兽守在旁边不敢离开，直到，养足精神恢复灵力气息后才醒了过来，一醒过来才知她已经睡了三天了。<br/>　　<br/>　　    不过，她想了一下，这里面三天，外面也就才一天，于是，也不急着出去，而是研究起那金莲来。<br/>　　<br/>　　    天机老人让我来登仙梯这里，该不会就是冲着这个来的吧？毕竟她体内的有着上古青莲，而这金莲，看着似乎也不简单的样子。<br/>　　<br/>　　    她眉头轻拧着，想到天机老人那说一半不说一半的话，不由的双手环胸盯着那金莲苦思着，最后，她目光微闪，露出了一丝笑意。<br/>　　<br/>　　    “不管是不是为这金莲而来，这金莲是好东西倒是真的，若是用来入药炼丹，会成为什么样的丹药呢？”她轻喃着，不免期待起来。<br/>　　<br/>　　    目光一扫，视线落在那莲蓬处，脚尖一点，红色身影掠过，将那莲蓬摘了下来。<br/>　　<br/>
