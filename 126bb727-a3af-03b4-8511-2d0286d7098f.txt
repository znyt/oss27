        连星儿本能地抓住这只捂住她嘴巴的手腕，而对方的手腕又粗又硬，让她一只手完全抓不稳。
<br/>
<br/>        当她想上另一只手，给这人来个过肩摔时，对方大手一伸，直接握住她的两只手腕压向她的头顶。
<br/>
<br/>        紧接着一道力度，将她推至墙上。
<br/>
<br/>        后背与墙壁的撞击，让连星儿感到硬生生地痛楚，忍不住“啊”地叫出了声。
<br/>
<br/>        不知道是不是她的这个声音刺激到了对方，还是别的什么不明所以然的缘故，对方突然抬起另一只手，令她防不胜防地直接摸到了她的大腿上。
<br/>
<br/>        布料“撕拉”的声音，在鸦雀无声的教室里响起。
<br/>
<br/>        连星儿刚想大呼救命，嘴巴瞬间被对方的大手捂住。
<br/>
<br/>        背后仍旧是冰冷的墙壁，而面前却是一个炙热魁梧的陌生男人。
<br/>
<br/>        就在她感到危险即将刺穿自己的身体时，一阵凉风突然扑面而来。
<br/>
<br/>        那只捂着她的大手瞬间从她嘴上挪开，连同这只大手的炙热魁梧的躯体也像是受到什么东西袭击了一样，向一旁飞去，“嘭”地一声撞得桌椅“哐啷”作响。
<br/>
<br/>        “星儿！”听到明司寒的声音时，连星儿几乎是闭着眼睛朝那声源的所在之处扑了过去。
<br/>
<br/>        明司寒伸手接住连星儿的身子，一手紧紧地将她搂入怀中，一手拿起手机打开了电筒功能朝刚刚那个男人照了过去。
<br/>
<br/>        男人本能地用手挡住光线，遮住了自己大半张脸，随即，他便立马从地上爬起来，飞快地踩上课桌，推开玻璃窗向外跳了下去。
<br/>
<br/>        明司寒搂着连星儿往窗户边走去，往外探头望去时，只见那男人一瘸一拐地越过花坛，向男生宿舍那边跑去。
<br/>
<br/>        这里是二楼，就算不死，这么高的地方跳下去，人的腿一定受不了。
<br/>
<br/>        而那男人的背影，他记住了。
<br/>
<br/>        明司寒皱起了眉头，单手紧握成拳。
<br/>
<br/>        没过几分钟后，自习室里的灯又亮了起来。
<br/>
<br/>        明司寒回过头来时，只见怀里的连星儿正低着头，身子微微颤抖着，偷偷地抹眼泪。
<br/>
<br/>        她的上身衣物倒是完整，只是下身的裙子裂了一道大口子，隐约能看到圆润微翘的臀线，以及大腿上两三道殷红的掐痕。
<br/>
<br/>        明司寒看得心疼不已，一手紧紧地揽着她，另一只手将她的脑袋按到自己的胸口上，情不自禁地低头，十分怜惜地轻吻了一下她的头顶，安慰道：“我一定会把他揪出来！”
<br/>
<br/>        连星儿吸了吸鼻子，眼泪不小心全蹭到了明司寒的衬衫上。
<br/>
<br/>        而此时此刻，她只觉在他怀中的感觉，和小时候被那个小哥哥护在怀中的感觉如出一辙。
<br/>
<br/>        好像时光冉冉，也没有让“物是”、“人非”。
<br/>
<br/>        因为裙子破了，走路容易走光。
<br/>
<br/>        明司寒直接将连星儿打横抱了起来。
<br/>
<br/>        他一只大手直接压着她的裙摆，搭在她的大腿上，另一只手则揽着她的后背，给了她一个最舒适的靠姿。
<br/>
<br/>        “我是不是很重？”连星儿怀里还抱着她和他的课本，声音软糯软糯地问。
<br/>
<br/>        明司寒勾唇一笑：“放心，我能一路不松手地把你抱回寝室。”
<br/>
<br/>        “谢谢你哦。”连星儿低垂着眸子，细若蚊蝇地说。
<br/>
<br/>        突然觉得这家伙，人其实还挺不错。
<br/>
<br/>        明司寒将连星儿送到了女生宿舍楼下，因为现在是晚上，他不方便将她送到楼上去。
<br/>
<br/>        不过，他要想进去，宿管员也不会阻拦。
<br/>
<br/>        连星儿仍旧郁郁寡欢，脸上惊魂未定，看样子是被刚刚那情形给吓得失了魂。
<br/>
<br/>        明司寒见状二话不说又将她横抱了起来，转身离开了女生宿舍。
<br/>
<br/>        “你要带我去哪里？”连星儿慌忙问道。
<br/>
<br/>        明司寒垂眸看了她一眼，莞尔一笑：“带你去看星星。”
<br/>
<br/>        “看星星？”连星儿顿时一脸懵。
<br/>
<br/>        他抱着她离开了学校，上了一辆出租车，向市区开去。
<br/>
<br/>        不知过了多久，出租车驶入了一幢单身公寓的地下车库。
<br/>
<br/>        明司寒付了车费，牵着连星儿那只柔弱无骨的小手直接下了车，然后往前方不过几米远的电梯口走去。
<br/>
<br/>        进了电梯后，明司寒又贴身站到了连星儿的面前，替她挡住电梯里的监控，避免她因裙子破了而走光的尴尬。
<br/>
<br/>        此刻，连星儿的怀中还抱着他们俩的书本，但周遭好像被明司寒身上所散发出的清冽气息给包围了。
<br/>
<br/>        在他的面前，这满满的安全感，让她此刻的心情舒畅了许多。
<br/>
<br/>        她低垂着头，小鸟依人般被他保护着。
<br/>
<br/>        电梯抵达顶层后，明司寒搂着连星儿出了电梯，没走几步便到了一张红褐色的防盗门前。
