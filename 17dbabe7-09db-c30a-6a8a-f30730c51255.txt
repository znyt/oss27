　　仙域世界之中，一道巨大的金色光柱散发着不朽的气息，一股凌天的意志，充斥在天地之间，镇压着天地苍穹，世界变得更加完善了。
<br/>　　“既然要合一，那便合得彻底些。”玄天一阵淡言道。
<br/>　　紧接着，他脑后那不断闪烁着紫金不朽之光的神轮，飞了进来，化成一方天道神轮，没入天道空间之中。
<br/>　　以岁月，命运，生死为基，以武道为主，统御天下万道，这样的天道，却是比普通天道更加完善了。
<br/>　　“轰！”
<br/>　　世界一震，天地大道仿佛变得更加清晰了，玄天双眼之中，闪过一丝明悟，天地奥妙，灵魂本源，武道意志，混元规则，无尽智慧……相互蹦跳了出来，闪现出各种灵光，各种火花。
<br/>　　咔擦！
<br/>　　玄天的心中，顿时响起一阵清脆的声响，他的气势，在不断飙升，轰的一声，冲破了混元的壁垒，一举迈入混元境，但玄天的积累、底蕴太深厚了，随着时间的推移，还在不断攀升，混元一重天，混元二重天，混元三重天！
<br/>　　“停！”
<br/>　　玄天一声大喝，他的气势顿时便停住了，但他的气息，却是还在不断攀升，境界虽然不变，但实力却还在增强。
<br/>　　良久，这种攀升才停了下来，玄天的境界，却也是实打实的到了混元三重天，也就是帝境三重天。
<br/>　　玄天心中闪过一丝明悟，原来，这便是混元啊。
<br/>　　不是手段越多越好，也不是实力越强越好，而是要那一丝的明悟。那种感觉，不可言，不可述，不可知，只能自己领悟，领悟了，便是成功了，不能领悟，别人说再多，也是毫无作用。
<br/>　　就像玄天，在最后的时刻，他明悟了自己的道是武道，至于其他的，不管是天地法相，还是紫金神轮，都是手段，有舍才有得，他舍弃了这些，却是明悟了混元的意境。
<br/>　　如今，他的实力不但没有减弱，反更强了。如今的他，不必再用天地法相，只因为他的随便一击，便比天地法相更强，他也不必再用岁月、命运，生死三道转轮，因为他随手便可以施展出这些手段。
<br/>　　“混元，就是这么简单！”玄天微微一笑，淡言道。
<br/>　　轰！
<br/>　　就在此时，九天之上乌云汇聚，雷闪电鸣，风云失色，乾坤倒转，一方浩瀚可怖的雷霆世界出现在众神眼前。
<br/>　　云层之中，狂风大作，云幕慢慢开始旋转，就在玄天的上方，渐渐出现巨大的旋涡模样。
<br/>　　这一幕对于洪荒众神们却是不再陌生了，因为上一次万劫魔尊突破混元之境，依旧如眼前一样。他们明白，这是混元之劫，只有突破混元之时，才会降下的天劫。
<br/>　　如今天劫已然出现，玄天自然也是突破了……混元。
<br/>　　“天劫，若是之前，也许能让我有些许忌惮，但如今，我已突破，却是不能再撼动我丝毫心神。”
<br/>　　话音刚落，便听得“轰隆”巨响，一声惊雷，赫然在天幕之中炸响。
<br/>　　从地面向上空望去，只见云层旋涡之中，电舞银蛇，撕裂虚空，电芒疯狂窜动，雷神轰轰，震耳欲聋，更有怪异的咆哮声传了出来，如天空张开狰狞大口，正欲择人而噬！
<br/>　　玄天突破伸手一招，将自己的本命神器武塔招了过来。他在武塔之中突破，武塔身为他的本命神器，自是随着法则的改造，晋升成了至宝，如今比一般先天至宝也不差，上面刻录了玄天的武道，更是吞噬了无数妖族与巫族，更有妖皇与祖巫，威力大增。
<br/>　　它经过玄天的不断锤炼，如今正好用来渡劫。
<br/>　　“去！”
<br/>　　玄天伸手往上一扔，武塔瞬间变大，九层武塔，犹如九重天，飞了起来，没入雷霆之中，一口便将无边雷霆世界吞噬，呼吸之间，劫云便被吞噬一空。
<br/>　　“这……”
<br/>　　洪荒众神顿时张大了嘴巴，目瞪口呆，想要说些什么，竟什么也说不出来，一个个在心里狂吼：“居然……就这么将天劫给……吞了！”
<br/>　　武塔之中传出噼里啪啦的声音，之后又是轰轰雷声，劫云在武塔之中发作，无数雷劫霹雳劈来劈去，震得武塔塔身剧震，不断传出“砰砰砰”的声响，就像是一柄铁锤在敲打钢铁。显然，武塔这是在利用雷霆锤炼己身。
<br/>　　雷霆乃是至刚至阳的狂暴之物，比什么神火都要强，锤炼武器那是最好不过了。
<br/>　　天劫仿佛是被玄天这幅态度给激怒了，天际黑云深处，滚滚裂雷轰鸣声中，一道道雷云旋涡急剧生成，其中一道道光柱从天而降，带着毁天灭地的威力，轰然击下，向着武塔与玄天击来。
<br/>　　武塔微微一晃，继续传出一道恐怖的吞噬之力，将这雷劫给吞噬一空。
<br/>　　就这样，雷劫连降九九八十一道，全都被武塔吞噬一空，其中好几次武塔都被雷霆炸得碎裂开来，但有着玄天的修复，每一次毁灭，便是一次新生，在不断毁灭与新生之中徘徊，终于铸就出这无上神器武塔。
<br/>　　如今的九重武塔，微微一晃，便能摇动诸天星辰，随便一震，亦能镇压苍穹。
<br/>　　天劫见丝毫奈何不了玄天，便也消失不见了。
<br/>　　天劫的消散，便代表着，玄天真正踏入了混元境界，天地开始承认他的境界。
<br/>　　玄天面无表情，突然，他的心神深处传来一阵声响，仿佛一道枷锁被挣断了，顿时，一道信息冲击到他的脑海之中。
<br/>　　“原来……如此。”
<br/>　　玄天喃喃道，紧接着，身形一晃，便消失于天地之间，不再去管洪荒其他诸神的态度了。
<br/>　　却说为何玄天会这样，那是因为他觉醒了自己真正的记忆。真正的记忆？不错，如今的玄天，才是真实的玄天，之前的玄天，却是拥有一份虚假的记忆的玄天。
<br/>　　原来玄天便是元始的逝我，在知道人族将出之际，元始便封印了逝我的记忆，制造了一份来自后世的虚假记忆。
<br/>　　后世记忆之中，玄天也是人族，而且还是一位准圣高手，在一次争夺宝物的大战之中，陨落了，这才有了洪荒的玄天。
<br/>　　在玄天的神魂之中，元始保留了大量的对道的领悟，只是封印了他的记忆，想让逝我之身，以人族的身份，重走一道，摆脱元始之道的影响，等真正大成后，逝我归位，元始势必会道行大增。
<br/>　　若仅仅只是逝我之身的修炼，虽然他一样会强大起来，但却只能如善尸永恒岛主那般，有了限制，而不会如恶尸万劫魔尊那般，走出自己的道路，突破混元。想要摆脱元始大道的影响，只有封印记忆，重走红尘，再修一道。如今看来，逝我却是成功了。
<br/>　　这一瞬，玄天也明白了为何自己能够在如此短的时间内便冲击到准圣圆满之境，原来是自己早有积累。
<br/>　　知道一切后，玄天反而释然了，之前他一直在猜测自己的修炼能够如此顺利，是否是有背后之人在算计自己，想要从自己身上图谋什么，如今看来，的确是有人在算计自己，而那人便是自己的本尊。
<br/>　　这一刻，玄天的心态恢复了平和，心境变得更加圆满了。
<br/>　　至于对本尊的想法，他也没有什么不满，因为他们都是同样的人，本尊的想法便是他自己的想法，当初封印记忆与其说是元始本尊的注意，还不如说是他自己的想法。如今，他已经成功了。
<br/>　　玄天微微一笑，不再多说什么，而是闭目修炼，巩固自己的武道。虽然觉醒了记忆，但对于他并没有什么影响，他依旧是他，是人族的武祖玄天，更是元始。
<br/>　　为玄为元，又何必区分？
<br/>　　……
