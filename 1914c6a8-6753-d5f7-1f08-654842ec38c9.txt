　　以御风境抗衡圣者的武者，有没有？有！
<br/>　　但是诺大的北域，能够真正找出来的，却是寥寥无几。
<br/>　　能够做到这一点的，可都是各大宗门秘密培养的天才，从小培养，才能做到这一点。
<br/>　　像是楚风眠这样一个，看起来根本就不是出自于各大宗门的人，能够做到如今的这一步，已经是足以让人吃惊了。
<br/>　　但是接下来，却是无人还看好楚风眠了。
<br/>　　刚刚陆莫出手，只是随意出手，并未出全力。
<br/>　　如今陆莫被激怒，全力出手之下，只怕是楚风眠，毫无反抗机会。
<br/>　　“给我死！”
<br/>　　陆莫一步踏出，身上的力量暴涨，所有的力量，集中在他的一掌上。
<br/>　　这一掌，突然凝聚无数灵力打出，猛然冲着楚风眠打了过去。
<br/>　　这一掌中的威力，仿佛如同一座大山，要将楚风眠完全碾碎，压死。
<br/>　　“啊，这是陆家的绝技，群山压神！”
<br/>　　“这一招是威力，只怕是达到地级武技的巅峰。”
<br/>　　“以陆莫的实力，打出这一招来，只怕是神力境四重的圣者，都要镇压，若是实力再差一点的，只怕是会被直接活活震死。”
<br/>　　围观众人脸色大变，纷纷后退，生怕是被波及到。
<br/>　　楚风眠抬起头，便是感觉到了一种无边压力。
<br/>　　如同绵延不绝的群山，向着他镇压而来一样。
<br/>　　楚风眠的眼神中，露出几分轻蔑，冷笑一声道。
<br/>　　“哦？想要镇压我？天穹想要镇压我，天穹都要破碎，更何况是群山之力！给我破！”
<br/>　　楚风眠凝聚灵力，一拳打出。
<br/>　　“轰隆！”
<br/>　　一声巨大无比的轰鸣声。
<br/>　　再在场无数人惊异的眼神之中，这连绵不绝的群山之力，在楚风眠的一拳之下，彻底破碎。
<br/>　　这陆家的绝技，群山压神，就这样在楚风眠的手中，直接破碎。
<br/>　　同时楚风眠又是一拳，猛然打了出去，这一拳，直接是向着那陆莫，轰了过去。
<br/>　　就连陆莫自己，都没有想到，楚风眠居然是可以抵挡得下他这一招群山压神。
<br/>　　面对着楚风眠突如其来的一拳，他的脸色都是惊慌无比，没有任何来得及抵抗，这一拳已经是砸在了他的胸口之上。
<br/>　　“噗！”
<br/>　　陆莫的胸口，都被砸出一个拳印来，他直接被打飞出去，口中吐出了无数鲜血，直接是倒在了地上。
<br/>　　败了。
<br/>　　这陆家的天才，陆家的二公子陆莫，就这样的败了。
<br/>　　惨败。
<br/>　　甚至都不是楚风眠的一招之敌。
<br/>　　这一幕，仿佛如同一颗惊雷一般，在众人的心中惊醒。
<br/>　　“怎么可能！陆莫居然败了。”
<br/>　　“一招，仅仅一招陆莫就败了，此人的实力到底要多强才行。”
<br/>　　“陆莫在这一届惨叫考核的人中，已经称得上是小有名气了，有可能踏入人榜的人物，如今居然就这样惨败了。”
<br/>　　无数围观众人，面露呆滞，看着楚风眠，彻底的惊呆了。
<br/>　　本来他们以为，楚风眠面对陆莫，只是在逞强罢了。
<br/>　　没想到，楚风眠居然真的可以击败陆莫，还是这样的干脆，这等实力，简直足以比拟大宗门中培养出来的天才了。
<br/>　　“这一届的新人中，居然是出现了如此黑马。”
<br/>　　“能够一招击败陆莫，看来他一定足以踏入到人榜之中了，甚至可能踏入人榜前一百的地步。”
<br/>　　“如今展现出的这等实力，才是人榜前一百的地步吗？”
<br/>　　听到众人的议论声，楚风眠心中暗自想到。
<br/>　　这人榜，也是北邙学院之中的一个榜单，其中评选的，便是北邙学院的外门弟子的名次。
<br/>　　一般武者通过考核，都是会成为北邙学院的外门弟子，只有极少数特殊情况，直接被一些长老看重，直接进入内门。
<br/>　　这种可能性，万不足一。
<br/>　　所以大部分武者，都会先成为北邙学院的外门弟子，在一步步晋升内门。
<br/>　　外门弟子中的佼佼者，便是就可以跻身人榜，每年北邙学院，都会举行比武。
<br/>　　以实力，来确定名次。
<br/>　　能够登上人榜的弟子，都是外门弟子中的佼佼者，地位崇高，在外门一手遮天，可以获取无数资源。
<br/>　　放在外面，都算是一方人物了。
<br/>　　不过人榜，仅仅是最低的榜单，在往上，还有这地榜，天榜。
<br/>　　地榜武者，都是北邙学院的内门弟子，而天榜武者，则都是北邙学院的种子弟子。
<br/>　　像是天榜武者，在北邙学院的地位，只怕都不比一些长老要低了，拥有大权，虽然是弟子，却已经步入高层，权利极大了。
<br/>　　如今楚风眠展现出来的实力，虽然不足他真正的十分之一，但是却只能排到人榜百名左右。
<br/>　　这北邙学院的底蕴，比起他想象的还要深厚。
<br/>　　就算是楚风眠动用全力，只怕也只能跟地榜中的一些人抗衡，天榜武者，还远远不是楚风眠可以抗衡的存在。
<br/>　　如今楚风眠，在北邙学院中，不能说毫无地位，但是只能算是一个小人物，不过楚风眠迟早也要踏平了北邙学院。
<br/>　　楚风眠将来要毁灭七大宗门，北邙学院，又算得了什么。
<br/>　　“二少爷！”
<br/>　　黑市入口，突然出现了一尊老者，这老者，一眼看见那倒在地上的陆莫，突然脸色一变。
<br/>　　“都滚开！”
<br/>　　这老者，一身灵力猛然外放开来，向着楚风眠这边冲了过来，将周围的无数围观众人，都给直接镇开。
<br/>　　“这是陆家的管家，陆信！”
<br/>　　“传闻他的境界，已经是足以达到巅峰圣者，只是一直压迫，没有突破。”
<br/>　　“居然是这尊凶人来了！”
<br/>　　“我们速速离去，免得被波及到。”
<br/>　　看到这陆信前来，周围不少围观武者都是急忙退让开，免得被波及到。
<br/>　　这陆家之人，个个都是极为嚣张，这陆信的嚣张，更是不比陆莫要差。
<br/>　　这样一尊凶人，他们可不敢得罪。
<br/>　　“信叔！”
<br/>　　陆莫看着陆信前来，脸上露出几分喜色，急忙的大喊道。
<br/>　　“信叔，就是此人打伤的我！”
<br/>　　陆莫一边大喊，一边指着楚风眠。
<br/>　　陆信的双眼，顺着陆慕的方向，看向了楚风眠道。
<br/>　　“小辈，就是你目中无人，敢打伤我们陆家的人？”
