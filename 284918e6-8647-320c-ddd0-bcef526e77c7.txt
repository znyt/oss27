        “那你为什么这一个多月都不给我个音讯？”童筱颖哑声地质问。
<br/>
<br/>        傅夜沉一脸无奈地回答道：“我以为你还在生我的气，万一，你又说一些让我气到吐血的话，怎么办？我可不想让你把我气死了，独留这么年轻的你守寡，更不想你再嫁给其他男人当妻子！”
<br/>
<br/>        童筱颖喜极而泣地攀住傅夜沉的脖子，呜咽道：“大哥，对不起。”
<br/>
<br/>        “你终于不生我的气呢？”傅夜沉剑眉微扬。
<br/>
<br/>        “不生气了。”她泪眼婆娑看着他，可怜兮兮地吸了吸鼻子，“就算你现在打我、骂我，我都不生气了！”
<br/>
<br/>        “我不想打你，也不想骂你。我现在只想吻你，你也不生气？”傅夜沉俯身凑到她耳边呢喃轻问。
<br/>
<br/>        童筱颖的思绪一滞，仰头望向他的那一刻，他便已经低下头来，准确无误又像是个意外般很自然而然地噙住了她这如花瓣般娇嫩的樱唇。
<br/>
<br/>        当唇与唇相接触，这个吻止渴了他这一个多月来伪装起来的思念。
<br/>
<br/>        她一定不知道，他手机里有多少条未发给她的草稿信息。
<br/>
<br/>        信息里，他一遍又一遍地反复编写着：筱颖，你还在生我的气吗？筱颖，你不要生我的气了。筱颖，你就不能主动向我妥协一次吗？筱颖，你打个电话给我好不好？筱颖……
<br/>
<br/>        在得知她一声不吭地跑来法国找他，天知道他有多高兴。
<br/>
<br/>        当他应酬完，走完今天会见贵宾的一半流程，就迫不及待地去房间里找她，想要带她一起参加最后的压轴宴，想要将她介绍给那群豪门贵族，想要让所有人都知道她是他深爱的太太。
<br/>
<br/>        谁知，这小女人竟然又一声不吭地跑了。
<br/>
<br/>        这才让他感到气急败坏，一肚子的火气不打一处来。
<br/>
<br/>        满足的轻叹声从傅夜沉两片薄唇间逸了出来，然后他情难自控地向怀里这小女人展开了更深切的掠夺、吞噬她那会让人疯狂的唇瓣。
<br/>
<br/>        他一边拥吻着她，一边带动着她的身子往屋内旋去。
<br/>
<br/>        房门自动关上的那一刻，她身上裹着的那条白色浴巾在他大手的拉扯下倏然落地。
<br/>
<br/>        翌日，清晨。
<br/>
<br/>        雨后的阳光，混合着泥土与鲜花的气息，穿透薄薄的窗帘投射而来的光影落在床上人儿粉嫩的脸颊上。
<br/>
<br/>        童筱颖那红扑扑的睡脸可爱得让傅夜沉忍不住想咬上一口，亲自品尝一下她这令他食髓知味的甜美。
<br/>
<br/>        他点漆的黑眸，隐隐浮现着激情的光彩，傅夜沉十分专心地凝视着童筱颖的睡颜。
<br/>
<br/>        小别胜新婚，导致他一夜未曾合眼却不觉得疲惫，兴奋与期待反而随着日光的爬升愈冒愈多。
<br/>
<br/>        筱颖窝在白色的被单下，香肩和藕臂露在空气里，上面还有他昨晚烙下的吻痕。
<br/>
<br/>        突然，他的手机铃声响起，就在他马不停蹄地从裤兜里翻出手机挂断后，结果还是惊扰了身边的小女人。
<br/>
<br/>        “早安。”
<br/>
<br/>        傅夜沉支起脑袋，侧躺着身子，顺势在童筱颖的眉心上落下一个早安吻。
<br/>
<br/>        童筱颖迷迷糊糊地抬起手来，揉了揉惺忪的睡眼，一副想继续睡，又睡不着的样子，可爱得让他的欲望又蠢蠢欲动起来。
<br/>
<br/>        “大哥。”她轻轻地唤了声。
<br/>
<br/>        “嗯？”
<br/>
<br/>        “我是不是在做梦？”
<br/>
<br/>        “为什么这么说？”他温柔地问道。
<br/>
<br/>        童筱颖这才真正苏醒过来，凝望着傅夜沉的俊脸，接着回答道：“就感觉是在做梦，很不真实。”
<br/>
<br/>        “那你掐我的脸，试试看我会不会痛。”傅夜沉主动把脸凑了过去。
<br/>
<br/>        童筱颖忍俊不禁地掐住傅夜沉的脸颊，却舍不得用力。
<br/>
<br/>        “老婆。”
<br/>
<br/>        她忍不住偷偷地笑起来。
<br/>
<br/>        “叫声老公来听听。”
<br/>
<br/>        “不要。”
<br/>
<br/>        “那以后都不许吃事后药了，对你身体不好。”
<br/>
<br/>        “好。”她这回回答得倒是十分爽快。
<br/>
<br/>        “我年级也不小了，已经到了当爸爸的年龄了。”他呢喃，轻吻了下她的粉唇，微笑着看着她娇羞的模样，又俯下身去，坏坏地抵着她的唇瓣低语，“宝贝老婆，给我生个小情人，好不好？”
<br/>
<br/>        语毕，他捧着她的脸，温柔的吻堵住她的小嘴，指腹不轻不重地摩挲着她细致的颈项与粉颊，仿佛一阵阵的电流“呲呲”地流窜她里每一个细胞，让她娇羞地泛起轻颤与红润。
<br/>
<br/>        因为想再要她一次，所以他的吻越来越深，带着浓浓的情意与狂欢，似乎想要将她拆之入腹。
<br/>
<br/>        结果，原本这美好的“晨运”，又被那手机铃声给打断。
<br/>
<br/>        傅夜沉索性不予理会，托起童筱颖的后脑勺，将这个吻加深，随后又转战她那可爱的耳朵，细细地描绘，轻轻地啃噬。
