　　雷战率先出手。
<br/>　　身为东荒十八城人人称道的天才猎妖师，死在他手上的妖兽多不胜数；败在他手中的其他天才武者，不计其数。
<br/>　　他未尝一败，只有在裂天剑派的第一轮考核中，才以仅五分之差，低了萧逸一名。
<br/>　　轰...轰...轰...
<br/>　　只见他手中雷光锤凌空一扬，无数雷电从锤子中激射而出，狂暴无比。
<br/>　　雷电还未打出，仅仅是聚拢着，已然让得周围的地面崩裂，肆虐至极。
<br/>　　耀眼的蓝光，更是令得其他武者几乎睁不开眼来。
<br/>　　“去。”雷战大手一挥，无数雷电暴走般朝萧逸打去。
<br/>　　下一刻，无数雷电吞没了萧逸。
<br/>　　“哼，死吧。”雷战冷笑一声。
<br/>　　他只想立刻打败萧逸，证明他才是第一。
<br/>　　“在我的雷电之下灰飞烟灭，也算你的荣幸了。”雷战对自己的攻击充满了自信。
<br/>　　下方，一众武者吃了一惊，“好强。”
<br/>　　“那雷光锤本身就是极具攻击性的武魂，以雷战半步洞玄的修为打出，先天九重都只有秒杀的份儿。”
<br/>　　“那萧逸，上一轮的考核第一，怕是要死了。”
<br/>　　众武者议论纷纷。
<br/>　　“萧逸。”林劲等人惊呼一声。
<br/>　　比武台上，忽然出现了一道淡淡的声响，‘散’。
<br/>　　话音刚落，原本吞没萧逸的雷电，瞬间散去。
<br/>　　“嗯？”雷战双眼微眯。
<br/>　　雷电散去，露出了萧逸的身影。
<br/>　　一层层清风剑罡，包围了他身体数丈之内，轻松抵消了雷电。
<br/>　　“有些本事。”雷战赞了一声。
<br/>　　“不过，观你身体周围的剑罡，应该是武技所发出的吧，而且还是玄阶以上的武技或者剑法。”
<br/>　　“可我，还未使用武技，刚刚的雷电，不过是热身罢了。”
<br/>　　“一旦我使用武技...”
<br/>　　“不必废话了。”萧逸打断了他的话，同时看了眼下方的林劲等人，看到了他们脸上担忧的神色。
<br/>　　“呵。”萧逸发自内心地笑了笑，道，“如今，有几个莫名其妙地家伙，正在担心着我。”
<br/>　　“虽然认为他们是白担心。”
<br/>　　“但，我最不喜欢的，便是被人担心。”
<br/>　　“所以，雷战，算你倒霉了，我要马上结束战斗了。”
<br/>　　淡淡的声音中，是萧逸对林劲四人的打趣，也是示意他们不必担心。
<br/>　　可这些话听在雷战耳中，却是赤裸裸的轻蔑。
<br/>　　“萧逸，我要你死。”雷战脸上杀意密布，大喝一声，“雷暴。”
<br/>　　雷暴，玄阶高级武技。以雷属性武魂施展，暴烈无比，威力比一般的玄阶巅峰武技还要强上数分。
<br/>　　“哼。”萧逸怡然不惧，低喝了一声，“升龙。”
<br/>　　霎那间，萧逸气势大增。
<br/>　　先天九重的修为，在升龙的增幅下，虽不至突破到洞玄境，却远超半步洞玄。
<br/>　　那暴烈的‘雷暴’打来，只见他微微打出一拳，雷暴瞬间消散。
<br/>　　“该我了。”萧逸脚步一动，瞬间朝雷战攻去。
<br/>　　“好快。”雷动吃了一惊，喝道，“雷动如山。”
<br/>　　雷动如山，玄阶高级武技，乃是防御力极强的一招武技。
<br/>　　一层雷电铠甲，覆盖在雷战身上，看起来威势不凡。
<br/>　　然而，当被萧逸的拳头轰了一下后。
<br/>　　咔...咔...咔...
<br/>　　雷电铠甲，顷刻间破碎。
<br/>　　而后，萧逸的拳头余威不减地轰在雷战身上。
<br/>　　嘭的一声，直接将雷电击飞。
<br/>　　噗，雷战站稳身子后，吐了一口腥血，脸色煞白。
<br/>　　“怎么可能这么强。”雷战满脸不可置信之色。
<br/>　　“气势直逼洞玄境，威力远超我的武技，这怎么可能。”雷战咬了咬牙。
<br/>　　“你要败了。”萧逸淡淡地说着，身影再次消失在原地。
<br/>　　“不...不可能。”雷战怒吼着，“东荒十八城，多少狠人死在我手上；多少北山榜有名的天才，在我雷光锤中饮恨。”
<br/>　　“萧逸，你不过是个名不经传的小子，有和资格败我。”
<br/>　　“三千雷煞。”雷战嘶吼着，脸色忽然变得疯狂起来。
<br/>　　一股黑色雷电凭空而现，宛如虚空长雷，又如黄泉暗雷。
<br/>　　诡异中，带着瘆人的可怕威力。
<br/>　　三千雷煞，本就是玄阶巅峰武技；七日前，雷战突破此武技，将其炼至大成。
<br/>　　如今，此招的威力，绝对达到了地阶武技的范畴。
<br/>　　下方，一众武者似乎都被这惊心动魄的战斗所吸引，完全忘了要去挑战其他站桩者。
<br/>　　“好强的雷电，这雷战真是个怪物。”
<br/>　　“我感觉，自己被这雷煞碰一下就会死掉的；那萧逸，这次怕是凶多吉少了。”
<br/>　　其他武者议论纷纷。
<br/>　　林劲也是脸色骇然，“糟了，雷战这招武技，怕是洞玄境武者都难挡。”
<br/>　　“咔咔。”铁牛的双拳握得劈啪作响，撼地神牛的虚影就要凝聚。
<br/>　　柳嫣然手中灵器凭空而现，眼中杀意隐然。
<br/>　　“我去将萧逸救下来，以我的速度....”秦飞扬就要化雕。
<br/>　　“你们别冲动。”林劲喝了一声，拦下三人。
<br/>　　“这是考核，也是萧逸自己的战斗，我们不能插手。”林劲沉声道，“再者，我们不是雷战的对手。”
<br/>　　“那些雷煞，足以将洞玄境以下的一切事物，毁成齑粉。”
<br/>　　比武台上，雷战的三千雷煞不断凝聚。
<br/>　　这似乎是他的禁招，他释放着，自身也承受着极大的压力。
<br/>　　很显然，雷煞虽然厉害，但他自己也承受着反噬的风险。
<br/>　　渐渐的，当他快要承受不住压力，脸色无比苍白；而雷煞也激增到一个可怕的威力时....
<br/>　　一个拳头，一只白泽却有力的手臂，凭空而现。
<br/>　　萧逸，就这般直接站在雷战身前，浑身剑罡密布，一拳打向了雷战。
<br/>　　“轰”的一声，雷煞竟被轻易散去。
<br/>　　“雷战，等你那么久，只是想让你实力全出，让你败得心服口服。”
<br/>　　“现在轰飞你，是因为不想让你承受更多的反噬，你还有后续的战斗。”
<br/>　　“强者，理应得到尊重；也应该得到正确的排名。”
<br/>　　萧逸淡淡地说罢，一拳轰出。
<br/>　　在雷战不可置信的目光中，一拳将其击落比武台。
<br/>　　“噗。”雷战吐了一口腥血，在下方站稳身子后，满脸战意地盯着萧逸。
<br/>　　下一秒，抱了抱拳，道，“谢了，但，终有一天，我会亲手打败你。”
<br/>　　说罢，雷战转身离去，到角落边坐下恢复伤势。
<br/>　　另一边，林劲等人瞪大了眼睛。
<br/>　　“萧逸胜了，赢了雷战。”林劲惊讶地说道。
<br/>　　“好家伙，不愧是我秦飞扬的队友，有些本事。”秦飞扬得意地说道。
<br/>　　“萧逸很强嘛。”柳嫣然美眸闪烁，尽是喜意。
<br/>　　铁牛摇了摇脑袋，没说什么。
<br/>　　比武台上，萧逸对四人淡淡一笑，表示不用担心。
<br/>　　半晌，他收起了笑意，看向了远处正在恢复伤势的雷战，脸色变得凝重。
<br/>　　刚才那一战，看似轻松。
<br/>　　但他很清楚，雷战刚才的战力达到何等骇人地步。
<br/>　　就算自己先天九重，用上了升龙后，修为远超半步洞玄，体内真气也比雷战多得多。
<br/>　　但，雷战刚才那一招武技，却绝对足以完虐只用真气和清风剑罡的自己。
<br/>　　这就是一门强大武技的好处。
<br/>　　修为相同，或者相差不大时，拥有强大武技的一方，绝对占优，甚至是轻松胜利。
<br/>　　最后，还是靠着肉体力量，才一拳轰碎了雷煞，并将雷战轰下台。
<br/>　　不过，刚才的拳头上，附加了清风剑罡，以作掩饰，密密麻麻的剑气覆盖着。
<br/>　　倒不用担心自己的肉体力量被别人发现，别人顶多会以为他使用了什么强大的武技，例如本就子虚乌有的地阶武技。
<br/>　　数分钟后，一众武者才从刚才萧逸与雷战的激烈战斗中反应过来，并开始向其他站桩者发起挑战。
<br/>　　至于萧逸这个区域，自从雷战落败后，再也没有人敢上台挑战。
<br/>(本章完)
