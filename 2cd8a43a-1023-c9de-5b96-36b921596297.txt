王聪道，“夕绾，你不要听外面的流言蜚语，他们都是嫉妒你，所以想诋毁你的名声，清者自清，没必要为了他们而难过。”

<br/>        那些不怀好意的人绘声绘色的讲黄色故事，说她已经被玩过了，怎么玩的，大家都喜欢八卦，尤其对这种带有色彩的故事尤其的敏感，估计坊间都传遍了。

<br/>        幸亏夏爸及时赶过来了，如果夏爸晚了一步，她也不知道现在的自己会怎么样。

<br/>        吴校带着A大的学生们都走了，夏夕绾感觉自己躺了两天了，所以掀开被子下床走走。

<br/>        走出自己的病房，夏夕绾来到了回廊里，很快，她在前方看到了一道高大英挺的身影，是陆寒霆！

<br/>        夏夕绾滞了一下，没想到会在医院里看到他。

<br/>        他怎么来医院了？

<br/>        他右边的胳膊怎么了？

<br/>        夏夕绾是医生，一眼就看出了他右臂的异常，他的右臂好像遭遇了重大的创伤，现在无力的垂在那里。

<br/>        夏夕绾澄亮的瞳仁狠狠的收缩着，他的右臂是不是…废了？

<br/>        他出什么事了？
