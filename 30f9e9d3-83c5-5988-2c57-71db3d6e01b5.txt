　　第62章
<br/>　　柳芷立在程大雷面前，双目幽怨，楚楚可怜。
<br/>　　其实她这些日子过得不是很好。程大雷之前不是没想过柳家不好驯服，只是他觉得，只要自己和柳芷建立好良好关系，柳家看在大小姐面子上，也不会闹得太狠。
<br/>　　这一点，他想错了。柳芷的确是柳家曾经的千金小姐，可是沦落到青楼的大小姐，还是原先柳家的小姐嘛。
<br/>　　她说她是清清白白身，也得有人信才行。纵然没有失身，可总与男人喝过酒吧，总在人前跳过舞吧，这……已经够得上辱没先祖了。
<br/>　　人总是会往黑处想的，古往今来都一样，所以这几天柳芷的日子，大概就是拔毛的凤凰不如鸡。
<br/>　　这是程大雷想错的第一点，他还想错了另外一点。前生他的日子分外悲催，总觉得女人是要小心意哄着，送花送包送车送房，总是将对方摆在一个高高在上的位置。到了这个三妻四妾的时代，他也想着娥皇女英齐人之福，可心理上还是和前生一样。
<br/>　　事实上，这是不对的，在这个时代得到一个女人，要比程大雷所想的容易。二两银子一个娇滴滴的小丫鬟，比驴便宜，陪房侍寝最后也还是丫鬟，妾室的身份都够不上，就像红楼梦里的赵姨娘，儿子都生了，待遇也不过和高级丫鬟一样。就算是明媒正娶，两头牛也足够换一个小家碧玉，当然随着对方身份的不同，彩礼也是不同。
<br/>　　而程大雷的身份还是山贼，正如果看上了谁，只要他有办法抢过来，抢过来就是抢过来了。就像苏樱，真如果程大雷狠得下心，生儿育女也得是她。此外，不管是娘娘还是公主，能抢到都是一样。
<br/>　　这时代你想做林黛玉，可没有几个贾宝玉喔。
<br/>　　柳家被逼到最后，想到这样一个不是法子的法子，以柳芷来换取他们的安身。自然有人反对，孟子云是肯定不同意的，但对于柳家几个老人来说，其实不算什么损失，毕竟，献出的只是一个沦落风尘的柳家之耻。
<br/>　　程大雷怔了半晌，将柳芷额头一缕乱发撩到耳后，柳芷身体颤抖一下，接下来却是一动不动。
<br/>　　“留下啦！”程大雷大手一挥，开了恩准。
<br/>　　山下柳家人如蒙大赦，开始络绎不绝上山，山上的人也松口气，这种心理类似于大家一起掉进粪坑，谁也别笑话谁臭。
<br/>　　不过想要上山也不容易，总需要一一点清人数，登记名字，划分登记，明了各人原先的工作。
<br/>　　如此，天很快便黑下来，山下只剩下几个人，一堆孤火。
<br/>　　“柳老，我们也上山吧。”
<br/>　　“天色已不早，柳老也好些日子没有好好休息了。”
<br/>　　“呵，你们先去吧，我慢慢走，不着急。”
<br/>　　柳慕云笑着说一句，抬头向山上望了一眼，程大雷也看着他，却总觉得，那张被火光映照的脸有些别得东西。
<br/>　　紧跟着，柳慕云做了一件谁也没想到的事情，他扯开身边人，后退几步，猛地发力，以头撞地，临时之前口中高喊一句。
<br/>　　“柳家历代列祖列宗，暮云来见你们了！”
<br/>　　鲜血四溢，满山皆惊，山上山下一片讶异欲死的沉默。
<br/>　　程大雷也睁大眼睛，满脸难以置信。
<br/>　　可柳慕云的尸体就倒在地上，人死不可复生，这件事真得不能再真。
<br/>　　士不可以不弘毅，任重而道远。仁以为己任，不亦重乎？死而后已，不亦远乎？
<br/>　　原来世上真的有这种人，为了那所谓的荣耀，死而后已。
<br/>　　山上山下，柳家所有人都将目光望向程大雷，脸上是复杂的情绪，眼神却在表达同一个意思：你逼死了柳慕云！
<br/>　　在人们心中，柳慕云的形象忽然高大起来，他为了让大家活下去，选择了屈辱，但却用自己的死证明柳家的荣耀。
<br/>　　可程大雷总觉得事情不是这样的，柳慕云那抹古怪的笑容背后，总似乎有些别的东西。
<br/>　　心中忽然咯噔一下，程大雷脸上也冒出一抹诡异的笑容，呐呐道：“老头子，真的要玩得这么狠么……”
<br/>　　程大雷用自己的手段，击碎了这些人的尊严，没有尊严的人，是很称之为人的，这样他们上山之后，便很难再反抗程大雷，管理起来也容易些。
<br/>　　可柳慕云却用自己的死，在这些人的心里埋下一根钉子。这些人在以后的无数日子里，都会记起柳慕云的死相，这根钉子，会在某天发作，变成引线，开启爆炸。
<br/>　　一死全了自己的名声，一死给程大雷种下祸根，可以说一死两得，但用死亡的代价换来这些，是否真的值得。
<br/>　　这才是真正的厚黑学，对别人黑，对自己更黑。
<br/>　　相比之下，程大雷在厚黑学上的造诣，简直可以说一窍不通。
<br/>　　原来，那抹笑容是嘲讽的笑，在笑程大雷的年轻和无知。
<br/>　　那么，你就看着，我怎么将这根钉子拔出来吧。
<br/>　　半晌，程大雷悠悠的叹了一句之前说过的话：封建礼教害死人哟。
<br/>　　无论如何，这些人晚些时候，在山寨的校场聚起，山寨人口确定在98，距离程大雷的目标还差两人。
<br/>　　程大雷在最高处，所有人都看着他，目光十分复杂，有依赖，有悲伤，有仇恨却又不得不屈服的耻辱。
<br/>　　嘟，获得恐惧值10
<br/>　　嘟，获得恐惧值11
<br/>　　嘟，获得恐惧值……
<br/>　　原来这样也能获得恐惧值，当别人恨你的时候，同时也在怕你。
<br/>　　大家都在等着程大雷说些什么，徐神机几人也是一样，他们感受到了人心不稳，急需安抚，有时候语言是不错的安抚工具。
<br/>　　可说些什么呢？程大雷暗自想，是准备之前很早就准备好的说辞，『落地为兄弟』『大家为一个共同目标走到一起』之类的，还是想些新的……
<br/>　　最后，在所有人的期待中，程大雷挥了挥手，口中吐出两个字。
<br/>　　“睡觉。”
<br/>　　也就在当天夜里，柳芷搬进了程大雷的房间，她洗白白擦香香，甚至还问了族人妇女中，一些关于洞房花烛的事。
<br/>　　当程大雷走进房间，看到炕上端坐的柳芷，整个人都不好了。
<br/>　　二人目光相对，一样紧张恐惧。
