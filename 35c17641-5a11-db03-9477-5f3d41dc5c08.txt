　　白磷手雷正常的爆炸半径是十五米，现在因为地下室过道的关系被压缩，此时这只有几平方米大的狭小空间中被白磷手雷变成了人间炼狱。
<br/>　　只有亲眼看到发生在别墅地下三层这几平方米范围内的一幕，或许才会明白，为什么白磷手雷会跟达姆弹一样，遭到国际禁用，为什么这种武器不会列装到军队中，为什么只会出现在恐怖分子，还有只为钱而战的雇佣兵手里。
<br/>　　没有灼人皮肤的炽热高温，但是那几个被白磷手雷沾染到的影子部队队员却是发出撕心裂肺的叫喊声，这种痛苦的哀嚎声让人头皮发麻。
<br/>　　防弹玻璃背后，西伦科特看着全身冒着磷火，哀嚎着朝着玻璃房跑来，但最后还是倒在玻璃房外的影子部队队员，尸体还在被磷火继续焚烧的影子部队队员，眼神冷漠，没有丝毫的感情波动。
<br/>　　西伦科特是个冷酷的人，这些影子部队队员的命都是他给的，是他赋予了他们活下去的权利，所以他们为他卖命也是理所应当。
<br/>　　把这些孤儿送入影子部队训练基地，并且把他们培养成合格的战士，西伦科特花了十几年的时间，但是在白磷手雷面前，仅仅只是几秒钟的时间，他们就变成尸体，并且逐渐被焚烧成了灰烬。
<br/>　　罗昊、李然、周柏，还有几个其他国家特种部队队员握着自动武器，快步朝着玻璃房的方向冲来，他们避开了地面上影子部队队员的尸体，避免被沾染到尸体上的磷火，受到波及。
<br/>　　防弹玻璃房中，西伦科特冷漠的朝着玻璃房外的罗昊、李然、周柏这些人看了眼，然后拉开车门，钻进汽车，把车子发动。
<br/>　　“西伦科特！！”
<br/>　　罗昊大吼一声，抬枪对着玻璃房打出一个点射。
<br/>　　“哒哒哒！”
<br/>　　“啪啪啪！”
<br/>　　子弹没有能击穿玻璃房的高强度玻璃，只是在上面留下几个白色印子。
<br/>　　“妈的！”
<br/>　　罗昊嘴里咒骂了一声，立刻跑到玻璃房门锁的位置，对着门锁又是两枪。
<br/>　　“当当！”
<br/>　　门锁上溅起两点火星，但是门锁却没有坏，依旧紧锁着玻璃房。而此时玻璃房内，西伦科特驾驶的那辆防弹GLK已经进入了漆黑的隧道中。
<br/>　　联合行动司令部“鹰巢”中，詹姆斯、云熙、宋萱还有其他国家的军事代表眼睛紧紧盯着他们前方的大屏幕，能不能抓到西伦科特，或者说直接干掉西伦科特就在此一举了。
<br/>　　“我来！”
<br/>　　一名“雷鸣”特种部队队员手中拿着霰弹枪走上前来，霰弹枪中所装填的是独头弹，这种弹药就是专门用来打门的，再坚固的门锁，也是一枪搞定！
<br/>　　其他人退后一步，给那名“雷鸣”特种部队队员让出位置，“雷鸣”特种部队队员端起霰弹枪对着门锁就是一枪。
<br/>　　“当！”
<br/>　　哗啦！
<br/>　　“雷鸣”特种部队队员握着套筒把枪用力一甩，甩出枪膛中的弹壳，重新推上一颗独头弹，再次瞄准了门锁。
<br/>　　“砰！”
<br/>　　“当啷！”
<br/>　　又挨了一枪的门锁掉落在地上。
<br/>　　罗昊上前，一脚踹开玻璃门，直接追进了那条漆黑的隧道，追了有一两百米的距离，罗昊对着黑暗中那两点红色尾灯，打出一个长点射，子弹落在隧道两边的墙壁上，发出“噗噗”的闷响声，红色尾灯很快就消失在了罗昊的视野中。
<br/>　　看着西伦科特就这么在自己眼皮子底下逃走，罗昊心中的郁闷可想而知，这是跟西伦科特交手这么多次，第一次跟他这么近距离的接触，可最后却还是被他跑了。
<br/>　　“鹰巢。”
<br/>　　罗昊一边往隧道入口的方向走，一边按住耳麦跟联合行动司令部取得联系，问道：“卫星能监控到这条隧道的出口吗？”
<br/>　　“Negative！”
<br/>　　詹姆斯遗憾的回答道：“这条隧道在山体内，卫星拍不到隧道的走向，无法推断出隧道的出口在哪里。不过我们会严密监控附近的山路，查找可以车辆。Alpha1，直升机还有十分钟抵达，你们可以从别墅中撤离了。”
<br/>　　“Roger！”
<br/>　　罗昊松开耳麦走出了玻璃房。
<br/>　　地下室中，其他人的脸色都不怎么好看，每个人心中都跟罗昊一样郁闷。
<br/>　　李然走上前，拍了拍罗昊的肩膀安慰道：“他逃得了初一，躲不过十五，我们总会抓住他的。”
<br/>　　罗昊点了点头，现在只能寄希望于联合行动司令部可以通过卫星监控，找到西伦科特。不过罗昊相信西伦科特再次出现的时候，肯定会试着化妆，来掩饰自己原来的面貌，以此来躲开联合行动司令部的追踪。
<br/>　　准备离开地下室的时候，李然注意到了那个镶嵌在墙壁内的冷藏柜，李然走到还有着阵阵白色寒气往外翻涌而出的冷藏柜前，看着冷藏柜中那三个原本放置三瓶P病毒的凹槽，伸出手，丈量了一下凹槽的长宽，若有所思。
<br/>　　“怎么了？”罗昊对表现出思考状的李然问道。
<br/>　　“你觉得会是什么东西放在这里？”
<br/>　　李然指了指冷藏柜中排列整齐的三个凹槽，说道：“这三个凹槽，高四十公分，宽七公分，圆柱状。而且很明显，原来这个冷藏柜是关闭的，是西伦科特在离开前才打开了冷藏柜，并且拿走了里面的东西，你觉得会是什么东西？”
<br/>　　罗昊皱眉，仔细思索，心中虽然有了大胆的猜测，但是却不敢肯定，对李然说道：“希望我的猜测是错误的，如果我的猜测是正确的，事情就麻烦了。”
<br/>　　“看来我们应该又是想到了一起去了。”
<br/>　　李然取出“手机”，对着冷藏柜“咔嚓咔嚓”的拍下一组照片。李然手中这个跟智能手机差不多大，集无线上网、拍照、摄像于一体的机器，是专门给特种部队在行动过程中在现场收集线索和取证所用的。而这个机器除了这些基础功能外，在连接上无人机系统后，还能控制无人机进行空中侦察。
<br/>　　拍摄下冷藏柜的照片后，罗昊、李然、周柏等人撤出了别墅地下室跟其他人进行汇合。
