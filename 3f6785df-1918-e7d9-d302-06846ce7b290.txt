　　锵的一声。
<br/>　　萧逸收回暴雪剑，又是一股热血从冰承天身上溅出，洒落在城墙之外。
<br/>　　冰承天，已然生机断绝，死得不能再死了。
<br/>　　眼眸瞪得极大，显然死不瞑目，对于死于萧逸这个年轻人手上，很是不甘。
<br/>　　嗖的一声。
<br/>　　萧逸拎着他的尸体，身影后退，落于雪地上。
<br/>　　刚才让冰承天短暂陷入失神状态的，自然是魂师的手段。
<br/>　　不过并非魂技。
<br/>　　只是‘魂念诀’上，对‘魂力’的最简单应用。
<br/>　　不过，这丝魂力并不太强，正常来说，根本不足以对冰承天这个层次的武者造成太大的威力。
<br/>　　若是正常战斗的话，刚才那丝‘魂念’攻击，几乎不会对冰承天有任何影响。
<br/>　　但冰承天刚才毕竟受了不轻的伤势，又一心逃遁，加之不知道魂师的诡异手段。
<br/>　　这才着了道。
<br/>　　简单来说，就萧逸现今的魂师修炼程度，魂师手段，只能凭着诡异莫测，用作奇招。
<br/>　　真正让萧逸击杀冰承天的，还是‘冰纹’的开启。
<br/>　　三千三百三十三丈气泉的一瞬间爆发，让得他的战力，飙升到极境三重巅峰。
<br/>　　另外比较重要的一点，是武道力量上的压制。
<br/>　　冰承天虽掌握了三成武道力量。
<br/>　　但，如同剑道分多种，分强弱。
<br/>　　寒冰之道亦是一样。
<br/>　　他的重冰武道，显然远远比不上萧逸的寒冰剑道。
<br/>　　若给寒冰武道分层次的话。
<br/>　　寒冰剑道，在顶尖行列。
<br/>　　而重冰剑道，只在三流行列。
<br/>　　萧逸‘冰纹’的开启，已经拉平了冰承天的极境三重差距，甚至犹有过之。
<br/>　　加上武道上的压制，便以摧枯拉朽之势，击杀了冰承天。
<br/>　　也就是说，即使萧逸不用魂师的手段，照样能杀冰承天。
<br/>　　只不过，这样冰承天将逃回冰武王国境内，萧逸也将耗费多一些的时间去追杀他。
<br/>　　萧逸并不愿意徒增麻烦。
<br/>　　言归正传。
<br/>　　此时，萧逸拎着冰承天的尸体，退回到雪地上。
<br/>　　而寒冰城上，寒冰城城主，以及一众兵士反应了过来。
<br/>　　“混账，给我拿下此恶贼。”寒冰城主怒喝一声。
<br/>　　寒冰城，以极寒之地与冰武王国，各取一字，命名而成。
<br/>　　可想而知冰武王国对此城的重视。
<br/>　　极寒之地，妖兽数量之多，实力之强悍，超乎想象。
<br/>　　这里简直就是一座无边无际的巨大‘妖兽森林’。
<br/>　　故，寒冰城这边，实则经常爆发严重的兽潮。
<br/>　　自然的，寒冰城这边的武者力量，非常强大。
<br/>　　甚至于，除了冰武王都以外，寒冰城算是整个冰武王国武者力量排在前列的大城。
<br/>　　城主是一位天元巅峰武者，守城士兵，清一色破玄境以上，地元境多不胜数。
<br/>　　此时，寒冰城主一声令下，无数守城士兵立马集结。
<br/>　　虽冰承天并未彻底回到寒冰城内，未算回到冰武王国境内。
<br/>　　但，冰承天作为冰武王国大供奉，就这般被击杀于城墙之外。
<br/>　　这在寒冰城主看来，乃是赤裸裸的挑衅。
<br/>　　城墙边缘，那一股洒落的血迹，染红了城墙的石阶。
<br/>　　也染红了寒冰城主的双目。
<br/>　　“胆敢杀我冰武国大供奉，必受我冰武王国无尽追杀。”
<br/>　　寒冰城主大喝一声，便要领兵出击。
<br/>　　恰在此时，一股滔天杀意，笼罩了整个寒冰城城墙。
<br/>　　杀意来自于萧逸。
<br/>　　“我乃冰缘大会第一名，冰承天夺我冰心圣果，这便是下场。”
<br/>　　“你冰武王国若要报复，尽管遣人来追杀。”
<br/>　　“来一个，我杀一个。”
<br/>　　冰冷的宣言，在萧逸口中吐出，传遍整个寒冰城。
<br/>　　先不说自己并未进入寒冰城杀人。
<br/>　　就是进了，寒冰城主也奈何自己不得。
<br/>　　对于真正的强者来说，所谓的世俗律法，只是个笑话罢了。
<br/>　　冰冷的宣言，让得包括寒冰城主在内的一众寒冰城武者心头一惊。
<br/>　　在他们眼中，面前城外不远处的桀骜年轻人，似极了一把冲天利剑，锋芒无匹。
<br/>　　他们意识到，这是一个能轻松斩杀极境三重实力的大供奉的绝世强者。
<br/>　　若是惹怒了他。
<br/>　　怕是倾尽寒冰城所有力量，也无法浇灭这股怒火。
<br/>　　咕，寒冰城主咽了口口水，最终还是停下了手头动作，站与城墙之上，满脸忌惮之色。
<br/>　　城墙之外，萧逸冷漠地看了他们一眼。
<br/>　　随后转过头，取下了冰承天身上的乾坤戒。
<br/>　　微微检查了一番，顿时一喜。
<br/>　　冰心圣果，果然还在冰承天手上。
<br/>　　“嗯？”这时，萧逸皱了皱眉头。
<br/>　　“冰冥幽蟒？”
<br/>　　萧逸在乾坤戒内，自然发现了大量修炼之物。
<br/>　　天材地宝、妖兽精血、内丹、寒冰之髓等等。
<br/>　　以及一些品阶不错的武器。
<br/>　　但最让萧逸注目的，还是里头的三头巨兽尸体，正是冰冥幽蟒。
<br/>　　冰冥幽蟒，八级初阶妖兽，实力介于极境一、二重左右。
<br/>　　无论是精血还是内丹，皆是至寒至阴之物。
<br/>　　故，这种妖兽的精血，武者别说拿来吞服，就是用来侵泡，都不可能。
<br/>　　萧逸皱了皱眉头，冰承天击杀这三头妖兽做什么？
<br/>　　萧逸大手一挥，将另一边的八道身影的乾坤戒吸摄到手中。
<br/>　　略微看了一眼，脸色忽然一变。
<br/>　　“六头暴雪冰鲸，八头雪地金刚猿。”
<br/>　　“暴雪冰鲸，身躯庞大，精血与内丹，皆蕴含着远超同阶级妖兽的庞大力量。”
<br/>　　“雪地金刚猿，精血与内丹，带着澎湃的狂暴气息。”
<br/>　　“三种妖兽的精血内丹，皆非寻常之物，无法正常使用。”萧逸皱眉自语。
<br/>　　“但若是加上冰心圣果…”萧逸脸色一变，“难道是...冰极丹？”
<br/>　　作为炼药本事过人的炼药师，萧逸单从材料上，就能大致猜出能炼制什么丹药。
<br/>　　萧逸摇了摇头，没兴趣再思索下去。
<br/>　　大手一挥，一个灵气屏障，陡然升起，包裹了四周。
<br/>　　事实上，以他现今实力，他想要暗中做些什么，等闲武者，根本无法发现。
<br/>　　但谨慎起见，他还是布下了屏障。
<br/>　　太阴太阳之眼，陡然出现。
<br/>　　一个无形漩涡，吸收了冰承天，以及那八人的武魂。
<br/>　　冰承天的武魂，乃是蓝色巅峰。
<br/>　　另外八人，亦是蓝色品阶。
<br/>　　早在许久前，萧逸就已经很少使用太阴太阳之眼吸收别人的武魂力量了。
<br/>　　实在是以现今控火兽的品阶，青色以下武魂，已几乎无用。
<br/>　　青色武魂，还能发挥一些功效。
<br/>　　青色之下的，吸收再多都没有任何效果。
<br/>　　而蓝色品阶的，萧逸可不会放过任何一个。
<br/>　　嗖的一声，做完一切，萧逸御空离去。
<br/>　　屏障，也渐渐消散。
<br/>　　原地，只剩下冰承天以及另外八人的尸体。
<br/>　　寒冰城外，大雪纷纷。
<br/>　　雪花，渐渐覆盖了这九具尸体。
<br/>　　寒冰城上，寒冰城主脸色煞白，脚步一个不稳，差点跌倒。
<br/>　　“完了…完了…大供奉，还有国主麾下八大冰武卫，尽皆殒命。”
<br/>　　“冰心圣果也被抢走了。”
<br/>　　“国主的怒火，谁能承受？”
<br/>　　寒冰城主，脸色一片灰丧，看着远方那道飘然离去的身影，陷入了呆滞。
