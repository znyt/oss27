　　“事不宜迟，你们尽快行动！”
<br/>　　苏莫沉声道，随即，他环视众人，指着十数人道：“你……你……还有你们留下来！”
<br/>　　苏莫选择了一些留下来的人，这些人之中，有亘古魔国的魔野，有天星神朝的星无澜，以及其他十余人。

<br/>　　“随我出发！”公孙长绝并未多言，朗喝一声，率先离开了第三城。
<br/>　　而后，约莫八十余名天才，纷纷仅随而去，离开第三城向第八城杀了过去。

<br/>　　“什么？”一见公孙长绝带着大批人手杀向第八城，古魔太子的脸色顿时为之一变。

<br/>　　“回城！”古魔太子毫不犹豫，立刻朗喝一声，率领麾下众人返回第八城。
<br/>　　公孙长绝见此，也没有再前进，稍稍停顿下来之后，便率人返回第三城。

<br/>　　那第一城、第五城、第六城、第七城和第九城，不知道有什么打算，一直在按兵不动。
<br/>　　古魔太子撤回，龙羽等第二城的人，彻底的放心了，立刻加快速度攻击。
<br/>　　渐渐的，第四城越来越无法抵挡，大量的天才被重创，仓皇逃穿。
<br/>　　不多时，第四城之人，只剩下了十几人，连无极公子都受了不轻的伤。
<br/>　　无极公子的实力，的确是很强大，一身星辰之力，比之星无澜要强大的多。
<br/>　　但是，双拳难敌四手，在围攻之中，无极公子口中不停的喷吐出鲜血。
<br/>　　苏莫的身后不远处，星无澜看到无极公子的惨状，深深的吸了口气，他并未去救援。
<br/>　　因为，以他一人之力，根本无济于事。
<br/>　　他留在第三城，到得最后，还有一丝能胜利的可能。
<br/>　　又过了片刻，第四城彻底的败了，虽然无极公子和一些人还再拼命的反抗，但是城中的阵基已经被彻底摧毁了。
<br/>　　在城中阵基被摧毁的那一刻，整个第四城迅速土崩瓦解，变成了一片废墟。
<br/>　　而所有第四城之人，全部被笼罩十座城池的大阵，强行排斥了出去，淘汰出局。
<br/>　　龙羽和贺兰殇等人，没有在外多留，率领剩下的人，立即返回了第二城。
<br/>　　接下来，各大城池都陷入看平静之中，第二城的人经过一番大战，不少人受了伤，在进行短暂的休整。
<br/>　　……
<br/>　　神昆小世界之外，各大势力之人，均是没有离开，在静静的观看这神昆小世界中发生的一切。

<br/>　　不仅如此，在这魔雾形成的画面之中，每座城池内的情况，都清晰的显露而出。
<br/>　　天星神朝之人，个个面色难看，因为第四城败的太惨了，无极公子都惨遭淘汰。
<br/>　　不过，他们也不是必败无疑，因为天星神朝的天才还有几人。
<br/>　　万化神殿、雷焱神王殿、无双神殿等人的面色，也很不好看，因为千选战一开始，苏莫就将第三城中他们的人赶了出来，甚至还出手击杀了一人。

<br/>　　这一次千选战，他说服了亘古魔国、不死神朝、幽冥神朝、霸天神朝、天星神朝这五大神朝。
<br/>　　这五大神朝愿意在千选战之中，让弟子擒拿或者诛杀苏莫，但是，苏莫的运气非常的好，居然随即被分配到了第三城。
<br/>　　虽然第三城之中，也有一些这五大神朝的天才，但是，这些人并不敢出手。
<br/>　　目前，最主要的是，苏莫非常聪明，身在第三城的消息，其它几座城池还不知晓。
<br/>　　不过，帝一魂也不急，苏莫身在第三城的消息，很快就会暴露，到时候必然是众矢之的。
<br/>　　……
<br/>　　第三城之中，苏莫和众人在商讨接下来的行动。
<br/>　　“苏莫，我觉得，接下来我们要先试出第一城、第五城、第六城、第七城和第九城的虚实！”魔野目光凝视着苏莫，沉声说道，他不想让苏莫和第二城联手攻击第八城。
<br/>　　“此言在理！”
<br/>　　公孙长绝点了点头，道：“第八城实力很强，而且古魔太子的实力，没有几个人能挡住，我们当务之急，是摸清另外几城的情况。”
<br/>　　“不错！”
<br/>　　“是要搞清楚另外几城的情况！”
<br/>　　“另外几城一直毫无动静，不知道有没有在暗中策划！”

<br/>　　不过，思来想去，想要知道龙腾和叶凌天的所在，还需要他亲自现身。
<br/>　　只要这两人知晓他在第三城，一定会与他联合起来。
<br/>　　“第五城有动静了！”
<br/>　　就在此人，有人突然开口说道，众人闻言，纷纷飞身而起，查看第五城的情况。

<br/>　　这两人，其中之一是霸天族，另外一人是妖族，应该是鲲鹏族之人。
