　　“轰隆隆！”
<br/>　　两者相撞，爆发出强烈的震动，轰动乾坤，风云退避。箭光湮灭，金色剑气不停，毫不犹豫的斩在了后羿身上。
<br/>　　“轰！”
<br/>　　后羿先以霸箭抵消一部分剑力，之后再以心力消散一部分，最后剩下的只能让自己的祖巫之躯抵挡了。
<br/>　　但，就算如此，他还是被这霸道的金色剑光劈出数千里之远。此时，他胸膛之处有一道深深的剑气，还在霸道的毁灭着他的生机，他全身上下鲜血淋淋，气势一阵萎靡，实力变得虚弱下来。
<br/>　　先前他耗尽全力爆发，终于抵挡住了帝俊的这一剑，但同时也耗尽了自己的全力，若是此时帝俊再给他一剑，莫说帝俊，就算是一个普通金仙的修士，也能够一剑解决他。
<br/>　　至于为何不是金仙以下的，那是因为，金仙以下者，连祖巫的肉体都砍不动，除非是其他特殊的方法，否则很难斩杀祖巫。。
<br/>　　帝俊暴怒一剑，威力恐怖如斯！
<br/>　　“嗯？竟然突破了，该死！”
<br/>　　帝俊一阵意外，本以为他自己的全力暴怒一击，就算是普通的准圣都能斩杀，更不用说是一位大巫了。但是大巫突破了就是两说了，因为祖巫本身肉体就十分强悍，堪比一些灵宝。若是再突破祖巫，肉身那就更是强悍了，堪比上品先天灵宝。再加上后羿本身实力就极为强大，抵挡住这一剑的机会虽然不大，但也还是有的。
<br/>　　“既然如此，那便再赐你一剑，给本帝死！”
<br/>　　意外归意外，既然一剑杀不死，那便两剑。帝俊是不会放过这个杀自己儿子的凶手的。
<br/>　　“铮——”
<br/>　　一道高昂响亮的剑吟之音响起，金色剑光再次划破天际，向着后羿斩去。他一出手，便是毫不留情。
<br/>　　“尔敢！休伤我巫族兄弟！”
<br/>　　在这生死幻灭的关键时刻，随着一声大喝，虚空一阵波动，剩下的八大祖巫终于赶到了。他们出手毁灭了剑光，救下了重伤垂死的后羿。
<br/>　　“帝俊，你敢杀我巫族祖巫，难道是想与我巫族开战吗？”
<br/>　　帝江祖巫一步踏出，走到前面，对着帝俊质问道。
<br/>　　“哼！帝江，你巫族之人，射杀我九个儿子，我今天便要他死，你又能奈我何？”
<br/>　　妖帝帝俊冷哼一声，手提金色帝剑，看着帝江等人，杀意惊天。
<br/>　　“可恶，还我儿命来！”
<br/>　　就在此时，东皇太一与妖后羲和等率领数百妖神也纷纷从三十三天之上赶了过来，见到自己的儿子被杀了只剩下一个，顿时大怒。
<br/>　　“母后，呜呜！兄弟们都死了，只剩下我一个了，呜呜，是他，是他杀了兄弟们！”
<br/>　　十太子陆压看见羲和，顿时将自己的遭遇哭了出来。之后指着后羿喊道。
<br/>　　“放心，母后定会为你们报仇。”
<br/>　　羲和一脸悲痛，看向帝江等祖巫，眼中杀机浓郁，毫不掩饰。
<br/>　　“帝江，此话你们怎么说？”
<br/>　　“哼，他们该死。妖族十太子是命，洪荒无尽生灵就不是命了吗？你十个儿子降下无尽太阳真火，焚山煮海，烧死无尽生命。妖掌天，巫霸地，你妖族之人胆敢在我巫族大地之上放肆，就要付出代价！”
<br/>　　帝江祖巫冷冷道，面对妖族的重要强者，毫不畏惧。
<br/>　　妖帝帝俊皱了皱眉，看着洪荒大地，无数火焰纵横，他不管其他，冷声道：
<br/>　　“本帝不管其他，将后羿教出来，否则，巫族今后没必要存在了。”
<br/>　　“哈哈！你要战，那便战！我巫族有何曾惧谁？说的那么好听，就好像你妖族赢定了似的。”
<br/>　　听了这话，帝江好似听到了这世上最好笑的笑话，他哈哈大笑一声，目光冷厉，大声喝道。
<br/>　　这次帝俊却没有在说话，他直接挥动手中的金色帝剑，向着帝江等凌空劈去。
<br/>　　帝俊的这一剑，直接将两族最后的引火线，点燃了，双方的大战，在这一刻，彻底爆发了。
<br/>　　东皇太一带领一众妖神，气势磅礴如山河般倾泻下来，妖云滚滚，无尽战意，浩荡天地，呼啸着，接天地之势，向众巫族强者席卷而来，瞬息之间，便到了近前，顶尖强者的浩荡威压，压着乾坤，令虚空为之震荡。
<br/>　　“哈哈哈！这么热闹的事，怎么能够少得了我人族呢？”
<br/>　　就在此时，一阵大笑由远及近，轰动乾坤，震动苍穹。却是人族武祖玄天带着人族强者赶来了。
<br/>　　按理来说，人族完全可以等两族打得两败俱伤之际，再杀出来，收拾残局，到时候自然就少了许多麻烦。但，玄天并不这么想。
<br/>　　人族与巫、妖怪两族之间的仇恨，那是刻骨铭心的血泪仇恨，那是种族之战，不死不休之战，只有自己亲自出手灭了两族，方解人族心头之恨。
<br/>　　再说，如今人族的实力，完全能够压倒任意一族，但若想要同时对付两族强者，还是有一定难度的。但是，三族之中，各怀鬼胎，此时妖族太子更是死在巫族手中，他们根本不可能联手。
<br/>　　是以，人族根本不惧两族，三族混战，谁怕谁啊？人族从来还没有怕过他们。更何况，人族想要成为天地主角，不杀不拼，不在血与战之中崛起，可能吗？
<br/>　　只有经历过血战的人族，才能经受得住未来的考验。
<br/>　　“怎么，你们人族也想插一脚？”
<br/>　　帝俊皱了皱眉头，看着人族强者的出现，有些意外，看着人族这些年出现的无数强者，他脸色有些难看道。
<br/>　　谁又曾想到，之前只是一个小小的后天种族，便能发展成如今这种称霸洪荒的大族，若是他早知如此，哪还会让人族成长起来，就算是拼着被女娲惩罚的危机，也定要灭了人族。
<br/>　　“怎么，你们不会是忘了我们之间的仇恨了吧。你们忘了，我人族可不会忘。三族之间，不是你死，就是我亡，如今是该有所了结了。”
<br/>　　玄天凭立虚空，面对两族强者，他面不改色，淡淡道。
<br/>　　“也对，你我之间，早已经是不死不休的局面了。哈哈，那便战吧！”
<br/>　　帝俊身为一位帝者，从不觉得自己妖族会弱于他人，既然别人都不怕，他还怕什么。
<br/>　　“战！”
<br/>　　巫族同样气势高昂，战意惊天动地，将九霄之上的层层云彩都震散开来，风云为之一滞，乾坤为之一荡，日月都为之失色。
<br/>　　“杀杀杀！”
<br/>　　人族之中，人族武者气势惊人，杀意惊天，面对两族强者，他们毫不畏惧，之前的种种仇恨，都在此刻化为心中无限的战意，爆发出来。
<br/>　　三族大战，就此展开！
