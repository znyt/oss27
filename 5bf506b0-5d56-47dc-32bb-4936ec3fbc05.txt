　　清晨，天还没有完全亮起的时候，刘哔就从床上爬了起来。
<br/>　　不是他不想睡，而是外面传来‘乒乓’的声音，实在是让他没有了一点睡觉的心思。
<br/>　　动作麻利的穿好衣服，刘哔一只手拿着镜子，另一边挑起自己的嘴唇查看了一下，嘴里的獠牙，似乎比昨天变得稍微长了一些。
<br/>　　懊恼的放下镜子，坐在床边想了一下，又拿起桌上的工具仔细修剪了一下指甲，直至确认不会有什么问题了，他才从自己的房间走了出去。
<br/>　　旭日初升，红彤彤的太阳将天空都染得变色，院子里空空荡荡，一缕缕晨缠绕而下，衬托的小院如同仙境一般。
<br/>　　环视了一圈之后，刘哔找到了制造出杂音的罪魁祸首，是正在准备早餐的文才。
<br/>　　“师弟，这么早就起来了啊，你先溜达一会，早餐还要等等才能做好。”听到屋内的动静，文才转过头来，看见了站在门口的刘哔。
<br/>　　伸展身体的动作一顿，听见早饭这个字眼，刘哔脸上的肌肉抽动了几下，心中传来一股抗拒感。
<br/>　　“师兄早啊，不用做我的早饭了，我有点事情要出去一趟，正好在外面吃了。”摆了摆手，有些急切的回应了一声，他实在是没有什么胃口。
<br/>　　反正他原本的计划，也是要外出一趟，早点晚点，也都没多大的区别。
<br/>　　“啊？什么事这么着急，吃完饭再出去啊。”文才一边说着，一边往灶台里添了一把柴火，灶火瞬间燃烧的更加旺盛了。
<br/>　　摆了摆手，刘哔没多做解释，直接打了一盆清水，开始洗漱了起来。
<br/>　　没有得到回应，文才也不气恼，依旧专心准备着锅里的饭菜，嘴上却是止不住的叨咕起来。
<br/>　　“我说师弟啊，我们师兄弟每个月的零花钱又不多，你别老从外面吃啊，家里又不是没有饭，从外面吃多浪费啊……”
<br/>　　吐出口中的漱口水，刘哔面无表情，看了一眼还在喋喋不休的文才过后，径直从义庄大门走了出去。
<br/>　　他第一次发现，原来一个男人也可以啰嗦到这种地步。
<br/>　　……
<br/>　　似乎是因为时间太早的原因，街上的行人还不是很多，一层雾气从空中蔓延着，对视线稍稍产生了一丝影响。
<br/>　　这倒没有什么太大的关系，反正以刘哔强化过的体质，即使在雾天，也能看的很远。
<br/>　　顺着道路不急不缓的溜达着，逐渐梳理着心中尚未完全成型的计划，一边不断的四处张望，似乎在寻找着什么。
<br/>　　走了有一段路程，刘哔咧开嘴笑了一声，他终于找到了目标，一个卖牛肉汤的早餐摊位。
<br/>　　唔，他记得系统说过，丧尸的特质是不食五谷，只吃肉，羊肉汤这种东西，应该是可以吃下去的吧。
<br/>　　心中思索着，刘哔走到摊位旁，嗅着空气中飘来的浓郁肉味，抽动了几下鼻子，眼中闪过一抹犹豫。
<br/>　　没有那股反胃的感觉，这是个好兆头，但实际吃起来，就不知道会是什么样子了。
<br/>　　“老板，来碗羊肉汤，不要香菜，不要辣椒，不要汤。”犹豫了一下，他还是想要试试，反正这又不是家里，实在不行，就不喝好了。
<br/>　　老板是一个长相憨厚的中年大汉，听到这样的要求，有些奇怪，但也没有多问，应了一声之后，动作麻利的准备了起来。
<br/>　　摆了这么多年的早餐摊，他什么样的客人都见过，但还是第一次见到不喝汤的人，要知道牛肉汤的滋味，全都在汤里。
<br/>　　“十块钱，您是打包还是在这吃？”准备食材的功夫，老板抽空询问了一句，脸上挂着客气的笑容，询问道。
<br/>　　从兜里摸出一枚小银角，直接扔在了老板装钱的铁盒里，这钱，是他昨天刚从文才那里顺来的。
<br/>　　毕竟初来乍到，系统也没给他什么提供什么钱财，那只能他自己去寻找解决办法了。
<br/>　　“从这吃。”接过老板找回来的一把铜板，刘哔仔细数了数，随后往兜里一踹，找了一个没有人的位置坐下。
<br/>　　之所以数的这么仔细，是因为他也不知道这个世界的货币制度，对华夏历史不太了解的他，能将银元，银角，铜板给分辨出来就已经很不错了。
<br/>　　收到的铜板一共九十枚，鼓鼓囊囊的装满了口袋，也就是说一个银角是一百枚铜板，而一个铜板相当于这个世界的一块钱。
<br/>　　就是不知道大洋是怎么换算的了，刘哔坐在小板凳上，心中琢磨了片刻，羊肉汤很快就端了上来。
<br/>　　说是羊肉汤，但是因为他特殊的需求，实际端上来的和熟羊肉片没有多大区别，看着还在散发着热气的羊肉片，刘哔吞咽了一口唾液，小心翼翼的尝了一口。
<br/>　　入口滑嫩，微微有点膻味，和前世吃过的羊肉相比，这股味道并不算精致，但对于刘哔来说，这简直就是不可多得的美味。
<br/>　　因为，没有任何奇怪的味道，他的猜想没错，纯肉质的食物，他是完全可以吃的。
<br/>　　确定了可以食用了之后，刘哔顿时两眼放光，风卷云涌的开吃了起来，虽然丧尸可以不进食，但也不代表不会饥饿啊！
<br/>　　这么长时间没有吃东西，他早就饿的不行了，此刻找到了适合自己的食物，自然是吃的不亦乐乎。
<br/>　　几分钟的功夫，半碗羊肉就彻底吃完了，意犹未尽的舔了舔嘴唇，刘哔从兜里掏出铜板，冲老板喊道：“老板，加肉！”
<br/>　　一碗……两碗……十碗……
<br/>　　桌上的碗不断更迭，老板和四周的一些客人，都被刘哔饿死鬼投胎一般的吃法给惊呆了，甚至到最后，老板说什么都不肯再卖给刘哔羊肉了。
<br/>　　以这样的吃法，正常人肯定会消化不了，老板也是为刘哔的身体感到担心，他怕这么吃下去，会出什么事情。
<br/>　　意犹未尽的摸了摸肚子，虽然还是有些饥饿，但他也不想太过于惊世骇俗，反正吃饭的问题已经解决，什么时候吃都是无所谓的事情。
<br/>　　“叮~恭喜宿主第一次成功进食，达成进食成就，奖励随即种族技能一个，请宿主即使进入技能界面领取。”
