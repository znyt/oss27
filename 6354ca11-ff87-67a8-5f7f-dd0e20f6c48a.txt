　　第97章
<br/>　　几乎是程大雷刚起身，秦蛮就被惊醒。他看到程大雷满头大汗的样子吓了一跳。
<br/>　　“大当家，出什么事了？”
<br/>　　程大雷摇摇头，将额头的汗水擦掉：“现在还不知道。”
<br/>　　“那大当家怎么知道出事了？”
<br/>　　程大雷没有吭声，身体靠在墙上，却是再无睡意。
<br/>　　那一夜，程大雷在城中大闹，很是积累了一波恐惧值，之后程大雷的恐惧值也在不断增长。但随着时间流逝，他收到的恐惧值自然也变得越来越少。
<br/>　　可就在刚才，程大雷突然收到大波恐惧值，其数量连程大雷也觉得恐怖。
<br/>　　而问题是，程大雷好好躲在屋里睡觉，根本什么都没有做。
<br/>　　本来恐惧值暴涨，意味他可以多抽几个人，但程大雷的太阳穴砰砰乱跳，总觉得不是什么好事。
<br/>　　究竟是出了什么事呢？
<br/>　　事情第二天早上就知道了。
<br/>　　程大雷昨夜没睡好，窝在屋子里补觉，忽然就听到院外罗子开大吵大闹的声音。
<br/>　　“姓程的，你给我滚出来，我与你一刀两命，不死不休。”
<br/>　　程大雷披着大衣出来，看到罗子开在院子里气得身体发抖，眼泪鼻涕一块儿往下流。
<br/>　　“他干嘛呢？”程大雷从屋里出来。
<br/>　　“不知道啊，早上出去回来就这样？”
<br/>　　“你们这群丧尽天良的山贼，我现在就去抱官，把你们绳之以法。”罗子开大呼小叫往外冲。
<br/>　　“扣了。”程大雷打着哈欠道。
<br/>　　秦蛮根本没动，林少羽一招就把罗子开摁在地上。可罗子开仍骂个不停，不住挣扎。
<br/>　　“好啦，好啦，究竟出了什么事，慢慢说不着急。”
<br/>　　“我把你们这群山贼，你们做了事，你们自己不知道么？”
<br/>　　“我给你脸了是不是！”程大雷一剑插在地上：“有话说清楚，再给我打哑谜，我送你归西。”
<br/>　　“呵，我若怕死，就枉读圣贤书，你们连贾家满门都灭了，还在乎我这条命么！”
<br/>　　“呦，还真有些读书人的性子，少羽，放了。”程大雷顿了顿，忽然道：“贾家满门被灭，是你刚才说的？”
<br/>　　“你们自己做的事你们不知道么！就在昨夜，贾家上下三十七口被杀尽……敢做不敢当么！”
<br/>　　程大雷和徐神机对视一眼，同时摇了摇头。
<br/>　　“可怜我那未婚妻，被你们糟蹋后，还割掉双乳，用她的血……在墙上画了一只蛤蟆。”说到最后，罗子开已经泣不成声。
<br/>　　程大雷拧起眉头，大家互相看看，心里都冒出同样一句话。
<br/>　　『被人冒充啦』
<br/>　　“好啦，好啦。”程大雷把罗子开拉起来：“什么你的未婚妻，人家明明是贾家的媳妇。再说了，我们昨天一夜没出门，你又不是不知道。”
<br/>　　“你不是被神仙传艺，会五雷法，有千里眼顺风耳，有神仙法术。”罗子开抬起头。
<br/>　　程大雷深吸一口气：“子不语怪力乱神。”
<br/>　　“难道真不是你们干的？”罗子开。
<br/>　　“不要理他，大家进屋商量商量。”
<br/>　　几人抛下失魂落魄的罗子开，回到屋里凑在一起商量。
<br/>　　“有人冒充我们做事，大家估计会是谁？会不会是高飞豹干的？”
<br/>　　“不会，山贼打劫不过是混口饭吃，能不杀人尽量不杀人，断人香火的事情绝不会干。”徐神机。
<br/>　　“那会是谁呢？”
<br/>　　“不管是谁，城里一定有咱们所不知道的第三股势力。”刘悲。
<br/>　　“是呐，一夜之间灭人满门，咱们昨晚竟然半点动静没听见，手法比咱们漂亮多了，是伙猛人呐。”
<br/>　　程大雷叹口气，心中的阴霾却挥之不去。
<br/>　　的确是伙猛人，不吭不响宰了三十几人，把人家刚过门的媳妇糟蹋死，还割掉对方双乳，潇洒的在墙上画个小蛤蟆，扬长而去。
<br/>　　有这三十几条人命震着，程大雷的恐惧值蹭蹭往上涨，早前定下的十万目标早就达成。
<br/>　　“无论他们是谁，我们出不了城，他们也绝对出不了城。”程大雷把腰上的佩剑缓缓插入鞘中：“敢冒我的名字，找到他们，干掉他们。”
<br/>　　程大雷撒出人去，连他也不在罗家躲着，六个人都出去找人。
<br/>　　城内如今正撒下大网找他们，此刻程大雷也顾不得许多，冒着险满城找人。
<br/>　　结果一无所获，也不能说丝毫没有收获，至少是没被城内搜寻的官兵抓住。
<br/>　　外面的消息，他们可是听见不少。这几天整座黑石城沸沸扬扬，说得最多的就是程大雷三个字。
<br/>　　如今的黑石城，大家提起程大雷三个字就又是恨又是怕。茶馆的说书先生，也编出故事，只说这程大雷青面獠牙，恶贯满盈，八岁屠母，九岁弑父，这种人就是一千年出一个的大恶人，日后就该天雷轰的。
<br/>　　黑石城的才子聚在一起，提起这件事就咬牙切齿，整天在茶馆青楼作诗大骂程大雷。
<br/>　　『有朝一日刀在手，杀掉蛤蟆去喂狗』
<br/>　　『刀怒斩程大雷，山豪迈成云霄』
<br/>　　『一人我饮酒醉，不杀蛤蟆誓不回』
<br/>　　这群才子整日慷慨激昂，就连当初和程大雷动过手的陆哼也被捧为英雄。以他为首，城里专门为杀掉程大雷城里了一支乡勇军。有城中的才子，武夫，以及各家各户出的家丁组成，人数足有三五百众。
<br/>　　绣楼内的大小姐小媳妇挺身而出，把自己的脂粉钱都拿出来酬军。足不出户的千金大小姐用绣帕抹着眼泪，口口声声的说：“哪位少侠能割了程大雷的狗头，奴家愿以身相许。”
<br/>　　青楼卖笑的歌女也不甘寂寞，黑石城最大的青楼玉春楼对外宣称，谁能杀掉程大雷，于春楼对其终身免费。
<br/>　　以前程大雷是恶名远播，这次是彻底臭大街了。他拥有的恐惧值虽然蹭蹭地长，但程大雷心里可无论如何也高兴不起来。
<br/>　　每天走在街上都能听见骂自己的声音，程大雷倒也不在乎被人戳脊梁骨，关键是这事并不是他干的。
<br/>　　这一日，程大雷一伙人晃悠在街上，一来是找到那伙假冒他们的人，二来也是想混出城去。
<br/>　　城里抓的紧，日子是越来越不好过了。
<br/>　　突然，北城门大乱，据说有伙人快马破门而出。
<br/>　　
<br/>(本章完)
