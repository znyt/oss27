　　在这宫殿之中，一共站着十几名武者，每个人身上的气息，都是极为恐怖，在宫殿的地上，赫然摆放这一个人头。
<br/>　　这人头的主人，早已经死去，可是这人头看起来却还是如同活的一般，还显现着几分生机，这人头的主人，至少也要是一尊超越生死的老祖。
<br/>　　可现在这尊人头，就是被轻易的丢在地上，无人愿意去多看几眼。
<br/>　　“这是附近守王宗的老祖，此人不愿意降服我六道魔门，便是就被我杀了，这王炎这个废物，让他去对付古家，就连区区一个古家的二小姐都杀不了，这种废物，的确也不配成为我们六道魔门的人！”
<br/>　　一尊武者冷笑的开口道，手一挥，这人头便是被他握在了手中，正是他，杀了这样一尊超越生死的老祖，特意将他的头带了回来，显现威严。
<br/>　　“古家家主的实力倒是不弱，若是他出手的话，王炎的确是没有机会。”
<br/>　　又一名武者开口道。
<br/>　　“不过王炎精修暗杀之道，只要是不在古家家主的身边刺杀，随便找个机会，杀一个生死境八重的小家伙，也是轻而易举吧，莫非他真的愚蠢，在古家家主的面前动手了。”
<br/>　　“哼，根据消息，古家家主根本没有动用，王炎是被一个生死境巅峰的武者杀的。”
<br/>　　又是一名浑身血红的武者，冷冷的开口道。
<br/>　　“生死境巅峰的武者，怎么可能杀的了王炎！王炎就算是在废物，也是修行的我们六道魔门的功法，同境界中不可能有人是他的对手才对，除非是那些七大宗门中真正的天才，就算是在宁州古城中的那三位监察使，都不可能是他的对手。”
<br/>　　就是那手握人头的武者，惊讶的开口道。
<br/>　　“王炎那个废物，难道就连那三个监察使都打不过？”
<br/>　　“不是三位监察使出手的，是一个年轻人出手的，你们看！”
<br/>　　那浑身血红的武者缓缓的抬起手，从他的面前便是凝聚出了一道水镜，在这水镜的背后，正是显现出了楚风眠的身影。
<br/>　　正是刚刚楚风眠阻止那王炎，将他一举擒下的那一幕，不过就当楚风眠的身躯，刚刚清晰的一刻，这水镜，突然破碎开来。
<br/>　　“哼，这宁州古城的附近，果然还隐藏这一群魔头，六道魔门的人，很好，正好就让我去会会这所谓六道魔门的弟子！”
<br/>　　坐在宫殿之中的楚风眠，也是清晰的感觉到，有人在推测他的身形，他的嘴角露出几分讥讽笑容，突然楚风眠身边的空间都是破碎。
<br/>　　猛然之间，就在这昏暗的宫殿之中，一处空间，突然破碎开来。
<br/>　　“怎么回事？”
<br/>　　“什么人？”
<br/>　　这突然的空间破碎，令这整个宫殿之中的武者，都是纷纷瞩目过来，一眼望去，在这宫殿之中的武者，最弱的几人，也都是有着生死境巅峰的境界。
<br/>　　其中大部分的武者，都是已经超越了生死的老祖，其中最为强大的两位，都是有着生死台阶第六重，炼心之境的境界，足以是跟那古家家主，平起平坐。
<br/>　　没想到，在这小小的宁州古城周围，居然还隐藏有着如此之多的强者。
<br/>　　若是这些人愿意，毁灭了宁州古城，都不需要太多时间。
<br/>　　“诸位不是正好想要来找楚某吗？楚某来的，怎么诸位都不欢迎吗？”
<br/>　　楚风眠从破碎空间之中，一步踏出，冷笑这看着在场武者，突然之间他的目光，看向了这昏暗宫殿的最深处的王座。
<br/>　　他的目光，正好是停留在了这王座上的人影上。
<br/>　　视线落入到了那极为妖艳的女子身上。
<br/>　　“洛汐姑娘，好久不见了。”
<br/>　　楚风眠笑着开口道。
<br/>　　这坐在王座上的女子，不是其他人，正是当初在虚空海之中，有着一面之缘的洛汐。
<br/>　　正是碰到了这洛汐，楚风眠才发现，六道魔门的道统还是被传承了下来，不过没想到，现在居然再次碰到了。
<br/>　　不过现在的洛汐，可是要比起当初在虚空海的时候，要强大的多，一身的实力，已经是在达到了炼心之境，甚至是只差一步，就要达到了生死台阶的第七重，天命之境的层次。
<br/>　　比起那古家家主的境界，还要更加强大几分。
<br/>　　几个月不见，已经是今非昔比了，比起楚风眠的变化还要大。
<br/>　　“原来是楚公子。”
<br/>　　洛汐看到楚风眠的到来，她的脸色都没有任何的变化，似乎是早就预料到了这一切一般。
<br/>　　“小子，就是你杀了王炎？”
<br/>　　洛汐的话音未落，站在下方的那提着人头的武者，便是向着楚风眠大声怒喝起来。
<br/>　　“王炎那个废物，死了也就死了，不过你既然敢杀了我六道魔门的弟子，今日我也不可能放过你。”
<br/>　　“现在我给你两个机会，一，你现在加入六道魔门来，我们可以绕过你这一次。”
<br/>　　“第二呢？”
<br/>　　楚风眠嘴角微微上扬，看向这开口的武者，这武者的境界，在生死境巅峰的层次，不过他身上的气息，显然是要比起寻常的生死境巅峰，要强大的多。
<br/>　　起码那宁州古城的三位监察使，若是出手的话，这武者几乎三招之内，便是足以将那三人轰杀。
<br/>　　他手中的人头，正是证明了他的实力。
<br/>　　不过，这种实力。
<br/>　　在楚风眠的眼中，依然是跟蝼蚁，没有任何的区别。
<br/>　　“第二，就是现在杀了你，让我将你的精血吞噬！”
<br/>　　这武者咧嘴一笑道，眼神中已经显现出了几分杀意。
<br/>　　“洛汐姑娘，看来你手下的人，倒是也有些不知死活之辈，我先解决了他，再跟你好好叙叙旧吧。”
<br/>　　楚风眠看着洛汐，笑着开口道，随后他的目光，便是转过头，看向那武者，平静的开口道。
<br/>　　“我选第三。”
<br/>　　“就是杀了你，吞噬了你的精血！”
<br/>　　“找死！”
<br/>　　听到楚风眠的话，那武者再也没有犹豫，一拳之间，已经是轰击了过来。
<br/>　　这一拳的威力，简直是比起许多碎命之境的老祖，还要强大的多，力量凝实到极点，直指楚风眠的胸口轰杀过来。
