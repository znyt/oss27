        第二天，明司寒特意给自己定了个闹钟，赶在了连星儿之前起来，还电话叫了酒店服务员把早餐送上来。
<br/>
<br/>        所以，连星儿起床后洗漱完毕，一出房间，就能闻到早餐的香味，在无形地将她召唤到了客厅。
<br/>
<br/>        “吃过早餐后，我陪你一起去游玩北城的各个景点。”明司寒将一次性筷子递向连星儿。
<br/>
<br/>        连星儿在茶几的另一边盘腿坐下，接过筷子后，就毫不客气地开吃。
<br/>
<br/>        她给他的感觉，就是一种天不怕地不怕，甚至有恃无恐，谁也难不倒她的样子。
<br/>
<br/>        “你不怕我对你动歪念头吗？”明司寒坏坏地笑了笑，打趣道。
<br/>
<br/>        连星儿刚刚“嘶溜”了一根拉面入嘴，微微抬眸，瞪了明司寒一眼，漂亮的脸蛋上好像写满了两个字：你敢？
<br/>
<br/>        “你……”明司寒接着欲言又止。
<br/>
<br/>        因为他发现，连星儿每次在吃饭的时候，都不会跟他讲太多的话。
<br/>
<br/>        食不言，寝不语。
<br/>
<br/>        或许，大概是她家的教养吧！
<br/>
<br/>        两人安安静静地吃过早餐后，连星儿就背了个小背包在背上。
<br/>
<br/>        明司寒忍不住说了一句：“我给你背吧！”
<br/>
<br/>        “不用。”
<br/>
<br/>        “我可不想被路人说我没绅士风度。”明司寒仍旧找了个借口，将连星儿的背包给抢了过来。
<br/>
<br/>        连星儿睐了明司寒一眼，没再多说什么了。
<br/>
<br/>        或许是因为有了明司寒的陪伴，连星儿发现，那些“黄牛”不再那么粘她了，也不再有一些单身的男旅客主动过来跟她各种搭讪。
<br/>
<br/>        而且，明司寒还有一个用处，那就是“拍照”。
<br/>
<br/>        连星儿背包里带着的自拍杆，都没用武之地了，因为，她很满意明司寒给她拍的照片。
<br/>
<br/>        “你学过摄影啊？”连星儿问。
<br/>
<br/>        “业余爱好而已。”明司寒云淡风轻地回答，心底却在说，你连星儿可是我拍的第一个与我没有血缘关系的异性。
<br/>
<br/>        “嗯，还不错。”连星儿看着手机里，明司寒给自己的拍的照，十分满意地点点头。
<br/>
<br/>        看到连星儿脸上的笑容，明司寒也情不自禁地跟着笑了。
<br/>
<br/>        有个时候，趁着连星儿没注意，明司寒偷换了自己的手机给她拍。
<br/>
<br/>        他也不知道自己为什么会有这种稀奇古怪的举动。
<br/>
<br/>        一路玩下来，他们一张合照也没有。
<br/>
<br/>        连星儿没提要一起拍合照，明司寒自然也不会开这个口。
<br/>
<br/>        北城的所有景点游玩下来，连星儿用了半个月的时间，而在这半个月里，明司寒不仅充当她的摄影师，而且还主动给她提包撑伞拿水，变身“ATM”机随时“取款支付”。
<br/>
<br/>        到了最后一天，明司寒一觉醒来后，并未看到连星儿的身影，只是当他在客厅茶几上找到自己的手机时，手机上贴了张便利贴，上面字迹娟秀地写了两个字“谢谢”。
<br/>
<br/>        而当他把手机屏幕解锁时，才发现，简讯里多了一条转账信息，来自于连星儿的手机号。
<br/>
<br/>        她把这段时间，游玩的门票、车费、食宿费等等一切由他主动支付的开销，全部打还给了他。
<br/>
<br/>        明司寒一脸错愕。
<br/>
<br/>        她这是什么意思？
<br/>
<br/>        还有，她是怎么解锁他的手机屏幕的？
<br/>
<br/>        就在他准备打电话过去质问连星儿的时候，套房里的门铃突然响了起来。
<br/>
<br/>        明司寒不得不放下手机，先去开门。
<br/>
<br/>        他没看猫眼，直接将门给打开后，看到来访的人，顿时脸色铁青。
<br/>
<br/>        “爷、爷爷……你怎么来呢？”
<br/>
<br/>        “那个女孩还在吗？”爷爷明护国走进屋后，一脸严肃地问。
<br/>
<br/>        明司寒摇头道：“她不在。”
<br/>
<br/>        “这个女孩，爷爷已经查清楚她的底细了，你不可以跟她在一起！”明护国斩钉截铁道。
<br/>
<br/>        明司寒顿时一脸茫然：“？！”
<br/>
<br/>        连星儿回去后，把自己的所见所闻说给了妈咪童筱颖听，还把自己的相片全都打印了出来，配上文字制成了“北城之旅”的日记相册。
<br/>
<br/>        把自己的事情忙完后，连星儿再次登录游戏账号时，发现她的师父“余寒犹厉”不仅主动跟她解除了师徒关系，而且还删了她的好友？！
<br/>
<br/>        连星儿连忙给师父在游戏上私信，结果显示，对方已将她的游戏账号拉黑。
<br/>
<br/>        隔了一段时间，连星儿就会上线看看国服排名，她的师父依旧榜上第一，只不过，她仍旧无法跟他再联系。
<br/>
<br/>        这件事情虽然让连星儿感到纳闷，可终究只不过是个网友，起初会在心底惦记着，但慢慢地沉淀了之后也就没那么上心了。
<br/>
<br/>        不过，最近，她的“小纽扣”倒是又上热搜了，而且还是口碑特好的热搜。
