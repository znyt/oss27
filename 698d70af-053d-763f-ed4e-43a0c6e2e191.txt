　　大伙一听，立刻朝叶少阳望去。碧清的话，也是承认了叶少阳之前的推测，只是……如果按照碧清后面的说法，那就不是八门中有一生门这么简单了。门上的卦象，与八门移动的方位，是在不停组合变幻的，也就是说，八门移动八次，上面的卦象变幻八次，才有一次正确的组合，说的浅白一点，生还的希望并不是八分之一，而是八八相乘，六十四种卦象，六十四分之一……如果靠蒙的话，几乎可以说就是去送死了。
<br/>　　就算是毛小方，都有点怀疑叶少阳是否真能办到，别的人更不用说，不抱太大的希望。
<br/>　　“我找到规律了。”叶少阳微微一笑。
<br/>　　结果大伙都还没出声，碧清的声音再度响起来：“不可能，这八门遁甲，是我潜心修炼多年，从那五行杀阵中推演而出，自成一阵，几无破绽，你不可能找得到规律！”
<br/>　　叶少阳笑了笑，道：“我们打个赌怎么样，如果我能找到生门所在呢？”
<br/>　　碧清冷笑一声，“为什么要跟你打赌，赢了我有什么好处？”
<br/>　　“我告诉你关于青牛祖师的一些事，还有他的去向。”叶少阳道，“只要你放我们走，我就告诉你，怎么样，反正你跟我们本来也无冤无仇，我们走了，对你也没损失不是。难道你还怕我们搬救兵来对付你不成？”
<br/>　　碧清沉默了一会，说道：“你试试吧。”
<br/>　　叶少阳左右看去，大伙都在望着他，一脸的不可置信。“呃，你们相信我，我不是乱说的，喏，你们看着，这八道门每变幻一次，需要的时间是五秒钟左右，我们这几个人，动作快一点，是绝对足够的，如果不够，就你们先去，我再等下一拨，也就是几分钟的事。”
<br/>　　说完之后，叶少阳逐个看去，发现大伙是一脸困惑地望着自己，登时有些纳闷。“怎么呢，我说的这么明白……你们还是听不懂么？”
<br/>　　“什么秒钟分钟的，什么意思？”卢晓清道。
<br/>　　叶少阳愣了一下，方才明白过来，这个时代，西洋的时间应该是还没普及，分钟秒钟的概念，一般人没听说过，怪不得他们听不懂了。“好吧，秒钟……就是眨眼的工夫，大概是眨十下眼睛，分钟……”一时间没想到合适的形容，挠了挠头说道，“反正从一次生门出现，到第二次出现，中间应该相隔半柱香的时间，这下听懂了吧。”
<br/>　　叶少阳转头观察几道门的运转，说道：“下一次生门出现，在三分钟……嗯，一盏茶的时间之后，你们做好准备，到时候一起进去，怎么样？”
<br/>　　一时间没人回答，叶少阳又追问了一声：“有什么问题？”
<br/>　　“少阳子，八门运转的规律，你是怎么推断出来的？”
<br/>　　“这……说起来有点复杂，没时间解释了。”随着八门运转，不断朝着中间挤压，他们所站立的空间越来越小，叶少阳估摸着，最多也就是十几分钟，阵法就会蚕食掉所有空间，也就是说，供他们逃生的时间，只够生门开启两次。
<br/>　　两次之内，所有人都必须离开，不然真的就是死路一条。
<br/>　　梧桐盯着他，说道：“我不是不相信你，只是……事关大家的生死，必须有绝对的把握，所以……”
<br/>　　“我有把握。”叶少阳冲她点了点头，表情也是十分严肃，“我不可能拿你们的性命开玩笑。我们，做好准备吧。”
<br/>　　大伙互相望去。梧桐和妙心各自点了点头。
<br/>　　拼一把，只能拼一把。这是他们最后的机会。
<br/>　　“叶少阳，我怎么知道你没有坑我们？”陈晓宇十分不放心。
<br/>　　“你可以不走，或者自己走。”叶少阳耸了耸肩，“我本来也没想救你。”
<br/>　　陈晓宇狠狠看他一眼，沉吟起来，目光无意间落在了自己的妖仆猴王身上，想到了什么……
<br/>　　“时间快到了，你们准备好了！”叶少阳目光不断从八扇玄门上扫过，密切注意着卦象的变动，出声提醒大伙。
<br/>　　“我们几个，一起走吗？”妙心问道。
<br/>　　“尽量，如果时间不够，就你们先，我最后一个，因为只有我能断出下次生门开启的时间。”
<br/>　　紧张地等待了两分钟，叶少阳提醒大家注意，跟上自己的脚步，随着八扇玄门的旋转，紧紧跟着其中一扇门。
<br/>　　“十九八七六五式三二一……进去！”自己猛地闪到一边去，推了毛小方一把，将他推进了玄门之中。
<br/>　　人进去玄门之后，立刻就消失不见。
<br/>　　妙心是第二个，飞快地进去，之后是梧桐、卢晓清。
<br/>　　在这一瞬间，叶少阳想到不知道过去了几秒，为了稳妥起见，没有跟上去，反正还有一次机会。
<br/>　　总算走了几个……
<br/>　　叶少阳松了一口气，转头看去，身后站着陈晓宇，肩上趴着他的猴王。
<br/>　　“你为什么不进去？”叶少阳皱眉问道。
<br/>　　“呵呵，我要跟你一起。”陈晓宇眼睛看着别处，说道。
<br/>　　“什么意思？”
<br/>　　“你为什么让他们先进去，自己不进去？”陈晓宇冷笑两声，“说什么时间不够，把机会让给人家，你跟他们好像也不熟吧，你有这么好心，打死我也不信！”
<br/>　　叶少阳微微一笑，“那你说是为什么？”
<br/>　　“因为，你自己也吃不准，自己推演出来的对不对，所以，你想拿他们做实验，想看他们进去有没有事，如果没事，那就是没问题，你下次再走不迟……哼哼，叶少阳，我不傻的，我早就看穿了你的伎俩！”
<br/>　　叶少阳暗暗叹了口气，这人……心肠恶毒地简直没救了，冲他无奈一笑，说道：“就你这人，你相信我，你绝对不得善终！”
<br/>　　陈晓宇哼了一声，还在自己的分析洋洋得意。
<br/>　　又等了几分钟，总算到了生门第二次开启的时候，叶少阳早就等着这机会，一闪而入。陈晓宇也一直在注意着他，当然不会让他自己跑了，紧跟着叶少阳钻了进去。
<br/>　　在进去的一瞬间，叶少阳很想反手推他一把……
