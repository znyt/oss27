　　第577章
<br/>　　目前程大雷还摸不清楚燕不归的目的，毕竟，一个正常人是不容易搞懂一个疯子的想法。
<br/>　　而程大雷……偶尔也算个正常人吧。
<br/>　　在他的认知中，燕不归已经要逃了。他想杀燕不归的意愿并不是很强烈，借此破阵而出的意愿却是很强烈。
<br/>　　眼见燕不归要逃了，程大雷佯装追击，暗中寻找着破阵的机会。
<br/>　　宋寒之封住了剑阵，他不想二人中任何一个闯出来，都是魔头，都是一丘之貉，一块儿死了干净。
<br/>　　书山剑阵像个大口袋，装下了程大雷和燕不归二人，二人在剑阵中不停挣扎，却挣不开这口袋。
<br/>　　剑阵当然在消耗，不断有人死去，程大雷和燕不归身上也多了许多伤口。这样下去，不是程大雷和燕不归先后被杀死，便是二人把布阵的七十二人全部杀光。
<br/>　　但看眼下的形势发展，明显是前者发生的可能性更大些。
<br/>　　而且程大雷和燕不归没有流露出半点想要合作的意思，他们都在寻找机会杀掉对方。剑阵中央，完全是一团乱战。
<br/>　　李莫愁立在高处，远远看着这一幕，口中呐呐道：“事情怎么会这样，事情怎么会这样……”
<br/>　　李婉儿仍旧昏迷不醒，李莫愁想寻机会唤醒她，可她还要警惕四周，避免别人打上来，目前，她手里一直摁着几根银针。
<br/>　　还真是个有福之人呐。李莫愁看了李婉儿一眼心中感慨。周围打得生生死死，结果如何谁也说不好，可她仍陷入昏睡中，似乎周围的一切与她无关。
<br/>　　气运……究竟是在程大雷身上，还是在这个熟睡的帝国公主身上。
<br/>　　程大雷感觉头越来越疼了，在剑阵中，他已施展平生解数，可仍无法突破这剑阵。就算燕不归加入后，结果也是一样。
<br/>　　不过，程大雷也不是完全没有法子，他还有压箱底的一招并未拿出来，这一招本来是为夫子准备的，难道现在就要动用么。
<br/>　　罢罢罢，看现在的情况，一时半刻也想不出太好的法子。再撑下去，说不得事情会演变到哪一步，眼下就拿出来，给这些人见识一些真东西吧。
<br/>　　“呔！”
<br/>　　程大雷大喝一声，目光落在宋寒之身上。宋寒之一激灵，这货用这种眼神看着我干什么，难道想要杀我，可你够得着我么。
<br/>　　“与我纳命来。”
<br/>　　座下的黑牛开始狂奔，缩短与宋寒之的距离，百步，五十步，三十步……程大雷面前的人越来越多，层层阻隔，避免他和宋寒之正面接触，现在双方之间还有不短的距离。
<br/>　　宋寒之松了口气：我就知道你碰不到我。
<br/>　　匹夫剑在这一刻脱手而出。
<br/>　　如强弓射出的劲箭，斩断清风，太快了，快到宋寒之根本无法反应。
<br/>　　阿飞快剑第三式：百步飞剑。
<br/>　　都这么多年过去了，程大雷也不可能只靠两招剑法混。现在他已领悟了阿飞快剑的第三式，名曰：百步飞剑。
<br/>　　百步之内，取人性命如探囊取物。
<br/>　　当然，真如果说在一百步距离内，程大雷逮谁杀谁，有些夸张了。不过，经过刚才的一轮冲锋，他和宋寒之的距离已经缩短，到了他出剑必杀的距离。
<br/>　　这是必杀的一剑，宋寒之挡不住。
<br/>　　一剑洞穿他的心脏，血汩汩涌出来，他的身体咚地一声倒地，张大嘴巴，最后一句话想说却没能说出口。
<br/>　　“飞剑术再次现世了。”
<br/>　　夫子和孟玄青替他说出了这句话，二人呆呆立在原地，脸上的表情难以描述。
<br/>　　剑道最高境界是什么，永远不是什么所谓的无招胜有招，心中无剑掌中也无剑莫名须有的话。任何一个执剑者或许都有一个梦想，千里御剑，取人头如割草。
<br/>　　当然，程大雷现在连百步都做不到，更不必提千里了。不过，这还是令在场的两位高人大惊失色。
<br/>　　匹夫不可夺志，三军不可夺帅。程大雷一招飞剑取人头，令剑阵的运转出现短暂的停滞。
<br/>　　这就是程大雷仅有的机会，机会一闪而逝，若是抓不到，程大雷可就真的陷入绝境，毕竟，他现在手中连一柄剑都没有。
<br/>　　身子腾起，一跃数丈，向宋寒之尸体倒地的方向扑去。
<br/>　　这其实是极其冒险的行动，身体腾空之后，双脚便失去了立足之地。不得借力，纵有天大力气，也无法发挥出来。
<br/>　　抢的就是他们一愣神的时间。
<br/>　　眨眼便是刹那，程大雷落地之后夺回了自己的剑。他心情大定，几个砍杀便冲出了剑阵。如此，也算自己破了剑阵么？
<br/>　　程大雷心里其实明白，自己的实力距离破阵还有很大差距，不过就是斩将夺旗，令对方军心大乱，简单的军中战法。
<br/>　　不过，终究是闯出来了。程大雷松了口气，抬头就看到夫子和孟玄青。
<br/>　　“两个老家伙，现在轮到你们了。”
<br/>　　程大雷咬牙骂了一声，提着剑就往山上闯。
<br/>　　结果令程大雷完全没想到，夫子和孟玄青对视一眼，竟然整整齐齐撤开了。
<br/>　　他们太老了，虽然偶尔也提起壮年时的事，可心里早已没有年轻人的血勇。最关键的是，程大雷刚才那一招百步飞剑，的确吓到了他们。
<br/>　　程大雷闯上了山，与李莫愁汇合。
<br/>　　“大当家，这是怎么会事？”李莫愁到现在还是一头雾水。
<br/>　　程大雷撇撇嘴，道：“回头再聊，婉儿怎么样？”
<br/>　　“应该只是中了迷药，并无大碍。”
<br/>　　“先将她唤醒。”
<br/>　　李莫愁随身也有醒神的药物，行走江湖总缺不了这些。
<br/>　　李婉儿悠悠醒转，看到程大雷时愣了愣，她感觉自己像做了一场很长时间的梦，醒来后，便出现在这里了。
<br/>　　程大雷简单说明了事情缘由，最后叹了口气。
<br/>　　“今天的事，怕不能善罢甘休。”李莫愁同样皱起眉头，除了大开杀戒，似乎也没有别的办法。
<br/>　　“不怕，你们扶我起来。”
<br/>　　李婉儿站了起来，目光看着山下密密麻麻围上来的。
<br/>　　“我乃是帝国明玉公主，你们谁敢杀我？”
<br/>　　声音并不大，却传遍整座山。
<br/>　　“我是帝国公主，你们谁敢杀我。”
<br/>(本章完)
