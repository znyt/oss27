　　“禁锢！”
<br/>　　一声淡漠的声音在这片虚空响起，顿时，天地仿佛被禁锢了，世界在这一刻仿佛停止了，天地一片寂静。
<br/>　　轰！
<br/>　　混沌道君幻化出巨大手印，向着那片虚空中不断挣扎着的天刀抓去。
<br/>　　“嗡！”
<br/>　　虚空一颤，天刀刀芒大盛，切开宇宙，显出星门，想要逃跑。天刀内有着不弱于人的思想，它知道自己打不过眼前这人，便想逃走。
<br/>　　“哼！想要逃跑，哪有那么容易？”
<br/>　　道君哼哼一声，巨手不停，伸入虚空之中，抓住那想要逃跑的天刀，将之禁锢。一把抓了回来。
<br/>　　“这柄刀不错，不过，这神邸吾不喜欢，还是灭了吧。”道君这样说道。
<br/>　　顿时，他眼中神光闪烁，看向天刀，只见其内部深处有一个小只的五彩凤凰，它此刻正在剧烈挣扎着，想要挣脱道君的禁锢，可是，混沌道君所设的禁锢，又是那么容易挣脱的吗？
<br/>　　“灭！”
<br/>　　不给它任何说话的机会，一个淡淡的字从道君口中说出。
<br/>　　顿时，无数神光化为一道道光芒，射向天刀内的神邸，将它包裹，神光锋锐，就好似无数剑气在纵横。
<br/>　　“啊！”
<br/>　　一声惨叫，从天刀身上传出，那神邸瞬间便被神光泯灭。
<br/>　　道君将天刀收起，便看向手中的巨大血球。想了想，他将自己的神念涌入血球，破开它的元神。一瞬间，无数陌生的信息纷纷向他涌来。
<br/>　　良久，混沌道君睁开双眼，淡淡吐出几个字：
<br/>　　“不死天皇！”
<br/>　　不死天皇？不错，他手中血茧，就是一只名为不死天皇的仙凰，此刻的不死天皇，正在涅槃之中，处于最关键的时刻，一旦成功，他便会蜕变，境界突破，实力大增。可惜，他遇到了混沌道君，才刚刚苏醒，还没来得及动手，便被道君封印了。
<br/>　　同时，从这血茧的记忆之中，混沌道君也知道了这个世界是一个怎样的世界了。
<br/>　　遮天界！
<br/>　　这是一个修仙盛行的世界，无数星域，修仙者无数。大世之争，天骄如雨，万类霜天竞自由！
<br/>　　登天路，踏歌行，弹指遮天！
<br/>　　无数天骄争锋，决出最终的大帝至尊。
<br/>　　帝者，威凌天下，独尊万古！
<br/>　　这个世界的修行境界分别为：轮海、道宫、四极、化龙、仙台。其中仙台又分十重，仙一半步大能、仙二大能圣主、仙三斩道王者、仙四半圣、仙五圣人、仙六圣人王、仙七大圣、仙八准帝、仙九大帝至尊、仙十红尘仙。
<br/>　　混沌道君将它与洪荒等级对比了一下，发现此方世界的准帝相当于洪荒的金仙，大帝至尊相当于太乙金仙，红尘仙便相当于大罗金仙。
<br/>　　但此方世界经过大战，道则有损，长生物质严重流失，准帝不能长生，大帝亦不能长生，唯有达到红尘仙这一境界，方可长生。
<br/>　　这让混沌道君不得不感慨，洪荒之修比起此处的修士，不知道要幸福多少倍。但有得必有失，洪荒虽算随便即可长生，但其修行速度与自身战力却又远比不上此方世界的同级修士。这尽管与此方世界的大道容易领悟有关，但不可否认这是一件事实。
<br/>　　此方世界，大帝只可活数万年，而洪荒之中，随便闭个关，就是数万年时间。随便领悟一个神通，也是数万年时间。而遮天界中，修士修炼一个神通，却只需几年或者几个月。
<br/>　　这一对比，混沌道君不得不感慨，这其中的差距还是很大的。真不知道两界的修士是该幸运还是该悲哀？
<br/>　　“这个世界很有意思。我喜欢。”
<br/>　　道君说了这么一句话。生活在此方世界的修士，因道则有损，故很容易领悟大道规则。并不像洪荒那样，世界发展已经趋向于完美了，道则不显，法则不现，十分难领悟。
<br/>　　因世界法则的不同，混沌道君很难确定哪方世界的大道更好。但这一切对他并没有太多影响。他远走混沌，本就是为了参悟不同世界法则，从而掌控诸天万界的法则本源。
<br/>　　此刻，遮天界的法则容易学习，这对他更有利，他自然不会放过这种机会，使自己变得更强大！
<br/>　　混沌道君抬头看向上方，他看到上方有着一道封印，想必出路就在那里。他单手一划，瞬间便将封印划破一道巨大口子。
<br/>　　一步踏出，便已出现在封印之外。外面是一片汪洋，漆黑如墨，这是一片黑海。混沌道君刚才所站的地方，正是汪洋之下。
<br/>　　混沌道君再次一步踏出，不知过了多远，等再次出现，却是已经出现在一座岛屿之上。岛屿通体漆黑，露出水面。
<br/>　　混沌道君知道，这不是岛屿，而是一条脊椎骨，横亘在此处，巨大无边，长达上万里。
<br/>　　这是一头星兽，而且还是幼年体，以吃星核为生，古来少有。可惜，它是死的。
<br/>　　道君抬头望去，看见的却是无边无际的黑色汪洋，这里死气沉沉，海中无鱼，更无妖魔，是一片死海，寂静无声。
<br/>　　此处显然是自成一界。
<br/>　　一界陵！
<br/>　　他再次抬头看去，上方还是一道封印，他此刻还是在地底世界之中。划开封印，沿着一道漆黑的洞口，穿过无尽深渊，来到一处海眼
<br/>　　此海眼为荧惑古星北海眼，但早已干涸，并无水泽，有的只是无数的沙粒与巨石，很多都被冲刷得很干净，大多为赤红色。
<br/>　　有人说，这些都是血浸过导致的。当年一战，整颗古星辰都被染红了，若非有仙宝保护，恐怕整片太阳系都不复存在了。
<br/>　　混沌道君继续向上，在此处，是十八层地狱中的第一层，为阴阳牢，分为上下两层，护卫前值与补充，这是一种绝妙的构想。利用天地自然之力镇压两大魔王。
<br/>　　在这里，混沌道君遇到了昔日被释迦牟尼镇压的两大妖魔，道君翻手之间，便镇压了他们。正是鳄祖与大成身体的神邸念。
<br/>　　当然，在镇压之前才，他们肯定是不服气的。遇到道君，便以为遇到了新人，想要欺负。最终结果很明显了，他们被道君用度心诀度化了。
<br/>　　道君轻轻一拂，尘埃尽去，一块巨大的石柱咔咔作响，坠落一层石粉之后，九颗星辰闪闪发光，闪耀了出来。
<br/>　　“十八层地狱的星空坐标……”道君讶然，果然内藏玄机。
<br/>　　其余十六层地狱所在的八颗星辰都如这般，分为上下两层牢狱，分布在八大古星。
<br/>　　混沌道君了然，将其星空坐标收起，待以后有时间再去看看。
<br/>　　待他从地心深处来到地表，见到的是一片残破的古庙遗迹。古庙后方的宏伟大殿与宫殿群早已化为废墟。只有一间小小的古庙，依然屹立于此，让人有一种平淡归真的感觉。
<br/>　　正是大雷音寺！
<br/>　　……
