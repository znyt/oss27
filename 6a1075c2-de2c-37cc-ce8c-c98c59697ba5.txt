　　叶少阳只好大致说了一下，碧清和包子都震惊不已，评论了一番，叶少阳没什么精神，去洗了个澡，回房间倒头就睡，昨晚后半夜才睡，又起早坐飞机，真的筋疲力尽了。
<br/>　　他做了一个梦，梦中，自己好像站在一座桥上，看着自己认识的人一个个走过去，对着自己招手道别，但自己
<br/>　　醒来已是晚上了，叶少阳揉揉眼睛，坐起来，发了一会呆，起身开门，本想去卫生间洗脸，一开门看到碧清和一个人坐在沙发上，听见开门声，那人回过头来，是林三生。
<br/>　　“你怎么来了？”看到他，叶少阳有点吃惊。
<br/>　　“早就来了，看你睡觉，我就没叫你。”
<br/>　　叶少阳还没开口，碧清从沙发上站起来，快步走过来，有些激动地说道：“这件事你怎么早不跟我说！”
<br/>　　“什么事？”叶少阳没反应过来。
<br/>　　“阻击黎山老母！”
<br/>　　叶少阳这才回过神来，耸肩道：“我不是才回来吗，还没来及告诉你。”
<br/>　　碧清看看他，又看看林三生，道：“什么时候行动？”
<br/>　　“你这么激动干啥，跟你有什么关系！”叶少阳挠着头。
<br/>　　“当然有关系，你不要装糊涂，那个苏沫是我一缕精魂，我只有收了她，才能圆满，我知道你一直不想我这么做，但这次你不要阻拦我，我也尽力帮你们！”
<br/>　　叶少阳深吸一口气，责怪地看了林三生一眼，本来这件事他是不想告诉碧清的，不过既然林三生已经说了，那也没办法，不可能不带她去。
<br/>　　“什么时候行动？”叶少阳也问了一声。
<br/>　　“我们现在立刻启程去泰山，你把老郭叫着，硬拼的话，我们未必是对手，需要取点巧……先过去布置一番吧，据我了解，他们大概两三天后到，还是可以布置一下的。哦，你刚回来，看你气色不好，不然你先休息两天，我们先去。”
<br/>　　“说什么呢，你的事，我可不敢耽误，明天一早出发吧。”
<br/>　　几人坐下来，先商量大致计划。
<br/>　　因为是正经事，碧清彻底把电脑手机扔一边了，全程参与讨论，比林三生还要积极。
<br/>　　谈到后面，叶少阳问道：“咱们先确定一下，这件事怎么收场，我们扰乱封禅仪式，他们肯定要拼命，到时候怎么办？”
<br/>　　“跑。”林三生道，“我们的目的，就是破坏封禅仪式，让仪式失败，然后我们就跑。”
<br/>　　“他们肯定知道是你干的。”
<br/>　　“知道又怎样，他们就算再怒，明面上也不能把我怎么样，否则空界大乱，大家都玩完。”林三生想了一下说道，“不过前提是我们一定要跑的掉，如果正面对上，被打死也是白死。一句话，只要离开了那个环境，各自回位，就再也打不起来了。”
<br/>　　叶少阳问道：“那如果跑不掉呢？”
<br/>　　“你的意思是？”
<br/>　　“底线是什么？”
<br/>　　林三生阴着脸，右手做了一个刀子往下砍的动作。
<br/>　　叶少阳深吸一口气，走到沙发上靠了下去。
<br/>　　四个人一起看他，都不说话。
<br/>　　林三生走过来，道：“这是自保，真到那一步，不是你死就是我活。”
<br/>　　叶少阳默默点头。
<br/>　　“你担心什么？”林三生道，“如果有机会，我会亲自动手，不让你动手。”
<br/>　　叶少阳白了他一眼道：“是兄弟就别这么说，你既然找我，我自然全力以赴，我只是感慨一下。”
<br/>　　当晚讨论之后，林三生就回去了，他要继续搜集情报，按照他的说法，黎山老母一行已经到了人间，目前去了南方，寻找封禅必须用到的一些珍惜物品，所以，时间还有。
<br/>　　林三生走后，叶少阳就给老郭打了电话，简单说了事情，问他有没有空明天一起去，结果老郭犹豫起来。
<br/>　　“你要有事走不开就算了。”叶少阳也觉得有些为难，毕竟老郭是有家的，不像自己这个单身汉，说去哪里都可以。
<br/>　　“没问题，不过我要跟你大嫂说一声，这样吧，我来订票，明天早上电话叫你。”
<br/>　　挂上电话，叶少阳心中对老郭抱了一份愧疚和感动。
<br/>　　叶少阳几乎一晚上没睡，坐在床上发呆，好几次忍不住想给周静茹和谢雨晴发信息，都忍住了，既然她们已经选择了新的生活，自己去打扰她们，只会让她们为难，还是算了。
<br/>　　天亮的时候，老郭发来信息，已经定好了机票，九点钟，飞往山东泰安，叶少阳赶紧起床洗漱，这才发现碧清在泡澡，跟她说了出发的事，碧清一听，立刻就化成莲花本尊，钻进他口袋里。
<br/>　　收拾好东西，也七点多了，叶少阳带着碧清、瓜瓜和包子一起出门，在飞机场见到了老郭，也没说太多，一起取票然后安检，进去等飞机的时候，叶少阳才把自己最近的经历、还有林三生的计划说了一遍，只是没说谢雨晴走了不回来的事。
<br/>　　他不太想跟人提起这件事。
<br/>　　“对付黎山老母和摇光仙子……”老郭深吸一口气，沉吟半晌，道：“好在这里是人间，是我们的主场，办法还是有的。”
<br/>　　叶少阳表示赞同，如果是在空界，阐教根深蒂固，根本没得玩，林三生估计也是看到这一点，才想到去搏一把。
<br/>　　飞机落地，到了泰安。
<br/>　　从机场出去，一路上能两边墙壁上有各种跟泰山有关的广告，机场还专门有旅游咨询点，老郭在一边解释，整个泰安市没任何地方出名，全靠着泰山。
<br/>　　“要说这泰山，真的是风水宝地，但这风水不能为俗世所用，化不成龙气……这里的风水，是仙家所有，特别适合修行，天下间，没有那个名山大川的灵力超过这里。”老郭一边走一边解说。
<br/>　　“你怎么知道这么清楚？”叶少阳好奇。
<br/>　　“我当年结婚度蜜月，带你大嫂来的就是泰山啊，唉，一晃这么多年过去了，孩子都这么大了。”老郭感慨地摇了摇头。
<br/>　　叶少阳想着他刚才说的话，纳闷道：“要这么说，这泰山应该有门派才对，为什么从古到今都没听说过有什么人在这里开宗立派？”
