的建筑前停了下来，门口是一块硕大的巨石，上面赫然写着四个大字：长青道馆！<br/>　　<br/>　　三人下了车，古纯阳捋了捋花白的胡须，对叶尘介绍道：<br/>　　<br/>　　“叶师，这里便是我那位朋友的道场了！我这朋友名叫桑木，人称桑木道长，道法修为和我不相上下，只不过我擅长炼器，他擅长画符……”<br/>　　<br/>　　说到这里，古纯阳微微一顿，接着又道：<br/>　　<br/>　　“当然了，和叶师的道法相比，那肯定是不值一提！但是在萨市这种偏僻之地，也算是有些名声……”<br/>　　<br/>　　古纯阳说着，正要领着叶尘和楚飞烟，进入长青道馆，然而就在这个时候，一排黑色悍马车，从远处呼啸而来！<br/>　　<br/>　　“嘎~~”“嘎~”“嘎~”……<br/>　　<br/>　　足足八辆悍马车，直直的冲到长青道馆门口处，齐刷刷的戛然而止，刚好堵住了长青道馆的大门，将叶尘三人挡在了外面。<br/>　　<br/>　　“咔！”“咔！”“咔！”……<br/>　　<br/>　　八辆悍马车的车门几乎同时打开，而每辆悍马车上都跳下五六名黑衣大汉，最终足足下来了三四十人！<br/>　　<br/>　　这些人个个气质凛然，显然都不是那种普通的街头小混混。<br/>　　<br/>　　“葛大师到！”<br/>　　<br/>　　随着一名黑衣大汉高喊了一声，从中间的那辆悍马车上，又走下了一名约莫五十来岁的光头男子。<br/>　　<br/>　　这光头男子的造型十分奇特，不光没有头，而且无眉无须，两眼如鹰，眼眸中精芒闪闪，一看就知道不是善茬。<br/>　　<br/>　　光头男子下了车，先是摸了摸自己的光头，继而又看到后面的叶尘三人，顿时桀桀一笑，声音尖锐刺耳，难听无比，<br/>　　<br/>　　“你们三位是来长青道馆求符的吧？回去吧！今天过后，长青道馆就不存在了！今后在萨市，只有我葛洪灵的通灵院！桀桀桀！”<br/>　　<br/>　　光头男子冲三人说完这话之后，再次狂笑了数声，大步朝长青道馆里面走去，再次朗声道：<br/>　　<br/>　　“桑木老儿！贵客驾到，还不赶紧出来迎接！”<br/>　　<br/>　　古纯阳听到光头男子的话之后，顿时脸色大变，喃喃道：<br/>　　<br/>　　“看来桑老鬼这次麻烦不小呀！”<br/>　　<br/>　　叶尘也不由得眉头微微一皱，他也没有想到，他们来的竟然如此不巧，不过既然是古纯阳的朋友有难，他自然不能袖手旁观，于是摇了摇头，淡淡道：<br/>　　<br/>　　“走吧，咱们也进去瞧瞧！”<br/>　　<br/>　　ps：今天遇到突情况，实在抱歉……<br/>　　<br/>　　(本章完)<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
