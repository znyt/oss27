　　“这……这是什么茶？竟会有如此神效？！”
<br/>　　大殿中人，纷纷惊骇不已，他们只是轻轻吸了一口大殿之中飘散的清香气，顿时就让他们陷入悟道的状态。这怎么可能？他们还从没有听说过如此逆天的茶呢？
<br/>　　若说是让一般的仙人陷入悟道状态，那样的茶虽然珍稀，众人虽然会惊喜，但也不至于到达惊骇的地步。因为那只能算是一株非常顶级的的先天灵根茶树。
<br/>　　但，他们是什么人，他们可是准圣强者啊。要知道，修士越是强大，就越是难以顿悟。而现如今他们只是轻轻嗅了一鼻这茶溢出来的清香，就有如此神效，那若是喝上那么一口，那还了得？
<br/>　　真不知道是这会是怎么样的灵根，才能接出具有如此逆天神效的茶叶？
<br/>　　“此茶名为混沌悟道茶，乃本座当年游历混沌时见到的一株混沌灵根。能让人陷入顿悟之中，这些年，也帮助了本座不少。今日就以此款待各位道友，各位可满意。”元始看着陷入震惊的众人，微微一笑，淡淡道。
<br/>　　而且这煮茶用的水也不简单，那可是十大真水中的天一真水啊。当年这可是孕育造化青莲的神水，被元始等人全部收了，这些年他就是用此神水来煮茶喝的。若是让别人知道，真不知道该说他这是浪费呢？还是奢侈呢？
<br/>　　“混沌灵根？”
<br/>　　“这怎么可能？竟然是混沌灵根？怪不得会有如此神效？”众人惊叹不已。
<br/>　　“道友好福气，好运道啊！”镇元子大仙感叹道。
<br/>　　“好了，众位道友赶紧喝了吧，若不然，待过些时辰，此茶效果减少了，那就不妙了。”元始对着众人轻轻说道。
<br/>　　说完，他也端起一杯茶，送到嘴边轻轻呡了一口。虽然他喝过不少次，但每一次喝这茶，他都不得不感叹，这茶不愧是混沌灵根，每一次都会有不同的味道，有不同的感悟。
<br/>　　听到此话，各位修士也来不及感慨，直接端起眼前的茶杯，轻轻喝了起来。
<br/>　　茶水入口，也不苦涩，直接化为一道清流，钻入喉咙之中，在众人身体之中走了一遍，最后来到他们的元神之中，散了开来。
<br/>　　顿时，各种感悟，如潮水般，蜂拥而来。
<br/>　　同时，在外面，这些大神们的身上，全都散发着祥瑞的光彩，各种异光闪烁，道光流转，道韵自生。
<br/>　　顿时，大殿中的众神，全都陷入顿悟之中。场面一下子变得安静下来了。
<br/>　　……
<br/>　　“呼——”
<br/>　　良久，众人才从顿悟之中清醒过来，纷纷轻呼一口气。众人脸色通红，霞光满面，好似遇到了喜事。
<br/>　　“圣人真是好造化，喝你一杯茶，抵得过自己修行数百年呐。”红云道人开口感慨道。
<br/>　　“好茶，真是好茶，这算得上是洪荒第一灵茶了。”
<br/>　　“真不愧是混沌灵根，果然名不虚传！”
<br/>　　“悟道茶，不愧‘悟道’之名。这算是名至实归了。”
<br/>　　……
<br/>　　“感谢圣人馈赠！”喝完茶，众修士也不忘感谢元始一番。
<br/>　　“不必多礼，既然茶也喝了。那么，接下来，吾等是否也该论道一二？”
<br/>　　“善！”
<br/>　　“大善！”
<br/>　　众人纷纷称善，他们自然不会不同意，他们来此的目的就是论道听讲圣人之道，如今能够在论道之前喝到一杯悟道茶，也算不虚此行了。
<br/>　　之后，一如预料之中那般，众多洪荒顶级修士坐在一起，开讲大道，论述法理。但见一道道的天地规则，凝聚宛若实质一般的粗大光柱，自每一个人的身上迸发而出，冲破玉虚宫的穹顶，直达九天之上，撼动风云。
<br/>　　众人都是当时洪荒最顶尖的强者，几乎个个都有准圣中期或者以上的修为或神通，若是单纯论道也就罢了，但是，论道之时，彼此之间也在相互试探，庞大的气息，各种道则在不断碰撞，于碰撞之中验证各自的大道，生成莫名异象，不停地闪烁明灭。
<br/>　　众人之中，法力最为高深者，当属元始。元始乃混元大罗金仙三重天巅峰的修士，自然最强。其次乃老子、通天、接引、准提、帝俊、太一、望舒等人，他们都是斩去两尸的修士，最后才是剩下的几人，他们都是将要斩去二尸的修士。
<br/>　　于大道碰撞之中明悟真理，同时能够吸收他们的大道，化为己用。其中，元始的元始大道一出，顿时仿佛化为一条宽大的大道河流。河流之中，万道浮现，众道合流，万象更新。其他各大修士的大道与之碰撞，不能掀起一点儿浪花，便会被大道河流同化，化为河水，吞噬消化，化为己用。
<br/>　　刚开始，还是众人相互碰撞论道，但到了最后，实在是元始的大道太强了，不管是什么，它都来之不拒，全部同化吸收。以至于后来众人联手压制元始。
<br/>　　但就算如此，元始一样不怕，同时，他心中还有些欣喜。这样也好，就不用我一个一个地去找你们了。元始的元始大道横空，仿佛一个天地烘炉，不管来多少，它都能炼化。
<br/>　　同时，不仅仅只有元始有所得，其他的修士一样从元始的万道合流之中领悟了不少，与自己的大道相互印证，让自己的道行大增。
<br/>　　不同的人，对道的领悟自然也各有不同。就算是同一条道，各人也会有不同的理解。论道的目的，就是要将自己不理解的给理解了，将别人对自己有用的，学会了。
<br/>　　同时，这种论道更能直观的感受比较各自的实力道行的深浅。讲究的就是要，以我道压服你道，那便算是取胜了。
<br/>　　“诸天万界，亿万大道，尽归元始！”
<br/>　　不知过了多久，众人都还沉浸在大道海洋之中。突然，元始一声大喝，气势陡然间拔高，大有吞噬天地、执掌万道之威，竟是他以一人之道，强行压迫其余众人之道，使其节节败退。
<br/>　　“无量天尊！”
<br/>　　面对元始的盖世神威，众大神们眼见自己的大道被镇压，心中暗暗吃惊的同时，也连忙睁开双眼，默念道号，自行退出争斗。
<br/>　　“无趣！”
<br/>　　元始撇撇嘴，顿时大感无趣。见众人都收回了各自的大道，当即不动声色的随之收手，其余诸人也各自罢手。
<br/>　　不过，话又说回来了，此番论道，诸神齐出，大道显化，诸神收获良多，其中元始收获最多。
<br/>　　论道结束，众位大神们也告辞离开了，他们此番论道，收获良多，需要尽快回去消化。只有完全消化吸收，那才算是自己的。这个道理谁都懂。
<br/>　　更何况，在见识到了元始的强大后，他们也没有心情多待在此处了。
<br/>　　但，就在最后众神即将离开之际，元始却淡声喊道：
<br/>　　“后土、玄冥、望舒三位道友留下。”
<br/>　　……
