　　“还是修为太低了，若如今我有半步洞玄的修为，今日的战斗便不必这么被动惊险。”
<br/>　　萧逸暗暗想着，但又无可奈何。
<br/>　　虽然自己能够炼药，可是先天丹是有杂质的，根本无法连续服用；九转炼体丹虽然没有杂质，但那是一品丹药，对现在的自己功效不大。
<br/>　　还有就是修罗战体，需要的妖兽精血太多了，自己一时三刻也凑不齐。
<br/>　　再就是冰鸾剑武魂，虽然让自己修炼速度极快，可是，再快也需要时间。
<br/>　　归根结底，自己最需要的就是时间。
<br/>　　只要给自己足够的时间，修为一定会有一个飞跃。
<br/>　　不过他似乎忘了，他来这个世界不过数月时间，却修炼到先天三重修为，已经是非常变态的了。
<br/>　　“呼....”萧逸在原地坐下，打算先稳下伤势。
<br/>　　两天后，待得伤势稳定，他绕了个大圈，经过柳絮城，又回到了百武城。
<br/>　　当然，他早就料到，自己和钟无忧一战，必然会引出波澜。所以，在进城前，他做了些乔装。
<br/>　　所谓的乔装，只是戴了个蓑笠罢了。
<br/>　　沿途所过，不时能听到武者和猎妖师们在谈论两天前的战斗。
<br/>　　“诶，你们听说了吗？两天前，天才猎妖师易霄大败无忧公子。”
<br/>　　“据当时观战的武者说，那场战斗，惊心动魄，精彩至极。”
<br/>　　“当然听说了。如今整个北山郡，谁不知道紫炎易霄之名。”
<br/>　　“可惜阿，我无缘看到那么精彩的战斗。”
<br/>　　“说起来，甚至有人说紫炎易霄足以进北山榜前十了，排名还要比无忧公子还要前呢。”
<br/>　　“.....”
<br/>　　不少人在议论纷纷。
<br/>　　倒是萧逸自己暗暗无奈，什么时候自己多了个紫炎的外号了。
<br/>　　......
<br/>　　没有多作理会，萧逸直接来到猎妖殿，趁着没人注意，快速接了两个任务。
<br/>　　这两个任务，正是击杀天残地缺和黑白无常。
<br/>　　天残地缺和黑白无常，早就在猎妖殿的悬赏任务中。
<br/>　　只不过，一直没人能奈何得了他们，一般人又不愿意得罪暗影楼，这两个任务便一直没人接。
<br/>　　反正两天前，自己将他们击杀了，任务点不要白不要，当然要回来领。
<br/>　　来到任务处，萧逸交出令牌。
<br/>　　任务处的人员看了眼，顿时脸色一变，瞪大眼睛看着萧逸。
<br/>　　当然了，现在的萧逸戴着蓑笠，蓑笠下还有个面具。
<br/>　　“别出声，我只是来交任务。”萧逸快速说道。
<br/>　　任务处的这个人员礼貌地点了点头，道，“好的，我这就帮您记录。”
<br/>　　半晌，这个人员将令牌递还给萧逸，礼貌道，“这位武者，两个四级任务，共60任务点，这是您的令牌和赏金。”
<br/>　　萧逸点点头，接回令牌，刚准备离开。
<br/>　　这个任务处人员连忙道，“这位武者慢着。”
<br/>　　“有事？”萧逸皱眉问道。
<br/>　　“是这样的，记录显示，您现在的任务点总共有1500点，可以成为我们猎妖殿的执事，只是问问您是否愿意加入而已。”
<br/>　　“猎妖殿的执事？”萧逸面露疑惑，随即摇摇头，“不必了。”
<br/>　　也不知道猎妖殿有什么规矩，或者成为执事有什么要求，萧逸现在暂时不想在这里浪费时间。
<br/>　　“好的，这位武者慢走。”
<br/>　　任务处的这位人员礼貌地说了一声，随即继续忙自己的事。
<br/>　　萧逸刚准备离开，忽然....
<br/>　　一道莫名的力量猛地束缚了他，让他无法动弹。
<br/>　　这股力量强大而飘渺，猎妖殿中猎妖师来来往往，却无任何一人发现他的异状。
<br/>　　只有他自己，脸色大变，心中升起一股不妙的念头。
<br/>　　“何人在禁锢我？”萧逸站在原地，暗暗惊诧。能无声无息地禁锢他，而且不惊动猎妖殿内这么多工作人员和猎妖师。
<br/>　　此人的修为，必定很强。
<br/>　　正当萧逸脑海中快速思索时，一个猎妖殿工作人员忽然走到他面前。
<br/>　　来人是个女子，年龄约莫二十出头，面容算不上绝色，但也算是美女。
<br/>　　萧逸眉头一皱，据他的观察，面前之人，之前并未出现在猎妖殿内任何一个办事处，好像是刚才才忽然出现的。
<br/>　　“易霄先生。”女子附身在萧逸耳边，低语道，“我们总执事有请，请随我来。”
<br/>　　女子的话音刚落下，自己身上的禁锢力量也瞬间消失，可以动弹。
<br/>　　“总执事？”萧逸眉头紧皱，面前女子的修为，不过是先天境。
<br/>　　刚才禁锢自己的，难道就是那位总执事？
<br/>　　所谓总执事，便是一个猎妖殿的负责人。
<br/>　　北山郡分别有九大总执事，一个分殿主。
<br/>　　九大总执事就是九座猎妖殿的负责人，各自掌管一座猎妖殿。
<br/>　　而在九大总执事之上，便是分殿主。
<br/>　　在北山郡，这位分殿主，几乎是一个传说，常年坐镇北山郡郡都，统领全郡猎妖殿。
<br/>　　权利之大，凌驾于各大势力之上，包括北山慕容家乃至暗影楼。
<br/>　　其修为，传闻除了裂天剑派掌门外，便是他最强。
<br/>　　至于九大总执事，在各自猎妖殿的十数城范围内，也是响当当的大人物。随便一个都是破玄境武者。
<br/>　　在女子的带领下，萧逸穿过猎妖殿的任务处。
<br/>　　原来在猎妖殿里头，还有一个内堂，应该是属于猎妖殿人员平日居住，或者内部办公的地方。
<br/>　　女子在一间正中央的房间中停下，敲了敲门，才莲步轻移地走了进去。
<br/>　　萧逸也走了进去。
<br/>　　房间很大，左右两旁摆满了书架。
<br/>　　书架上放满了卷宗、档案和一些资料。
<br/>　　正中，是一桌案，籍。
<br/>　　女子对着老者鞠了一躬，而后自顾坐到另一旁的桌案上，处理自己的工作。
<br/>　　猎妖殿的工作人员，似乎总是很忙。
<br/>　　萧逸看了眼老者，发现自己根本看不透老者的气息。
<br/>　　在自己眼中，老者简直就如一汪深不见底的水潭，深不可测。
<br/>　　如果没有猜错的话，这位就是百武城猎妖殿的总执事了。
<br/>　　百武城附近十数城中，势力最强、修为最高之人。
<br/>　　“坐。”老者淡淡地合上了书籍，抬头看了眼萧逸。
<br/>　　“不必了，小子站着就行。”萧逸淡淡地摇摇头，道，“不知总执事唤我来有何事？”
<br/>　　老者将书籍放到一边，而后盯着萧逸，眼中泛过一丝满意。
<br/>　　“不错，在我面前能做到不卑不亢，紫炎易霄，名不虚传。”
<br/>　　萧逸皱起了眉头，不知这总执事到底何意。
<br/>　　“如果总执事没有事吩咐，易某还有要事，便先行离去了。”
<br/>　　在这种深不可测的武者面前，萧逸并不打算多留。
<br/>　　“呵呵。”老者忽然笑了笑，道，“易霄，不必拘谨。我叫你来，并没有恶意。只是想问问你，有没有兴趣成为我们猎妖殿的执事。”
<br/>　　“执事？”萧逸眉头紧皱，面对老者，他自然不会像面对普通猎妖殿人员那样随便敷衍一句‘不必了’。
<br/>　　只是，他更不想浪费时间。
<br/>　　老者似笑非笑地点点头，道，“易霄，我很看好你。只要你愿意，我马上就能帮你办好各种手续。”
<br/>　　“成为本殿执事，荣耀、权利、地位，你唾手可得。当然，你需要付出一丁点儿代价。”
<br/>　　“代价？”萧逸面露疑惑之色。
