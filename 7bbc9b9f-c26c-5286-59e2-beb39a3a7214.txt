　　这个叶少阳也是懂得一些，这些所谓的毒砂，是将沙子跟毒液一起搅拌，然后在锅里炒，让毒液完全附着在沙子上，这样只要沙子还在，毒性也就还在，而且一般用来做毒砂的都是刺激性的慢性毒，跟人身体皮肤接触，会让人皮肤慢慢溃烂而死……
<br/>　　选择毒砂用来守墓，是因为毒砂可以长久保存，只要机关不坏，千年之后毒性依旧存在，还有一个重要原因：毒砂在流出来之后，会不断堆积，最终将墓道堵塞，想要出去，就必须得从毒砂里爬出去，可能等人爬出去，浑身也已经溃烂得只剩下皮包骨头了。
<br/>　　毒砂哗哗地流下来，因为地处开阔，总算是没有将出口封锁，而是向着四周摊开，等了有半分钟左右，从雕塑里不再往外流出毒砂。
<br/>　　“我们这怎么回去啊。”毛小方看着前方墓道，如今好像是多出来一个长约十来米的沙滩，四周岩壁光滑，无处借力，想要过去，只能从这“沙坑”上走过，但是十来米长的沙坑，人又不会飞檐走壁……
<br/>　　“都是你，乱碰什么！”妙心对着陈晓宇呵斥道。
<br/>　　陈晓宇脸皮一红，也不好反驳，挠着头想了一会，说道：“这好像也不算太费事，我们待会从下面出来，找个东西垫在上面，勉强跨过去就是了，这件事交给我。”
<br/>　　话说到这里，妙心也不说什么了，主要是事情已经发生，不管把他怎么样，都已经于事无补了。
<br/>　　大伙在一起简单商量了下，决定还是继续下墓探查，至于这些神像为什么都变成了笑脸，既然搞不清楚，暂时还是不要去想了。
<br/>　　民间有句话：见怪不怪，其怪自败。放在灵异事件上也是很有道理的。
<br/>　　邪物千变万化，怪事千丝万缕，一时之间，法师根本弄不清楚有些事背后的真相，尤其是在这种闭塞的环境里，只能是兵来将挡，先做好准备，继续做自己的事，让妖邪自己跳出来……
<br/>　　说到底，这其实也算是一种自信的表现，就拿眼前遇到的情况来说，如果是一般盗墓贼，甚至法力一般的法师，一旦遭遇眼前的情况，估计早就不敢再下去了，要么想办法离开，要么就弄清楚这些神像微笑背后的秘密，不然绝不敢再贸然下墓。
<br/>　　但是对于这帮人来说，因为艺高人胆大，也就无所谓了。
<br/>　　叶少阳提着三色莲花灯走在最前面，在墓道里照亮，在走过转角，一路往下之后，墓道两边不再有那些诡异的雕像，没有那些眼睛盯着，大伙都感到舒服了很多，于是一路走下去，前方走不多远，是一个九十度的直角转弯，过去之后，没走多远又是一个这样的转弯。
<br/>　　大伙一路走下去，也不知道转了几次这样的弯，都愈发感到不对劲：这转弯怎么没完没了？叶少阳更是疑惑，这到底是古墓还是特么的自己家小区的消防通道？
<br/>　　“不对啊，之前，我们好像走过这里。”毛小方猛然站住，环顾左右，纳闷地说。
<br/>　　叶少阳心中也是咯噔了一下，左右看去，墓道有三四米高，顶部和四周都没有任何装饰，看上去平淡无奇，也没什么特征可以记住。
<br/>　　不过，不光是毛小方心中有那种重游故地的感觉，大家差不多都有。
<br/>　　妙心默不作声，摸出了一把铜钱，一边往下走，一边不断撒下铜钱，再次转过了几个弯之后，妙心手里的铜钱还没有洒完，突然愣住了：
<br/>　　在前方不远，一个台阶上赫然就有自己刚刚撒过的铜钱。
<br/>　　妙心深吸了一口气，喃喃地吐出了三个字：“悬魂梯。”
<br/>　　叶少阳倒吸冷气，悬魂梯三个字，对他来说并不陌生：这是一种在墓穴中偶尔会出现的神秘现象，一条楼梯，转了四个弯之后，还是回到原点。也就是说，如果一直走下去的话，这楼梯根本就走不完……
<br/>　　叶少阳当初跟四宝他们一次下墓的时候，就遇到过这种事，之后叶少阳还特意上网搜过一次资料，知道这悬魂梯并不是华夏人的专利，在国外，有一个非常专用的名字，叫做“潘洛斯阶梯”，电影《盗梦空间》里也出现过这种神秘的楼梯：
<br/>　　楼梯不断往下，转弯三次，到第四次转弯的时候，再次回到原点……这当然是一个悖论，在西方，不断有科学家研究过潘洛斯阶梯，据说有人造了出来，但是经不起检验。
<br/>　　在华夏，这种所谓的楼梯差不多都是出现在古墓里，一般是利用古墓特殊的地形修筑，悬魂梯分成两种，一种是按照数字“８”的形态，修成一个永远走不出去的怪圈，不管往哪个方向走，最后都还是回到起点。不过这种悬魂梯没有任何意义，也只能用来迷惑人，一般在这墓道上都还有各种机关埋伏，这样当人被困在悬魂梯中，到处摸索的时候，会触发机关而死……
<br/>　　还有一种悬魂梯，并不是真正的悬魂梯，而是利用地形，造出了一个阵法，再利用一写灵异方面的因素，影响人的判断，说白了就是一种障眼法。
<br/>　　听上去很神奇，不过真说起来，也就是个加强版的鬼打墙，让人以为路在前面，但是转来转去，又回到原点，但是在走的人看来，却真的是一条直路……
<br/>　　这种悬魂梯，因为靠的是灵异力量的加持和阵法掌控，因此，一定会有阵眼缩在，只要找到阵眼，堪破之后，阵法自己也就破了。
<br/>　　最简单的破除悬魂梯的办法，就是提一只公鸡，割开喉咙，倒提着一路走下去，来到之前走过的前面有血迹的地方，在合围的地方再转过身往回走，再走一圈，就可以闯出这个阵法。
<br/>　　不过叶少阳一干人仗着实力强，提前也没有太多的准备，一时间也没地方找公鸡去。
<br/>　　别的办法，当然也有，但是都需要能够持续消耗的法药用来破阵，叶少阳开始考虑用什么法药试试。
