　　第574章
<br/>　　霸王点金枪舞开，风雨不透，连绵枪影攻向程大雷。程大雷顿时只有招架之力，而无还手之功，他只能一退再退。
<br/>　　但现在的情况是周围猛虎环伺，一心一意要杀掉程大雷。程大雷的身后已无退路，他必须向前。
<br/>　　王无敌在白沙岛苦修多年，书院一直压着他不让他出世，本就是当做杀手锏来用的。一杆枪十几年不出世，第一次出世时，那锋芒定然光照苍穹，无人能敌。
<br/>　　这次为了杀死程大雷，连铁枪无敌的王无敌都动用了，可见书院杀人的决心。
<br/>　　的确好强呐。
<br/>　　程大雷心中感慨一声，倒是无多少惧意。今天本就是一场硬仗，他本也没打算简单便宜的过了晃一圈。
<br/>　　王无敌很强，枪得蛮横，但他也并非程大雷遇见的最强的对头。
<br/>　　更何况，程大雷本来就是个杀贼。
<br/>　　不能后退，程大雷便向前，用手中的剑开路。
<br/>　　所谓一寸长一寸强，一寸短一寸险。面对对方凌乱的枪法，程大雷大开大合的闪避很容易被对方所趁。所以，他选择迎难而上，用手接住了王无敌的霸王枪。
<br/>　　霸王点金枪重三十六斤，很明显，他强在速度、反应、眼力，但是，他的力气并不是很大。
<br/>　　真正的霸王枪，讲究一力降十会，一招霸王盖顶已经可以横扫天下，无人能敌。王无敌用的枪法，并不是真正的霸王枪，充其量也不过是霸王枪的衍生枪法。
<br/>　　程大雷用左手握住霸王枪，当真是险之又险，可也骇人得紧。那凌乱枪法烟消云散，只是一杆枪被捏在手中，王无敌面红耳赤，却撼不动程大雷分毫。
<br/>　　程大雷左手灵犀一指，接住他的长枪，右手直来直往的快剑，一剑洞穿王无敌的喉咙。
<br/>　　你名唤无敌，可不代表你就当真无敌喔。
<br/>　　程大雷上岛之后，第一次杀人，先拿书院压箱底的高手洗剑。九绝又怎样，一个个杀，总是能杀得完的。
<br/>　　程大雷左手提枪，右手握剑，猛回头一声高喝。
<br/>　　“谁来！”
<br/>　　众人一激灵，恍惚看到一头沾满血的猛虎从山林中窜了出来。
<br/>　　夫子从程大雷身上，再次想到气数这个词。他、孟玄青当年都是名动一方的角色，哪怕是宋寒之、张六、王无敌、孙飞鹰、周沧海如此多的这些人，都有自己的荣光时刻，也曾八仙桌上我最少，群英汇里独占鳌头。可今时今日，并不是从前他们风光露脸的时候。
<br/>　　程大雷这头猛兽今日气运旺得不行，很显然要鳌里夺尊。
<br/>　　且看看，能不能摁下他要抬起来的龙头。
<br/>　　杀。
<br/>　　程大雷往山顶闯，山下的人围过来，这白沙岛的山其实并没有多高，峰顶也不过百十多米。在很短时间内，程大雷身边围着的人便越来越多。
<br/>　　王无敌虽然没有亲手杀了他，但毕竟用自己的死缠住了程大雷。
<br/>　　此刻，书山剑阵已成。
<br/>　　这是七十二人组成的完整剑阵，每人都至少有顶级实力。三十六人就能困住燕不归，二十一人就能困住程大雷，何况现在是七十二人。
<br/>　　七十二人各持青锋剑，白袍临风而动，当真是飒飒威风。
<br/>　　看到剑阵已成，所有人都松了口气。毕竟，还从未有人能从这剑阵中逃脱。
<br/>　　程大雷深吸一口气，必须打点起全部剑阵。的确，二十一人的剑阵就能围住他，但当时程大雷要护着刘发财，难免陷入腹背受敌的状况。可如今，他却只有一个人，而他的剑也的确够快。
<br/>　　究竟能不能破阵，程大雷想试一试。
<br/>　　七十二按照特定的方位站定，但运转起来时，也按照某种特定的规律。其实，书山剑阵还是可以破的，七十二人是按照地煞七十二之数，传说，还有天罡地煞的一零八之数，之上还有周天群星三百六十五剑阵。
<br/>　　当初创造书山剑阵的一批人，都是十九国里面精英的精英，每人都有惊才绝艳的本事。他们本想凑齐三百六十五之数，或者是一零八的天罡地煞阵，然后，发现做不到。
<br/>　　从哪里找同心同德的一零八人，或者是三百六十五人，于是才退而求其次，定了七十二之数。
<br/>　　当然，对天下许多人来说，七十二已完全够用了。可这也给剑阵留下了破绽，令它是可以被攻破的。
<br/>　　对天罡地煞，什么七十二一零八程大雷完全是一窍不通，所以这剑阵对他而言，是相当够用。
<br/>　　不过，从来破阵就不止一种法子，还有一种，便是以蛮力破之。任你千百般变化，我自大力砸下，砸个稀巴烂。
<br/>　　程大雷的确也没破阵的力气，他如果往前冲，前后左右就要面临十几柄剑，剑剑锋利，剑剑要人命。
<br/>　　不过，程大雷也并非孤家寡人，他还有个同伙呢。
<br/>　　被剑阵围住后，书院打算慢慢磨，一直磨死程大雷。程大雷口中一声呼哨，那黑牛得到讯息，狂奔而来。
<br/>　　山峰很矮，山势很平缓，给了黑牛狂奔的空间。那辆水陆两栖的推土机再次发动，轰轰隆隆而来。
<br/>　　剑阵开始变化，散开了一条通路，没有试图阻拦这头暴走的黑牛。当然，估计也拦不住。
<br/>　　啧。
<br/>　　程大雷看着这一幕，发出一声轻叹。剑阵的组织运转能力，的确出乎他预料，如此看来，想靠着这头黑牛破阵，怕也不是太容易的事。
<br/>　　待黑牛扑到程大雷身边，程大雷翻身上了牛背，拍拍它的丑脑袋，道：“老伙计，来与我一起冲杀出去。”
<br/>　　程大雷开始破阵。
<br/>　　仗着这头黑牛的蛮力往前冲，的确给书山剑阵造成了巨大的压力。主持大阵的宋寒之面沉似水，当真也没有想到程大雷能想出这法子。
<br/>　　在他的指挥下，剑阵给出相应的变化，由外向内一层层转着圈子。程大雷冲向何处，人群便往何处聚集。
<br/>　　程大雷一次有一次冲锋，却都被挡了回来。
<br/>　　啧，看来加上这头黑牛，自己的力气也不够大呀。
<br/>　　“六爷，你进阵挡住他。”宋寒之担心程大雷破阵，如此吩咐道。
<br/>　　正这时候，一道声音忽然响起。
<br/>　　“张六，你在哪里。”
<br/>(本章完)
