　　第321章
<br/>　　女人心海底针，这句话诚不我欺，翻译过来便是，女人有时候柔得像水，但水里不知时候藏着针，当你心无旁骛，一口饮下时，里面的针就会扎得你滋滋冒血。
<br/>　　程大雷脊背冒汗，一道冷气从尾椎骨直抵天灵盖。
<br/>　　苏樱仍抹着眼泪，可在程大雷看来，却有种鳄鱼眼泪的感觉。
<br/>　　“我也不是那容不下旁人的妒夫，你若有心，便将她收了房，从此能多一个人照顾你，我也能省些牵挂。”
<br/>　　程大雷此刻还真有些摸不着苏樱的套路，恍然想起，这可是个附带明察秋毫属性的奇女子，世间根本没什么事能瞒得过她。现在的情况，自己是选择坦白从宽，牢底坐穿，还是抗拒从严，回家过年。
<br/>　　程大雷义无反顾的选择了后者，说到最后，苏樱扑哧笑了一声，倒是笑得程大雷浑身凉意。
<br/>　　不过，苏樱的话，却提醒了程大雷另外一件事。
<br/>　　山寨目前有一千多号弟兄，绝大部分是赤条条的光棍汉，这本身就是不安定因素呐。自己这当大哥的，军事问题要解决，生活问题也不能放任自流。
<br/>　　人有七情六欲，山寨的弟兄偶尔也会去城里吃花酒，对于这种事程大雷也只能假装瞧不着。当初在长安城时，程大雷也不是完全守身如玉，他和长安城几个当红头牌，都保持着不错的关系。
<br/>　　可如何解决这个问题，程大雷还没想到法子。总不能让手下人去抢附近城镇的黄花大姑娘吧。不如去戎族那边买些女奴过来……程大雷觉得这也不是自己的行事风格。
<br/>　　目前还有强敌环侍，对于这件事，程大雷暂时只能放下。
<br/>　　林少羽和林冲的伤势好转的很快，没过几日，就可以拄着拐杖支撑着下床。不得不说，系统给予的药品还是蛮有效的。
<br/>　　进入冬天，天气一日一日变得冷了。蛤蟆寨白天训练，晚上程大雷继续主持人手给大家授课，甚至还有城里掳来的私塾先生教大家识字。
<br/>　　脱盲班和军事班同时授课，每到晚上，一群大老粗便摇头晃脑，口中哼着些之乎者也。
<br/>　　林冲和林少羽也加入到听课大军，每看到林少羽全全神贯注的模样，程大雷就会觉得不寒而栗。
<br/>　　未来的他，究竟会走到哪一步？这一切对他而言，究竟是福是祸？
<br/>　　林少羽走到今日，程大雷觉得自己有不可推卸的责任。他随口编造了一个弥天大谎，说的人有口无心，听的人却奉为金科玉律。
<br/>　　于是，在授课时，程大雷总会侧重聊一些林少羽用得着的东西。
<br/>　　如果注定要趟入这乱世，多学一些东西终究是没错的。
<br/>　　到这一日，琴川迎来了第一场冬雪，浩浩荡荡，铺天盖地，雪停时望过去，山川大地都覆着白绒般的大雪，而河道也结了坚冰。
<br/>　　程大雷一直不希望发生的事情，终究是发生了。
<br/>　　……
<br/>　　琴川关，莫鸣米看着城下的积雪，脸上终于露出一丝笑容。长久以来积蓄在心中的郁气都随着这场雪而烟消云散。
<br/>　　“将军，斥候刚刚传来的消息，大河之上已经结冰，马匹可以通行。”严笛。
<br/>　　程大雷一直派人盯着琴川关的动静，莫鸣米其实也一直盯着蛤蟆寨的动静。双方都面带微笑，却都防备着彼此。
<br/>　　程大雷开辟商路，训练士兵，建设山寨……这些事情，莫鸣米并不是一无所知。他只是在等待，等待一个合适的机会，将程大雷这讨人厌的蛤蟆一脚踩死。
<br/>　　现在，他终于等到了这个机会。
<br/>　　大河结冰，蛤蟆寨失去了天险，他在河上布置的所有防御失效。自己的大军可以直接登上蛤蟆寨，与蛤蟆寨的山贼展开肉搏战。
<br/>　　不计代价，不计损失，一切只为除掉程大雷。
<br/>　　想到即将来临的大战，莫鸣米身体里的血开始变得炙热起来。
<br/>　　“将军，咱们什么时候动手？”严笛问。
<br/>　　“再等几日，让河上的冰更坚固些。我们的重甲军，也该操练操练。”
<br/>　　蛤蟆寨那里，程大雷也知道了琴川关的动静。对于即将来临的一战，程大雷并不是丝毫没有准备。这必然是艰难的一战，也当是最后一战，此战过后，将决定谁是琴川一地的主人。
<br/>　　蛤蟆寨从训练状态瞬间转入战斗状态，秣兵历马准备迎接即将到来的战斗。
<br/>　　目前最大的问题，就是冰封的河道，程大雷心中一动，既然河道被封，自己把冰面砸开不就可以了么。
<br/>　　一面令手下盯着琴川关那边的动静，一面程大雷开始带领手下兄弟开凿冰面，在冰上挖了一个个大小不一的冰窟窿，由此阻止骑兵队伍的推进。
<br/>　　大家都干得热火朝天，连日来的训练，显现出不错的成果。士兵的执行力，战斗意志都有不错的提升。
<br/>　　不过，这项工作只进行了一天，程大雷就决定结束了。因为头一天辛辛苦苦开凿的冰洞，第二天就冻得结结实实，甚至可以说更加结实。
<br/>　　程大雷无奈的接受这是一个愚蠢的决策，偶尔开凿一个冰窟窿来捕鱼还是可以的，但要说把冰面凿开，阻止大军推进，几乎相当于扯淡。
<br/>　　不过人有失手，马有失蹄，就算一个圣人也同样会犯错，更何况自己一个山贼。
<br/>　　如何防御，成了当前的重中之重，晚上的军事培训班成了战略会议室，大家群策群力，思考着如何应敌。
<br/>　　讨论就难免伴随着争吵，大家拍桌子踢板凳，如果不是程大雷一直在场，很多次都差点动起手来。
<br/>　　李行哉每次都出现，眼前这一幕令他难以置信，他还从未见过一个军事会议是如今这个模样。
<br/>　　这……或许就是程大雷的出奇之处。
<br/>　　但令李行哉琢磨不明白的是，这么重要的会议，山寨的第一军师徐神机却每次都不出现。
<br/>　　想来想去想不明白，李行哉觉得，真实情况可能是以徐神机的能力，已经不需要参加这种程度的会议。
<br/>　　到这一日，安插在琴川关附近的斥候飞马来报，莫鸣米点兵一万，倾琴川关之力，随时都有可能出兵。
