　　第519章
<br/>　　那坛子搁在脚底下，还嘎咕嘎咕作响。程大雷用一只脚踏在上面，心里琢磨着，这究竟是个什么玩意。
<br/>　　老尼姑也明白自己已身陷绝地，今日杀不了程大雷，自己怕从此地脱身都难。她当然不想死，但同时也不怕死。
<br/>　　她当时将一颗心横下来，双臂伸展，骨骼发出阵阵爆响。目光中的杀机宛若实质，面对程大雷已拿出了全部实力。
<br/>　　但程大雷今日占了天时地利人和，他又何惧之有。双臂晃了晃，将剑擎在面前，嘴角裂开，冷冷地笑了一声。
<br/>　　迅雷不及掩耳之间，程大雷暴起攻击，剑以决绝之势，撞向白衣老尼。
<br/>　　就此时，耳边忽然哞地一声，一团黑影扑向程大雷。
<br/>　　程大雷本来是锐不可当的一剑，就要和老尼姑硬碰硬，可见到这团黑影，却生生收回了剑，身子往后退。
<br/>　　牛棚里的黑牛找已等了太久，它一直没明白是怎样一回事。当然，它现在也不算很明白，只是看到程大雷出现，撞破牛棚窜了出来。
<br/>　　来就势不可挡，如果不是老尼姑退得及时，她非得被掀翻不可。无可奈何，她也只得后退，心中暗暗吃惊。以她的本事，在大山里打虎都不算太困难的事情，但这头黑牛却给她一种危险的感觉。
<br/>　　程大雷也很无奈，这黑牛一直缠着他，一副谄媚的样子。可是，这无形中，却挡住了他的攻击，本来今夜的情景，要宰掉老尼姑不难，可因为这头黑牛，却处处受限。
<br/>　　这黑牛还以为他要上阵，看谄媚的样子，是准备让程大雷骑上它冲锋。但这样的院子里，程大雷却不需要冲锋。
<br/>　　我都快胜了，你又冒出来做什么。
<br/>　　老尼姑却不会被这黑牛所干扰，顿时抖索威风，将全身本事施展开来，扑向程大雷。面对这一幕，她颇有种大快人心的感觉：瞧瞧，连这黑牛都在帮自己。
<br/>　　“大当家，退！”
<br/>　　院门，响起秦蛮洪亮的声音。程大雷立刻反应过来，几个纵跃便隔着墙跳了出去。那黑牛动作更是蛮横，用牛角撞塌墙出现在院子外。
<br/>　　老尼姑本想跟着窜出去，却被一阵阵弩箭，硬生生逼了回来。
<br/>　　程大雷和秦蛮汇合，看到院外已经有一排排士兵，将整座院子团团围住。刚才动手的时候，他已听到外面的动静，只是没时间分神查看外面的情形。
<br/>　　程大雷和老尼姑在这里打得热闹，秦蛮也不会没有半点反应。上次老尼姑在蛮字军的攻击下走脱，被秦蛮视作奇耻大辱，最近这些日子，发了疯的训练手下人。今天得知老尼姑闯了进来，他第一时间不是愤怒，而是高兴。
<br/>　　雪耻的时候到了。
<br/>　　立刻调了三千兵过来，将城主府团团围住，这小小的院落外，就有七八百人。
<br/>　　程大雷捋顺了呼吸，冲秦蛮道：“你们替我压好阵，待我进入宰了这老娘们。”
<br/>　　秦蛮拦住了程大雷，道：“大当家，千金之子坐不垂堂，以您的身份，不该冒险。而要杀这老尼姑，属下已有法子。”
<br/>　　程大雷皱起眉头：“什么法子？”
<br/>　　秦蛮伸手一指，道：“只要大当家舍得了这院子。”
<br/>　　程大雷抽了抽鼻子，已经嗅到火油的味道，他看到秦蛮已经派人将隔火沟挖好。
<br/>　　“既然你已经准备好了，我还能说什么。难道我一座院子也舍不下么。”
<br/>　　秦蛮眼放寒光，目光看着院门，道：“动手！”
<br/>　　秦蛮憋着一股邪火，他手下蛮字军也不例外。上次跑了老尼姑，蛮字军差点成了蛤蟆城的笑话。尤其是关鱼的鱼字军，一直有竞争压过蛮字军的意思。碰到这机会，还不好好奚落蛮字军一番。蛮字军的弟兄也是无话可说，毕竟，军中做事，胜者为尊，败者不如犬。
<br/>　　压到现在，这股怨气终于有机会发泄出来，人人如狼似虎，按照秦蛮的命令开始行动。
<br/>　　装满火油的罐子抛了出去，然后是燃烧的火箭，瞬间，整座院子就燃起熊熊大火。
<br/>　　程大雷摁住剑柄，他并不觉得如此安排，就可以杀掉老尼姑。如果老尼姑硬抗着闯出火海，程大雷要给她致命一剑。
<br/>　　秦蛮发现了程大雷的动作，他摇摇头道：“今日不用大当家动手，请大当家做壁上观。”
<br/>　　“莫要大意，这老娘们可不简单。”
<br/>　　“大当家放心，今日若让她逃了，我蛮字倒着些。”
<br/>　　秦蛮终于拿出些豪横劲儿来，铁了心要在此地宰掉老尼姑。
<br/>　　一片火烧，烧红了半边天空，外面的人都觉得烟熏火烤，何况身处火海当中的老尼姑。
<br/>　　这院子内本就堆满了稻草，如今一经火燃，立刻燎烤起来。白衣老尼几次想往外闯，都被拦了下来。眼中被红色填满，进无进路，退无退路。体内一阵阵燥意往上涌。
<br/>　　白衣老尼已经情知不妙，最后的目的，便是拉着程大雷同归于尽。最后关头，她将希望搁在那只毒蛙身上。
<br/>　　一脚踏碎了陶罐，却看到那只凶神恶煞的毒蛙已经一动不动，不知是不是已经被烤死。
<br/>　　她知道这毒蛙不能接触身体，但如今已顾不得太多，用手拎起来想投入水井中。但手刚碰到毒蛙的身体，半条胳膊已经红肿起来。
<br/>　　这蛤蟆的赌性好生了得，便是已经被烤得半焦，还能在片刻之间废掉自己一条胳膊。
<br/>　　转眼间就是毒气攻心，老尼姑身子晃了三晃，感觉头晕眼花。她也知道自己时辰到了，强用真气压住体内毒气，恶狠狠的咬着牙。
<br/>　　“程大雷，我便是死，也得拖你一起下地狱。”
<br/>　　身体跃起，冲出火海，身上袍子已被火点燃，但此刻她不管不顾，口中大吼：“程大雷，你在哪里，你给我去死。”
<br/>　　程大雷骇了一跳，这老尼姑此时已经是个火人，凶神恶煞，如同妖魔。
<br/>　　但秦蛮的表情却冷得很，用不大不小的声音道：“放箭。”
<br/>　　箭若雨下，老尼姑手舞足蹈，如同中魔，用手中的拂尘敲打羽箭。
<br/>　　可她的速度越来越慢，手中的拂尘渐渐挡不住纷纷而至的羽箭。
<br/>(本章完)
