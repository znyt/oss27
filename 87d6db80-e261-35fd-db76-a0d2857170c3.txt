　　“怎么回事？”
<br/>　　罗昊在电话里向云岳了解情况。
<br/>　　“是若依打来的电话，她在电话里也没有具体说，只说小战在学校跟高年级学员打架，上去两拳就把人家的鼻子给打断了。”云岳在电话中说道：“好像还是小战先动的手。”
<br/>　　“蓝丫头说是我家小子先动的手？不是别人先惹得他？”
<br/>　　罗昊对蓝若依的说法感到有些怀疑，自己家那小子平时虽然不安分，会调皮捣蛋，可是绝对不会主动去打架惹事。
<br/>　　“我跟小熙现在就去红旗学校一趟。”
<br/>　　罗昊说完就挂断了电话，在罗昊跟云岳通话期间，云熙已经洗漱完并且换好了衣服。
<br/>　　“你赶紧换衣服、洗漱，我去开车。”
<br/>　　云熙抓起桌子上的车钥匙就匆匆下楼去。
<br/>　　……
<br/>　　红旗学校，校长办公室。
<br/>　　老校长夏洋好几年前就已经从位置上退了下来，现在只是挂名的荣誉校长，真正的负责整个红旗学校大小各种事宜的，是红旗学校的现任校长沈琛。
<br/>　　沈琛是文职军官出身，算是我国对特种作战研究比较早的一批人，博士学历，国内读完博士就去了国外深造，对国内外的特种部队和特种作战都有很深的了解，以及自己独特的见解。回国后，在某军区特种大队从事特种作战研究，后来被夏洋发现，挖到了红旗学校担任教员，罗昊、齐凡、李然、蛰龙这些人都是他的学生。
<br/>　　老校长从龙穴把罗战和云悠带到红旗学校来的事情，沈琛是知道的；罗战和云悠的爸妈是谁，沈琛也知道；罗昊是什么样的脾气，沈琛更是了解的清清楚楚。
<br/>　　所以，从事发到现在，沈琛一句话都没有说，始终皱着眉头，他实在太了解“混世魔王”罗昊的脾气了，这家伙这次可是又要来“大闹天宫”了。
<br/>　　“校长，这事儿必须要严肃处理。”
<br/>　　近身格斗教员诸钟伟指着自己身边被罗战打断了鼻梁，整个鼻子肿的跟胡萝卜似的学员，对沈琛说道：“这学员跟学员之间交流学习，怎么能对同学下这么重的手！”
<br/>　　严肃处理？怎么严肃处理？
<br/>　　沈琛的眉毛都快要打结拧在一起了。云悠和罗战作为插班生被夏洋带进学校，别人不知道他们的身份，沈琛还能不知道？沈琛要是严肃处理他们两个，回头罗昊就敢把自己的办公室给拆了！
<br/>　　“那是他欠揍！”
<br/>　　罗战挥着拳头就要往前冲，指着那个被自己打断了鼻梁骨的学员说道：“谁让他欺负小悠儿的，我只打断他的鼻梁骨都算是客气了！”
<br/>　　站在旁边的蓝若依一把拦住罗战，问道：“小战，到底是怎么回事？你们不是上格斗课，怎么就打起来了？”
<br/>　　“罗战，我没事，你别惹事了。”
<br/>　　躲在罗战背后的云悠拉了拉罗战的衣服，说道：“大家都是同学，算了。”
<br/>　　“谁跟这混蛋是同学，跟他做同学我都觉得丢脸！”罗战这次没有听云悠的劝，瞪着藏在诸钟伟身边的诸磊，恶狠狠地说道：“这件事不能就这么算了，你就给他白欺负啊？别以为这诸钟伟是他叔叔，他就能无法无天，今天我不把他打成孙子，我就不是罗战！”
<br/>　　“校长，你也看到了，这个罗战态度恶劣，打了同学还不知道认错。”诸钟伟对沈琛说道：“这样的学员就应该开除出学校！”
<br/>　　“对，应该把他们两个都开除！”
<br/>　　因为被罗战打断了鼻子，诸磊说话的时候，瓮声瓮气的。
<br/>　　“嘁，说得我好像很想要赖在这里一样，就这地方，我还不愿意待呢。”罗战满不在乎的说道。
<br/>　　罗战这话一出，不仅是沈琛，就连旁边的蓝若依脸色都是一变，虎着脸对罗战说道：“你胡说八道什么呢！？”
<br/>　　罗战这小子从小就聪明，一看这周围气氛不对，就知道自己说错了话，闭上嘴站在一旁不再说话。
<br/>　　“小罗战，你来说说，到底是怎么回事，为什么把诸磊的鼻子给打断了？”
<br/>　　沈琛这个时候开口向罗战询问起事情的原委来，诸钟伟领着诸磊来自己这里告状，只说罗战在格斗课上把诸磊鼻子打断了，但是其中具体的原因，诸钟伟却没有说明。
<br/>　　“那是他欠揍！”
<br/>　　罗战毫不客气的往办公室里那张真皮沙发上一坐，晃荡着两只脚。
<br/>　　“他怎么欠揍了？”沈琛继续问道。
<br/>　　罗战看了眼沈琛办公室里的冰箱，说道：“我渴了。”
<br/>　　“你这小子，就跟你爸一个德行。”沈琛看着罗战，就像是看到了当年的罗昊一样，笑着从椅子上站起身来，打开冰箱门拿出一罐可乐，递给罗战说道。
<br/>　　“他是‘混世大魔王’，我是‘混世小魔王’，那必须得一样，要不然不是丢了他的脸？”
<br/>　　罗战毫不客气的接过了沈琛递给自己的可乐，说道：“还有小悠儿的！”
<br/>　　“你小子，倒是一点也不客气。”沈琛手指虚点了一下罗战，又从冰箱里拿了一罐可乐，递给云悠。
<br/>　　“谢谢校长。”
<br/>　　云悠道了声谢，从沈琛手里接过可乐。
<br/>　　“你姐姐可比你懂礼貌。”沈琛坐在罗战旁边，问道：“现在口不渴了，是不是可以说了？”
<br/>　　“她随我们妈妈，我随我们爸爸，正常！”罗战喝了一口可乐，说道：“不就是看不惯我们两个‘插班生’的身份，故意找麻烦。这小子知道我们两个是插班生，好几次带着人来挑衅我跟小悠儿，但来的时候，我家老头子说了，退一步海阔天空，忍一时风平浪静，我也就忍了，可谁知道，他不但不知道收敛，还蹬鼻子上脸，趁着这次两个年级的混合格斗教学课，点名要挑战我们。”
<br/>　　听到罗战说罗昊告诉他们姐弟俩“退一步海阔天空，忍一时风平浪静”的时候，沈琛差点就笑出声来，他很难想象这两句话会是从罗昊嘴里说出来的。
<br/>　　按照罗昊的脾气，教育罗战的模式应该是这样的，“儿子，记住了，别人怎么欺负你的，你就要怎么欺负他。他打你一拳，你就要还他两拳，他欺负你一次，你就要欺负他两次！”
