　　几个大唐商人正在讨论着关于这次商业洽谈的见闻，见彭城过来，一个个顿时露出笑容打起招呼：“老彭，你这个外部经济贸易司的司长大人什么时候能有空？咱们一起出来坐坐如何？”
<br/>　　“哎，我也想啊，不过段时间外经贸才刚刚成立，正是多事的时候，等什么时候把事情理顺了，由某作东如何？”彭城打着哈哈，将哈德林克引到自己身边，然后用一种十分熟络的语气说道：“不过你们也别说某不照顾你们，我身边的这位就是拜占庭的哈德林克先生，你们今后产品能不能进入拜占庭就看你们的了。”
<br/>　　番子与大唐因为习惯的不同，他们并不怎么使用大人这个称呼，所以彭城一直以先生来称呼哈德林克，而哈德林克也不反对这样的称呼。
<br/>　　“哈哈哈……，老彭就知道你最够朋友！”丝绸商人郑克明的明中露出一丝了解，哈哈大笑着对彭城拱了拱手，然后又对着哈德林克拱了拱手：“哈德林克先生，在下有礼了！”
<br/>　　哈德林克学着其他人的样子，对着郑克明等人拱拱手，用生硬的大唐官话一字一顿的说道：“你们好！”
<br/>　　郑克明等人显然没有料到这外国老番子还会这么一手，一个个有些意外的愣了片刻，而后同时举起拇指：“哈德林克先生，厉害！”
<br/>　　“哈哈哈……”哈德林克像是完成了一项十分伟大的成就一样，得意的笑着，末了说道：“大唐的官话实在是太难学了，见笑，见笑了！”
<br/>　　“哈德林克先生，来，我给你介绍一下，这位便是大唐数一数二的丝绸商人郑克明；这位是玻璃商人钱富；这位是……”
<br/>　　彭城本就是长安的商贾，虽然不是最顶级的，与郑克明等人混的不是一个圈子，但一些简单的信息却难不住他，一会儿功夫就把面前的几人全都介绍了一遍。
<br/>　　哈德林克只一听几人所经营的商品，脸上立刻绽放出灿烂的笑容，再次拽了一句半生不熟的大唐官话：“幸会，幸会啊！”
<br/>　　看来这老番子为了今天应该是下了不少的功夫呢，彭城与郑克明几人对视一眼，心中已经有了一丝明悟，也对今天晚上的这一次商业洽谈充满了希望。
<br/>　　要知道，如果这老番子不想与大唐作生意，或者说作成生意的想法不是很强的话，他完全用不着学习大唐官话，以他拜占庭贵族的身份，也的确不大需要如此与大唐的商人拉近关系。
<br/>　　虽然说大唐与异族之间在某些理念上存在差异，但在官与商的地位差异上却依旧有着很多的共通之处，就社会地位来说，异族的贵族虽然看重钱，但同样不需要这样讨好一群商人。
<br/>　　所以在短暂的对视之后，郑克明第一个开口说道：“哈德林克先生难道对大唐的丝绸也有兴趣？”
<br/>　　“是的，我很有兴趣，只是不知道郑先生的丝绸价格是什么样的？”哈德林克倒也没有绕圈子，郑克明一问，他立刻就反问了一句。
<br/>　　“按照大唐的标准，每匹的价格大概要十五贯左右”郑克明想都没想就回答道。
<br/>　　这个价格是提前商量好了的，已经是大唐境内丝绸价格的五倍，不过相对于拜占庭国内丝绸制品的价格，依旧便宜了不少。
<br/>　　彭城站在一边见他们已经开始谈起生意，知道自己在这里再待下去就会比较尴尬，于是对着哈德林克笑了笑说道：“哈德林克先生，你们先聊着，本官失赔一下。”
<br/>　　哈德林克也不是傻子，他当然知道彭城是什么意思，不过他现在也没有什么理由来挽留彭城，于是只能点头说道：“彭大人尽管自便，我还要与几位先生好好聊一聊。”
<br/>　　“失赔！”彭城微微一笑，又对着郑克明几人点了点头。
<br/>　　“大人请便！”郑克明等人同样也没有挽留彭城，拱了拱手目送他离开。
<br/>　　而等到彭城一离开，哈德林克也恢复了正常的神情，看着郑克明说道：“郑先生，我很想知道，为什么卖给我们的丝绸价格会如此高昂，据我所知，在大唐一匹丝绸的价格应该在两到三贯左右才对。”
<br/>　　“非也非也，这个价格其实并不贵，要知道，在大唐境内销售的丝绸其实是有官府补贴的，而出口到国外则不然，这是一种资源外流，官府会因为资源外流而取消一些政策上的扶持，而且补贴的那一部分费用也会取消。”
<br/>　　“哈德林克先生，我们是商人，并不是慈善家，我们的手下需要养活大量的工人，如果没有一定的利润，我们同样不会将商品卖出去。”
<br/>　　郑克明与哈德林克相对而立，侃侃而谈，把前段时间大家聚在一起商量好的说词一股脑儿的倒给了他。
<br/>　　丝绸是用蚕丝织成的，蚕丝是由蚕吐出来的，蚕是虫子，是可再生的，而蚕吃的树叶同样也是可再生的，哪来的资源流失？哈德林克明知道郑克明说的并不是那么一回事儿，但却没有办法反驳，大唐是一个强大的国家，他没有办法用武力来威胁，所以只能任由那个姓郑的家伙在那里胡说八道。
<br/>　　倒是那个玻璃商人钱富比较有眼色，见哈德林克脸色有些难看，打着哈哈加入了进来：“哈德林克先生，老郑这人说话比较直，您不要见怪。但从某些方面来说，他说出的这个价格不真不算是贵，您可以想想看，我们作为大唐的商人，其实在很多方面都享有优惠政策的，比如说运输，我们可以借助铁路，这比你们在长安买完了东西运回去要省事的多吧？再比如安全，你以这个价格从我们这里买东西，我们可以给您安安全全的送到边境，保证不会出一点问题，而如果您自己独立运输，安全方面又如何保证？”
<br/>　　哈德林克的眉头皱了皱，不得不说，钱富的这一翻话还真说到他心里去了，运输和安全方面的问题的确是他是所担心，如果真的能像钱富所说，用铁路将这些东西运到边境的话，这个价格还真不是不能接受。
