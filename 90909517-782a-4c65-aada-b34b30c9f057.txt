　　第615章
<br/>　　程大雷窝在车厢一角，手脚都被绑着，也是动弹不得。
<br/>　　他的目光无可奈何的落在周桃身上，见周桃将原先那马夫的衣服扒掉自己换上。她先脱掉自己的衣服，顿时露出一具吹弹可破，玉雪晶莹的娇躯。程大雷目不转睛的望着，周桃也完全不在意，宛若将程大雷当做死人一般。
<br/>　　“卿本佳人，奈何为贼。”程大雷悠悠叹了一声。
<br/>　　突然，周桃掌心多了一柄匕首，冷锋扫过，斩断程大雷额头三缕黑发。
<br/>　　“好刀。”程大雷情不自禁的赞道。
<br/>　　本想吓得程大雷闭嘴，谁料程大雷恍然不觉扑面而来的危险。周桃也有些无语，道：“我再进一进，你的命就要交待在此地。”
<br/>　　“能死在姑娘刀下，当是我的造化。”
<br/>　　看着程大雷无赖的样子，周桃的确难以想象，他便是那位搅动天下风雨的琴川之主。索性不去理他，自顾自换上马夫的衣服。又取出贴身的一个小包袱，将一样白色药粉拍打在脸上。她一番收拾，程大雷吃惊得睁大眼睛。
<br/>　　只见短暂片刻间，这周桃脸颊胖了一圈，头发乱糟糟的打结，脸上还有细细密密的麻子。看上去就像个模样普通的年轻后生，那还是刚才勾魂夺魄的蛇蝎美人。
<br/>　　鬼脸婆婆，名不虚传呐。
<br/>　　她驾着马车出发，程大雷也不知道她要带自己去哪儿。可她既然不杀自己，至少证明自己有更大的作用。至于她抛下柳追风，定然是因为柳追风在她心里没那么重要了。
<br/>　　二人就隔着一道轿帘，程大雷自己占了轿厢，他抱着既来之则安之的心思，倒是要看看周桃要带自己去那里。
<br/>　　到正午时分，周桃驾着马车来到一处镇子。她在路边店铺买了一屉包子，看她的意思并不打算与程大雷分享。
<br/>　　程大雷饥肠辘辘，但也只能看着周桃在那里大块朵颐。在镇子中心有棵大槐树，两三人合抱不过来，树冠一蓬乌鸦窝，正午时乌鸦哇哇的叫着。
<br/>　　马车停在大槐树下，春寒料峭，也没什么人。等到四周无人时，周桃身子附在树干上，几下便攀上树顶，从乌鸦窝中掏出一样物事。
<br/>　　程大雷奇怪的看着这一幕，心里算定，这当就是陆长迢丢失的那方军印了。不过有些事情程大雷还是想不明白，这方军印对于陆长迢来说十分重要，但对普通人来说，其实也没那么重要。说是见印如见人，但也仅仅是说说而已。
<br/>　　那么这周桃千辛万苦盗了军印，又费尽心思藏在此地，究竟有什么目的。
<br/>　　程大雷本来就好奇心旺盛，眼下是更加好奇了。
<br/>　　周桃驾着马车出了镇子，越走天越黑，等到日落西山的时候，她将车停在河边一芦苇荡前。
<br/>　　口中发出三声喜鹊叫声，过了片刻，芦苇荡中也想起三声鸟鸣。程大雷心里琢磨着，这当是她与同伙接头的暗号，不过程大雷想不清楚：谁会是这鬼脸婆婆的同伙。
<br/>　　从芦苇荡中走出一个黑脸大汉，络腮胡子，提一柄鬼头刀。
<br/>　　“姑娘，东西带到了么？”
<br/>　　“嗯，他还好么？”
<br/>　　“老大一直在等姑娘。”
<br/>　　黑脸汉子看了一眼马车，道：“车上有什么东西？”
<br/>　　“等到了地方再说。”
<br/>　　看周桃如此鬼鬼祟祟，所图当然不小。程大雷也琢磨不清楚她在图谋什么，不过程大雷觉得，等他们把自己带到地方，就应该可以接近真相了。
<br/>　　继续往前走，天不知不觉暗了下来。一轮明月生在中天，如水的月光洒下来，大地像镀了一层银膜。
<br/>　　前方出现一片梨园，中间有座农神庙。程大雷被绑在马车里，也没有向外看，但外面的声音渐渐乱起来，应该是人越来越多。
<br/>　　“莫老大今日开这场绿林会，遍请西北豪杰，想是做一件大事的。”
<br/>　　“怕莫老大这次有大心思，胃口不小呐。”
<br/>　　程大雷掀开轿帘，悄悄向外面张望一眼，顿时吓了一跳。在这片不大的梨园中，窝了上千人，他们身上都带着兵器，打扮得模样千奇百怪，该都是京州凉州二地的绿林人物。
<br/>　　有人来迎周桃，周桃直接牵着马车走进农神庙。庙中点着火把，显出一座狰狞可怖的塑像。一个男人站在塑像前，看到周桃时目光柔和起来。
<br/>　　“桃妹，你回来了。”
<br/>　　周桃用清水洗脸，恢复了本来模样，她将那装着军印的包袱递给男人。男人接过来看了一眼，脸上立刻露出喜色。
<br/>　　“好好好，有这东西在，就可叫西北绿林道心服口服。”
<br/>　　周桃道：“莫郎，我还带来另外一件东西，你怕是绝没想到他是谁。”
<br/>　　“桃妹卖什么关子，既然你这么说，这个人想来十分重要了。”
<br/>　　周桃笑而不语，将男人引到马车前，伸手拉开轿帘。外面的光线透进来，程大雷看清楚了男人的模样，同时对方也看清了他。
<br/>　　呀。
<br/>　　两人同时睁大眼睛，心里想得是同样一句话：怎么是他。
<br/>　　这男人不是旁个，乃是西北万马堂的首领莫将难。万马堂在西北还是有很大势力的，这莫将难号称西北绿林的头把交椅。曾经正义教起事的时候，绿林道也跃跃欲试，这万马堂还想拉拢程大雷。当然，最后没讨到好果子吃。
<br/>　　莫将难在秦蛮手中狠狠碰了钉子，灰头土脸的离开。
<br/>　　这对程大雷来说不是一件很重要的事，如果不是查看对方的信息，他连莫将难是谁都不认识。但这件事对莫将难来说，可以说是奇耻大辱，程大雷的样子，怕他一辈子都忘不掉。
<br/>　　他咬紧了牙，发出阴森的冷笑：“程当家，好久不见，没想到是我吧。”
<br/>　　程大雷现在被绑得像粽子相仿，委实也动弹不得，他呵呵笑了一声：“好久不见，没想到在这里碰见，咱们……还真是有缘呐。”
<br/>　　想起往事，莫将难咬牙切齿，道：“多谢桃妹将他带过来，他落在我手中，我非让他明白生不如死是什么滋味。”
<br/>　　倒是周桃拦住了莫将难，道：“莫郎，这人留着还有些用处，先容他活两天。”
<br/>(本章完)
