　　第607章
<br/>　　衬着飘忽的火焰，白无双长叹了口气，道：“都是往事，不提也罢。”
<br/>　　野原火目不转睛：“先生莫怪我多心，我的心意先生已经知道，可我也要知道先生为何要帮我？”
<br/>　　白无双理解野原火话中的意思。只不过有些事，已经过去很久，他从没对任何人说起过。也就是今天没有办法，他才肯打开脑海中尘封的记忆，说起那些不愿提起的往事。
<br/>　　算下来，这白无双与白衣老尼、孟玄青、夫子是同一类人，他们都是十九国的后人。当初十九国覆灭之后，十九国后裔组成反帝联盟，想着能推翻李氏的统治。不过十九家势力，也是各自说各自的话，各自有各自的利益。所谓的反帝联盟，根本没做过一件事，便自己闹得不可开交，后来分崩离析。
<br/>　　李氏渐渐坐稳了江山，所谓推翻李氏也就成了无稽之谈。他们只不过也就是做些暗杀捣乱的勾当。这白无双乃是故燕国的后人，知道从帝国内部推翻已经是不可能，于是便只身来到草原。许多年经营，终有一番成就。
<br/>　　野原火静静听着，道：“你想复国？”
<br/>　　白无双摇摇头：“故国旧梦，不过是一场笑话。帝国风雨飘摇，但最后取帝国而代之的，定然不会是十九国的后人。我已到这把年纪，过去的朋友和敌人，早已死得所剩无几，便就剩我孤家寡人活在世间。说不得某天，一阵风就能吹走我的性命。当初，我的亲人都死在帝国追杀中，现在我想要的只是毁灭。帝国这座老房子快要塌了，我愿意用这把老骨头给它加把柴，让它烧得干干净净。”
<br/>　　野原火轻轻点头：“我大概信了白先生的话，白先生今天既然来找我，当应该已经有了主意。怎样做，才能吞掉呼延氏？”
<br/>　　白无双的确是有准备而来，他道：“呼延氏曾经是戎族共王，只要消灭他再征服整座草原也不是难事。大王如果有吞天下的心，老朽就甘当大王的马前卒。呼延力手下有两员大将，人人都有万夫不敌之勇。三日之内，我取其中一人的人头，算是老朽给大王的投名状。”
<br/>　　野原火也知道呼延力手下大将，一名鹰，擅长连环箭，能射连环十三箭，堪称神术。另一位名叫蛮雄，双臂有千钧之力，堪称戎族第一勇士。野原火如果向呼延力动手，这二人就是最棘手的存在。
<br/>　　“我等白先生的消息，等先生的消息到了，我这边就可以动手。”野原火道。
<br/>　　远处有马蹄声响起，一队人举着火把，口中呼道：“大王，大王……”
<br/>　　白无双起身行了一礼，道：“大王，老朽暂且告辞，人头不日就能献上。”
<br/>　　他身影一闪，已经消失在黑暗当中。野原火的手下赶到，他们都是出来寻找野原火的。野原火今天出去这么长时间，手下难免有些担心，便顺着火光找到此地。
<br/>　　野原火和诸人一起返回营地，心中还在琢磨着白无双的话。如果有白无双相助，自己想做的事情，就能尽快完成。但到现在，野原火也并没有完全信任白无双，一切就看接下来的消息，他是否能顺利为自己献上人头。
<br/>　　……
<br/>　　草原之上，一眼望过去，入目都是荒草。
<br/>　　程大雷和小白狼流浪在草原上，屈指算来已经有些日子。二人捕鱼打猎为食，填饱肚子倒是没有问题。不过一直餐风饮露，二人活得和野人也差不多。
<br/>　　关键是程大雷也失去了『耳朵』和『眼睛』，草原上连个人影都晃不见，就更不用提打探到什么有用的消息。
<br/>　　这一日，程大雷从湖里打了两条鱼来，在湖边烤着吃。
<br/>　　小白狼也并非不能过苦日子的人，只不过这样一日日无聊待着，终究有几分枯燥。
<br/>　　“我们要等到什么时候？”小白狼道：“不如我们回蛤蟆城吧？”
<br/>　　程大雷翻着烤鱼，也轻轻叹了口气。此地距离蛤蟆城也并没有太远，回家也用不了多长时间。只不过，不杀燕不归，他心中这口气实在咽不下去。
<br/>　　“等一等，再等等。”
<br/>　　“但我们这样要等到什么时候？”
<br/>　　程大雷挠挠头，心里确实也没有太好的主意。他心里盘算着再进一次戎族王城，不过上次的事情已告诉他，想要单枪匹马进王城杀人，几乎是没可能的事情。
<br/>　　“不如我们抓个人，打听一下附近消息。”
<br/>　　“说是如此说。”程大雷道：“可在这鸟不拉屎的地方，抓个人知道的事情还不如我们呢。”
<br/>　　说着说着，程大雷叹了口气，道：“实在不行，只能打道回府了，回去之后，打探清楚这里的消息，再找机会动手。”
<br/>　　说是如此说，可程大雷也明白，如果这次杀不了燕不归。事情一旦放下，以后可能就再没杀他的机会了。回蛤蟆城日子自然能过得轻松些，可让他放过燕不归，也并不是一件容易的事。
<br/>　　这段时间，程大雷和小白狼就流浪在女儿湖附近，距离戎族王城倒不是特别远。二人已经有了离开的准备，只不过想在临走前，再试一试。
<br/>　　……
<br/>　　北蛮部大帐。
<br/>　　野原火屏退左右，看着面前的柳褐岩，他的帅案上还摆着一方盒子，里面有一颗血淋淋的人头。
<br/>　　白无双果然有不弱的手段，按照事先约定，已经拿下蛮雄的人头。这份投名状，已大概能令野原火相信他投诚的心。
<br/>　　这件事目前还属于机密，不能让任何人知道，就连柳褐岩过来传信，也是悄悄摸进来的。
<br/>　　“大王，长老还送了一封密信。”
<br/>　　野原火将信打开，他勉强认字，大概能读懂信上的内容。
<br/>　　用兵攻打戎族王城，并不是一件容易的事。就算能够打下来，怕也是损兵折将严重。于是白无双为野原火想了个法子，他找个由头，让呼延力走出王城，在草原上狩猎。野原火可以派精兵赶来，长路奔袭，杀掉呼延力，拿下呼延氏。
<br/>　　按照白无双的计策，这件事并不是不可做。
<br/>(本章完)
