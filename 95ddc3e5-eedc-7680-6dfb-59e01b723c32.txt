　　“废话不多说，我要的员工条件已经写得很清楚了，在我这里干，一天给你二十颗晶石。你们都知道吧？不符合规矩的就别浪费大家时间自己乖乖离开这里吧。”林峰道。
<br/>　　“我知道，我之前就是矿工！你看我的手！”
<br/>　　一个人开口道。
<br/>　　林峰扫了眼这个家伙，都是茧子，“很好，你可以留下。”
<br/>　　“耶！”
<br/>　　这个人居然还高兴的跳起来了。高兴的像个猴子。很显然，他为自己找到了一个生计很高兴。
<br/>　　“我也是！”
<br/>　　“我全家都是矿工！”
<br/>　　其余人顿时一个个自高奋勇。
<br/>　　“很好，你们都留下，你们几个呢。”林峰道。
<br/>　　“我是黑铁战士三阶的。”
<br/>　　“我是二阶的。”
<br/>　　“很好，你们都留下。”林峰道。
<br/>　　“我，我是矿业大学的学生。”
<br/>　　一个戴着眼镜的人道。
<br/>　　“矿业大学的？”
<br/>　　林峰扫了对方一眼，不是黑铁战士。
<br/>　　矿业大学林峰知道，很厉害的学校，好像是211吧。
<br/>　　“你们矿业大学好像学的都是地质勘探，或者环境监测之类的东西吧。”
<br/>　　这学生点了点头，“是的。”
<br/>　　“那我们老龙口矿业公司用不着，我现在要的就是愿意吃苦耐劳的矿工，技术人员不需要。不过我觉得也许以后你会有用武之地。”
<br/>　　林峰无情拒绝了对方。
<br/>　　这个学生顿时满脸黯然的离去了。
<br/>　　林峰没有任何同情，他虽然富有，但也不会养废物，这种高知识分子现在，短时间内来说，完全没用了，对方一肚子的知识，但现在没有仪器，那也是白搭。
<br/>　　来的一共三十个人，就这个家伙被淘汰了，其余二十九个不是曾经就是矿工就是本身就是个黑铁战士。
<br/>　　不要以为是个黑铁战士就敢杀丧尸敢见血了，这都因人而异的，有些人也是运气使然变成黑铁战士，但实际上自身也没有多大战斗力，宁愿去挖矿也不愿意离开基地去和丧尸搏斗。
<br/>　　林峰把他们全部带到了老龙口。
<br/>　　“哇，你们看，这里还有坦克。”
<br/>　　三十个人一开始一看要离开曙光基地还是很害怕的，可是现在他们一个个一点都不害怕了，因为看到了坦克。
<br/>　　“狗子，这些人以后就你来管了，你来当督察员。”
<br/>　　林峰道。
<br/>　　“我去，我岂不是那头头了？”陈勾笑着道，“是不是我还得找个鞭子，就好像电视剧里演的那样，这群家伙不干活我就一鞭子下去。”
<br/>　　“少屁话，人家是我们员工不是我们奴隶。”林峰无语道，“你带他们去熟悉下矿场吧。”
<br/>　　“得嘞。”
<br/>　　陈勾立即开着车带着一群人去矿场了。
<br/>　　李广带着手下坐镇基地。
<br/>　　林峰见此就干脆去附近探探情况，看看老龙口附近有没有什么危险，不过逛了一大圈，怪物的确有，但没有大规模怪物的聚集点。可以说他的老龙口基本上算是安全稳定下来了。
<br/>　　黑龙镇也是如此，寒鸦那边为了补充战死的战士会，这几天也是招揽人物，短短几天时间人员又是上千人之多。
<br/>　　而军部打下来的几个据点和其余势力打下的几个点也都彻底的保护了下来。
<br/>　　转眼的时间，七天就过去了。
<br/>　　这七天林峰就三点一线的跑，要么黑龙镇要么老龙空要么就是曙光基地。空投也都没放过都是些食物。
<br/>　　而这一日，林峰正在老龙口内呆着，他在想办法，怎么能够把曙光基地内的人给他弄到老龙口来，毕竟若是没人来的话，那么林峰的一切计划都是白费的，这要想赚钱，那首先得有人不是。
<br/>　　不过现在的人似乎除了曙光基地，什么地方都不肯去。要想把他们给弄到一个没有五十米高墙保护的地方，似乎难于登天。
<br/>　　最后，林峰想到了一个点子，要想让曙光基地的人放心他林峰的地盘，那么只有一条路可以走，那就是要让所有人知道他林峰的实力，知道这是他林峰的地盘，哪怕没有五十米巍峨的高墙，也足够可以保护他们的安全。
<br/>　　林峰想到此，就直接要动身前往了曙光基地。刚这样想着。
<br/>　　忽然刷刷刷。
<br/>　　就是一根箭射了过来，直接射翻了李广的一个小弟，这个小弟足足四阶实力，可直接被这跟箭给爆头了。
<br/>　　“有人偷袭！”
<br/>　　李广的小弟反应很快顿时大喊道。
<br/>　　然后李广就和寡妇跑了出来。寡妇还在那里穿裤子，李广还在那里系皮带，这两人光天化日的，也不知道在屋内脱裤子看嘛，莫非是脱裤子斗地主吗？
<br/>　　“峰哥，妈的，有人偷袭我们地盘？”
<br/>　　林峰目光微微眯起，“这可不是偷袭，而是光明正大的来抢地盘来了。”
<br/>　　在他视线之中，来的人不多，也就那么七八个，这七八个人根本没有任何要躲藏的意思，就这样笔直走来，其中五个男的两个女的，其中一个女的还背着弓箭，刚才就是她射的箭。
<br/>　　“林峰是吧，今天这里就是你的死期。”其中一个男人眼中杀机毕露。
<br/>　　“你知道我的名字？”
<br/>　　林峰略感以外，“你们是战门的？”
<br/>　　“你无需知道太多，只要知道，我们要拿走你脖子上的脑袋！”那人嗤笑道。
<br/>　　“这小小林峰居然还敢蹦跶，听说实力很强，死在他手上的人不在少数了，我倒要试试看，这小子有多大的实力。”
<br/>　　其中一个大块肉冷笑着就是走了过来，看都不看李广其余人，仿佛在他眼中，李广等人就是不值得一提的狗屎一样。
<br/>　　他对林峰勾了勾手指头，“林峰，你我单挑，我让你三招。”
<br/>　　“单挑？”
<br/>　　林峰呵呵一笑。
<br/>　　“在这？”
<br/>　　“没错，怎么，我敢到你的地盘找你单挑，你要玩阴的也没问题，不过就那边那几个废物，怕就算是偷袭，也根本伤害不了我。”
<br/>　　“你特么说谁废物呢。”
<br/>　　李广顿时暴怒道。
<br/>　　“区区五阶，你不是废物谁是废物？”
<br/>　　大块头扫了他一眼不屑道。
<br/>　　“你……”李广还要说什么被林峰给阻拦了。
<br/>　　林峰冷笑的看着大块头，“看来你对自己很自信。”
<br/>　　“林峰，少跟我装逼了，我知道你的实力，不就是八阶嘛，呵呵，不要以为八阶就很厉害了，我也是八阶！”
<br/>　　大块头说着就放出了自己一股无形的气势，气势看不见摸不着，但就是可以感觉的到。
<br/>　　这一刻，这个大块头就好像是一个张牙舞爪，龇牙咧嘴的怪物一样。
