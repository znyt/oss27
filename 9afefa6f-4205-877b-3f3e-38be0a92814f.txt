<br/>    当然，对于九九玄功的领悟，卓青瑶还是不可能比得上陈明的。
<br/>    
<br/>        其他三个亲传弟子看向陈明，果然啊，只要学到了师父的一招半式，就足以在整个大陆横着走了。
<br/>    
<br/>        刚刚来到御剑宗就收获了这样的神兵，陈明对于御剑宗的兴趣也是越来越大。
<br/>    
<br/>        陈明看向御剑宗的山门，准备踏入其中。
<br/>    
<br/>        就听见风妖老祖笑道：“你以为御剑宗是那么好进的吗？”
<br/>    
<br/>        陈明回头问道：“怎么了？”
<br/>    
<br/>        风妖老祖道：“守护御剑宗的便是我，我有多强，你可以尽情的发挥自己的想象，那么真正的御剑宗山门，想要进入，更是难比登天。当年御剑宗将要覆灭之时，御剑宗的宗主曾经设下了一道禁制，在山门之后，有着九块石碑，便是御剑宗三百三十三种御剑之术其中的九种，要在看到石碑之后的一个时辰之内施展出这九种御剑术，才能通过，否则，就会触发御剑宗的护山大阵，万剑诛心之阵。除非是当年御剑宗九峰都有后人存在，不然，根本不可能打开这御剑宗的禁制，进入其中。”
<br/>    
<br/>        陈明听了一番，笑道：“原来只是施展九种御剑术而已，听起来也没有多么难吗？”
<br/>    
<br/>        风妖老祖哈哈大笑道：“曾经古仙门也不是没有派人来这里，试图打开这禁制，不过却都是铩羽而归，最多只能进入九人，而且每个人还只能进入一次，若是进入第二次，便会触发这万剑诛心之阵，他们不仅要选择九位天赋非凡的天才，只需要一个时辰便能领悟御剑术的天才，这样的天才，别说是九位了，就是一位，也难以找出。”
<br/>    
<br/>        “所以，能够打开这御剑宗山门的，只有御剑宗的后人，不过，御剑宗的后人早就已经死绝了。不然，你以为我为什么愿意将这御剑宗给你，就是给你，你也进不去！”
<br/>    
<br/>        陈明笑了笑道：“原来只是九个时辰参悟九种御剑术，这难度，也太低了。”
<br/>    
<br/>        陈明道完，直接踏步进入了其中。
<br/>    
<br/>        风妖老祖接着道：“失败的下场可是万剑诛心，知道什么叫做万剑诛心吗？杀人，还要诛心！就是古仙门的老君，也未必敢对抗这万剑诛心。听那些老君说，他们都是准备在自己寿元将近之时，再来这御剑宗试试，若是能够破开这禁制，也算是给他们后人留下了一份造化，就是老君，也是将死之时才敢来，看在你将定风珠还给了我的份上，现在退开，还来得及。”
<br/>    
<br/>        陈明看向风妖，认真道：“不是我说，风妖，你好歹也是一方君主，但是你眼中的天下，也太小了。”
<br/>    
<br/>        陈明踏步，站在第一层石阶之前，一块石碑出现在石阶之上，陈明扫了一眼，开头便是一句话：天下剑道，半道我手！
<br/>    
<br/>        天下若有一剑，我当执掌其半！
<br/>    
<br/>        这御剑宗的口气，还真是相当的大啊，不过，我喜欢！
<br/>    
<br/>        陈明看向这上面所写的御剑术：驱风剑。
<br/>    
<br/>        所谓御剑术，便是飞剑的驾驭之法，陈明看过一眼，师德还有上万，够了，一千师德消耗掉，手中万道天竹一扫，一柄长剑从拂尘之中飞出，赫然便是驱风剑！
<br/>    
<br/>        这一刻的陈明，已经是人剑合一，驱风剑驾驭狂风，掀起一阵风尘。
<br/>    
<br/>        风妖老祖看向陈明，风依旧在吹，他本就没有人形，也看不出他的表情，若是他是人形，不知道脸上会是什么吃惊的表情。
<br/>    
<br/>        陈明继续踏步，再上一阶，万道天竹再次挥舞，雷霆奔涌，一柄长剑自万道天竹之中飞出，落在地面上，赫赫雷霆之威不绝于耳，这时的陈明，再次人剑合一！
<br/>    
<br/>        陈明继续向前，如同闲庭散步，只是看上一眼，下一刻，便是已经直接人剑合一，万道天竹一扫，一种御剑术已经是施展而出。
<br/>    
<br/>        剑道境界，先是凝聚剑气，之后便是剑心通明，然后便是人剑合一，人剑合一之后，劈出一剑道境横生，便是剑意。
<br/>    
<br/>        人剑合一有多难，看卓青瑶就知道，其他三位亲传弟子可是知道卓青瑶历经了多久的苦修。
<br/>    
<br/>        卓青瑶自然更是清楚。
<br/>    
<br/>        而现在，陈明一步便是一次人剑合一，而且还是不一样的剑法的人剑合一！
<br/>    
<br/>        九步踏完，九次人剑合一，陈明此刻，已经是站在了御剑宗之内，并没有什么万剑诛心之阵，这只能说明一点，御剑宗的禁制，已经破了。
<br/>    
<br/>        狂风在呼啸，风妖老祖亲眼见证了一个人九次人剑合一，而且还是在一瞬之间，“世间，竟有悟性如此恐怖之人.......莫非是，先有此人，再有剑道不成？怪不得巨阙认可了此人！”
<br/>    
<br/>        四位亲传弟子亲眼见证了这一幕，对于陈明的崇敬，可谓是已经深到了骨子里。这腿是真的粗，粗的有点抱不住了！
<br/>    
<br/>        陈明回头，看到四个亲传弟子现在眼中疯狂地冒着小星星，嘿嘿嘿，主角又怎么样，见到我陈仙师，还不是得纳头就拜！
<br/>    
<br/>        凌仙看向李素衣，“要不是他是我师父，我有点想再拜他为师一次。”
<br/>    
<br/>        李素衣则是在琢磨，为啥上一世没有遇到陈明呢，甚至连陈明的名字都没有听说过呢？难不成，陈明早就预料到了上一世将会重复，现在这一世才是真正的盛世，所以压根懒得出现。
<br/>    
<br/>        对此，李素衣也只能说，“深有同感！”
<br/>    
<br/>        陈灵则是甜甜的一笑，眼里全是小星星，“师父好帅！”
<br/>    
<br/>        卓青瑶横了三个没出息的小家伙一眼，“瞧你们那点出息！走了，去看看这御剑宗，哦，不，我们燕山还有些什么好玩的东西。”
<br/>    
<br/>        卓青瑶走到山门前的石碑前，抽出背后的巨阙，一剑铲下一层石块，而后在平整的石块上，落下金钩铁划的两个字：燕山。
<br/>    
<br/>        陈明看了一眼，嗯，这写字的水平，相当的不错，比陈仙师强多了，以后写字的事就全部交给卓青瑶了吧，嗯，就这么愉快的决定了。
<br/>    
<br/>        绝不是我陈仙师想要偷懒啊，实在是字太丑，拿不出去。
<br/>    
<br/>        自今日起，御剑宗归于燕山，燕山大宗主，卓青瑶亲笔手书。
