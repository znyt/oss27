        金钥匙关系到大哥傅夜沉和连华生的生死，这两个歹徒是来抢金钥匙的话，他们抢到金钥匙后，横竖都会杀了她。
<br/>
<br/>        既然最后的结果都是死，那么她就信守承诺地去死。
<br/>
<br/>        苏漫雪见童筱颖一心求死，连忙劝道：“童筱颖，快说，那把金钥匙在哪里？把金钥匙给他们！”
<br/>
<br/>        “……”童筱颖紧咬牙关，不再看苏漫雪一眼。
<br/>
<br/>        “那你先说！不然我杀了你！”挟持着苏漫雪的男人，将匕首又落回苏漫雪的脖子上。
<br/>
<br/>        苏漫雪感觉到脖子上的那抹冰凉后，顿时一动也不敢动了，一回想起这匕首上有毒，吓得支支吾吾地说道：“在、在……在我的……我的那个……行李箱里！”
<br/>
<br/>        “去！把它找出来给我！”男人拿刀挟持着苏漫雪，往门外走去。
<br/>
<br/>        童筱颖现在一点脱身的法子也没有，早上起来，她就把手机落在了卧室里。而且，现在，她身上没有任何东西可以用来防身。
<br/>
<br/>        “你确定你不说？”拿枪指着童筱颖额头的歹徒再次问道。
<br/>
<br/>        童筱颖偏过头去，紧闭双唇。
<br/>
<br/>        歹徒突然大步上前，一手反剪住童筱颖的双腕，推着她往厨房里走去。
<br/>
<br/>        厨房里有个小水池里养着当天的新鲜活鱼，歹徒直接将她拽到水池旁，再次疾言厉色地质问：“你到底说不说？”
<br/>
<br/>        童筱颖默不吭声。
<br/>
<br/>        歹徒拿着手枪的那只手，直接揪住了童筱颖的头发，将她往池子里的水面上摁。
<br/>
<br/>        水面上的动静，惊得小池子里鱼四处逃窜。
<br/>
<br/>        “还是不说，是吗？好！我看你能忍多久！”歹徒凶神恶煞道，下一个瞬间，便将童筱颖的脑袋往水池里摁。
<br/>
<br/>        童筱颖被这歹徒压得动弹不得，带着鱼腥味的水瞬间充满呼吸道与气管，缺氧的窒息感让她痛苦地摇了摇头，无声地呜咽。
<br/>
<br/>        就在她觉得自己即将因为缺氧而晕过去的时候，那歹徒又将她的头拿出水面。
<br/>
<br/>        童筱颖满满地吸了两肺的空气，呛得直咳嗽。
<br/>
<br/>        “说不说？”歹徒不耐烦地问。
<br/>
<br/>        童筱颖继续咳嗽，其他话，她什么也不想说。
<br/>
<br/>        歹徒再次将她的头摁入水中。
<br/>
<br/>        这歹徒的力气很大，她一个女人压根就无还手之力。
<br/>
<br/>        童筱颖连咳嗽带呛水，更多的池子里的水灌进了喉咙，眼睛惊恐地往外凸，耳朵因为入水过深而胀痛。
<br/>
<br/>        开始她是瞬间的疼痛，最后就成了尖厉的耳鸣。
<br/>
<br/>        当童筱颖快要陷入昏迷的时候，脸上还带着绝望呐喊的神情，头皮一阵生痛，这歹徒拽着她的头发，又将她从水里给拉了出来。
<br/>
<br/>        “既然不肯说，那么你去死吧！”
<br/>
<br/>        “嘭——”地一声。
<br/>
<br/>        歹徒语毕后，直接朝童筱颖开了一枪。
<br/>
<br/>        童筱颖吓得惊魂未定，脸色惨白得如同死尸。
<br/>
<br/>        然而，她只听到了枪声，却未感觉到自身有任何疼痛感。
<br/>
<br/>        歹徒将枪收好，直接放开了童筱颖，二话不说地转身离开了厨房。
<br/>
<br/>        童筱颖猛地回过神来，发现刘婶就躺在离自己身旁的不远处，连忙爬过去查看刘婶的情况。
<br/>
<br/>        幸好，刘婶只是被人打晕了过去，并未大碍。
<br/>
<br/>        童筱颖心里松了口气，跪坐在地上，将刘婶的上半身抱入了怀中，轻轻地摇了摇，唤道：“刘婶、刘婶，你醒醒！醒醒啊！”
<br/>
<br/>        而此时，耳畔突然响起了一阵掌声和脚步声。
<br/>
<br/>        童筱颖循声望去，只见连华生薄唇微扬，一边拍着手，一边朝她走过来。
<br/>
<br/>        连华生微笑着赞叹道：“你果然信守承诺！”
<br/>
<br/>        所以说……
<br/>
<br/>        刚刚的那一切，都是假的喽？
<br/>
<br/>        童筱颖恍然大悟后，突然间肚子又开始痛了。
<br/>
<br/>        只不过是为了证明她是不是信守承诺，居然拿她的性命开玩笑！
<br/>
<br/>        童筱颖轻轻地将刘婶放了下来，等到连华生朝她走近的时候，她使出浑身所有的力气站起身来，猝不及防地扬起手，直接掴了连华生一巴掌。
<br/>
<br/>        这“啪”地一声巴掌声，比刚刚那“嘭”的枪声还要响得透彻。
<br/>
<br/>        连华生被童筱颖这一巴掌打得一脸懵然。
<br/>
<br/>        童筱颖瞪着连华生，呼吸间浅而急促，极其不舒服地按住腹部。
<br/>
<br/>        她吃痛地用贝齿咬住下唇，努力让自己站直身子，然而双腿却在隐隐地颤抖。
<br/>
<br/>        为了给他这一巴掌，她耗尽了最后一分气力，慢慢地向冰冷的地面跌落
<br/>
<br/>        连华生见状，大步上前，伸手接住了童筱颖虚弱的身子。
<br/>
<br/>        童筱颖只觉自己腹部里那强烈的绞痛袭来，眼前渐渐发黑，周遭眩晕而狂乱，苍白的嘴唇微微干裂，呼吸再次变得急促了起来。
