　　“嗯？这么说你之前遇到过我战门成员？”
<br/>　　说者无意闻着留心，他们三人是根据追魂索命蛊的才是一路追了过来，遇到了这个尸魁老者，虽然在对方身上察觉不到追魂索命蛊的气息，但是追魂索命蛊也不是无敌的，有些强者可以隐蔽上面的气息。
<br/>　　所以他们三人刚才也不确定这个尸魁老者是不是他们要追杀的目标。
<br/>　　可是眼下，这个尸魁老者一句话，那就等于承认了，说他之前见过战门成员。
<br/>　　他们战门成员可不是什么大白菜，随随便便就可以遇到的。
<br/>　　“看来就是你个老匹夫杀了我们战门成员！”
<br/>　　三人顿时眼中杀机毕露。
<br/>　　“杀你们战门成员？”
<br/>　　尸魁老者糊涂了，刚才那小子不是活得好好的，怎么就杀了他了。
<br/>　　他虽然有点疑惑，但是三个战门成员可不给他时间，而是三人联手朝他冲了过来，三人根本没有任何的手下留情，一出手就是拿出了自己的绝杀。
<br/>　　“岂有此理！你战门还真是欺人太甚！当真以为老夫是好惹的嘛！？？？？’
<br/>　　尸魁老者气的暴跳如雷，太气人了，自己好歹也是个前辈也是个七阶的，这群六阶的娃娃，居然就把他当泥巴捏的。
<br/>　　之前那个追着他打，眼下又冒出三个。
<br/>　　“前辈！”
<br/>　　也就在这时。
<br/>　　哒哒哒哒哒。
<br/>　　一辆直升机飞了过来，然后从上面下来了几个人。
<br/>　　尸魁老者看到他们顿时眼睛一亮，“你们这群人来的正好，还不出手？”
<br/>　　这从直升机上面下来的几人不是别人，正是王梅的手下，寡妇。在寡妇身后就是嗜血绅士和坦克，这都是他们的代号，他们曾经都是中东的雇佣兵。
<br/>　　自从地鼠去僵尸墓里找千山血莲失败，他们调查曹龙之死也失败，就寻思着来找王家背后的组织尸傀门，希望他们能够帮忙出手调查一下。
<br/>　　尸魁老者就是他们之间的联络人。
<br/>　　结果刚来到这就看到有人对尸傀门围殴。
<br/>　　几人虽然不知道发生了什么，但是尸傀门是王家的仰仗，欺负尸傀门就等于要灭他王家，他们几人毫不犹豫就是开枪射击。
<br/>　　哒哒哒哒哒。
<br/>　　步枪的子弹威力极大，虽然千年僵尸不怕这个玩意，但是可不代表人这个血肉之躯不怕。
<br/>　　三个战门成员立即面色一变纷纷闪躲。
<br/>　　“玄冰盾！”
<br/>　　其中一人立即是放出一面寒冰盾牌。将那些子弹全部挡住。
<br/>　　“好家伙，这群人也是组织之人？”
<br/>　　坦克目光微微眯起。
<br/>　　“别管他们到底是谁，反正是我们对手，出手。”寡妇道。
<br/>　　“知道。”
<br/>　　坦克二话不说就是几枚手榴弹丢了过去。
<br/>　　“你有灵器，我有热武器，看谁狠。”
<br/>　　嗜血绅士则是拿着一把长剑冲了过去。
<br/>　　“找死！”
<br/>　　三个战门成员勃然大怒，其中一人顿时和嗜血绅士交战起来。
<br/>　　嗜血绅士和他实力都是六阶，两人斗了个旗鼓相当。
<br/>　　“哈哈，战门的杂碎，你们厉害又如何，别忘了，你们只是区区外门成员罢了！还真以为天下无敌？”
<br/>　　尸魁老者冷笑道。
<br/>　　“前辈你没事吧。”
<br/>　　寡妇来到他身边问道。
<br/>　　“无碍，只是需要一些女人来补充下消耗的元气。”尸魁老者桀桀阴冷道。
<br/>　　“这个前辈放心，到时候我们王家肯定给您送十个处女过来。”寡妇道。
<br/>　　尸傀门保护他们王家，作为回报，王家也要满足尸傀门的一些要求，因为他的异能有些副作用，必须要处女之血方可以抵消副作用。
<br/>　　女人是极阴之体，处女更是其中的极品。所以这么多年来，王家一直给尸傀门提供极品。
<br/>　　“除了处女之外，我还要丧尸，最起码五十头！我你们王家要的试验品，被这群王八蛋给破坏了！只能重新开始炼制。”尸傀门老者冷声道。
<br/>　　而这个时候林峰才来到这里。
<br/>　　顿时看到一个直升机和一群人在那里大战。
<br/>　　“这尼玛什么情况，怎么忽然来了这么多人，忽然就干起来了。”
<br/>　　林峰完全不知道，这群人干起来，还都是因为他。
<br/>　　“那几个家伙看起了似乎有点熟悉啊。”
<br/>　　林峰目光落在了三个年轻人身上。
<br/>　　“对了，他们的衣服和我之前杀的那个战门的家伙有点相似，气息也有点相似。”
<br/>　　林峰恍然大悟。气息是因人而异的，有阴柔，有阳光等等各式各样的气息。
<br/>　　当然普通人的气息都差不多。
<br/>　　但组织中人的气息就不一样了。
<br/>　　比如说这个尸魁老者简直就是个老阴逼，一看就不是什么正经人，给人的气息就是邪恶的。
<br/>　　有的人修炼的功法不同，内力不同，导致释放出了的气息也是不同。
<br/>　　“这几个机会莫非也是战门成员？难道他们是来追杀我的？那个什么追魂索命蛊？”
<br/>　　林峰一下子想到了关键。
<br/>　　“既然如此，那我就看好戏，等他们打个两败俱伤我再上。”
<br/>　　场中。
<br/>　　尸魁老者目光阴冷的看着场中。
<br/>　　“你的手下不行了。”
<br/>　　寡妇瞧去，也是眉头皱起。
<br/>　　他几个手下都是雇佣兵，身手很强的，但是却也依然敌不过对方。
<br/>　　“前辈，现在都21世纪了，科技改变世界。”
<br/>　　寡妇忽然一笑。
<br/>　　然后对头顶上的直升机做了个手势。
<br/>　　顿时。
<br/>　　直升机上出现一个人，驾着一把火神炮就是瞄准了战门三人。
<br/>　　哒哒哒哒哒。
<br/>　　瞬间火神炮怒吼，几千发子弹倾泻而出，直接把一些树木全部射到了。
<br/>　　“什么？”
<br/>　　三个战门成员面色大变，其中一人再次丢出了玄冰盾。
<br/>　　然而仅仅是坚持了两秒钟，咔擦，那盾牌就直接裂掉了。
<br/>　　然后整个人被打成了马蜂窝，倒在了血泊之中。
<br/>　　在绝对的强大火力面前，哪怕是他们也根本无法阻挡。除非他的实力已经达到了极其强大的地步，不过很显然，这群人不过外门成员罢了，还没那么强大的实力。
<br/>　　“不好！”
<br/>　　其余两人面色大变。
<br/>　　然后更多的子弹磅礴而来。
<br/>　　“拼了！”
<br/>　　两人怒吼着就是各自拿出来自己的绝招。
<br/>　　“不好，，他们要同归于尽！！”
<br/>　　尸魁老者面色大变立即是朝旁边跑。
