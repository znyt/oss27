　　茫茫森林中，一颗巨树顶端，两名老者盘坐其上，正是内院两名长老，袖袍随风轻轻摆动，看上去极其玄妙。
<br/>　　忽然，两人同时张开双眼，面面相觑，眼中都带着一丝不可置信。
<br/>　　苏长老微微一笑，道：“今年，琥老头特招进学员的这两名新生，可真是格外的有意思啊...”
<br/>　　“...不过，叫慕清霄的小家伙，估计隐匿了修为，实力恐怕能排进内院前十，连我们都看不出一丝端倪，身上肯定有特殊的宝物。”
<br/>　　说着，庆长老又道：“不过，其余的新生，以至于新生前五，表现都有些差劲啊，才两个时辰，火能都被抢的差不多了。”
<br/>　　“这一届火能猎捕赛，估计无人能挡住这两个小家伙，内院的这群小子，估计会栽的很惨啊！”
<br/>　　苏长老轻笑一声，笑声中充满了玩味，同时眼中闪过一丝赞赏。
<br/>　　“这一届猎捕赛估计不会持续多久，继续看下去吧。”
<br/>　　说着，两位长老对视一眼，皆是一阵轻笑，旋即再度闭上双眼。
<br/>　　...
<br/>　　...
<br/>　　寂静的森林，枝叶迎风抖动，带起一阵沙沙声响。
<br/>　　在森林深处，五道身影行走在枯叶遍地的地面上，在他们的胸口左上角，都佩戴着一枚刻有塔状图案的徽章。
<br/>　　“真是倒霉，都两个小时了，就遇到三支队伍，其中两支队伍，新生的火能都被别的老生夺走，照这样下去，我们连参加猎捕赛的本钱都赚不回来。”
<br/>　　其中一名老生，忍不住开口抱怨道。
<br/>　　“继续找，新生总共有五十名，他们没有地图，两个小时不可能走出森林，估计在哪里晃悠呢，我们还有机会。”
<br/>　　队伍中的领头老生，皱了皱眉，对抱怨的老生呵斥一声，嘴上虽这么说，但心里也没底。
<br/>　　“队长，你就不觉得奇怪么，在两个小时内，我们居然没遇见老生的队伍，究竟怎么回事？”
<br/>　　“...老生队伍就四支，猎捕赛森林不小，遇不到也很正常...”
<br/>　　忽然，领头老生脚步顿住，抬起头来，只见在数十米外的巨树阴影下，站着两道身影，一名男子，一名女子，显得有些诡异。
<br/>　　“队长，快看快看，是先前的那两名新生，这小子可真潇洒，什么来路啊？”
<br/>　　“我没瞎，不用你来提醒。”
<br/>　　老生语气平淡，眼睛盯着两人，却隐隐觉得有些不对劲，但想到对方是新生，能有什么不对的，估计是自己想多了吧。
<br/>　　于是，他便带着身后四名老生，向着两人走去。
<br/>　　看到老生队伍后，慕清霄与小医仙相视一笑，但都没有离开树下，目光望着一步步走来的老生们。
<br/>　　片刻，当老生来到五米内，慕清霄道：“几位，找了这么久，你们不累吗，不如坐下歇息一会？”
<br/>　　“......你们就是新生口中的特招生吧，将晶卡交出来，可以少受点皮肉之苦，千万别做多余的事情，免得我们下狠手。”
<br/>　　领头老生见两人不逃，虽然觉得奇怪，却也省了许多时间，直接开口讨要火晶卡。
<br/>　　闻言，慕清霄微微摇头，似笑非笑，道：“你们就不觉得，身体使不上劲，斗气也无法凝聚吗？”
<br/>　　“不妙，快退！”话语刚落，领头老生就要退后。
<br/>　　可就在下一秒，瞳孔一缩，骤然色变，因为他发现，自己真的使不上劲了，而且身体软趴趴的，有些发麻。
<br/>　　随后身体踉跄后退几步，一屁股坐在枯叶遍地的地面上，一脸懵逼，另外四名老生亦是如此，脸色都难看极了。
<br/>　　“嘻嘻，十米内的空气中都充满了麻醉散，无色无味，传播在空气中，能够暂时封住斗者的经脉，就算是大斗师，都有中招的可能哦。”
<br/>　　站在身后，小医仙俏皮一笑，扬了扬手中的玉瓶。
<br/>　　听到她的话语，老生们是彻底懵逼了，知道自己等人落入圈套了，在迦南学院中这么久，他们自然知道麻醉散是什么东西。
<br/>　　可麻醉散是四品药物，两个新生怎么会有？
<br/>　　领头老生知道自己等人栽了，坐在地上不在挣扎，至于其他老生依旧不信，面红耳赤的挣扎着。
<br/>　　可任他们如何的尝试，都无法使出斗气，整个身体都发麻起来，难受得紧。
<br/>　　“为何，你们两个没有中招？”
<br/>　　闻言，小医仙不禁翻了个白眼，语气清冷，道：“废话，当然是我们事先服下了解药。”
<br/>　　脚步声沙沙响起，慕清霄来到老生们面前，居高临下，道：“把火能交出来，免得我动手。”
<br/>　　果然，听到火能两个字，老生们的脸色再度变化。
<br/>　　“你们要火能干什么？”
<br/>　　完全理会老生，慕清霄卷起一颗石子，轻轻一掷，伴随着一阵清脆的断裂声，远处的两个树木便被截断。
<br/>　　“你们有十秒的考虑时间，否则就把你们变得与那两颗树一样。”
<br/>　　领头老生一听，就知道后者不是在开玩笑，毕竟火能猎捕赛，只要不杀人，废除手脚却是可以的。
<br/>　　想到这里，五名老生额头上皆是冒出一层冷汗。
<br/>　　轻风拂过，带起一片枯叶，慕清霄上前一步，面无表情，抬脚就向领头老生的膝盖处踩去。
<br/>　　“别动手，我们给你！”
<br/>　　望着慕清霄如常的脸色，领头老生心中皆是一寒，面前这位新生，绝对比他们还狠。
<br/>　　虽然极其不愿意，但总比被废了的好，他们在内院不过是中下的存在，若真是被废了，以后处境，简直不敢想象。
<br/>　　见慕清霄的脚微微一动，老生不敢拖沓，手在纳戒上一抹，一张蓝色的火晶卡就出现在手中，随后递了过去。
<br/>　　接过火晶卡，慕清霄瞥了一眼其上的数字，总共有四十七点，其余老生稍微少了一些，不过，收获已经算不错了。
<br/>　　微微点头，慕清霄也取出黑色火晶卡，贴在一起，轻轻一搓，就将上面的数字划到自己火晶卡上。
<br/>　　当五张火晶卡全部划完，算上先前掠夺的两只队伍，黑色火晶卡上已经有着六百一十的字数。
<br/>　　六百一十的火能，哪怕对于九成老生来说，都是一笔巨款。
<br/>　　“仙儿，把火晶卡给我。”
<br/>　　微微颔首，小医仙就将火晶卡取出来，递了过去。
<br/>　　拿着火晶卡，轻轻一划，小医仙的火晶卡上的数字‘五’就变成了四百零五。
