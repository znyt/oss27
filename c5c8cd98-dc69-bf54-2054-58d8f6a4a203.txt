　　她做梦也没有想到，
<br/>　　出卖自己的那个人，竟然会是一直被她当做亲哥哥一般的冥沧海！
<br/>　　“为什么？冥沧海！我一直把你当做我的亲哥哥看待，你为什么要出卖我？”
<br/>　　冥沧荨双眼通红，不由得大声质问起来。
<br/>　　冥沧海冷冷一笑，
<br/>　　“为什么？你还问我为什么？冥家所有人都知道我喜欢你，可是你呢？却一直对我视而不见，我让爷爷替我向你提亲，也被你狠狠拒绝！”
<br/>　　“我知道，在你的眼中，我冥沧海不过就是一个下人的儿子，哪里配得上你这样的天之娇女？”
<br/>　　“既然我不得到你，那我就毁了你！”
<br/>　　冥沧海状若癫狂，犹如疯了一般，发泄了一通之后，又忽然桀桀一笑，
<br/>　　“不过，你放心！大公子已经答应过我，不会立刻杀了你，至少再你死之前，我会让你体验一下作为女人的乐趣，桀桀桀~”
<br/>　　轰隆！
<br/>　　冥沧荨听到这番话，顿时娇躯一阵巨颤，险些一头栽倒在地。
<br/>　　就在这时，那位冥家的大长老，也突然开口了，似笑非笑的望着金老，
<br/>　　“金兄，我知道你修为高深，可是如果我和火云天君联手，你自信有几分把握呢？”
<br/>　　“我……”
<br/>　　金老听到这话，脸色顿时煞白无比。
<br/>　　眼前的这两个人，无论是冥家大长老，还是万杀门的火云天君，实力都不在他之下，若是两人联手，他连一成胜算都没有，几乎肯定会被对方当场轰杀！
<br/>　　大长老看到金老这幅表情，知道他已经动摇，继续又道：
<br/>　　“金兄，识时务者为俊杰！只要你立下誓言，今后支持大公子上位，你以后依然是冥家的客卿长老，甚至连我这个大长老的位置，都可以让给你，如何？”
<br/>　　“这……”
<br/>　　金老听到这话，顿时开始纠结了，
<br/>　　诚如之前冥沧山所说，他本就不是冥家的人，不过是冥家请来的客卿长老，受冥家老家主的命令，来保护冥沧荨的安全。
<br/>　　当前这个局势下，倘若他继续坚持，不但救不了冥沧荨，还会搭上自己的老命，实在太不值得了！
<br/>　　一念及此，金老回头望了一眼冥沧荨，脸上露出一抹歉意，
<br/>　　“沧荨小姐，对不起了！”
<br/>　　说完这话，金老脚下一顿，直接跳出战圈之外，
<br/>　　“今日之事，老夫就当没有什么也没有看到，你们请便吧！”
<br/>　　冥沧山见状，顿时哈哈一笑，
<br/>　　“好！金老果然识时务！”
<br/>　　说到这里，冥沧山又望向另外那几名护卫，
<br/>　　“连金老都退出了，你们几个还要坚持吗？”
<br/>　　哗啦啦！
<br/>　　五名元婴高手护卫，不由得对视了一眼，当即纷纷将手里的兵刃丢到了地上，然后迅速退让到了一旁。
<br/>　　蹬蹬蹬！
<br/>　　看到这一幕，冥沧荨再次一脸倒退了数步，面如死灰，身体摇摇欲坠。
<br/>　　很显然，接二连三的背叛、打击，让冥沧荨已经处于了崩溃的边缘。
<br/>　　就连叶尘，看到这一幕，也不由得微微摇头叹息，
<br/>　　难怪前世的冥沧荨，后来会变成一个杀人不眨眼的大魔头！
<br/>　　任谁经历过这种背叛，心里都不会太好受吧？
<br/>　　虽然叶尘并不知道，后来的冥沧荨是如何活下来的，但必然经历了许多磨难，这才导致她从一个阳光可爱的小女孩，成为令整个修真界都闻风丧胆的冥帝！
<br/>　　不过，既然叶尘遇上了，自然就不会让这种悲剧再发生。
<br/>　　一念及此，叶尘走上前，伸手拍了拍冥沧荨的肩头，冲着她微微一笑，
<br/>　　“放心！有我在，我不会让他们伤你一根寒毛！”
<br/>　　众人看到突然站出来的叶尘，顿时全都一愣，继而轰然大笑，
<br/>　　“我没有听错吧？一个只有元婴境四重的小子，竟然也敢口出如此狂言！”
<br/>　　“这家伙怕不是个傻子吧？”
<br/>　　“真是笑死我了！”
<br/>　　……
<br/>　　众人仿佛听到了世上最好听的笑话，纷纷嗤笑起来。
<br/>　　这也难怪，先不说三位化神天君在此，即便是其他人，除了冥沧海只有金丹境巅峰之外，其他随便挑一个人出来，都是元婴境的实力，即便是众矢之的的冥沧荨，也有元婴六重啊！
<br/>　　大公子冥沧山不由得眉头一皱，冲着一旁的冥沧海道：
<br/>　　“这小子是谁？好像不是我们冥家的人吧？”
<br/>　　冥沧海瞥了叶尘一眼，轻蔑一笑，
<br/>　　“他呀，不过就是一个路人罢了！不值一提！”
<br/>　　冥沧荨也没有没有想到，在自己众叛亲离，所有人都背叛她的情况下，唯一一个站出来支持她的人，居然是这个才刚刚认识不久的叶尘！
<br/>　　“叶公子，谢谢你的好意，不过你还是快点走吧！”
<br/>　　感动归感动，但冥沧荨也不认为，叶尘能帮到自己，于是果断拒绝。
<br/>　　叶尘却没有动，脸上依旧挂着淡淡的笑容，道：
<br/>　　“那个，沧荨，其实我的实力还挺不错的，如果你愿意的话，我可以帮你们把这些人全杀了！”
<br/>　　哗！
<br/>　　叶尘这话一出口，周围先是经过片刻的寂静之后，继而再次哄笑起来，而且比之前笑的更加热烈。
<br/>　　“哈哈哈！你们听到了吗？这小子说，可以把我们全都杀了！”
<br/>　　“哇，我真的好害怕哦！”
<br/>　　“看来还真是个傻子！”
<br/>　　……
<br/>　　众人再次纷纷嗤笑起来，尤其以冥沧海笑的最为夸张，
<br/>　　“冥沧荨！你是不是很感动？这小子为了你，竟然连死都不怕呢！”
<br/>　　叶尘缓缓抬起头，瞥了一眼冥沧海，眼眸中闪过一抹寒色，
<br/>　　“悖主弃义！该杀！”
<br/>　　说完这话的同时，叶尘虚手一抓，
<br/>　　轰！
<br/>　　下一刻，
<br/>　　冥沧海忽然感觉身体完全不受控制，似乎被一股难以抵抗的伟岸力量擒住，整个人不由自主的就飞了出去。
<br/>　　冥沧海顿时心中大骇，刚想要爬起来，叶尘一抬脚，便踏在了他的后背之上，令他动弹不得。
<br/>　　“大公子，救我！！”
<br/>　　冥沧海此刻已经来不及去思考，对方为何可以如此轻易将他擒住，强大的危机感，让他第一时间向后方的冥沧山求救起来。
<br/>　　冥沧山顿时一愣，显然也没有反应是怎么回事。
<br/>　　其实不光冥沧山，就连其余三位化神天君境界的强者，也都没有反应过来，叶尘是如何出的手？
<br/>　　不过，等他们的目光，聚焦到叶尘的身上，发现他的确只有元婴境四重的修为，脸色再次露出轻视之色。
