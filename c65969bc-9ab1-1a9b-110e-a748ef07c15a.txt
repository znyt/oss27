　　“嗯？”
<br/>　　元始意识观看着这些五颜六色的一团团规则本源，它们交错混杂，你中有我，我中有你，甚至很难说清有多少团本源。
<br/>　　因为有些本源，是彼此交错的。
<br/>　　如“五行”与“金”、“木”、“水”、“火”、“土”这六大本源就有许多重合。
<br/>　　再如“天罚”与“雷电”，这两条本源也有许多重合。
<br/>　　其中，许多本源都让他感到熟悉亲切。那是他已经参悟透的法则本源。
<br/>　　白发原始并没有多管，他直接找到了如同一张大网，包裹着整个神秘空间，闪烁着黑白色的因果本源与闪烁着电光与无尽威严的银白色的“天罚本源”。
<br/>　　他此次的目标便是这两团本源。
<br/>　　“融入。”
<br/>　　白发原始先将意识融入到因果本源之中。
<br/>　　轰！
<br/>　　浩瀚的讯息冲击自己的意识。它浩瀚、博大、广阔、诡异、无限，它充满了神秘。原始不再多管，迅速刻下自己的灵魂烙印，仅仅数个呼吸的时间，便成功烙印了，这是因为他在因果一道上的积累深厚的原因。
<br/>　　之后原始不停，又耗费一盏茶的时间，在天罚一道本源上刻下自己的灵魂印记。
<br/>　　“呼——”
<br/>　　“还好，没有出现神秘意外。”原始深吸一口气，缓缓道。
<br/>　　“可惜，其他的还不能参悟，一次性不能烙印太多，不然自己的身体会承受不了，毕竟我不是本尊。不过，这两条也足够了，我的道是元始大道，可不是天道。天道可以作为参考，但不能占据绝对部分。”
<br/>　　最后，再次看了一眼两大本源规则以及整个本源空间之后，原始略微有些不舍的离开了这神秘的空间，将自己的意识返回到身体之中。
<br/>　　在原始没有察觉到的本源空间深处，在他走后，一道虚幻的身影飘出，他面容苍老，眼神淡漠，看着原始的意识离开本源空间，眼中闪过一丝异色。之后，他再次消失。
<br/>　　轰隆隆！
<br/>　　一股强大的圣威惊动整个洪荒大地。
<br/>　　天地震动，上至九天，下至幽冥，须臾之间，两道光彩本源于九天之上破空而出，落入白发原始头顶，他的身体一震，一道紫色玄气随即破体而出，三道光华交缠在一起，之后落入元始身体之中。
<br/>　　顿时，一股凌然震撼的恐怖威压，瞬息之间，充盈在整个浩瀚天地。
<br/>　　乾坤倒转，风云失色，苍穹震动，星辰倒垂，星光灿烂。天花乱坠，地涌金莲，一片片的灵气狂涌而来，化作各种异象。金龙腾云，凤凰舞天，麒麟镇世，青龙探爪，白虎撕天，朱雀啸天，玄武覆海……
<br/>　　一时之间，天音飘飘，仙音袅袅，道音震耳，神音繁繁，万花朵朵，馨香四溢，万兽齐鸣，祥瑞之气洒向整个洪荒大地，紫气浩荡，铺满天涯。
<br/>　　广阔的洪荒大地之上，突然演化出条条大道。一时之间，洪荒大地上，草木凡兽，奇珍异兽数不尽的灵物被点开灵智，更有资质逆天的直接得以化形蜕凡。
<br/>　　“鸿蒙初判有声名，炼得先天聚五行。
<br/>　　顶上三花朝北阙，胸中五气透南溟。
<br/>　　群仙队里称原始，玄妙门庭话未生。
<br/>　　漫道香花随辇毂，沧桑万劫寿同庚。
<br/>　　冥冥因果由吾掌，煌煌天罚为我生。
<br/>　　诸果之因元天王，万古至尊第一圣。”
<br/>　　“吾，乃盘古氏玉清原始天罚圣主，今日成圣！”
<br/>　　轰！
<br/>　　一股强大、广博的圣威威凌天下，甚至这威压越来越强，几个呼吸后，突破圣人一重天，来到圣人二重天，一直到二重天巅峰，才缓缓停了下来，不再增长。此时，洪荒之中，凡准圣以下者，全部跪拜伏地，朝着昆仑山方向不停跪拜。
<br/>　　同样的圣威，同样的祥瑞之景，洪荒众生已经不是第一次看到了，上一次是洪荒道祖，那是天地之间的第一位圣人，而如今，第二位圣人出世，结果也没有出乎洪荒众生的预料，是盘古正宗，三清之首，众人大师兄元始成圣。
<br/>　　不过，他们猜对了，同时也猜错了。是三清之首，众人的大师兄成圣了，是天罚圣主原始成圣了，但不是一元复始的元始成圣。
<br/>　　原始与元始虽是同一个名字，也是同一个人的不同分身，但今后它却代表着两个不同的意义。
<br/>　　原始的成圣异象让整个洪荒都陷入了莫大的震惊之中，虽然众人知道早晚会有人成圣，而且也知道会是哪些人成圣，但众修士却没有想到，原始成圣会如此早。如今距离紫霄宫讲道结束才过去区区一百余年，原始这就成圣了，这的确让人异常震惊。
<br/>　　对于原始成圣这件事，自然是有人高兴，有人不愉快。最高兴的当然是他的两位兄弟通天与老子，不高兴的自然是与他有过过节的那些修士了。比如，妖族中的妖帝帝俊、东皇太一、娲皇女娲与羲皇伏羲。
<br/>　　天庭之中，帝俊阴沉着脸，脸色变得极为难看，看着紫霄宫方向，他眼中闪过几丝愤恨与不甘，更有忌惮之色。对于他来说，原始成圣对他可不是什么好事啊。毕竟自己等人与三清有过争斗，如今人家都成圣了，报仇之事，看来是无望了。对于他来说，三清是唯一让他难堪的人，也是他们第一次被人欺负得无法还手。对于不周山上那件事，他或许永远无法忘怀。
<br/>　　“大哥，怎么办？”他的身旁，东皇太一同样阴沉着脸，沉默了一会儿，对着帝俊问道。
<br/>　　“以后不要再提这件事了。三清之恨，我等也应该忘记。”沉默了一会儿，帝俊深深倒吸一口气，淡淡道。
<br/>　　“这怎么可能？我那混沌钟……”太一不甘地大叫道。
<br/>　　“不行也要行，人家都是圣人了，你能有什么办法？除非……”
<br/>　　“除非什么？”太一急声问道。
<br/>　　“除非我们这边也有人成圣。”帝俊眼光闪烁了一下，默默道。
<br/>　　“我们不是有女娲道友吗？”太一不解道。
<br/>　　“二弟，女娲终究是外人，外人终究不可相信啊。谁也不能保证她成圣后还能帮助我们？要知道三清是有三人，而女娲只有一人啊。将来发生什么事，谁也说不清？”帝俊对着太一深深道。他这个二弟什么都好，就是太过容易相信他人了。
<br/>　　“啊？那怎么办？”太一急声道。沉默了片刻，太一道：“大哥，不是还有红云吗？他那鸿蒙紫气可不是道祖亲传的，他也不是道祖的弟子。我们是不是可以……”
<br/>　　帝俊眼神闪烁几下，轻声道：“这件事需要从长计议，现如今还是考虑一下原始成圣这件事。”
<br/>　　帝俊既没有赞成，也没有不同意。显然，在他心中，也早打起了红云老祖身上的鸿蒙紫气的注意。
<br/>　　帝俊将目光投向洪荒大地，眼光深邃，不知道在想些什么。
<br/>　　……
