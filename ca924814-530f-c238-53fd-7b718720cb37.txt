　　紫云洞外，族人们围得水泄不通。
<br/>　　毕竟是三年才一次的盛事，族人们也想看看家族最出色的子弟们，到底有了多大的成长。
<br/>　　“出来了，他们出来了。”一个眼尖的族人高声叫着。
<br/>　　“看他们一个个都笑容满面的，必定是修为大涨。”
<br/>　　族人们七嘴八舌地讨论着。
<br/>　　十位家族子弟的亲人们更是满脸期待地追问着。
<br/>　　“子木，如何了？”
<br/>　　“壮儿，现在到哪一重修为了？”
<br/>　　“....”
<br/>　　“逸儿。”三长老满脸慈爱地看着萧逸。
<br/>　　“三长老放心，逸儿在里面收获颇丰。”萧逸笑着说道。
<br/>　　“那就好。”三长老没有追问萧逸的修为，只是点了点头。对于他来说，萧逸十天前给他的惊讶已经足够了，无论萧逸在紫云洞内收获如何，他都很满意。
<br/>　　然而，在这一片片欢声笑语中，有两人的脸色变得十分难看。
<br/>　　一个是萧若寒，一个是他父亲五长老。
<br/>　　“到底怎么回事？”五长老愤愤地说道，“以若寒你的黄阶武魂，绝对能凝结气丹，怎么会在最后一天忽然吸收不了灵气？”
<br/>　　萧若寒摇了摇头，低声道，“我也不知道，但我感觉，最后一天，是那小废物搞的鬼。”
<br/>　　“不可能。”五长老道，“小废物没有那个本事，他的赤阶武魂还有低下的资质，都远远不是你的对手。”
<br/>　　萧若寒反驳道，“可是，前些天他忽然实力大增又该如何解释？我就不信他真的能在半月内连跳四重，还有他那些古怪的招式。我认为，这其中一定有鬼。”
<br/>　　五长老沉默了一下，沉声道，“先不管这些，你无法突破到后天境，后面的事虽然变得麻烦，但幸好我早有准备。”
<br/>　　“准备？”萧若寒微微疑惑。
<br/>　　“嗯。”五长老阴笑一声，道，“若寒，我问你，以你如今的实力，可有把握赢小废物？”
<br/>　　“自然是有的。”萧若寒肯定道，“虽然我不知道在紫云洞的这些天，他的修为成长到什么地步，但绝对不可能突破到后天境。”
<br/>　　“只要他不是后天境，以我的火云蟒武魂，足以发挥出碾压他的实力。”
<br/>　　萧若寒颇有心计，经常算计他人，倒是猜出了萧逸一些情况。
<br/>　　“好，你有把握就行。”五长老说罢，忽然走到了所有族人面前。
<br/>　　“众位族人，请听我一言。”在深厚的真气支撑下，五长老的声音极为高亢。
<br/>　　族人们热闹的气氛瞬间停止，纷纷看向他。
<br/>　　“这家伙，又想搞什么鬼？”萧逸和三长老同时皱起了眉头。
<br/>　　“族人们。”五长老一副大义凛然的样子，高声道，“所谓家不可一日无主，萧家家主之位空缺已久，家族每况日下，我身为萧家长老，绝不愿意看到萧家日渐衰落。”
<br/>　　“所以，我提议，重选家主！”
<br/>　　五长老的话太突然，让族人们一下子有些反应不过来。
<br/>　　其实，萧家家主之位虽然空缺，但三长老作为代理家主，这些年将萧家打理得极好。不说蒸蒸日上，但却也威风不减当年。
<br/>　　当下有族人质疑道，“萧家家主之位何时有空缺过？家主还是那位家主，只不过如今他不在罢了。而且，五长老你今日作词提议，莫非是你想当家主？”
<br/>　　五长老当即严肃地回答道，“家主失踪已久，名存实亡。而且，为免非议，我现在就当着所有族人的面保证，我绝不会当这家主。”
<br/>　　五长老继续道，“我身为长老，自当以家族为先。萧家现今需要的，乃是一位有魄力的领导者。我虽未老，但年纪已大，早已没了年轻人的雄心壮志。”
<br/>　　族人们道，“难得五长老深明大义，但家主之位何等重要，岂能说选就选，必须从长计议才是。”
<br/>　　“这一点我当然知道。”五长老道，“所以，现在只是为日后之事做准备。而准备之一，便是推选出一位有能力的少家主。”
<br/>　　“少家主？”族人们顿时面露怪异之色，议论纷纷。
<br/>　　一旁的三长老暗骂一声，已经明白了五长老的用意，立刻喝道，“五长老，你不要太过分了，半月前你已经处处为难逸儿。今日，明知逸儿就是少家主，还想重选少家主，你真当我萧重好欺负不成？”
<br/>　　事涉萧逸，三长老立马发怒，一身真气澎湃而出，大有大打出手之势。
<br/>　　五长老似乎早就料到三长老的表现，不慌不忙地说道，“三长老稍安勿躁，萧家家主之位，向来是有能者居之。当年家主他修为冠绝全族，他的儿子萧逸成为少家主也无可厚非。”
<br/>　　“可现在家主他失踪多年，杳无音讯，若还让萧逸当少家主，岂不是对其他家族子弟太过不公平了。若萧逸有能力，少家主之位自当由他来当，可若他无能，那我就绝对不能坐视不管，免得日后他耽误了家族的发展。”
<br/>　　“说到底，我只是不想我们萧家日后在一个无能之辈手上破落掉罢了。”
<br/>　　五长老一连窜话语，说得情深意切，大义凛然。不知道的，还真会以为他是一个为家族殚精竭虑，处处着想的人呢。
<br/>　　“说得倒好听。”三长老反唇相讥道，“逸儿十天前才大败萧子木，他如何没有能力了？”
<br/>　　五长老笑道，“既然三长老认为萧逸有能力，那就检验一下好了。”
<br/>　　“如何检验？”三长老问道。
<br/>　　“很简单。”五长老道，“萧逸必须再次接受所有家族子弟的挑战。”
<br/>　　“挑战？”三长老皱起了眉头。
<br/>　　“不错。”五长老笑道，“少家主之位，必须由最出色的子弟担任。今日，我们萧家的各大天才从紫云洞内修炼出来，一个个修为大涨，雄心壮志，气度不凡。若能从中选出更好的少家主，日后哪怕家主仍旧没有音信，我萧家也不至于一直群龙无首，可尽快挑选出新的领导人。”
<br/>　　五长老一番话，说得天衣无缝，言语中无不表示着为家族考虑，顿时让族人们没了质疑。
<br/>　　当然，从某些角度来看，他今天的提议并没有错。
<br/>　　“看来，不给你点教训，你真的会越来越得寸进尺。”三长老已经压抑不住心头的怒火，当即就要出手。
<br/>　　萧逸一把拉住了他，随即站了出来，高声道，“我接受任何子弟的挑战，依五长老你的意思，何时开始检验呢？”
<br/>　　萧逸很清楚，所谓的所有子弟挑战，根本就只有萧若寒一人。
<br/>　　毕竟，自从十天前自己轻松打败萧子木后，其他子弟已经自觉不是对手。唯有萧若寒，有能力与自己一战。
<br/>　　而且，相信在很多族人心里，也很想知道，到底是萧若寒这个年轻一辈第一人强，还是自己这个忽然崛起的少家主更强。
<br/>　　“爽快。”五长老高声道，“难得萧逸你深明大义，我佩服。你刚刚从紫云洞出来，想必已然劳累，为了能让你发挥出更强的实力，便给你三天时间休息，三天后，家族比武场，接受挑战。”
<br/>　　“哼。”萧逸心中冷笑，“嘴上说得好听，背地里又不知在搞什么阴谋。”
<br/>　　不过，他还是说道，“好，就依五长老你的意思。”
<br/>(本章完)
