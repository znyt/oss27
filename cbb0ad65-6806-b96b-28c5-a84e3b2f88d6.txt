　　“这只丧尸有些奇怪，大家小心。”放下望远镜，楚涵出声道。
<br/>　　超级丧尸中的小头领当然不同凡响，至少也是会和他们之前遇到的所有超级丧尸都不同，更强也更神出鬼没，最重要的是恐怕还有什么不为人知的能力。
<br/>　　但楚涵在这些基础之上，再一次提醒众人小心，足以说明这丧尸是大敌。
<br/>　　并且奇怪这个形容，更极少会出现在楚涵口中，他一向运筹帷幄，连他都觉得奇怪，事情的发展很有可能会超出他们掌控。
<br/>　　众人都将警惕心提升到最高，陈少爷更是将全部心神都放在了那只丧尸身上，随时会射·出第二发绝命狙击枪的子弹！
<br/>　　当下不是最佳的射击时机，这只丧尸在这片范围不慎被陈少爷命中从而擦伤脸部，这时候必然会有所提防，更何况在它不注意的时候都能躲开最致命的攻击，仅仅给它造成了擦伤，可想而知此时有了提防的情况下，想要命中它有多困难。
<br/>　　而被世人称之为神之狙击的陈少爷，更不是一个看到目标看似就在射击范围就会扣下扳机的笨蛋，他的命中率之所以会无限接近于百分之百，还有太多射击方面的细节。
<br/>　　目标的闪躲，速度，防御……
<br/>　　当下的风速，湿度，气压……
<br/>　　射击的捕捉，预判，时机……
<br/>　　有太多的讲究，不懂的外行恐怕会在这时候急性子的跳出来大骂陈少爷为何不开枪，懂的人则是连呼吸都谨慎无比，生怕漏掉任何一个细枝末节。
<br/>　　双方的对峙并没有持续太久，或者更准确的说是狼牙这方的谨慎对峙，那只丧尸则是一切未知。
<br/>　　就在何峰接过楚涵递来的望远镜刚抬起观察的时候，却猛然发现望远镜内远处的那座房屋顶层，已经空无一物！
<br/>　　“消失了！”蒋天庆速度更快，第一时间就出声提醒：“那只丧尸再往基地中心跑！”
<br/>　　由于速度实在太快，在那丧尸离开楼顶的时候，众人的大脑都没能跟上对方的移动速度。
<br/>　　楚涵当然也是第一时间，他的眼神跟着那个身影不断移动，很快就看到那只丧尸停留在了一个丧尸群极多的区域，那里也是迷宫战区，却是新兵还没来得及清理的地方。
<br/>　　黑压压的丧尸群大·片的占据了视野，光是那一块地方恐怕就挤了靠近好几十万的数量！
<br/>　　那只丧尸头领就站在一座房顶上，身姿很明显。
<br/>　　“它想干什么？”蒋天庆没由来的紧张起来。
<br/>　　楚涵的身躯也紧绷，随时准备行动！
<br/>　　基地的扬声器内还在放着大多数人都听起来断断续续的古怪旋律，那些低阶丧尸依然在这种声波中一动不动，迷宫区的新兵群起而攻之，大肆的收割着这些丧尸，战绩傲然。
<br/>　　但下一秒。
<br/>　　“吼——”
<br/>　　一声持续不断的咆哮从那只丧尸头领的口中发出，它大张着嘴朝着下方地面上那些不动弹的丧尸群咆哮着，表情带着一种诡异的愤怒，非常像训练时教官在训人的样子。
<br/>　　它的声音必然不如扬声器响亮，但音波传递很远，嘶吼时它的耳朵在不断颤动。
<br/>　　很快以这只丧尸头领为中心，周围密密麻麻的丧尸群从内而外的开始重新动了起来，从上方看就像是水波荡漾，当然是黑色的水波。
<br/>　　很快那一方区域的丧尸竟然全体脱离了静止不动的状态，扬声器内散发而出的古怪声音对它们不再有效果，从最开始的僵硬行动还有些犹豫和迟缓，渐渐的随着那只丧尸头领的咆哮声响起到后期，这群丧尸竟然就这么逐渐恢复如初！
<br/>　　“吼！”
<br/>　　“吼吼吼！”
<br/>　　它们集体咆哮出声，声音中带着某种愤怒，更是疯狂的朝着原定的方向挤压涌去！
<br/>　　比之前刚涌进迷宫战区时的样子，更为狂暴！
<br/>　　看到这一幕的人都一颗心渐渐下沉，指挥室内的众人更满脸不可置信，那只丧尸头领竟然将这批丧尸唤醒了！
<br/>　　咆哮完的丧尸头领并没有跟着那几十万的丧尸群一同移动，而是回头朝着指挥室看了一眼。
<br/>　　一直用望远镜观察的何峰，猛地心脏漏跳了一拍，他看到了刚才那丧尸头领回头一瞥的眼神，那是……
<br/>　　轻蔑。
<br/>　　是的！那只丧尸竟然露出了人类才有的眼神！
<br/>　　它在轻蔑谁？
<br/>　　它知道这间指挥室内，是主导这场战役的一群人，是针对了尸潮而刻意准备的狼牙参谋部。
<br/>　　所以它在轻蔑狼牙战团所做的一切，都毫无意义？
<br/>　　丧尸头领并没有在原处停留太久，很快就朝着另一处丧尸群的聚集地而去，这样的举动也说明，丧尸头领唤醒的丧尸群，已经不受扬声器内的声音影响，更不需要它时刻注意。
<br/>　　瞬间得出这一理论的指挥室内众人，一下子心急如焚！
<br/>　　“糟糕！丧尸群要重新动起来了！”
<br/>　　“新兵反应很快，但时间太短了，迷宫战区至少还有四百万低阶丧尸！”
<br/>　　“要是都被唤醒……”
<br/>　　正当所有人都大惊失色之中，蒋天庆忽然惊叫出声：“楚涵哥呢？”
<br/>　　众人都被蒋天庆的惊叫声吓了一跳，反应过来后左右一看，此时的指挥室内哪还有楚涵的身影？
<br/>　　而然很快，蒋天庆再次一声大叫：“陈少爷？你去哪？！”
<br/>　　又是他第一个发现情况，还抓了个正着，不愧为敏锐度最强的侦查队队长。
<br/>　　再次被一惊又一吓的众人连忙将视线转移，却发现陈少爷已经快速收拾好了简易装备，手中扛着绝命狙击枪，背上背着大量绝命狙击枪的专属子弹。
<br/>　　他动作不停，准备完毕后立即朝下走，头也不回的回答道：“复命。”
<br/>　　陈少爷没有留在此处解释说明的时间，他必须立即赶到目标的射击范围找到最佳定点，但他没有楚涵那么变·态的速度和体能，还没被蒋天庆察觉就已经追了上去，但一次往下跳两到三米他还是可以做到的。
<br/>　　于是紧接着，陈少爷也快速消失在了众人面前，而听到他之前回复的那两个字，以狼牙战团内部的默契，其余人自然瞬间明白。
<br/>　　复命，复谁的命？
<br/>　　当然是楚涵。
<br/>　　陈少爷这是要跟着楚涵一起上前线去打丧尸头领啊！
