　　第533章
<br/>　　黄老汉表情一怔，万没想到刘发财竟说出这番话。他脸上一苦，随之道：“程当家，到这个关头您就别开玩笑了。谁不知道您是天下贼之首，只要您肯出手，定能吓退群贼。程当家，您可一定要救一救我黄家。”
<br/>　　刘发财哼唧两声，谁也听不清他口中说的是什么。他自己心里道：如果我当真姓程，想要料理几个围庄的山贼应该也不费什么劲，但我是个赝品呐。
<br/>　　“黄老伯不用慌，不用慌，容我想个办法……”
<br/>　　刘发财脑门嗡嗡响，他的确绞尽脑汁在想。可想的不是应对山贼的法子，而是自己脱身的法子。刘发财是有几分机智，但面对今日情景委实是第一次。
<br/>　　“程大雷，你若是没种出来，待老子杀进去，要将黄家满门杀绝。”
<br/>　　“你出来呀，与我兄弟大战三百回合。”
<br/>　　大门外，山贼一声声喊着，声音越来越大，一直传到大厅。黄老汉心如刀绞，脸上本就不多的胡子都快被揪干净了。
<br/>　　庄外，长须鬼盯着紧闭的大门，道：“怎么姓程的还不出来？”
<br/>　　“哼，还不是被吓破了胆，怕了爷爷的大枪，他再不出来，我便杀进去。”年轻气盛的张猛道。
<br/>　　“怕是没那么简单。”饿死鬼朱刚道：“想那姓程的，该不是贪生怕死之人。”
<br/>　　“诸位爷，诸位爷……”
<br/>　　一名小喽啰从外面闯进来：“报告诸位爷，兖州城的官兵杀了过来。”
<br/>　　鹰嘴岭几位当家都是一惊，孙山起下意识道：“莫非是我们这次行动走漏了消息。”
<br/>　　“不会。”朱刚摇摇头：“就算走漏了消息，从兖州赶到这里也不是一时半刻能办到的，怕是有别的缘由。”
<br/>　　方石隆领着兖州兵到了黄土镇，真好与鹰嘴岭的土匪撞个正着。
<br/>　　“怎么会事，不是说山贼只有两个人么？”方石隆。
<br/>　　“这，这……”黄为善也摸不清是什么情况：“小人不知啊。”
<br/>　　“哼，等我料理了这件事再与你算账。”方石隆握紧手中大刀：“来，弟兄们与我杀平了这波山贼。”
<br/>　　两方人马都有些稀里糊涂，但今日既然在黄土镇遇上了，一兵一匪，也就只有厮杀一条路走。
<br/>　　双方举着火把，在黑夜里厮杀一阵，黄土镇的老百姓不敢开门，都窝在屋里挤在墙角，吓得瑟瑟发抖。
<br/>　　毕竟是兖州兵占优势，一路冲杀到黄家的大门前，两方人马对峙。
<br/>　　方石隆骑在马上，小臂中了一枪，此刻血流不止。不过他也算豪横，不去管受伤的小臂，睁大虎目盯着朱刚一伙人。
<br/>　　“哪里来的小贼，见到本将军还不快快束手就缚。”
<br/>　　张猛哇哇大叫，口中道：“老子是鹰嘴岭的大王，想死的，放马过来一战。”
<br/>　　方石隆不是很愿意搭理他，目光扫了一眼黄为善：“程大雷还在你黄家么？”
<br/>　　“在……应该在！”
<br/>　　“好！”方石隆不在意鹰嘴岭几个山贼，看重的还是程大雷：“速速将院子围了，不许放走院子内任何一人。”
<br/>　　兖州兵立刻开始行动，他们包围了半边院子，鹰嘴岭的山贼也同样围了半边院子。兵与匪向来水火不容，然而此刻却莫名其妙达成某种合作。
<br/>　　“程大雷，你已被包围了，识相的快快出来受死。”
<br/>　　这个时候，程大雷和刘发财还窝在大厅内，外面的动静自然传到这里。他们也都知道兖州兵杀来了，但没想到，兖州兵一到，就点名要杀程大雷。
<br/>　　官也要杀，匪也要杀，这就是程大雷如今的处境。
<br/>　　程大雷看了一眼刘发财，拍拍他的肩膀道：“兄弟，人家叫你出去呢。”
<br/>　　“呃……”刘发财也没想到事情搞得这样大，他看了眼程大雷，发现二人身形相貌竟然有五六分相似。
<br/>　　“兄弟，说实话我与你一见如故，咱们撞在一起就是上天注定的缘分呐。这样，你假扮我的模样，出去糊弄过他们。等我躲过这一劫，去天下间宣扬兄弟你的侠义之名。对了，兄弟你叫什么名字来着？”
<br/>　　“刘发财。”
<br/>　　“嗯，刘发财简直就是天下第一大好人呐，绝对不会看着兄弟去死。”
<br/>　　“不不不。”程大雷忙否认：“程大雷才是天下第一好人，为兄弟去死，你义不容辞。”
<br/>　　“不不，兄弟，刘发财才是好人，程大雷简直就是个王八蛋。兄弟，为了我你就去死吧。”
<br/>　　“应该你去死，刘发财才是王八胆，程大雷是那义薄云天，仁义无双的第一好人。”
<br/>　　黄老汉看着二人谦让，道：“二位好汉，你们起码出去一个吧，你们再不出去，他们就要打进来了。”
<br/>　　的确，外面的喊声越来越热闹，山贼和官兵一个比一个骂得厉害。看这架势，最多一刻钟时间，他们就会杀进黄家。
<br/>　　“住口！”
<br/>　　程大雷和刘发财同时回头，喝住了黄老汉。
<br/>　　黄老汉被二人凶神恶煞的样子吓住，一时不敢吭声。
<br/>　　“黄老贼。”程大雷呵呵冷笑：“真当爷爷不知道你的鬼把戏么。你与我说，那官兵如何来的？”
<br/>　　“这……”黄老汉向后退了半步，惊得哑口无言。
<br/>　　程大雷此刻也琢磨过味来，黄为善早晨就出去了，兖州兵这个时候正好到了，简单想一想，就不难发现池中端倪。
<br/>　　“呀！”刘发财惊叫一声：“原来官兵是你引来的！”
<br/>　　程大雷冷笑连连：“黄老头啊黄老头，你打得如意算盘。那山贼和官兵岂是好打发的，一旦双方打起来，你黄家卷进去，怕你这黄家老小都要尸骨无存。可怜啊可怜，大祸当前，你还犹然未觉。”
<br/>　　黄老汉惊得脸色苍白，细想程大雷的话，他越发害怕。
<br/>　　本想靠这次机会，成黄家一步登天的机会，可没料到，事情竟然会演变到如此地步。
<br/>　　是呐，天下大乱，上至庙堂，下至江湖草莽，都想从这场乱世中分一杯羹，就连黄老汉这土财主，便也有了吞天之意。
<br/>　　可是，他不知道，有胆子上这赌桌容易，但如果想离场，却难如登天。·
