　　第547章
<br/>　　无怪程大雷有这种似曾相识的感觉，当年幽州城有个叫罗铁的人，是杨龙停的义子。同样号称幽州第一快剑，也拿了一把铜钱在程大雷面前砍来砍去，结果么……
<br/>　　程大雷不由去想，是不是用快剑的都有随身携带铜钱的习惯。自己的剑法也算不错，是不是也该在身上揣把铜钱。
<br/>　　看着程大雷皱眉思索的模样，钱观鱼冷哼一声，道：“怕了么，若你的剑斩不开铜钱，便乖乖滚下山去。”
<br/>　　这孩子还真是可爱呐……程大雷深吸一口气，挺直了腰杆：“抱歉，我的剑不是用来斩铜钱的。”
<br/>　　“那你的剑可以做什么？”
<br/>　　“我的剑只会杀人。”
<br/>　　“哈哈哈。”钱观鱼仰天长笑：“凭你手中哪把破剑也可以杀人。笑话，你能杀我么？”
<br/>　　程大雷手摁剑柄，这孩子可爱得还真是不好让人下杀手。可话赶话赶到这里，不开杀戒的话，会不会让你小瞧了我。
<br/>　　“抱歉了。”
<br/>　　手摁绷簧，匹夫剑锵啷一声出鞘。空气中只是剑光一闪，钱观鱼甚至没看清楚什么，程大雷已将剑收回鞘中。
<br/>　　随之，几缕黑发缓缓垂落，钱观鱼额头重重砸下一滴冷汗。
<br/>　　“好孩子。”程大雷叹了一声：“幸亏我没有出剑必杀人的规矩，不然你一条命就丢了。你瞧瞧，你运气多好。”
<br/>　　钱观鱼一张脸由白泛红，又羞又恼，一是因为程大雷的剑，而是因为程大雷的话。在程大雷眼里，自己就是个『孩子』，所谓的剑术也不过是孩子的小调皮罢了。
<br/>　　“还不快散开。”程大雷冷喝一声：“没规矩的娃娃，不懂得见大人么。”
<br/>　　程大雷不介意言语重些，什么书院第一，借此眼高于顶，目中无人，是该让他们明白，书院第一并不等于天下第一。也该叫醒他们，免得帝国读书人都这幅臭毛病，窝在屋里穷横。
<br/>　　也算是自己为天下读书人做件好事吧。
<br/>　　钱观鱼当真是无地自容，堂堂七贤十二秀，让人当做娃娃训斥。以后自己还怎么在书院混，师兄师弟怎么看自己，书院的鸡鸭鹅怎么看自己。
<br/>　　“呔，你忒狂了些吧。”
<br/>　　锵啷一声，钱观鱼拔出了佩剑，空中寒光一闪，向程大雷扎来。程大雷立刻退步，钱观鱼这一剑刺空。但仍如跗骨之蛆，紧追着程大雷不放，招招都是要害，步步都是杀机。
<br/>　　程大雷连避七剑，毫发无伤。心中的火气也冒了出来，口中喝道：“呔那娃娃，我不愿伤你，无非怜你年纪轻轻，不是惧了你书院。你要再不知好歹，我不介意送你回炉重造。”
<br/>　　七剑没有讨得半点好处，钱观鱼已知道他和程大雷之间的差距。但以他的年纪，能连环七剑逼的程大雷无法还手，成就其实已经算相当不错。
<br/>　　沉下心来再磨练几年，未必不能问鼎天下第一四个字。
<br/>　　可在此时此刻，他感受到的只有羞恼。退到牌坊下，面带红色，目露杀机。
<br/>　　一队人从书院涌出来，口中喝着：“怎么了，怎么了？”
<br/>　　“布剑阵。”钱观鱼喝道：“与我拿下此贼。”
<br/>　　书院有七十二书生剑阵，七十二柄书生意气剑联在一起，浩然正气直冲云霄，杀威不是一般的盛。
<br/>　　刘发财与楚云生窝在一边，心道：早知如此，直接一路打上山，也省的中间麻烦。
<br/>　　程大雷直面剑阵，看对方布成二十八人的小剑阵，人人白袍，手持佩剑，倒是整齐威风。
<br/>　　程大雷心底的豪横应激而发，仰面笑道：“这就是书院的待客之道么。好好好，就由我来会一会你们的书生意气。”
<br/>　　程大雷也有些烦躁，索性打一顿再说，回头还是看孟夫子有什么女儿孙女。
<br/>　　匹夫剑再次出鞘，在阳光下并不反光，只是剑身上流转着淡淡血红。从出世以来，这柄剑也不知斩过多少高手，以血养剑，早就将一柄剑养成杀器。此刻拿出来，周围的光线仿佛都暗了几分。
<br/>　　“先生，且慢。”
<br/>　　一道声音忽地响起，周围好像安静下来。程大雷看到一白须白发的老者拄着拐杖走过来。
<br/>　　“夫子。”
<br/>　　“夫子。”
<br/>　　“夫子。”
<br/>　　钱观鱼几个都乖乖的收起剑，像见到猫的老鼠一般站得笔直，一丝不苟的行礼。
<br/>　　程大雷盯着这老者，原来他就是天下扬名的孟夫子，看上去童颜鹤发，倒是有几分仙风道骨。
<br/>　　“先生请莫于小辈一般见识，恕了他们这次吧。”
<br/>　　程大雷也收起剑：“好说，好说。”
<br/>　　“夫子。”钱观鱼还有些不服：“他们是来闹事的。”
<br/>　　“混账，还不快谢过先生的不杀之恩。”孟夫子道：“自己去后山领责罚。”
<br/>　　钱观鱼不敢违抗夫子，行过礼后，乖乖的往后山去了。
<br/>　　孟夫子一撩袖子，道：“先生请到前厅说话。”
<br/>　　对方这么客气，程大雷也不能不给面子。他也依样还礼，随着孟夫子来到书院大厅中。
<br/>　　分主客落座，在一番客套之后，程大雷还是问出了自己最关心的事。
<br/>　　“夫子名扬东海，自然见多识广。我冒昧打扰，是想向夫子打听一件事。东海附近，可有姓华的，出名的大夫？”
<br/>　　夫子微微皱眉：“欧公子寻找华氏一族为何？”
<br/>　　程大雷眼前一亮，夫子没有说不知道，反而询问自己，很显然是知道有关的情报了。
<br/>　　“确实是家中有人病了，传闻华氏医术无双，所以才来撞撞运气，夫子既然如此问，该知道他们住在哪里？”
<br/>　　“呵呵，公子问我算是问对人了。那华氏一族就在离此不远的东海城，城外有一华家村，华氏一族就住在那里。”
<br/>　　程大雷大喜过望，一路走来，总算是找到些有用的消息。他起身告辞，立刻就要去华家村。
<br/>　　“先生且慢，明日书院有一场伏魔大会，到时候华家会有人来。先生等在此地即可，何必长途奔波。”
<br/>　　“伏魔大会？”程大雷一头雾水，难道这书院是儒家的壳子佛家的里子，他不解的问：“伏什么魔大会？”
<br/>　　“公子难道没听说？”夫子认真问：“那蛤蟆大王来东海了。”
<br/>(本章完)
