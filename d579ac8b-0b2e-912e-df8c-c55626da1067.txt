　　脑海中回想起黑雷劫出现的情景，慕清霄内心深处，至今都有些余悸，余悸的同时也有一股狂热。
<br/>　　当初大脑一热，想都没想就冲了上去，若是在来一次，他依旧会用肉身渡劫，用肉身渡一次劫，带来的好处简直不可估量。
<br/>　　回过神来，慕清霄的视线又回到拍卖台上。
<br/>　　“若是谁服用了这枚阴阳龙玄丹，那么日后在重伤火哦这命垂一线的时候，运气若好的话，这丹药便能赋予你破而后立的机会。”
<br/>　　说罢，老者笑吟吟的继续说道：“所谓的破而后立，便是打破以往的束缚，让人犹如蜕变一般，无论身肉身，灵魂或者斗气，都能更上一层楼！”
<br/>　　话语刚落，虽然有些修炼者仍旧不明所以，但潜意识里就觉得七品丹药牛逼。
<br/>　　“或许很多人对于彻底蜕变的界限有些模糊，那老朽在此向诸位详细说明。”老者轻笑一声，旋即开口道：“想必风尊者古灵之名，诸位都听说过吧？”
<br/>　　“风尊者，那可是闻名大陆的巅峰强者，堂堂斗尊级别强者，怎会没听说过。”老者的话语，顿时招来一片白眼。
<br/>　　慕清霄倒是颇有兴致的听了起来，他记得风尊者古灵，与药尘的关系貌似很不错，不过，斗尊级别，并不是他现在能接触的。
<br/>　　“既然诸位都知晓风尊者之名，想必也知道在成就斗尊前，古灵大人曾与一位宿敌的惊世大战吧。”
<br/>　　老者面露狂热，道：“在那场战斗中，尚还是五星斗宗的古灵大人，虽然成功击败对手，可自己也处于重伤散功状态，按照常理，即便他能活下去，实力也将不如往昔。”
<br/>　　说道此处，老者语气停顿了下，继续道：“可事实呢？在距离那场大战后七年，失踪已久的古灵大人再次出现在大陆上，而此时他的实力，却已经在斗尊级别。”
<br/>　　话语刚落，整个拍卖场都鸦雀无声，许多修炼脸庞上都涌上一抹惊骇，七年的时间，不仅恢复了伤势，而且实力飞跃，直接从斗宗晋阶斗尊，好恐怖的速度！
<br/>　　“想必诸位都已经猜到，古灵大人当初游历大陆时，也侥幸得到一枚阴阳龙玄丹，并且自己服用，那股药效在漫长的岁月中，一只潜伏在他体内，直到他重伤，破而后立成功后，他就在七年内，从斗尊晋阶为了斗尊！”
<br/>　　偌大的拍卖场中，寂静持续了片刻，随后就响起一片倒吸凉气的声音，迷茫的目光也消失不见，取而代之的是惊骇。
<br/>　　所有人看向阴阳龙玄丹的时候，眼中的垂涎已经彻底转变成了贪婪。
<br/>　　听到老者的吹嘘，慕清霄微微一笑，阴阳龙玄丹确实能使人破而后立，但也就一次，而且破而后立所带来的痛苦，可不是普通人能够忍受的。
<br/>　　“仙儿，我们走吧。”
<br/>　　“七品丹药，你不需要吗？”
<br/>　　慕清霄拉着她的玉手，神秘一笑，道：“自然是需要的，但不是现在，你觉得我们拍走了七彩韵灵果，某些人会放过我们吗？”
<br/>　　“这可是我们花八百多万金币拍获，难道他们......”小医仙捂着粉唇惊呼一声，隐隐猜到了些什么，但美眸中却满是不敢置信。
<br/>　　慕清霄眼底掠过一丝杀意，道：“所以我才说，你这妮子依旧是太善良了，世界上有太多黑暗，倒不如让你提前经历一番。”
<br/>　　走出拍卖场后，望着碧空如洗的蓝天，两人就向拍卖会大厅走去，道：“有七彩韵灵果，你能在短时间内晋阶大斗师，在拍卖会中，他们不会下手，别担心了。”
<br/>　　来到客厅，慕清霄就将顶级贵宾卡交给了待女，道明来意后，便被后者恭敬的请了进去。
<br/>　　“请两位大人稍等片刻，拍卖会即将结束，到时候主事人，会帮两位清理拍卖所得。”待女将两杯清茶放在桌上，随后就微笑着退了出去。
<br/>　　待女离去，小医仙手中出现一枚银针，小心翼翼的放进茶杯中，几秒后拿起，发现没有变色后，俏皮一笑，道：“嘻嘻，居然没下毒呢。”
<br/>　　说罢，便端起茶杯，轻轻抿上一口。
<br/>　　片刻后，茶水见底，几道脚步声就来到客厅，房门打开，只见一名干瘦老者，领着两名待女走了进来。
<br/>　　“先生是竞拍残图与七彩韵灵果的大人吧？我是这里的主事，叫我范管就行。”见慕清霄与小医仙都套着黑袍，看不出端倪，老者走上前来，微笑着说道。
<br/>　　慕清霄微微点头，并不想与他多废话，道：“现在可以取走东西了吗？”
<br/>　　“呵呵，自然可以，请先生稍等。”
<br/>　　范管事目光在慕清霄与小医仙身上扫过，旋即将顶级贵宾卡，一枚普通纳戒递了过来：“先生所拍的丹药，总共一千二百万金币，本行收购的物品一千万金币，前后总共二千二百万金币”
<br/>　　说着，范管事眼中闪过一丝莫名的笑意，道：“先生拍获的残图二十一万金币，七彩韵灵果八百二十一万金币，扣除手续费，卡中持有一千三百万金币，请您收好。”
<br/>　　接过黑卡与纳戒，残图与七彩韵灵果确实在其中，悄无声息的装进储存空间中后，慕清霄就起身离开，小医仙寸步不离。
<br/>　　望着慕清霄的背影，范管事若有所思，道：“真是冷漠。”
<br/>　　“看出他的身份了吗?”
<br/>　　待慕清霄离开，一道平淡的声音在接待室中响起，范管事回到一看，只见一名虎背熊腰的中年男子，不知何时已经坐在慕清霄刚才所坐的位置上。
<br/>　　看见中年男子，范管事弯腰恭敬道：“门主，属下完全看不出来，两人隐藏极深，显然都不想以面目示人，但他能拿出极品聚气散，完全有可能是一位五品炼药师。”
<br/>　　眉头皱了皱，中年男子微微点头，平淡的语气中却充满阴冷的杀意。
<br/>　　“派人暗中跟着，搞清楚他的底细，五品炼药师不可多得，想尽办法让他为我八扇门效力，若是不成，直接干掉，不能为我所用，自然不能便宜别人。”
