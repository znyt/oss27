　　“心脏之中，有奇物？”
<br/>　　楚风眠听到星玄的话，才突然察觉到，在这血邪魔帝心脏之中，的确有着一股精粹的生命力。

<br/>　　在这血邪魔帝心脏之上的伤口，都在一点点的修复这，虽然很慢，可楚风眠感觉到，一旦是让这些伤口全部修复完，就是这血邪魔帝重生的一日。
<br/>　　那个时候，才是真正的灾难。

<br/>　　楚风眠目光扫了过去，隐隐约约，在这血邪魔帝心脏之上，看到了一只蜈蚣的模样。
<br/>　　说是蜈蚣，却背生六翼，就是这一条蜈蚣，藏在血邪魔帝的心脏之中，散发出强横的生命力，在修复这血邪魔帝心脏的伤痕。

<br/>　　楚风眠脸色猛然一变，六翼蜈蚣，藏金精，是在这佛寺还未开启之前，楚风眠就从宝光之中看到的，两件炼体之道的至宝。
<br/>　　其中藏金精，被楚风眠从金身佛尊陵墓之中得到，这六翼蜈蚣，楚风眠也在寻找。
<br/>　　没想到这柳依蜈蚣，居然藏在血邪魔帝的心脏之中，看来是血邪魔帝故意而为，来维持生命力。
<br/>　　以至于这血邪魔帝，在数十万年后，心脏依然是保持跳动，甚至可能是再度复活。
<br/>　　现在阻止血邪魔帝复活的最好办法，就是收取心脏！
<br/>　　“始祖第二式！”
<br/>　　楚风眠身上战龙血脉之力，凝聚在了他的掌心之中，猛人一掌轰击而下，轰击在了这血邪魔帝的心脏之上。
<br/>　　轰！
<br/>　　在这始祖第二式轰击下去的一刻，从血邪魔帝的身躯之中，大量的魔意，开始凝聚，随着这心脏的跳动，化为了一道完美的屏障。

<br/>　　楚风眠始祖第二式的力量，轰击在上面，这十二万九千六百枚小晶体，都在剧烈的颤动这，但是这屏障的力量，却是丝毫没有减弱。
<br/>　　“力量还不够！”
<br/>　　楚风眠脑海之中立刻明白，这始祖第二式的力量，还不足以打破这十二万九千六百枚小晶体组成的屏障。
<br/>　　现在楚风眠已经是凝炼了半步不朽真身，自然懂得这一道道小晶体组成的屏障，到底是多么的厉害。
<br/>　　除非是一击，打破临界点，不然的话，在强横的轰击，轰击在上面，都会被十二万九千六百枚小晶体分散化解。
<br/>　　不过现在这血邪魔帝，毕竟只是一个死人，还是一个死了数十万年的死人。
<br/>　　现在血邪魔帝身上蕴含的力量，已经是不足他全盛时期的千万分之一。
<br/>　　楚风眠刚刚施展出的始祖第二式，已经足以微微撼动这血邪魔帝身上的屏障。
<br/>　　只要是更加强横的攻击，就有机会打破。

<br/>　　嗖！
<br/>　　立刻之间，从楚风眠背后的天涯剑匣之中，万道剑气同时飞了出来，楚风眠手中，祖龙至尊剑也被楚风眠握在了手心之中。

<br/>　　楚风眠身上的剑意，也全然加持在了这一剑之上，一剑打出，数十道剑气飓风，向着那血邪魔帝心脏之上十二万九千六百枚小晶体化为了的屏障轰击而去。

<br/>　　两股力量碰撞，这血邪魔帝骸骨坐下的王座，都在剧烈的抖动这，一枚枚小晶体的颤动，比起刚刚要厉害的多。
<br/>　　在这样的力量之下，楚风眠看见，这一枚枚小晶体凝聚出的完美屏障，已经是出现了一道道的裂痕。
<br/>　　这血邪魔帝心脏之上凝聚出的无敌屏障，已经是被打出了一个缺口，只差一点力量，就可以全部撕裂。

<br/>　　楚风眠看到这一幕，再度举起了剑锋，漫天星辰之力，从天而降，缓缓的凝聚在了这剑锋之上。

<br/>　　当这剑锋之上的星辰之力，凝聚到了极限的一刻，楚风眠的剑锋轰然落下了，耀眼的剑光，刺入到了这一枚枚小晶体组成的屏障之中。
