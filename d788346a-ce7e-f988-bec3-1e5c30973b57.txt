　　太阴星上，月桂树下，倾世佳人，缥缈飞仙。
<br/>　　一绝色丽人，犹如九天之上的广寒仙子，白衣飘扬，银发飞舞，手持银色灵剑，神色凝重地望着前方不远处的一个男子。
<br/>　　一玉袍男子，如统领九天十地的一尊仙帝，星光璀璨，月华笼罩，又如九天仙王，威压天地众生，凭立虚空，给人一种手掌生死，宇宙幻灭凭他一念的感觉。
<br/>　　若不是气氛不对，两人站在一起，一定会被认为是花前月下，郎才女貌。
<br/>　　这男子自然就是前来太阴星取星辰本源的道印天尊元始了。女子自称为太阴之主，妄图阻止元始的行为。
<br/>　　“何必呢？本座只是来取一点太阴本源而已，又不是要毁了这太阴星。你又何必如此固执。”元始凭立虚空，悠悠开口道。
<br/>　　“不行，太阴星不容有损！”绝世女子坚决不松口，坚持不让元始动太阴本源。
<br/>　　“看来是谈不成啰！那就别怪本座不会怜香惜玉了！”
<br/>　　元始再度开口，这一次语气却不若之前那般随意了，而是开始变得凌厉起来。太阴本源他必须得到，这关系到他能否突破的问题，不管是谁，阻我道者，杀！
<br/>　　本着先下手为强，后下手遭殃的原则，元始一点也不怜香惜玉，不管她是女神，还是仙子，直接动手。
<br/>　　“不周印！”
<br/>　　元始立于虚空，悠然开口道。
<br/>　　话音刚落，便见一道巨大的不周神山出现在虚空，一股滔天的天地威压压向那道绝色身影。
<br/>　　随着元始实力的增强，他在不周山上领悟的不周印早已变得威力巨大，威压丝毫不弱于当初元始上山之时的威压。
<br/>　　那仙子身影停顿片刻，一不小心，差点便被不周印的威压压倒。不过，她终究是一个实力高深的仙子，而不是一般的仙子。只见恍惚之间，她便反应过来，运转法力，抵抗这股滔天威压。
<br/>　　同时，她挥动手中仙剑，一道巨大的银色天幕出现在元始眼前，天幕之上，带着浓浓的月华之力、太阴之力。天幕顿时化作一道惊天利刃，斩向元始。
<br/>　　威力巨大，轰动星空，声势浩大，虚空动荡。
<br/>　　“嗯，不差！”元始不咸不淡地评价道。
<br/>　　“再接本座一印。阴阳印！”
<br/>　　陡然间，天空之上出现一道巨大的太极图，黑白两色太极图遮天蔽日，瞬间就遮住了这片天空，阴阳鱼相互游动，化作一个巨大的黑白磨盘，缓缓旋转，飞向那银色天幕。
<br/>　　那银色天幕就仿佛是一张纸，瞬间便被黑白磨盘磨灭，绞成碎片，之后又被吞噬。太极图不停，依旧向着那仙子飞去，要将她磨灭。
<br/>　　女子神色凝重，从刚一开始交手，他便知道眼前此人法力高深，远超乎自己的想象，她不敢轻视，打起十二分警惕出手应对。
<br/>　　手持银色仙剑，女子手捏太阴剑诀，剑气破空而出，剑光飞舞，剑意至阴。
<br/>　　“铮——”
<br/>　　一声高昂剑鸣迸发而出，流转的剑光凝结化作实质一般的银色匹练，绽放出最刺眼的华丽银光。
<br/>　　剑冷如冰，面沉如水，进步前进，冷剑劈斩，银色仙剑锋芒凌厉，携着至阴之力轰然劈向黑白磨盘。
<br/>　　“咔擦——”
<br/>　　太极图破碎，黑白磨盘碎裂，同时，那道冰冷剑光亦崩裂碎开，两者相互抵消。
<br/>　　但，就在此时，高空之上，又一道淡然的声音幽幽传来：
<br/>　　“寂灭指！”
<br/>　　“不好！”
<br/>　　仙子心中暗道一声，那仙子还没来得及反应，只见虚空之中，一道巨大的手指蓦然出现，手指浩大，如同一道天柱，轰然而下。
<br/>　　只见周围的空间出现一道道涟漪波纹，空间变得凝固，时间变得没有意义，一切都尽将结束。
<br/>　　“哗啦——”
<br/>　　伴随着一道声响，虚空破碎，如同一面镜子变成碎片，那绝世女子瞬间便被那巨大手指给击中。
<br/>　　“噗嗤！”
<br/>　　仙子倒飞而去，鲜红的小嘴之中喷出一道鲜血，血花宛如一道血色莲花，绽放于虚空，让虚空变得凄美无比。
<br/>　　“嗒！嗒！嗒……”
<br/>　　女子落地，脚步向后退数百步之后，身影才停了下来。可是她还没来得及回应，虚空之中，再次传来一道悠然的声音：
<br/>　　“四象印！”
<br/>　　瞬间，四道身影从虚空之中窜了出来，化为四象神兽，金色白虎，带着强烈的金戈杀伐之气，飞向女子。
<br/>　　赤色朱雀，化为一道滔天神火，焚裂虚空，涌向女子。青色青龙，化为一道巨大身影，腾云驾雾，探出巨大龙爪，拍向仙子。黑色玄武，涌起翻天巨浪，覆灭天地，涌向仙子。
<br/>　　四道身影，化为四道杀伐之气，全部杀向那绝色仙子。
<br/>　　“不好，月精轮，出！”
<br/>　　那绝色仙子不愧是高手，瞬间便做出决断，用上她的上品先天灵宝月精轮，这是她在太阴星深处找到的，威力巨大。
<br/>　　月精轮化为一道银色弯月，快速旋转飞舞，划破虚空，飞向四道身影。
<br/>　　“轰隆！”
<br/>　　月精轮化为弧形利刃，瞬间斩破青龙之躯，之后再斩向朱雀。朱雀也并没有坚持多久，再次被月精轮斩碎，神火化为零星碎火，散落一地。
<br/>　　之后，月精轮不停，携着斩灭朱雀与青龙的威势，斩向白虎。白虎掌杀伐，庚金之力迸发，与月精轮拼杀。
<br/>　　“铛！铛！铛！…”
<br/>　　显然，虚空传来阵阵金属交击的声响，月精轮并没有一击瞬灭白虎神兽。
<br/>　　隐于虚空之中的元始见此，低声笑道：“有点意思！”
<br/>　　看着下方正在与白虎拼杀的月精轮，再看一看正在操控月精轮的那道绝色身影，元始心念一动，跨过空间，瞬间便出现在女子身前。
<br/>　　“不好！”
<br/>　　那女子暗叫一声，瞬间便做出反应，操控月精轮放弃白虎，向着自己眼前这道人飞舞而来。
<br/>　　月精轮带着凌厉的月刃与至阴的寒气，跨破虚空，斩向元始。
<br/>　　元始眼也不抬，伸手向着左面虚空就是一道霸拳。
<br/>　　“轰隆！”
<br/>　　虚空震动，月精轮瞬间就被元始一拳崩飞，飞出很远，眨眼见便不见了踪影。
<br/>　　“怎么可能！”女子大吃一惊，竟然有人能够凭借双手打破月精轮，此人力量到底有多强？这一刻，女子心中升起一股强烈的寒意。
<br/>　　同时，元始右手也不闲着，右手伸出拇指，朝着那道还处于震惊之中的绝色仙子眉心一点。
<br/>　　动作看似缓慢，实则快速无比，电光火石之间，女子还没来得及反应，元始的拇指便已点中女子的眉心。
<br/>　　“元神，封！”
<br/>(本章完)
