　　不周之巅，看着三清拿着混沌钟离去的身影，四妖皇眼中闪过不甘、屈辱、憋屈、仇恨的神色，瞳孔深处更隐藏着一丝丝忌惮。
<br/>　　他们虽然不甘，甚至恨透了三清，但他们不得不承认，三清的强大，远不是他们如今能够对付的。特别是元始的强大，让四人有些绝望。
<br/>　　“这个仇一定要报！”
<br/>　　太一恨声道。帝俊虽然没有说话，但眼中亦闪过一丝坚定之色，显然，这事不会就这么简单就结束了。
<br/>　　“三清，老子，元始，哼！你们等着。今日羞辱吾等，此事不会就此罢休！”女娲恨恨道，眼中闪过一丝疯狂。
<br/>　　“我等，如今该如何？”
<br/>　　伏羲苦笑道，虽然他也很想报仇，但他是四人之中看得最清的一个，他们报仇的机会实在是太渺茫了。
<br/>　　帝俊眼中闪过一丝奇异光芒，对着三人说道：
<br/>　　“仅凭我等三人如今想要报仇，希望实在是太渺茫了，但，我们又不能不报仇。所以，我们必须联合起来。”
<br/>　　说着，他看向女娲与伏羲，接着说道：
<br/>　　“女娲道友，伏羲道友，你们对于我之前的提议考虑得如何？如今摆在我们面前的只有一个办法，只有妖族统一起来，才能聚一族之力，抗衡三清。”
<br/>　　“我们同意了，但你必须说话算话，我们兄妹俩需要成为妖皇之一。”
<br/>　　女娲之前对于这个意见还有些犹豫，她是一个不喜欢争斗的人，能不争斗她就不会去。但如今她实在是被元始三清逼急了，她毫不犹豫地就同意了帝俊的意见。
<br/>　　“好，只要两位道友同意就行，妖族立族之时，女娲道友可为娲皇，伏羲道友可为羲皇。”
<br/>　　帝俊眼中闪过一丝兴奋，他毫不犹豫地同意了，只要两人同意，他就可以实现妖族的大一统，他就可以成为真正的妖帝，至于许诺给他们的皇位，那只是一个位子而已。
<br/>　　“伏羲道友以为如何？”
<br/>　　接着，帝俊眼中闪过一丝历芒，看向伏羲，若是伏羲不识趣，他只好强行动手。
<br/>　　“既然我妹妹都同意了，那我这个做哥哥的，也就只能同意了。”伏羲苦笑一声，以如今这种情况，他还能怎样，只能同意。
<br/>　　“哈哈！好，既然如此，那就预祝我们妖族大统一，称霸洪荒！”帝俊大笑道。
<br/>　　“别的我不管，三清之仇，必须报。”女娲冷声道。
<br/>　　“可以。”
<br/>　　“那北冥宫的鲲鹏要怎么办？”太一道。
<br/>　　“鲲鹏一人，他还能抵挡我们几人不可。此次交战，让我们认识到自身的不足，我等如今最重要的是赶紧提升实力，至于北冥鲲鹏，哼，等下次紫霄宫听道后，再做计较。”帝俊听道鲲鹏，冷声道。
<br/>　　“大善！”
<br/>　　几人同意道。在他们眼中，妖族都快统一了，鲲鹏一人还能坚持多久？
<br/>　　“好，既然如此，那我们就先走一步。”
<br/>　　帝俊对着女娲与伏羲道了一声。便带着太一离开了。他们还需要急着回去疗伤。显然，在此次战斗之中，他们受伤不轻。
<br/>　　“哼！我们也走。”
<br/>　　女娲就想要离开，但看到不远处剩下的那即将枯萎的葫芦藤，想了想，心中想道：
<br/>　　“这葫芦藤能够结出七个葫芦，想来也应该是件宝物，依如今的情况，它用不了多久便会死去，既然如此，那我还是收了它吧。”
<br/>　　女娲一挥袖，将仙藤连根拔起，准备收入收入空间之中，但让她意外的是，，葫芦仙藤底部竟然还藏有一小块紫色灵土。
<br/>　　这紫色灵土为戊紫神土，乃是孕育先天灵根的最顶级的几种灵土之一。许多灵根都有寿命，很难一直存在。
<br/>　　这戊紫神土充满灵性，能够培养各种灵根灵药，亦能移植一些较为珍贵的先天灵根，能延长它们的寿命。而且炼制一些较为珍贵的仙丹，也需要这种戊紫灵土。这可谓是用途颇广的一种神土。
<br/>　　可以说，这种戊紫神土只比先天五壤等几种顶级的神土低一个等级，是不可多得的神土。
<br/>　　先天五壤乃息壤、定壤、释壤、衡壤与庚壤，是最为顶级的先天神土，只有开天时才会产生，用一点就少一点，属于不可再生资源。
<br/>　　女娲见此，心情稍微好转，挥手将戊紫神土与葫芦仙藤一起收走，这才与自己的哥哥伏羲一同向着山下走去，他们的洞府就在不周山山腰，并不像帝俊与太一，要回到太阳星。
<br/>　　……
<br/>　　“大兄，如今既然已经得罪了妖族，你为何不将他们一次解决。”回道昆仑山，老子对元始的做法十分不解，便对着元始问道。
<br/>　　“哈，不是吾不想，而是不能。”元始轻轻说了一声。
<br/>　　“不能？”老子不解，疑惑道，“为何？”
<br/>　　“妖族与巫族为下一量劫主角，他们不能死。”
<br/>　　“嗯？”老子瞳孔一缩，惊骇道：“大兄竟然能算出下一量劫？”
<br/>　　元始笑而不语，他当然算不出，但他能说自己天生知道，因为自己是穿越者吗？显然不能。
<br/>　　“那以后妖族要怎么办？”老子见元始不想扯这个话题，便问了另一个问题。
<br/>　　“不必担忧，他们不足为虑，只要时常关注他们动向便可。再说，就算他们来报仇，本座还会怕他们不成？”元始霸气道。
<br/>　　“是啊，我们三清还会怕他妖族吗？”
<br/>　　通天好战，坚信自己的力量，他赞同元始的说法。
<br/>　　“好了，不说那些了，来看看我们此次的战利品。”
<br/>　　元始单手一挥，便将七个葫芦放了出来。
<br/>　　七个葫芦，紫金、紫青、紫红、紫白、紫黑、紫黄、紫绿，七种色彩闪耀在空中，放出无比耀眼的光彩，将整个洞府照亮，变得美轮美奂。
<br/>　　“来，你们分别感受一下，自己与哪几个有缘，一人选两个。”元始对着两人说道。
<br/>　　看到七个葫芦，两人顿时露出一丝微笑，纷纷按照元始所说，用心去感应。
<br/>　　突然，老子睁开眼睛，将紫金色、紫黄色两个葫芦拿到手中，道：
<br/>　　“此二者与贫道有缘。”
<br/>　　在老子选后，通天也伸手将紫青色、紫绿色两个葫芦抓走，道：
<br/>　　“它们与贫道有缘。”
<br/>　　在两人选完后，元始看也不看，袖袍一挥，便将剩下三个全部收走，之后对着两人说道：
<br/>　　“混沌钟却是不能给你们了，但吾此处有两件不弱于先天至宝的存在，却是比混沌钟更适合你们。”
<br/>　　……
