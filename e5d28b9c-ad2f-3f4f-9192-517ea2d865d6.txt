　　第491章
<br/>　　程大雷倒是不知道，自己这一举动竟然给和珅带来如此大的心理震撼。此时，他也到了城头上，看着千马奔腾的场面，点点头道：“蛮好，蛮好嘛。”
<br/>　　此时和珅几人才注意到程大雷来了，和珅赶忙行礼，弯下腰来。
<br/>　　“大当家。”
<br/>　　程大雷摆摆手：“这下够了吧，不够我再给你弄点过来。”
<br/>　　“够了，够了。”和珅忙不迭的答应，心中更加确信这一幕是程大雷搞出来的，果然大当家是那仙人传人，有神鬼不知的手段。
<br/>　　程大雷目光看着城下的马群，却是起了别的心思。他击杀薛丁山后，还获得一次抽奖机会，一直没有使用，主要原因也是顾忌暴恶属性，给自己召唤出什么不得了的人物来。
<br/>　　不过，程大雷也觉得自己不能妄自菲薄，就算真召唤出某个在史书上留名的人物，自己目前是绝世，秦蛮也是绝世，再加上关鱼、赵子龙一帮高手，也未必不能干掉对方。
<br/>　　关键是有一次抽奖次数，放着不用的话，会一直牵着程大雷的心。
<br/>　　趁着今天这个机会，程大雷决定抽奖。
<br/>　　随着扭蛋机的出现，程大雷心中暗暗祈祷，希望这次系统可以抬抬手，不要给自己搞出什么了不得的英雄。
<br/>　　但天不遂人愿，随着一条系统提示，彻底摧毁了程大雷的希望。程大雷这次抽到的人物是绝世游侠，李莫愁。
<br/>　　李莫愁，古墓派大师姐，一柄拂尘杀人如麻，江湖人送外号赤练仙子。
<br/>　　程大雷盯着这条系统提示，内心有些无语，因为暴恶属性，自己这次抽到一个女魔头。但同样因为暴恶属性，系统会随机出现一位绝世高手。
<br/>　　果然没有意外，随着程大雷便又收到一条系统提示。
<br/>　　嘟，随机降临绝世刺客——聂隐娘
<br/>　　程大雷盯着最后出现的名字，一时张大嘴巴久久合不拢。刚才他的心情是无语，现在便是彻底崩溃。
<br/>　　聂隐娘何许人也，手持二尺青锋，能够刺白猿，斩虎头，长街之上杀人于无形之间。
<br/>　　这种实力怎么会只是绝世，程大雷觉得起码也是传说，当然，她的实力一定会有所削弱。但人家的职业可是刺客，干得就是偷鸡摸狗，千里去取人头的勾当。
<br/>　　李莫愁虽然也了不得，但完全没办法和人相提并论么。这笔买卖还是做亏了。
<br/>　　嘟，恭喜获得绝世座骑——赤兔
<br/>　　正在程大雷崩溃的时候，他又收到一条系统提示，程大雷一楞，就见城门外忽地跑过来一道红线。
<br/>　　来若一团燃烧的火，吸引了所有人的视线，只见赤兔行到城门前，仰首一声长嘶鸣，群马寂然垂首不发出任何声音。
<br/>　　和珅眼睛一亮，指着赤兔道：“这定是群马的王，最是神骏烈性，当是天下一等一的坐骑，只不过也最不好降服。”
<br/>　　从来名将爱宝马，美人爱有宝马的名将，只要搞到宝马，那美人也一块儿都有了。城头几个武夫见到这赤兔出现，眼睛已移不地方，听到和珅的话后，更是人人跃跃欲试。
<br/>　　“哈哈。”高飞豹搓着手大笑：“这是上天给我送坐骑来了，诸位兄弟莫争莫强，让我下去降服了此兽。”
<br/>　　“好啦。”程大雷摁住了他的肩膀：“你就别下去丢人了，关二爷，你去，若有本事降服它，以后这便是你的坐骑。”
<br/>　　“大当家稍后，某家去去便回。”关鱼将手中大刀交给身后喽啰，自己大步下了城门。
<br/>　　虽然他表现得很镇定，然而程大雷还是看到，他下台阶时都是一步并做两边走。毕竟，面对这样的绝世名驹，没有人可以无动于衷。
<br/>　　高飞豹、张肥几个人站在城门上往下看，程大雷既然点了关鱼，他们也不好去争。不过关鱼如果出了岔子，没有拿下这赤兔，他们也还有机会。
<br/>　　程大雷倒是没有扒着城头看的兴趣，这赤兔一看就是关二爷的官配，肯定不会出岔子，吕温侯不出世，谁敢和关二爷争。
<br/>　　程大雷心里计较着这次抽奖的得失，虽然多了个聂隐娘有些恶心，但获得赤兔这次抽奖也不能算亏了。击杀薛丁山就可以获得这样的奖励，如果把聂隐娘干掉岂非能获得更多奖励。
<br/>　　如今自己是装备盔甲坐骑都已齐全，但秦蛮、关鱼几个可都只是普通装备，如果可以多刷几个薛丁山这样的boss，也能给他们凑齐全套装备。如果能帮他们升到绝世，山寨的战斗力就又会往上提一截。
<br/>　　约有半个时辰，关鱼喜气洋洋的上楼，冲着程大雷是纳头便拜。
<br/>　　“二爷快快请起，不用行此大礼。”
<br/>　　关鱼喜不自禁，站在城门之上，目光环视周围，恨不得原地跳起来似的。当然，张肥、高飞豹几人看他的表情就不甚痛快了，越看关鱼越觉得今天的他有些欠揍。
<br/>　　“大当家，你这样做可是有些不地道啊。”高飞豹凑过来道：“你这明显是偏心，你如果点我，我也可以降服这红马。”
<br/>　　程大雷白了他一眼，不是很愿意搭理他，人最重要的就是要有自知之明呐。如果是高飞豹下去，定然是给这赤兔马刷经验。
<br/>　　“便请大当家给这宝马赐个名字吧，大当家看叫什么好？”
<br/>　　“依我看……”程大雷顿了顿，道：“就叫墨兔吧。”
<br/>　　“呃……”
<br/>　　所有人都愣住了，不可思议的看着程大雷。
<br/>　　“大当家，那好像是一匹红马？”和珅语气有些不太确定，不太确定程大雷是不是个色盲。
<br/>　　“哈，就因为是红马才要墨兔。没有想到吧。”程大雷道：“你们想不到，敌人也不会想到。他们怎么能想到，骑墨兔的大将竟然会骑一匹红马呢。”
<br/>　　在一片啧啧称奇中，程大雷下了城头，飞也似的回了城主府。
<br/>　　关鱼得到得意坐骑，心中是高兴了。然而，这可是程大雷用命换来的，那聂隐娘有鬼神不测之能，万一她突然冒出来，一剑把自己杀了怎么办。
<br/>　　传说中，那聂隐娘可是在闹市街头，砍人脑袋却无人察觉，为了自己的性命问题，程大雷不得不谨慎。
<br/>(本章完)
