        “昨晚，趁着混乱捅伤我的那个人找到了吗？”傅夜沉岔开话题，冷冷地问道。
<br/>
<br/>        连华生微微耸肩，瘪嘴道：“我已经替你，安排阿凯和阿峰去暗中调查这件事情了。”
<br/>
<br/>        “……”傅夜沉随之沉默了。
<br/>
<br/>        连华生顿了顿后，接着说道：“不过，也奇怪了，明明可以做到一刀致命，但对方似乎并非想直接要了你的命。阿沉，你不觉得这件事情会不会跟我们一直在查的那件事情有关？”
<br/>
<br/>        “你的意思是，昨晚的事，又跟那个冒险者游戏有关？”傅夜沉剑眉微蹙。
<br/>
<br/>        这三年来，他们内部一直未放弃过调查此事，但是这件事情，却让他们成为了无头苍蝇。
<br/>
<br/>        他和连华生只是辅助调查，提供有效的线索和证据给上头，并不负责对案件的跟踪，所以，傅夜沉和连华生对于这件事情也是束手无策。
<br/>
<br/>        “我的直觉告诉我，那个所谓的‘冒险者游戏’，所针对的人中包括你。但是，对方似乎又不想要了你的命，一直在玩弄你。也不知道，这到底是不是我的错觉。”连华生若有所思地说道，替傅夜沉包扎好伤口后，关心道，“这几天，你就不要去公司里，好好在家养伤。身体重要！”
<br/>
<br/>        “我明白！”傅夜沉点了下头。
<br/>
<br/>        连华生收拾医疗箱的时候，目光无意间落在了傅夜沉的左手腕上，见他一直戴着那条青丝手链，忍不住感慨道：“如果，你决定放下童筱颖了，就把手腕上那条手链给取下来吧！”
<br/>
<br/>        “……”傅夜沉下意识地摸了摸自己手腕上的青丝手链，回想起那日他强吻了筱颖后，还割断了她那一绺长发时的情形，俊脸上忍不住露出了苦涩的笑。
<br/>
<br/>        筱颖当时一定不理解他对她的所作所为吧！
<br/>
<br/>        那段时间，他忙着护国护民，差点还搭上了自己的性命，然而上天却仍旧待他不好，让他把自己心爱的女人给弄丢了。
<br/>
<br/>        筱颖，你在澳洲，过得好吗？
<br/>
<br/>        澳洲，悉尼大学附近的联排别墅区。
<br/>
<br/>        童筱颖手里捧着自己刚拌好的水果沙拉，肩膀和耳朵之间夹着手机，一边捏着水果叉，吃着盘子里的水果，一边神情清朗地越过书房走向侧厅的拱型阳台。
<br/>
<br/>        七月的悉尼，热情的阳光与不带湿气的微风，让这样的天气成为盼求不得的天堂。
<br/>
<br/>        “小卫，我们种的金合欢树都已经开花了，你什么时候可以回来呀？”童筱颖微笑着问道，放眼望向了楼下台阶前的花坛里，她和傅昼景三年前一起种那棵金合欢树。
<br/>
<br/>        一朵又一朵像棉团一样的金合欢花，睡在墨绿色的树梢间，随风摇曳。
<br/>
<br/>        “我现在负责的这个项目大概还要三个月左右才会完工，完工后，我就回来陪你。待会把金合欢花拍给我看看。”傅昼景温和地应道。
<br/>
<br/>        “嗯，好！”
<br/>
<br/>        “你真不打算继续读建筑硕士学位了吗？你的成绩真的很不错，上次你的导师还在跟我说，让我劝劝你，继续念建筑学硕士。”
<br/>
<br/>        “我已经拿到本科毕业证了，以后，我也可以帮你分担工作。而接下来，最主要的是我想要孩子，不想再读建筑学硕士了。”童筱颖一脸不害臊地说。
<br/>
<br/>        傅昼景忍俊不禁地说道：“嗯，也对，我们俩，是该要个孩子了。等我回来，我们一起要个孩子。”
<br/>
<br/>        “那……是不是代表了，我们要结束这种‘  Love（柏拉图式精神恋爱）’了！”童筱颖忍不住调侃起来。
<br/>
<br/>        三年前，傅昼景被查出得了恶性肾肿瘤做了切除手术，童筱颖经过配型成功后，毫不犹豫地把自己的一个肾，给了傅昼景。
<br/>
<br/>        医生建议他们双方禁欲一年，而他们这一禁，居然禁了三年。
<br/>
<br/>        其实，他们在澳洲生活的这三年里，童筱颖要念大学，傅昼景要工作，两人共处的时间，屈指可数，总是聚少离多。
<br/>
<br/>        在澳洲的各种开销，远远比在滨江城的开销还要大。
<br/>
<br/>        爷爷傅炎廷其实只为童筱颖支付了一个学期的学费，就连在澳洲买的别墅，爷爷傅炎廷也只是替他们付了首付而已，再后来，爷爷傅炎廷就再未过问过童筱颖和傅昼景的事情了。
<br/>
<br/>        傅昼景是那种很要面子的男人，哪怕爷爷傅炎廷再有钱，他也不会主动开口问爷爷傅炎廷要钱。
<br/>
<br/>        童筱颖就更加不会去开这个口了。
<br/>
<br/>        爷爷傅炎廷那边的经济来源断了后，接下来童筱颖的学费，两人的生活费，家里的房贷，成了两个人之间最大的几个问题。
