　　萧家矿脉，矿山之上，一少年盘膝而坐，面朝太阳。
<br/>　　看不清他的样子，但他的后背很伟岸，连阳光也被他遮挡了大半。背影下覆盖的阴影，仿佛是一处给人无限安全的庇护之所。
<br/>　　恰逢有两人这时跑上山，站在这阴影之下，看着那背影，一时竟看痴了。
<br/>　　确切来说，那背影并没有那么宽广，毕竟只是少年的身躯。但，它就是有一种能让人很舒服，很放松的感觉，仿佛一切风雨都无法跨过背影分毫。
<br/>　　那盘膝而坐的少年，忽然睁开了眼睛，并站了起来，阳光照耀在那帅气的脸庞上，很是迷人。
<br/>　　脸庞上同时挂着慵懒、坚毅和冷酷，更让人看得无法自拔。
<br/>　　不错，少年正是萧逸。
<br/>　　他最近都习惯了在矿山之上修炼，面朝太阳，沐浴在阳光下的空气中蕴含着比平时更浓郁的火属性天地灵气，能让他的控火兽武魂吸收天地灵气更快。
<br/>　　配合冰鸾剑武魂，双生武魂同时启动，修炼速度要比在矿山之下快。
<br/>　　“萧壮，萧子木，你们怎么跑上来了？”萧逸淡淡地问道。
<br/>　　刚刚跑上来的两人，正是萧壮和萧子木。
<br/>　　“嗯？发什么愣？”萧逸见两人呆呆地看着自己，不由皱起了眉头。
<br/>　　“阿。”两人顿时反应了过来，微微有些尴尬。
<br/>　　“今日不用去巡视小型矿脉吗？怎么有空来我这里。”萧逸问道。
<br/>　　萧壮不好意思地摸了摸后脑勺，回答道，“小型矿脉那边都没人敢去抢，我们也不用经常去巡视了。”
<br/>　　萧子木也笑道，“少家主你上次的事传开后，已经没人敢打我们萧家小型矿脉的主意了。毕竟你连慕容千军都敢杀，其他人可不敢来招惹我们。”
<br/>　　两人已经苏醒了数天了，当他们醒来时发现自己竟然已经突破后天一重了，兴奋得不了。而当得知这是萧逸的功劳后，对萧逸更是感激无比。
<br/>　　毕竟按他们的修炼速度，要突破后天一重起码要数年之后。
<br/>　　而武者的修炼，自然是年纪越小，修为越高越好。
<br/>　　再之后，当他们知道萧逸竟然杀了铁刀猎妖队的队长，连慕容千军也被追杀得如丧家之犬，更是对萧逸崇拜不已。
<br/>　　“你们还没告诉我来找我做什么？”萧逸微笑着问道。
<br/>　　“哦，差点忘了正事。”萧壮回答道，“五长老和萧若寒来了，四长老让你回山下。”
<br/>　　“哦？”萧逸眉头微皱。
<br/>　　萧壮回答道，“少家主你经常在山上，很少理矿脉的事，不知道也正常。矿脉中挖出的紫云铁，家族会定期派人来运回紫云城。这次负责运送的，就是五长老。”
<br/>　　萧子木也道，“今日五长老前来，所以我们所有子弟都没有外出呢。四长老怕五长老又找借口来找你麻烦，所以特地让我们来叫你回去。”
<br/>　　“哦，原来是这样，你们先回去吧，我等下跟上。”萧逸点点头。
<br/>　　“是。”萧壮和萧子木两人微微行了一礼，退了下去。
<br/>　　“呼。”萧逸深呼吸一口气，看了眼远方，那漫山遍野的紫云铁，着实很美丽。
<br/>　　“不知不觉，来矿脉这边差不多一个月了，如果没什么事，也该回家族了。”萧逸淡淡一笑。
<br/>　　景色再美，也比不过依依的一颦一笑，一声甜甜的‘萧逸少爷’。
<br/>　　....
<br/>　　议事厅中，四六长老以及各位执事早已入座。
<br/>　　五长老来临，众人自然停下手头工作，招待一番。
<br/>　　“少家主。”
<br/>　　“少家主。”
<br/>　　“....”
<br/>　　萧逸走下山，遇到的族人们都会恭敬地称呼一声‘少家主’，并微微行礼。
<br/>　　他们看萧逸的目光，是一种信服与信任。
<br/>　　“少家主。”
<br/>　　萧逸来到议事厅，自顾地走了进去，执事们纷纷叫了一声。
<br/>　　“萧逸。”四六长老满脸笑容。
<br/>　　萧逸随便找了个位置坐了下来。
<br/>　　在两位长老的下首位，是刚刚来到的五长老和萧若寒。两人见到萧逸的出现，竟同时脸色一变，脸上的憎恨之色一闪而过。
<br/>　　“小废物如今在矿脉这边竟有这么大的威信吗？”五长老心中暗道。
<br/>　　“五长老。”四长老率先出言道，“你远道而来，不如在矿脉这边多住几天。紫云铁方面，族人们会打理好，过几天你直接带走即可，免得你多费心神。”
<br/>　　“如此甚好，还是四长老想得周到。”五长老笑着回答一声。
<br/>　　这时，五长老继续说道，“其实，我这次前来，还有一事。”
<br/>　　“哦？五长老所为何事？”四长老问道。
<br/>　　五长老笑了笑，看向萧逸，道，“听闻，前些日子少家主得到了三颗妖血炎心果。老夫脸皮厚，特意来讨一颗给我儿若寒。”
<br/>　　萧逸心中冷笑，难怪这次会是五长老前来，原来运送紫云铁是假，想拿妖血炎心果是真。
<br/>　　萧逸直接说道，“五长老，你也知道你脸皮厚阿。不过，妖血炎心果都服用完了，不好意思。”
<br/>　　五长老听着萧逸丝毫不给面子的话语，顿时脸色一黑，但还是说道，“少家主说笑了，妖血炎心果，乃不可多得的灵果，紫云城已有多年没有人出售过了，可谓有钱也买不到。”
<br/>　　五长老继续道，“这等灵果，应当交由家族处置并分配。自然应当用于给予家族中最出色的年轻一辈。少家主你一颗，我儿若寒一颗，另一颗则交由大长老处置。”
<br/>　　萧逸耸耸肩，道，“我也认为你说得有理，但可惜，真的都服用完了。”
<br/>　　“少家主，你莫非是在为难我不成？”五长老脸色变得难看，道，“若少家主真那么小气，我买下来便是。像上次若寒进入紫云洞那般，少家主你出个价便是。”
<br/>　　妖血炎心果，太过稀有了，对后天境武者的作用又太大，任谁都想得到。
<br/>　　萧逸笑笑，道，“真的都服用完了，不信的话，五长老可以问四长老和六长老。”
<br/>　　“嗯？”五长老脸色一变，看向四、六长老。
<br/>　　两位长老点点头，道，“三颗妖血炎心果，其中一颗给了萧壮和萧子木恢复丹田，另外两颗萧逸已经服用了。其实嘛，以若寒的资质，就算没有那等灵果，修炼也不会有影响的。”
<br/>　　两位长老也不能把话说得太过，需给五长老一些面子，特意给他找了个台阶下。
<br/>　　孰料，五长老顿时大怒，“什么？竟然真的都服用完了？那等灵果，岂能由你们胡乱决定，萧壮和萧子木算什么东西，他们能比我儿若寒更有资格服用？”
<br/>　　“还有萧逸，你凭什么一人独享两颗灵果，这...待我回去家族，定当治你们一个以权谋私之罪。”五长老大吼着。
<br/>　　四长老本想给五长老一个台阶下，没想到五长老反而恶言相向，当即冷声道，“五长老，三颗妖血炎心果是萧逸夺回来的，他要给谁是他自己的事，他全部服用了，也没人能怪罪他。”
<br/>　　六长老直接出言嘲讽道，“这事我们早就禀报过大长老了，大长老说由萧逸自己处置那三颗灵果。五长老你就不必多此一举会家族去定罪了。“
<br/>　　“你...你们....欺人太甚。”五长老怒火中烧，狠狠地挥了挥袖子，道，“若寒，我们走。”
<br/>　　临走前，两人脸上的恶毒之色一闪而过，心中想道，“既然你们不仁，便别怪我不义了，一切都是你们逼我的。”
<br/>　　两人愤愤离开，其他长老和执事并没有注意到他们的神情。
<br/>　　但萧逸却注意到了，皱眉暗道，“这两个家伙，又在想什么诡计？”
<br/>(本章完)
