<br/>				本能的移开了目光低下了头，可当他做完这些时，才反应过来，不由一怔，再度抬头朝那小厮看去，却见那小厮一脸的憨厚之色的吃着果子，他不禁怀疑，刚才那一眼莫非是他的错觉？<br/>　　<br/>　　    “玉堂，你倒是说说，你这小厮怎么不一样啊？”另一名少年笑说着，神情带着好奇。<br/>　　<br/>　　    谢玉堂见他们一个个一副好奇的模样，便道：“他救过我一命，其实我们一家是前几天才搬到主家来的，不过在路上遇到了袭击，当时要不是他救了我，我估计就被杀了。”<br/>　　<br/>　　    听到这话，众人微讶：“这小厮救过你？看他的实力也不是很强啊！”<br/>　　<br/>　　    “原来你家是前几天才搬来的？难怪以前没见过你。”<br/>　　<br/>　　    凤九吃着果子，听着他们在那里聊着，神识却感应得到有人在暗处盯着，似乎在打量着在场的众名少年。<br/>　　<br/>　　    她心下不禁好笑的想着：这些少年有什么好打量的？莫不是，这阮家主母打着生日宴的名义，实际是想挑一挑未来女婿不成？<br/>　　<br/>　　    “哎，对了，你们有没听说易宝楼过几天有一枚七阶丹药要易换？”一名少年对众人说着：“我可是听说了，那是一名可以让仙圣巅峰突破进入仙帝级别的丹药。”<br/>　　<br/>　　    “这事我也听说了，我父亲和祖父他们昨天还在说着这事呢！怎么？你们家不会也盯了这枚丹药吗？”少年看向那人问着。<br/>　　<br/>　　    “哈哈哈，那是肯定的啊！只不过，我听我父亲他们说，那丹药不是拍卖的，而是要用同等价格的宝贝去易换的，除此之外，还有最主要的一条，那就是要登仙梯的信息一份。”<br/>　　<br/>　　    “登仙梯？那是什么？”一名少年不解的询问着。<br/>　　<br/>　　    “我怎么也没听说过这什么登仙梯？”另一人也说着，看向众人，问：“你们听说过吗？”<br/>　　<br/>　　    众人相视一眼，有的茫然，也有的沉思着，显然，有的是不知道的，有的则是知道一些的。<br/>　　<br/>　　    “能让仙圣巅峰突破进入仙帝的丹药？该不会也像其他丹药一样，吃了那么一枚之后实力以后就停在那里了吧？”<br/>　　<br/>　　    谢玉堂问着，他的消息没其他的人灵通，要不是他们说，他都不知有什么七阶丹药，更不知这城中的易宝楼在哪。<br/>　　<br/>　　    另一名少年道：“易宝楼的鉴定师放出的消息，就不会有影响，那可是一枚七阶上品纯净的极品丹药，可遇不可求。”<br/>　　<br/>　　    “既然那样珍贵，肯定很难得到，而且，这也没我们什么事啊！那样的丹药，我们又不能用。”谢玉堂兴趣缺缺的说着。<br/>　　<br/>　　    闻言，众人笑了笑，意味不明。也是，这丹药就算是落到谢家手中，也不会落到谢家旁系手中，更何况，还要登仙梯的信息，就更能易换了，就是他们家族的长辈昨天都在商量着这事呢！也不知到最后会被哪个家族得到？<br/>　　<br/>　　    见众人都在想着那七阶丹药，于是，阮家大公子便笑道：“今天难得人这么齐，不如，我们去练武场切磋一番怎么样？”<br/>
