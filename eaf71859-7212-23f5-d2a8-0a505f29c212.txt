　　东海市市局局长，二级警监，冯剑南有那个资格知道关于龙隐部队的事情，深深地看了眼肖天行，最后点了点头。
<br/>　　在从国安方面得到慕容飞的信息后，冯剑南就有预感，这件事情他们解决不了，龙隐部队的事情只有交给龙隐部队自己去解决。
<br/>　　龙隐部队特种作战基地。
<br/>　　赵天宇、云岳、朱致阳三个人一直守着赵天宇桌子上那部红色的专线电话。
<br/>　　这次“追魂”行动联动了国家安全部的资源，苏南省公安厅，东海市武警机动大队等多方面资源，唯独就隐瞒了东海市公安局。
<br/>　　东海市的情况持续不断地被传到龙隐部队，一切都在按照云岳所设计好的进行发展，他唯一担心的就是东海市警方脑子一热，一拍脑子就往上冲，这样就把慕容飞陷入到了危险之中。慕容飞的武器中只有一部分是实弹，剩余的都是空包弹，而东海市警方却是实打实的实弹，慕容飞的实弹不可能会对着东海市的警察射击，但是东海市警方的实弹可是会照着慕容飞招呼的。
<br/>　　桌子上，红色专线电话突然响起，打破了办公室中的寂静。
<br/>　　“总算是来了！”
<br/>　　赵天宇松了口气，快步走到办公桌旁，接起了电话，“我是承影。”
<br/>　　“承影，我是金雕。”
<br/>　　电话那头传来肖天行的声音，听到肖天行的声音，赵天宇丝毫不感到意外。云岳之前就预测过，打来电话通知龙隐部队，慕容飞“叛变”的人，百分之八十会是肖天行打来的。
<br/>　　“金雕，有事吗？”赵天宇若无其事的问道。
<br/>　　“承影，我这里发生了一些事情……”
<br/>　　随后，肖天行在电话中把发生在东海市的事情详细地跟赵天宇叙述了一遍。
<br/>　　“我明白了。”赵天宇深吸一口气，声音有些沉重的说道：“既然无常是龙隐部队的兵，就应该由龙隐部队来解决！你们控制住现场，我们的人立刻出发。”
<br/>　　肖天行打完电话，回到现场对冯剑南说道：“冯局，我联系过了，我的老部队会立刻派人过来。”
<br/>　　冯剑南点了点头，表示了解。
<br/>　　一旁的王荣浩这个时候不干了，这可是他升迁支队长的绝好机会，他怎么能就这么把这个机会拱手让人？王荣浩立刻走到冯剑南身边，说道：“冯局，慕容飞在里面只有一个人，再给我们突击队一次机会，这次我们一定能拿下他！”
<br/>　　冯剑南还没答话，舒磊这边就走了过来，对着王荣浩出示了一下自己的证件，说道：“国家安全部第九局舒磊。王副支队是吧？不好意思，这件事情你们恐怕不能插手了，因为我们国安方面有些话要问慕容飞，所以必须要活口。”
<br/>　　王荣浩张了张嘴还想要说什么，钱伟民这个时候走了上来，说道：“既然国安的同志说了，荣浩，把你的突击队暂时先撤下来，不要有什么动作。”
<br/>　　王荣浩牙齿咬得嘎吱作响，可是却没有任何办法，只能让突击队后撤，放弃对烂尾楼的突击行动，心中的不甘可想而知。挡在他面前的是挂着“国家安全部”这块金字招牌的大山，他根本撼动不了。
<br/>　　云岳等到赵天宇放下电话，对着他竖起大拇指，说道：“承影，没想到你还真有演戏的天赋，刚才那段表演真是滴水不露啊。”
<br/>　　“那是必须的。”赵天宇笑着说道：“等到我退下来了，我还打算去影视圈闯荡闯荡，说不定还能拿个什么影帝奖杯回来。”
<br/>　　“那你估计是没这个机会了。”朱致阳说道：“你觉得野狼会放过你？再过两年，等到狂龙接任了你‘龙头’的位子，你就等着去总参报到吧。”
<br/>　　“行了，战术中队那边谁去通知？”云岳看了眼赵天宇和朱致阳问道。
<br/>　　“还是我去吧。”赵天宇拿起桌子上的帽子，说道：“等到他们知道消息，肯定会非常震惊，这里也就我能压得住狂龙那小子。”
<br/>　　“也行，由你这个大队长出面，比我们两个更合适。”朱致阳点头道。
<br/>　　接到赵天宇从对讲机中传来的命令，战术中队队员立刻放下手上的事情，赶到直升机库集中。
<br/>　　赵天宇站在队伍前方，眼神严厉的从战术中队每一个人的脸上扫过，说道：“十分钟前，我接到了金雕的电话，在东海市发生了一件事情。无常跟境外军火贩子‘司机’进行交易，在交易过程涉嫌故意伤人，遭到东海市警方的围捕。”
<br/>　　听到赵天宇的话，直升机机库中战术中队的队员无不面露震惊，感到不可思议。
<br/>　　陈应天看着赵天宇说道：“承影，你不是在跟我们开玩笑吧？无常跟军火贩子交易，涉嫌故意伤人？这怎么可能！？”
<br/>　　“怎么不可能？这完全有可能！”
<br/>　　赵天宇表情肃穆，眼神凌厉，看着眼前的战术中队队员，说道：“事实就是事实，我也不相信这是真的，但事情确确实实发生了。这是现场照片，你们如果还有疑问，可以自己拿去看。”
<br/>　　赵天宇抬起拿在手里的牛皮文件袋，罗昊从赵天宇手中接过文件袋，把里面的照片拿出来一张一张翻看着，把所有的照片看完后，交给身边的齐凡。
<br/>　　现在的罗昊已经不再是当年的热血青年，具备了作为指挥官必须要有的理智和冷静，而且他不认为赵天宇会无聊到制作这些假照片来跟自己开玩笑。
<br/>　　罗昊保持着冷静对赵天宇问道：“到底发生了什么事情，无常应该在东海市陪佟千雪看病，为什么会突然涉入到军火交易和故意伤人中。”
<br/>　　战术中队的队员已经传看完了照片，心中的怀疑也已经是被他们所看到的照片冲散了他们心中大部分的疑虑，所有人抿着嘴唇，神色沉重，他们已经明白了后面将会面临什么。既然赵天宇把他们召集到这里，并且告诉他们事情详情，就意味着他们要去东海市“清理门户”了！
<br/>(本章完)
