　　叶少阳笑而不语。
<br/>　　来的宾客陆续入座。
<br/>　　有一些法术公会的人，带着几个工匠，在广场前面搭起了一个系着大红花的喜棚，棚下放着两个蒲团，前面还有香案，上面点着两只红烛，显然是待会新人拜堂用的。
<br/>　　坐了没一会，人陆续都来了，鬼婆这才现身，身上穿着一件红色的袍子，走到喜棚的前面，站了一会，等人到齐了，这才开始发言，先介绍了张晓寒和梧桐，称他们是法术界的金童玉女，天造地设之类的，她说完之后，人群中爆发出一阵鼓掌和道贺的声音。
<br/>　　妙心怼了叶少阳一下，道：“看到没有，大家都觉得张晓寒和梧桐是一对，你待会要是抢婚，是要引众怒的。”
<br/>　　“那又怎么样？”叶少阳从桌上的凉盘里抓了猪蹄子，大肆啃起来，头也不抬地说道：“那又怎么样？”
<br/>　　“你会被打死的。”
<br/>　　“那你让我怎么样呢，看着他们完婚？”
<br/>　　“梧桐现在山下客栈里，你要是去，还来得及。”
<br/>　　“等一会上来之后，一样来得及。”
<br/>　　妙心差点气死。
<br/>　　接下来有一些门派的宗师，上去献对联什么的，什么比翼双飞，永结连理之类的，都是拍法术公会的马屁，叶少阳完全不理。
<br/>　　天差不多快黑透的时候，广场上的红灯笼都点亮了，一派喜庆，开始有人起哄要看新娘子。
<br/>　　陈晓宇等人在鬼婆的指示下，放了一长串的鞭炮，山下听到鞭炮声，开始响起唢呐、笙箫一类的音乐声，渐行渐近，众人都转头朝山路上看去，四个腰上系着红腰带的汉子，抬着一顶花轿，身后跟着衣着光鲜的新郎官张晓寒，骑在一匹马上，意气风发。
<br/>　　在他们后面，跟着十几个奏乐的，敲敲打打地爬上山来。
<br/>　　“晚了。”妙心叹了口气，抓着叶少阳的手腕说道，“将来再找机会吧，你老实坐着继续吃吧。”
<br/>　　“嗯嗯。”叶少阳继续吃着。
<br/>　　花轿和张晓寒骑着的高头大马都进入广场，宾客全都起立，各种鼓掌和起哄，制造着喜庆的气氛。
<br/>　　张晓寒得意洋洋地抱拳还礼，翻身下马，在众人起哄的催促声下，掀开了花轿的帘子，梧桐坐在里面，身穿红色旗袍，头上罩着红盖头，在张晓寒伸手搀扶之下，似乎犹豫了一下，然后没碰他的手，自己走了下来。
<br/>　　立刻有伴娘走过去，扶着梧桐的手，一起走向喜棚。
<br/>　　鬼婆上前，安排两人在蒲团上跪下去，有傧相来宣礼，等待吉时。
<br/>　　陈晓宇这时候走到喜棚前面的空地上，抱了个四方揖，朗声说道：“在下陈晓宇，诸位都认识，今天是我师兄和师姐大喜的日子，现在吉时未到，我来给大家助助兴……茅山的叶道长何在？”
<br/>　　叶少阳无奈一笑，一边埋头继续吃，挥了挥手。
<br/>　　“好啊，既然叶道长在，那便请上来吧。诸位，不瞒大家，我与这叶道长之前结下梁子，说定了今天当众斗法，输的人，要向另一个当众下跪道歉，叶道长，可有此事。”
<br/>　　叶少阳站起来，嘴里还叼着一块鸡腿，三两下吃掉，赶紧把骨头扔出去，拍了拍手，往广场前面走去。
<br/>　　这样子，活脱脱就是一个二流子……本来这些宾客听说还有约斗，一个个兴致昂然，待看到叶少阳这个样子之后，立刻就无语了：这样一个货色，怎么可能是陈晓宇的对手？
<br/>　　陈晓宇看叶少阳上前，冷笑一声，目光一扫，找到了云春生和伏明子，拱手道：“二位前辈，我今日要与贵派弟子比斗，只因我二人有约在先，与茅山完全无干，几位仍然是我法术公会的座上宾。”
<br/>　　他是担心到时候叶少阳败了，自己逼他磕头认错之类的，有辱茅山的声望，因此提前跟茅山几位宗师打个招呼，堵住他们的嘴，免得到时候他们出来阻止，自己也不好坚持。他是一心要让叶少阳出丑的。
<br/>　　云春生转头看了看叶少阳，道：“他不是我派弟子。”
<br/>　　宾客之中，立刻响起了哄堂大笑。
<br/>　　“哈哈，敢情这个茅山天师还是冒充的！”
<br/>　　“看看他这样子，一会看看他怎么怎么跪地求饶。”
<br/>　　“不是茅山弟子便好，他一会要是不跪，我们也好说话。”
<br/>　　…………
<br/>　　叶少阳听到了众人的议论，他心里很清楚，陈晓宇之所以在婚礼开始之前发难，一方面是因为他实在看自己不爽，二来，也是为了给法术公会立威，当着这么多人的面，把自己踩在脚下……
<br/>　　叶少阳走到陈晓宇对面十几米远的地方，伸了个懒腰，对陈晓宇说道：“说话算话吗？”
<br/>　　“什么？”
<br/>　　“你之前说的，输的人跪地磕头。”
<br/>　　陈晓宇哈哈大笑，“当然！我只怕你到时候不肯。”
<br/>　　叶少阳对他招了招手，“来吧！”
<br/>　　来吧！自己也等这一刻很久了。想起陈晓宇之前的种种缺德的表现和对自己的挑衅，不能忍了。
<br/>　　陈晓宇怔了一下，道：“你的法器呢？”
<br/>　　“不用法器。”
<br/>　　陈晓宇咧嘴一笑。
<br/>　　张晓寒站在喜棚边上，冲陈晓宇喊道：“既然如此，你还等什么，速战速决，我等着他磕头认错！”
<br/>　　陈晓宇对着叶少阳冲了过来，叶少阳不用法器，他也不好用，双手结印，祭出了一道不定式符，打向叶少阳。
<br/>　　叶少阳侧身避过，陈晓宇却是贴身而上，一只手朝叶少阳面门抓来，叶少阳踉跄后退，抬手反击。
<br/>　　陈晓宇这一击虽然没中，但叶少阳的表现让他信心大增，抢攻而上。叶少阳非常艰难地躲避，几次差一点就被集中，但是都运气好地避过了。
<br/>　　“没意思。”张晓寒摇了摇头。
<br/>　　大伙也纷纷摇头，觉得扫兴，大声喊着让陈晓宇快点搞定他，他们都想看磕头认错的一幕。
<br/>　　陈晓宇越战越勇，逼的叶少阳手忙脚乱，自己也不断加快攻势，但是……他很纳闷的是，叶少阳明明看着已经快支持不住了，但自己总是差那么一点，陈晓宇不免有些着急起来。
<br/>(本章完)
