　　腐朽之气，本就是天地间最为霸道之际的力量。
<br/>　　天地间的一切，都逃不过腐朽，就连天地，天意，终有一天也会腐朽，凋零。
<br/>　　可如今，这腐朽之气却是被楚风眠直接吞噬，甚至是都给炼化。
<br/>　　非但是没有影响到楚风眠的身躯，反而是化为了精粹的力量，让帮助楚风眠凝聚远古战龙精血。
<br/>　　“轰！轰！轰！”
<br/>　　楚风眠的身上，无穷无尽的灵力暴涨开，他的境界，居然是在这吞噬了无数腐朽之气之后，再度突破。
<br/>　　一跃之间，达到了神力境五重的地步。
<br/>　　在他身体的最深处，那远古战龙虚影，也足足是凝聚了七千五百滴远古战龙精血了。
<br/>　　“好！果然斩杀这样的天才，才是最好晋升的手段！”
<br/>　　楚风眠心中也是无比舒畅。
<br/>　　楚风眠来到颜古龙帝的洞府中，先是斩杀了白骨公子，然后又是杀了鬼谷圣子，荒古门的五人，如今又杀了落长恨。
<br/>　　以他们八人身上的力量，在加上这枯朽天的腐朽之气，终于是让楚风眠的实力，再度晋升。
<br/>　　楚风眠现在所缺少的，便是境界，他的无数武技，灵器，血脉，都是最为顶尖的。
<br/>　　一旦是境界晋升，对他的实力提升是极为可怕的，楚风眠只要是实力在晋升一步。
<br/>　　就算是面对生死境高阶的武者，楚风眠也不是没有一战之力。
<br/>　　眼睁睁的看着楚风眠吞噬了腐朽之气，非但是没有任何的影响，反而是身上的气息还在节节攀升，到现在一跃突破。
<br/>　　枯朽天的表情，近乎可以以面如呆滞来形容，眼睛都要掉了下来。
<br/>　　他的腐朽之气，可是他枯朽之道修炼数十年，才凝聚出来的。
<br/>　　就刚刚那一缕腐朽之气，已经是他半数力量的力量。
<br/>　　就算是让一尊生死境高阶的强者前来，都不一定可以抵挡吧。
<br/>　　别说是吞噬，就算是抵挡，都困难无比，腐朽之道的强大，在强大的武者，都要忌惮三分。
<br/>　　可是现在楚风眠却是直接将腐朽之气都给吞噬，将其中的灵力，都给全部转变为他的力量。
<br/>　　“这，这已经不是人了！是魔头！不，就算是魔头也做不到，枯朽腐朽，乃是天地都无法逃脱的事，他怎么可能逃脱！”
<br/>　　枯朽天想破脑袋都想不到，楚风眠到底是如何可以做到这一切的。
<br/>　　这是他凝聚枯朽之道一来，第一次被人破解，在他心中无敌的枯朽之道，被人吞噬，这刺激，让他都直接陷入疯狂。
<br/>　　“枯朽天，别发呆了，快点出手，趁着这小子，刚刚突破，境界不稳，还极为虚弱，杀了他，他的一切奇遇，都是我们的了！”
<br/>　　战魔尊者长啸一声，直接催动百战魔神的力量，六只手臂中的六件兵器。
<br/>　　刀，枪，剑，戟，斧，矛！
<br/>　　六件兵器中，顿时都是蕴含其中锋利灵力，同时轰击过来。
<br/>　　楚风眠的突破，令战魔尊者都察觉到了一丝机会，抓着这个机会，猛然出手。
<br/>　　一尊武者，在刚刚突破的时候，便是最为虚弱的时候。
<br/>　　因为一旦是刚刚突破，身上的灵力肯定都会用在突破上，倒时候灵力亏空，反倒是最为虚弱的一刻。
<br/>　　这样的时候，便是最为容易击败的一刻。
<br/>　　“楚风眠，你实在是太贪心了！在我们的面前突破！正好是给了我们一个机会！今日你的死，也是死在了你的贪心之下，陨落吧，你这样的人，不应该存在于北域之中！”
<br/>　　沧澜霸也是大吼一声，猛然也是凝聚灵力，化为一拳，打了过去。
<br/>　　“战神七式！”
<br/>　　“轰！轰！轰！轰！轰！轰！轰！”
<br/>　　沧澜霸一连打出七拳来，凌空打出，每一拳的力量，仿佛都如同一颗太阳一般的耀眼，其中蕴含的力量，足以将在场的无数武者，都给撕裂。
<br/>　　在沧澜霸背后的战神虚影，也是同样随着他的动作，连连打出七拳。
<br/>　　足足是十四尊耀眼太阳，出现在天空之上。
<br/>　　随着沧澜霸的命令，其中蕴含的无数拳风力量，都向着楚风眠，轰击了过来。
<br/>　　这每一尊太阳，便是意味着一个沧澜霸的力量，足足十四尊太阳，便是沧澜霸一拳打出了十四倍的力量。
<br/>　　这样的力量之下，轰杀同境界武者，简直是轻而易举。
<br/>　　在看到楚风眠突破之后，战魔尊者，沧澜霸二人都是毫不留守，将一切的底牌尽出，为了彻底的灭杀楚风眠。
<br/>　　要打算趁着楚风眠，刚刚突破，虚弱的一刻，轰杀了楚风眠。
<br/>　　“虚弱期？你们以为楚某突破了，你们就有机会了？可惜，我的突破，只能是代表你们再也没有任何的机会！”
<br/>　　面对着轰击而来的无数武技，楚风眠脸色轻松到了极点。
<br/>　　“统统给我破！”
<br/>　　他大手一挥，一股灵力猛然震发出去，这一股灵力打出，便是化为了一道长虹。
<br/>　　一道灵力长虹，席卷天空之上，那十四尊太阳，在这灵力长虹面前，纷纷破碎，当场陨落。
<br/>　　这被打破的，更是沧澜霸的武道，他的武道，都在这一股灵力之下完全破碎。
<br/>　　整个人都是被打飞了出去，口中“噗”的一声，无数鲜血吐了出去。
<br/>　　“怎么会有如此可怕的力量！他不是刚刚突破吗？怎么一点没有虚弱，反而是力量更加可怕！”
<br/>　　战魔尊者看着沧澜霸就这样败了，一身武道都被打碎，简直是心神开始崩溃。
<br/>　　他本以为，如今楚风眠突破，是给了他们一个机会，趁着虚弱期，他们联手起来，爆发力量，足以将楚风眠彻底轰杀。
<br/>　　可现在，楚风眠的力量，哪里有着任何一点虚弱，这恐怖至极无法反抗的力量，简直是比起刚刚，还要强大无数倍。
<br/>　　“虚弱？”
<br/>　　楚风眠冷笑连连。
<br/>　　战魔尊者绝对想不到，楚风眠身上有着神脉。
<br/>　　寻常武者突破，的确是会陷入到一阵虚弱中，至少需要几天才可以彻底恢复过来。
<br/>　　所以在刚刚突破的一刻，便是最为危险的一刻。
<br/>　　但是楚风眠不同。
<br/>　　楚风眠有着神脉，他突破中消耗的灵力，都被神脉瞬间补充完毕，可以说楚风眠根本没有虚弱期这个概念。
<br/>　　一旦突破，下一瞬便是最为完美的状态。
<br/>　　他们二人打算利用楚风眠刚刚突破的虚弱来对付楚风眠，绝对是最为愚蠢的行为。
