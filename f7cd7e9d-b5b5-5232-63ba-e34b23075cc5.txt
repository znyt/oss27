        是个男人！
<br/>
<br/>        “你放开我！你是谁？救命！”下一秒，童筱颖意识到危险后，急得大声呼救起来。
<br/>
<br/>        因为挣扎，她挣脱开了一只手，往床边摸去，不管摸到了什么，都愤怒地往压在自己身上的这个男人后背砸去。
<br/>
<br/>        “啪嗒”一声，室内的吸顶灯忽然间亮了。
<br/>
<br/>        童筱颖还未来得及看清楚这个欺辱自己的男人，这男人就已经从她身上离开了。
<br/>
<br/>        再定睛一看的时候，只见一个戴着金丝假面眼镜的男人，手里持着一把黑色的枪，正抵在了那个八爷的太阳穴上。
<br/>
<br/>        “阿十，你这是什么意思？”八爷愣在床边，不敢轻举妄动地扯着嘴角问。
<br/>
<br/>        童筱颖怔怔地看着那个三年未见的十爷，除了他脸上那副金丝假面眼镜，她记得以外，对于这个男人，她真的快没什么印象了。
<br/>
<br/>        “她是婴宁的妹妹，你动她，就是动我的人。”十爷冷冷地呵斥道。
<br/>
<br/>        八爷却嗤笑了一声：“呵？我就说嘛！你们姓‘冷’的怎么就都嗜好这一类货色！弄了半天，原来是姐妹啊！”
<br/>
<br/>        “她不是傅夜沉的女人，她是傅夜沉的弟弟傅昼景的妻子，你抓她，威胁不到傅夜沉。”十爷面无表情地接着说道。
<br/>
<br/>        “怎么可能！她都跟傅夜沉拜堂成亲了！”八爷狐疑道。
<br/>
<br/>        十爷顿时瞪向了童筱颖。
<br/>
<br/>        童筱颖坐在床边，脑海里回想起季雅珊提醒过她的那些话，连忙矢口否认道：“和傅夜沉拜堂成亲的人不是我。”
<br/>
<br/>        “那是谁？”八爷微眯起眸子，直勾勾地盯着童筱颖。
<br/>
<br/>        童筱颖眼珠子一转，淡定道：“是另一个跟我长得一模一样的女人。”
<br/>
<br/>        “那女人是谁？”八爷不依不饶地追问。
<br/>
<br/>        童筱颖下意识地看了看十爷，见他剑眉微蹙，她被恫吓得咽了咽口水，弱弱地回答道：“她叫苏漫雪，现在是当红女星，不信的话，你们可以上网查查。”
<br/>
<br/>        八爷连忙掏出手机，在浏览器里搜索了“苏漫雪”，待苏漫雪的照片出来后，他又感慨道：“我就说，我看你怎么都觉得眼熟喽！原来你真的跟那个女明星苏漫雪长得一模一样！我说，你们姓‘冷’的，还真就嗜好这类女人啊！三个男人娶了三个几乎长得一样的女人！啧啧啧，果然是祖传下来的一条染色体导致的结果！”
<br/>
<br/>        其实，在八爷心里，也有些怀疑这女人的身份。因为，从今早到现在，傅夜沉那边竟然一点动静也没有。
<br/>
<br/>        这不是很奇怪吗？
<br/>
<br/>        十爷不耐烦地呵斥：“你啰嗦完了吗？”
<br/>
<br/>        “既然如此，我把就这个女人还给阿十好了！”八爷收起手机，装出一副落落大方的样子。
<br/>
<br/>        十爷这才利索地将手里的枪收回插入了腰后。
<br/>
<br/>        八爷松了口气后，双手一摊，耸了耸肩，一边往阁楼门外走，一边提醒道：“阿十，你若是想放这女人走，记得把这女人的记忆给抹除了。以免暴露了我们的身份，招来杀身之祸。”
<br/>
<br/>        待八爷离开后，十爷目光犀利地盯着童筱颖，一脸严肃地说道：“我会派人送你回去，前提条件是，你必须对这里的事情只字不提。你若敢泄露出去，死的人就不仅仅只是我了，还有雅珊。你考虑清楚了？”
<br/>
<br/>        “我明白。”童筱颖凝眉点了点头。
<br/>
<br/>        十爷随之从衣兜内掏出一块怀表，中指挂住表链，伸手放到了童筱颖的眼前。
<br/>
<br/>        “看着这块表。”他命令道。
<br/>
<br/>        童筱颖本能地盯住了眼前的这块怀表，随着这块怀表在空中的摆动，她也在不知不觉中哈欠连天。
<br/>
<br/>        “忘了在这里的一切……”
<br/>
<br/>        这个声音原本只是在耳边回荡，慢慢地变成了在脑海里回荡。
<br/>
<br/>        忘？
<br/>
<br/>        怎么可能忘掉！
<br/>
<br/>        小时候，她听外婆说，这世上还有一种“巫蛊”叫“幻术”，要想自己不中“幻术”，只需要心无杂念的静下心去提前入睡即可。
<br/>
<br/>        心无杂念……
<br/>
<br/>        童筱颖缓缓地闭上了眼睛。
<br/>
<br/>        就在她的身子往后躺去的时候，十爷连忙收回怀表，疾步上前一把抱住了她。
<br/>
<br/>        十爷将童筱颖慢慢地放倒在床上，然后拉过一旁的被褥，轻轻地给她盖上。
<br/>
<br/>        季雅珊此时刚巧走到阁楼门外，看到了这一幕，心里突然很不是滋味，但随即又将这种醋意给甩开，抬起手来，搭在门框上，关问道：“阿十，筱颖她要不要紧？”
<br/>
<br/>        “今晚，我会将她送出去。”十爷转过身来，接着说道。
<br/>
<br/>        季雅珊欣慰地点了点头。
<br/>
<br/>        “雅珊。”
<br/>
<br/>        “嗯？”
<br/>
<br/>        “你若想走，我便将你和筱颖一起送走。”十爷语重心长道。
