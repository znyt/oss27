　　第472章
<br/>　　草原上，一支队伍正准备扎营，埋锅造饭，今夜要在草原上度过。
<br/>　　并州的战局已接近尾声，各路诸侯都在准备撤离事宜，剩下的就只是些收尾工作。比如，程大雷率领的正义教残部还没被消灭干净。
<br/>　　当然，这已不是太重要的事情。
<br/>　　这支队伍的指挥者名叫况券祟，属于扬州李星的队伍，这况券祟有虎将之称，在李星手下很得重用。
<br/>　　这一次，便被李星派出来，追杀程大雷。
<br/>　　一场乱战，有人在战场上损命，史书吝啬得不肯加上一笔，有人却趁机捞足功勋，从此平步青云。
<br/>　　毫无疑问，况券祟便是后者。
<br/>　　“况将，您先吃些东西，咱们过上一夜，明日就可继续行军。”一名裨将上前讨好道。
<br/>　　况券祟适逢而立之年，最出名的是一脸胡须，他看了看天色，道：“歇一歇，先填饱肚子，明天再去杀那程大雷。”
<br/>　　手下陪着笑道：“将军，听说那程大雷可是不好对付，有万夫不当之勇，很多名将都在他斧下丧命。”
<br/>　　况券祟听罢，是哈哈大笑，手搁在自己大刀的刀柄上。
<br/>　　“程大雷了不了得，我是不知道。但以俺观之，世上庸庸碌碌，大多浪得虚名之辈。死在程大雷手中的人也是一样。程大雷之所以猖狂这么久，是因为他没遇到我，若是遇到我便要他知道我大刀的厉害。”
<br/>　　“将军的本事，我们当然都是知道的。”手下道：“不过，前两天得到的消息，有一伙人从并州边关入境，不知会不会是程大雷的人马。”
<br/>　　“是又怎样，一群乌合之众的山贼而已。他们来了更好，我还嫌军功不够呢。”
<br/>　　二人正在说着话，饭已经在锅里煮熟，此刻天将黄昏，一轮红日将要落山，给荒草萋萋的草原洒下一片火红。
<br/>　　在落日的余晖下，一波人马从北方而来。粗略看过去，大概有两千人左右。
<br/>　　“将军，有人来了，北方五里处。”裨将道。
<br/>　　“向他们打灯语，看他们属于哪路人马。”
<br/>　　“是。”
<br/>　　有人举起火把，向对方发出信号，但却没有得到任何回应。
<br/>　　况券祟皱起眉头，命令手下准备防御。
<br/>　　对方对况券祟发出的灯语视若无睹，在双方距离三里处，开始全速推进。
<br/>　　距离越来越近，已经可以看到他们身上的黑色皮甲，一杆鱼字旗猎猎而舞。
<br/>　　裨将大将：“将军，就是他们……他们是程大雷的人。”
<br/>　　“慌什么！”况券祟冷喝一声：“不过就是一群乌合之众，来多少，我杀多少。”
<br/>　　对方已杀到眼前，清晰得可以看到他们脸上的表情。每个人都睁大眼睛，一脸凶神恶煞，那眼中流露出饥渴，仿佛是在说：可算是找到你们了。
<br/>　　相比之下，况券祟的队伍却是人心惶惶，这场不期而遇的战斗，他们尚未准备好。
<br/>　　“杀！”
<br/>　　双方同时发出怒吼。况券祟骑马举着大刀，口中喝道：“与我杀！”
<br/>　　正此时，从敌军队伍中冲出一人，面若重枣，留有长须，此刻一双丹凤眼怒睁，拍马向况券祟冲来。
<br/>　　来若急急之风，眨眼便冲到况券祟面前，几个亲卫，被他轻易挑落马下。等况券祟反应过来时，大刀已出现在眼前。
<br/>　　咔嚓！
<br/>　　一声脆响，血淋淋的脑袋滚落在地上。况券祟的腔子还在马背上，晃了三晃，噗通一声跌在地上。
<br/>　　“将军死了，将军死了！”
<br/>　　况券祟坠马而死，帝国军发出一声声惊慌，人心惶惶，兵荒马乱。
<br/>　　而关鱼所率领的鱼字军，却人人龙精虎猛。这是关鱼亲手打造出来的队伍，也完美继承了关鱼的行事风格。话不多，却是够狠够猛，常常以山寨第一军自居。
<br/>　　从黄昏开始的战斗，在月照中天时结束，除了逃走一小部分人以外，大部分人被击杀在当场。草原上尸横遍野，空气中漂浮着一种刺鼻的血腥味。
<br/>　　“将军，歼敌一千人，还缴获一些马匹和粮草。”手下禀报道。
<br/>　　“马带上，每人携带三天的干粮，剩下的全部烧掉。”关鱼。
<br/>　　“是。”
<br/>　　关鱼骑在马上，目光扫了一圈，月明星稀，借着银白色月光，能看到一张张严肃的脸。
<br/>　　这支队伍自己整整训练了三年，今天可以说是第一次将他们亮出来，实话说，效果还不错。以一千兵力，击溃了两倍与自己的敌人。
<br/>　　“原地休整，埋葬弟兄们的尸体，吃饱饭后，我们去下一个地方。”
<br/>　　“是。”
<br/>　　与此同时，张肥和高飞豹正埋伏在一处山谷上。
<br/>　　“张三爷，你说会有人从这里过么？”高飞豹。
<br/>　　“我已打探清楚，这道山口是交通要道，他们要运送粮草辎重必定会走这里。”张肥肯定道。
<br/>　　正这时，斥候快马来报：“张将军，高将军，有人来了？”
<br/>　　张肥和高飞豹眼睛同时一亮：“来了多少？”
<br/>　　“是将军府的运粮队，大概有三千人左右。”
<br/>　　张肥和高飞豹对视一眼，同时点了点头。运粮队属于后勤队伍，战斗力不如作战队伍强。现在的情况虽然己方兵力不占优势，但却已经可以打了。
<br/>　　远远看着，有一队人沿着大道过来，绵延如一条长龙，一眼望不到边。
<br/>　　“高二爷，你去前面路上埋伏，痛打落水狗，我先去搅散了他们。”张肥已经握紧了自己的丈八蛇矛。
<br/>　　“咦，为什么不是你去埋伏，我负责主攻。”
<br/>　　正此时，张肥已经带着自己的张字军冲杀出去。
<br/>　　山坡下这帮运粮队，乃是负责给相府运输辎重，如今战斗快结束，大家都在准备撤退。
<br/>　　行到山口，冷不防山上喊杀声起，一群如狼似虎的恶人冲杀下来。为首一个黑脸大汉，更是猖狂，手中的长矛，碰见谁就砸得谁脑袋开花。
<br/>　　人群惶惶而乱，向四面八方溃逃。最大一波溃败的队伍，在前方遭遇高飞豹的伏击部队。这支伏兵的突然出现，更是击溃了他们最后的希望。
<br/>　　杀人遍野，丢下粮草辎重，侥幸的人逃了一条生路，没来的逃的，便成了这草原亡魂。
<br/>(本章完)
