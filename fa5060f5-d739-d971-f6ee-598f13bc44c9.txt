　　主府之内。
<br/>　　萧逸细细感知着面前这个巨大的血色圆球。
<br/>　　与其说是圆球，倒不如说是一个能量团。
<br/>　　单看外表，倒是和幻光洞府的那个魂力圆球差不多；大小一样，同样气息浑厚。
<br/>　　唯一不同的，只是幻光洞府的那个圆球，散发着氤氲幻光。
<br/>　　而这个圆球，则是通体猩红。
<br/>　　在萧逸的感知中，里头溢散的气血力量，其浑厚与精纯程度，与幻光洞府的那魂力圆球，在同一个层次。
<br/>　　“小贼，这东西能救怜星吗？”公孙火舞迫不及待地问道。
<br/>　　“不知道。”萧逸摇了摇头，“只能试试。”
<br/>　　“怜星姑娘。”
<br/>　　萧逸看了眼顾怜星。
<br/>　　顾怜星点点头，来到血色圆球之前。
<br/>　　圆球上的血色红芒，异常刺眼。
<br/>　　光芒，照耀了整个主府。
<br/>　　顾怜星闭目感受着。
<br/>　　半晌，顾怜星面露喜意，点了点头，“我的天星血脉在缓慢修复着。”
<br/>　　哗…
<br/>　　顾怜星手掌一摊，一抹星光力量凝聚于掌中。
<br/>　　这抹星光，很是稀薄。
<br/>　　不同于武魂的星光，武魂凝聚的星光，很是纯粹。
<br/>　　而这抹星光中，带着莫名的武道力量。
<br/>　　代表着，这乃是天星血脉中，血脉之力所激发的星光。
<br/>　　血脉，由先辈遗留。
<br/>　　越是强大的先辈，流传于血脉中的力量，便越是强横。
<br/>　　而越是出色的后辈，觉醒血脉的几率也越大。
<br/>　　强大的先人，因一身掌控的武道力量，而血脉不消，遗留后辈。
<br/>　　自然的，血脉之力中，会带有武道力量的意味。
<br/>　　“看来确实有效。”萧逸满意笑笑。
<br/>　　原本顾怜星的血脉异常虚弱，接近凋零。
<br/>　　她连半点血脉之力都调动不了。
<br/>　　现在，她能微微释放出一丝稀薄星光，证明这些气血之力确实有效。
<br/>　　“盘膝坐下吧，接下来我会引导这些气血力量溢出，随后打入你体内，修复天星血脉。”
<br/>　　“有劳萧逸公子。”顾怜星点点头，盘膝坐下。
<br/>　　萧逸伸出指尖，靠近那颗血色圆球。
<br/>　　他将牵引出这些血气力量，打入顾怜星体内。
<br/>　　之所以需要他引导，便是因为这是炼药师的手段。
<br/>　　否则，武者自己单纯吸收，力量只会存于体内，或者在体内四散。
<br/>　　而他的引导，则是将这些力量打入体内后，聚于天星血脉一点之上。
<br/>　　同时，整条血脉，千疮百孔，虚弱不堪。
<br/>　　也就是说，这条血脉，异常脆弱，哪怕是一丝的伤害都不能再受了。
<br/>　　即便是气血力量进入，也需要小心翼翼游走。
<br/>　　这等引导，哪怕是对等闲的皇品炼药师来说，亦是难度不小。
<br/>　　但萧逸对自己的炼药师手段，以及力量的掌控，还是极有自信的。
<br/>　　“萧逸公子，慢着。”顾怜星忽然说了一声。
<br/>　　“怎么？”萧逸靠近血色圆球的手指缓缓停下，疑惑地看着顾怜星。
<br/>　　顾怜星皱眉转过头，看了眼主府之外的八人。
<br/>　　“他们八人，也是在抽调这里的气血力量吧？”
<br/>　　“萧逸公子忽然抽调血色圆球内的力量，可会对他们造成影响？”
<br/>　　那八人，正在准备武魂的第二次觉醒。
<br/>　　此事，对于任何武者来说，都是重中之重，更别说是对于这八位绝世天骄。
<br/>　　“不会。”萧逸摇了摇头。
<br/>　　“他们八人，不过是借圆球内的气血力量，以增幅自己的血脉之力罢了。”
<br/>　　“这个血色圆球内的力量，精纯浑厚到极点。”
<br/>　　“他们八人顶多只能吸收些许。”
<br/>　　“别说他们区区八人，就是武道大能来了，全力吸收，都休想撼动这颗圆球。”
<br/>　　这颗血色圆球，其内蕴含的力量，怕是比几十个武道大能的一身元力加起来还要多得多。
<br/>　　那八人不过是圣王境巅峰修为，八人加在一起，吸个一年半载都不可能吸得完。
<br/>　　更别说他们不可能吸收这么久，顶多就能吸收几天。
<br/>　　“你修补血脉所需的力量，不比他们激发血脉之力多多少。”萧逸解释道。
<br/>　　“那就好。”顾怜星松了口气。
<br/>　　萧逸刚要有所动作，看了眼远处的八人，微微皱了皱眉头。
<br/>　　这八人，在阵法之内，盘膝而坐。
<br/>　　除了吸收着气血力量外，身上还散发着惊人的武道力量气息。
<br/>　　证明他们同时还在消化着武道圣丹。
<br/>　　就八人现今散发着的武道力量气息，已让萧逸面露惊骇之色。
<br/>　　武道力量的气息之强猛，远远超过了武道大能。
<br/>　　很显然，这八人用于武魂第二次觉醒的武道圣丹，绝不是外头那些长老的武道圣丹。
<br/>　　这八颗武道圣丹的主人，其修为，绝对超过了武道大能。
<br/>　　传言，十八府府主，皆是武道大能之上的强者。
<br/>　　看来，传言非虚。
<br/>　　“不愧是中域十八府，底蕴确实深厚。”萧逸淡漠自语一声。
<br/>　　事实上，单看十八府从上古时期结束便一直传承至今，无数岁月以来，一直屹立不倒，便可知他们底蕴之深厚。
<br/>　　顾怜星和公孙火舞，同时看向萧逸。
<br/>　　“萧逸公子的武魂，还未进行第二次觉醒吧。”顾怜星忽然说道。
<br/>　　“等这里的事一了，若萧逸公子愿意，可随我回天星府。”
<br/>　　“这些天，多得萧逸公子费心，怜星才续命至今，算起来，怜星欠了萧逸公子不少人情。”
<br/>　　“怜星找来一位武道大能替萧逸公子第二次觉醒武魂的能力还是有的。”
<br/>　　“不必了，算不得什么人情。”萧逸摇了摇头，婉言拒绝。
<br/>　　“我救怜星姑娘，也不是为了这些人情。”
<br/>　　“那小贼你来我天索府吧。”公孙火舞脱口而出。
<br/>　　“反正我没欠你人情，你也没欠我的。”
<br/>　　“呵。”萧逸笑笑，“既是互不相欠，我更不该去你天索府。”
<br/>　　一颗武道圣丹，即便是武道大能，也需要费尽心血去凝结温养。
<br/>　　即便这二人真的能请动，怕也要付出不小的代价。
<br/>　　“可是…”顾怜星和公孙火舞，同时认真地看着萧逸。
<br/>　　武者的第二次觉醒，需要武道圣丹，也需要强者的辅助。
<br/>　　她们很清楚，对于一个背后没有势力的武者来说，这些都几乎不可能办到。
<br/>　　而且，等闲势力也办不到，需得底蕴极深的大势力。
<br/>　　“两位好意，在下心领了。”萧逸笑笑，“开始引导吧。”
<br/>　　萧逸没有再继续这个话题，伸出手指，放在了血色圆球之上。
<br/>　　嘶…嘶…嘶…
<br/>　　一丝丝气血力量，从血色圆球中被牵引而出。
